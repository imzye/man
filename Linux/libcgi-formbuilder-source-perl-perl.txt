NAME
       CGI::FormBuilder::Source::Perl - read FormBuilder config from Perl syntax files

SYNOPSIS
           my $form = CGI::FormBuilder->new(
               source => {
                   type   => 'Perl',
                   source => '/path/to/form_config.pl',
               }
           );

DESCRIPTION
       This module allows you to specify the config for a CGI::FormBuilder object using Perl syntax. The contents of the config file will be "eval"ed and the hash ref returned will be used as the config
       for the object.
