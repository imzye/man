NAME
       cgnsconvert - convert HDF5/ADF files

DESCRIPTION
       usage  : cgnsconvert [options] InputFile [OutputFile] options:

       -a : write ADF file

       -h : write HDF5 file

       -f : force output if input format is same as output

       -l : expand links in ouput file
