DESCRIPTION
       cwt is a binary file to make rebuilding Docker images easier.

       koji It lists builds, base images, hash ids.

       build It builds images.

       git It works with upstream/downstream git repositories.

       utils Other actions tied  to  the  rebuild  (communication,  repository
       preparation etc).

       cwt should help a bit with the image rebuild workflow..
