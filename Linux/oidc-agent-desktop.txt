NAME
       oidc-prompt - interface to create user dialogs

SYNOPSIS
       oidc-prompt TITLE TEXT LABEL [INIT] [LIST_ELEMENTS ...]

DESCRIPTION
       bin/oidc-prompt: line 26: usage: command not found

       oidc-prompt -- An interface for prompting the user.

       This tool is intended as a internal tool of oidc-agent. Different

              oidc-agent components use it to prompt the user for information. The user should not call oidc-prompt.
