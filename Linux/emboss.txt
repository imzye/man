NAME
       taxgetup - Get parents of taxon(s)

SYNOPSIS

       taxgetup -taxons taxon -hidden boolean -outfile outtaxon

       taxgetup -help

DESCRIPTION
       taxgetup is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It is part of the "Ontology:Taxonomy" command group(s).

OPTIONS
   Input section
       -taxons taxon

   Additional section
   Advanced section
       -hidden boolean
           Default value: N

   Output section
