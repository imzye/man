NAME
       firestation - FireVision viewing and calibration GUI

SYNOPSIS
       firestation

DESCRIPTION
       This GUI application allows for inspecting a running FireVision systems
       by viewing images and visualizing color maps. Its second purpose is to
       calibrate the vision system, e.g. by creating mirror calibration files
       and creating color maps.
