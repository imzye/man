NAME
       fvfuseviewer - View FireVision images over the network

SYNOPSIS
       fvfuseviewer

DESCRIPTION
       This GUI application allows for viewing images from running FireVision
       applications. Note that the fvfountain plugin must be loaded to be able
       to view the images.
