NAME
       logique44 - evaluate logical functions

SYNOPSIS
       logique44 [OPTIONS] [BIT]... EXPRESSION

DESCRIPTION
       logique44  evaluates  EXPRESSION  for each set of BIT(s) binary values.
       BIT(s) are expected valid C identifiers.  EXPRESSION may  refer  BIT(s)
       integral variables.

OPTIONS
       -P, --print
              Print the parsed expression instead.

       -s, --skip-header
              Do not print the BIT(s) line header.

       --help Print  a  usage  message  and exit with a status code indicating
              success.

       --version
              Print version information on standard output then exit.

SYNTAX
       The allowed expression syntax is same as for express17(1).

EXAMPLES
              logique44 a b "a ^ b"

SEE ALSO
