NAME
       stiff - convert scientific FITS images to TIFF format

SYNOPSIS
       stiff <fits_image1> [<fits_image2> <fits_image3>]
              [-c <Configuration_file>] [-<keyword> <value> ...]

       stiff -d

DESCRIPTION
       STIFF is a program that convert scientific FITS images to the more popular TIFF, in 8 (grayscale) or 24 (true colour) bits per pixel.
       See http://astromatic.net/software/stiff for more details.

   Operation modes:
       -h, --help
              print this help, then exit

       -V, --version
              print version number, then exit

       -d, --dump
              dump a default configuration file

MANUAL
       The full documentation for STIFF is maintained as a PDF manual available at http://astromatic.net/software/stiff
