NAME
       tsswriteapp - Runs TPM2 writeapp

DESCRIPTION
       writeapp

       writeapp is a sample NV write application.  Provisions an NV location, then does two writes with password 'pwd' using a bound, salted HMAC session using AES CFB parameter encryption.

       Used to test minimal TSS build

       [-pwsess
              Use a password session, no HMAC or parameter encryption]
