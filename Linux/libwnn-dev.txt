JL_SET_JIKOUHO(3)                                                                                                                                                                    JL_SET_JIKOUHO(3)

名称

       jl_set_jikouho()         全候補(大、小)の内の次候補設定。
       jl_set_jikouho_dai()     全候補(大文節)の内の候補設定

形式

       #include "jllib.h"

       int
       jl_set_jikouho(buf, offset)
       struct wnn_buf *buf;          /* バッファ */
       int offset;                   /* オフセット */

       int
       jl_set_jikouho_dai(buf,offset)
       struct wnn_buff *buf;         /*バッファ*/
       int offset;                   /*バッファ*/

機能

       jl_zenkouho は、jl_zenkouho で取り出した全候補 のうち offset 番 目のものを選択する。

       jl_set_jikouho, jl_next, jl_previous を呼ぶ前に、jl_zenkouho, を呼ぶ必要がある。候補の設定は、jl_zenkouhoが呼ばれた文節に対し 行なわれる。jl_zenkouho_は、最後に呼ばれたものだけが有効である。

       jl_zenkouho_dai は、jl_zenkouho_dai で取り出した大文節全候補 の うち offset 番目のものにバッファの内容を置き換える。

       jl_set_jikouho_dai, jl_next_dai, jl_previous_dai を呼ぶ前に、 jl_zenkouho,を呼ぶ必要がある。候補の設定は、jl_zenkouho_daiが 呼ばれた文節に対し行なわれる。jl_zenkouho_dai は、最後に呼ばれ たも
       のだけが有効である。

リターンバリュー

       候補のオフセットを返す。

使用上の注意

       jl_zenkouho を呼んでから jl_set_jikouho, jl_next, jl_pervious を呼ぶまでの間に jl_zenkouho_dai を呼ぶと、jl_set_jikouho などは、エラーを返す。

       jl_zenkouho_dai を呼んでから、jl_set_jikouho_dai, jl_next_dai, jl_jprevious_dai を呼ぶまでの間に jl_zenkouho を呼ぶと、 jl_set_jikouho_dai などは、エラーを返す。
