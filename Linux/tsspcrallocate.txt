NAME
       tsspcrallocate - Runs TPM2 pcrallocate

DESCRIPTION
       pcrallocate

       Runs TPM2_PCR_Allocate

       Allocates banks for a full set of PCR 0-23.  Not all hardware TPMs support multiple banks or all algorithms

       [-pwdp platform hierarchy password (default empty)]

       +sha1   -sha1
              allocate / deallocate a SHA-1 bank

       +sha256 -sha256 allocate / deallocate a SHA-256 bank

       +sha384 -sha384 allocate / deallocate a SHA-384 bank

       +sha512 -sha512 allocate / deallocate a SHA-512 bank

              More than one algorithm can be specified
