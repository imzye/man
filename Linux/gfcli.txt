NAME
       gfcli - manual page for gfcli 0.3.0

SYNOPSIS
       gfcli [OPTION]... [URL]

DESCRIPTION
       Start a Gluster shell to execute commands on a remote Gluster volume.

       -o, --xlator-option=OPTION
              specify a translator option for the connection. Multiple options
              are supported and take the form xlator.key=value.

       -p, --port=PORT
              specify a port on which to connect

       --help display this help and exit

       --version
              output version information and exit

EXAMPLES
              gfcli glfs://localhost/groot

              Start a shell with a connection to localhost opened.

              gfcli -o *replicate*.data-self-heal=on glfs://localhost/groot

              Start a shell with a connection localhost open, with the  trans‐
              lator option data-self-head set to on.
