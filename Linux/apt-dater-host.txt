NAME
       apt-dater-host - host helper application for apt-dater.

SYNOPSIS
       apt-dater-host

DESCRIPTION
       This manual page documents briefly the apt-dater-host helper.  apt-dater-host is the host helper application for apt-dater for watching and updating the targeted application about apt-dater.
