NAME
       pmd2text - manual page for pmd2text 0.0.4

SYNOPSIS
       pmd2text [OPTION] INPUT

DESCRIPTION
       `pmd2text' converts PageMaker documents to plain text.

OPTIONS
       --help show this help message

       --version
              show version information and exit
