NAME
       dnssec-nodes - A graphical DNS visualization tool

DESCRIPTION
       A graphical DNS visualization application, specializing in DNSSEC.  The
       DNSSEC-Nodes application is a graphical debugging utility  that  allows
       administrators  to  watch  the  data being logged into a libval or bind
       logging file.

       Further information on this tool can be found on the DNSSEC-Tools  wiki
       page, of which this application is a component:

              http://www.dnssec-tools.org/wiki/index.php/DNSSEC-Nodes

SEE ALSO
       For more information on the DNSSEC-Tools project, see the project's web
       page:

              http://www.dnssec-tools.org/
