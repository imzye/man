NAME
       fx-completion - get shell completion for fx(1)

SYNOPSIS
       fx-completion --bash

DESCRIPTION
       Emit the completions script for the selected shell.  Currently, only bash(1) is supported.

       Additionally, this command provides an interface used by the completions script to get completions, the details of which are not documented here.

OPTIONS
       --bash Emit the completions script for bash(1).

EXAMPLES
       Add to your .bash_profile:

           source <(fx-completion --bash)

NOTES
       The fx-completion RPM package automatically installs shell completions for bash(1) system-wide, so direct invocation of this command is generally not needed.

SEE ALSO
       fx(1), bash(1)
