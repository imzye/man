NAME
       porestructure - Restructure Gettxt PO files produced by :doc:`poconflicts </commands/poconflicts>` into the original directory tree for merging using :doc:`pomerge </commands/pomerge>`.

SYNOPSIS
       porestructure [--version] [-h|--help] [--manpage] [--progress PROGRESS] [--errorlevel ERRORLEVEL] [-i|--input] INPUT [-x|--exclude EXCLUDE] [-o|--output] OUTPUT

DESCRIPTION
       See: http://docs.translatehouse.org/projects/translate-toolkit/en/latest/commands/pomerge.html for examples and usage instructions.

OPTIONS
       --version
              show program's version number and exit

       -h/--help
              show this help message and exit

       --manpage
              output a manpage based on the help

       --progress
              show progress as: dots, none, bar, names, verbose

       --errorlevel
              show errorlevel as: none, message, exception, traceback

       -i/--input
              read from INPUT in po format

       -x/--exclude
              exclude names matching EXCLUDE from input paths

       -o/--output
              write to OUTPUT in po format
