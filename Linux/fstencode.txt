NAME
       fstencode - manual page for fstencode 1.6.9

DESCRIPTION
       Encodes transducer labels and/or weights.

              Usage: fstencode in.fst codex [out.fst]

       PROGRAM FLAGS:

       --decode: type = bool, default = false

              Decode labels and/or weights --encode_labels: type = bool, default = false Encode output labels --encode_reuse: type = bool, default = false Re-use existing codex --encode_weights: type = bool,
              default = false Encode weights
