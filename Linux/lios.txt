NAME
       lios - Linux-intelligent-ocr-solution

SYNOPSIS
       lios [-h] [-s] [-v] [files [files ...]]

DESCRIPTION
       Lios  is a free and open source software for converting print in to text using either scanner, camera, or screenshot, It can also produce text out of scanned images from other sources such as Pdf,
       Image or Folder containing Images. Program is given total accessibility for visually impaired.  Lios is written in python3, and we release it under GPL-3 license.

COMMAND LINE OPTIONS
       A summary of options is included below.

       -h,  --help
              Display help and exit

       -v,  --version
              Show version information and exit

       -s,  --ocr-screenshot
              Take screenshot and ocr

FILES
       ~/.lios_preferences.cfg
              file with user configuration.

       ~/.lios_recent
              file with user last output.
