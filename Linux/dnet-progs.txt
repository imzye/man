NAME
       /etc/vmsmail.conf - VMSmail configuration file

DESCRIPTION
       /etc/vmsmail.conf is an ASCII file which contains names and values of configuration variables for the VMSmail gateway programs.

       There is one entry per line, and each line has the format:

              <name>=<value>

       Valid values for <name> are

              hostname  The name of the gateway machine (to put in the mail header). Defaults to the value returned by gethostname(2).

              username  The name of the user associated with sending mail from VMS to Linux.  Defaults to vmsmail.

              smtphost  The name of the SMTP mail host to send mail to. If this entry is omitted them vmsmaild will send mail by running 'sendmail' on the local ma‚Äê
                        chine.

       EXAMPLE
                 hostname=myhost.domain.net
                 username=vms
