NAME
       sphinx2nb - manual page for sphinx2nb 0.6

DESCRIPTION
       usage: sphinx2nb [-h] [-W] rst_file

       Convert Sphinx ReST file to Notebook JSON

   positional arguments:
       rst_file
              ReST file to convert

   options:
       -h, --help
              show this help message and exit

       -W, --warn-is-error
              turn warnings into errors
