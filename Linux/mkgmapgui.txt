NAME
       mkgmapgui - Graphical User Interface for mkgmap

SYNOPSIS
       mkgmapgui

DESCRIPTION
       mkgmapgui is a graphical user interface for mkgmap.  Although mkgmap has a lot of features, this GUI allows you to do the most common actions: convert an Osm file into an Img file and combine mul‐
       tiple Img files into one big Img file.

SEE ALSO
       For further information, visit http://activityworkshop.net/software/mkgmapgui/
