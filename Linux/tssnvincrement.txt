NAME
       tssnvincrement - Runs TPM2 nvincrement

DESCRIPTION
       nvincrement

       Runs TPM2_NV_Increment

       -ha    NV index handle

       -pwdn  password for NV index (default empty)

       -se[0-2] session handle / attributes (default PWAP)

       01     continue
