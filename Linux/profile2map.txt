       profile2map â€” Fuse profiler output converter

SYNOPSIS
       profile2map [OPTION] infile outfile

DESCRIPTION
       profile2map converts Fuse profiler output into Z80-style map format.

OPTIONS
       -h, --help
              give brief usage help, listing available options.

       -V, --version
              output version information.

       infile specifies the Fuse profiler output to be converted.

       outfile
              specifies the Z80 map file for output.

BUGS
       None known.
