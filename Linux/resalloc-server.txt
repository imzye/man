NAME
       resalloc-server - resource allocator server

SYNOPSIS
       Resource allocator for expensive resources

       Server part doesn't accept argumetns; It is just simple server script which is usually executed from systemd's unit.

       The script is expected to be executed by resalloc user, and it accepts configuration files /etc/resallocserver/*.yaml

AUTHORS
       resalloc was written by Pavel Raiskup <praiskup@redhat.com>.

DISTRIBUTION
       The latest version of resalloc may be downloaded from ⟨https://github.com/praiskup/resalloc⟩
