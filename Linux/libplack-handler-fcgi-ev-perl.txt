NAME
       Plack::Handler::FCGI::EV - PSGI handler for FCGI::EV

SYNOPSIS
         > plackup -s FCGI::EV --listen :8080 myapp.psgi

DESCRIPTION
       Plack::Handler::FCGI::EV is an asynchronous PSGI handler using FCGI::EV as its backend.
