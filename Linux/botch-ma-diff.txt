NAME
       botch-ma-diff - diff two packages files for differences in their multiarch values

SYNOPSIS
       botch-ma-diff [options] packages1 packages2

DESCRIPTION
       Check for differences in multiarch annotations between two packages files.

OPTIONS
       -h, --help
           Show help.

       --verbose
           Be verbose.
