NAME
       mml2musicxml â€” convert MMLlib MML files to MusicXML

SYNOPSIS
       mml2musicxml [-h] path

DESCRIPTION
       mml2musicxml converts files in the MMLlib Music Macro Language format to MusicXML, compatible with software like MuseScore.

USAGE
       [-h]       Show usage information.

       path       Path to the MML file to convert.

       /usr/share/doc/mmllib-tools/MMLFILE.rst
