NAME
       clutter-scan-immodules — program to scan immodules and generate configuration files

SYNOPSIS
       clutter-scan-immodules

DESCRIPTION
       This manual page documents briefly the clutter-scan-immodules command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       clutter-scan-immodules  is  a  prgram  that scans immodules for Clutter and generate a configuration file which is suitable to be used as /etc/clutter-imcon‐
       text/clutter.immodules
