NAME
       botch-packages-diff - create a diff between two Packages or Sources files

SYNOPSIS
       botch-packages-diff [options] inPackages1 inPackages2

DESCRIPTION
       print the binary packages whose fields differ between two packages files.

OPTIONS
       --verbose
           Be verbose.

       -h, --help
           Display help.
