NAME
       popfile-bayes - dumps the word matrix for a given message

SYNOPSIS
       popfile-bayes <files> ...

DESCRIPTION
       The bayes.pl script provides a way to see the word matrix for any text message fed to the script.

SEE ALSO
       popfile-insert(8), popfile-pipe(8).
