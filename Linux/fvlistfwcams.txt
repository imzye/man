NAME
       fvlistfwcams - Show available Firewire camers.

SYNOPSIS
       fvlistfwcams

DESCRIPTION
       This program lists the Firewire (IEEE 1394) cameras connected to the
       system and accessible to the current user.

EXAMPLES
       fvlistfwcams
           List connected Firewire cameras.

       fawkes(8)
