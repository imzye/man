NAME
       pub2odg - manual page for pub2odg 0.9.6

SYNOPSIS
       pub2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `pub2odg' converts MS Publisher documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
