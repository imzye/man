NAME
       pvfs2-showcoll â€“ show server storage collection

SYNOPSIS
       pvfs2-mkspace  [-s  data_storage_space] [-m metadata storage space] [-c
       collection_name] [-d dspace_handle] [-vkh]

DESCRIPTION
       The pvfs2-showcoll utility displays the configuration of  a  particular
       storage  collection.   This  utility  operates  directly on the storage
       database and does not utilize the server.

       The options are as follows:

       -s     Specify the path to the data storage space.

       -m     Specify the path to the metadata storage space.

       -c     Specify the name of the collection.  Without this option, a list
              of collections available will be displayed.

       -d     Specify the dspace handle to display.

       -v     Enable verbose output.

       -k     Display data in keyval spaces.

       -h     Display synopsis.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set  the  OrangeFS debug mask.  Possible masks are documented in
              pvfs2-set-debugmask(1).
