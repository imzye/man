NAME
       teeworlds - Teeworlds Client

SYNOPSIS
       teeworlds ["option value"]...

DESCRIPTION
       This manual page documents briefly the teeworlds command.

       teeworlds is the Teeworlds client. It is an online multi-player platform 2D shooter.
