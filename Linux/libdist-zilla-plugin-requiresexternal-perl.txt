NAME
       Dist::Zilla::App::Command::externaldeps - print external libraries and binaries prerequisites

VERSION
       version 1.009

SYNOPSIS
       On the command line:

           % dzil externaldeps
           man
           sqlite3

DESCRIPTION
       This is a command plugin for Dist::Zilla. It provides the "externaldeps" command, which prints external prerequisites declared with Dist::Zilla::Plugin::RequiresExternal.
