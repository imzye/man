NAME
       lambda-term-actions - Display lambda-term editing actions

SYNOPSIS
       lambda-term-actions

DESCRIPTION
       lambda-term-actions displays the list of actions that can be used in the ~/.config/.lambda-term-inputrc file.

       Jérémie Dimino <jeremie@dimino.org>
