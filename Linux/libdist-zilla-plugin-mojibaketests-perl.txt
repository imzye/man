NAME
       Dist::Zilla::Plugin::MojibakeTests - Author tests for source encoding

VERSION
       version 0.8

SYNOPSIS
       In dist.ini:

           [MojibakeTests]

DESCRIPTION
       This is an extension of Dist::Zilla::Plugin::InlineFiles, providing the following file:

           xt/author/mojibake.t - a standard Test::Mojibake test
