NAME
       grab_color_image - grab an image using libdc1394

SYNOPSIS
       grab_color_image [--guid=/dev/video1394/x] [filename.ppm]

DESCRIPTION
       Get one image using libdc1394 and store it as a portable pix map (ppm) image.

OPTIONS
       --guid specifies camera to use. default is the first camera found in the system.  --help print help message
