NAME
       true - Returns true, a successful exit status.

       Immediately  returns with the exit status `0`, except when invoked with one of the recognized options. In those cases it will try to write the help or version text. Any IO error during this opera‐
       tion causes the program to return `1` instead.

SYNOPSIS
       true [--help] [--version]

DESCRIPTION
       Returns true, a successful exit status.

       Immediately returns with the exit status `0`, except when invoked with one of the recognized options. In those cases it will try to write the help or version text. Any IO error during this  opera‐
       tion causes the program to return `1` instead.

OPTIONS
       --help Print help information

       --version
              Print version information
