NAME
       autoimport - Automatically add import statements to Python code

SYNOPSIS
       autoimport [OPTIONS] [FILES]...

DESCRIPTION
       Automatically add import statements to Python code

OPTIONS
       --version
              Show the version and exit.

       --config-file TEXT

AUTHORS

DISTRIBUTION
