NAME
       external-cryptobone  -  Administration tool to change firewall settings
       and sshd configurations

SYNOPSIS
       /usr/bin/external-cryptobone

DESCRIPTION
       The program external-cryptobone is called by the  GUI  external-crypto‐
       bone-admin to perform administrative changes as root. It can be used to
       enable the external cryptobone daemon and to set or  reset  restrictive
       firewall  settings as well as a hardened secure shell daemon configura‐
       tion.

OPTIONS
       none

FILES
       /usr/bin/external-cryptobone

SEE ALSO
       libclr(3), external-cryptobone-admin(8), cryptobone(8)
