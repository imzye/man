NAME
       pvfs2-viewdist â€“ display file layout

SYNOPSIS
       pvfs2-viewdist [-v] -f file name

DESCRIPTION
       The pvfs2-viewdist displays the layout of a particular file.

       The options are as follows:

       -v     Print version number and exit.

       -f     Specify the file to operate on.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set  the  OrangeFS debug mask.  Possible masks are documented in
              pvfs2-set-debugmask(1).

       PVFS2TAB_FILE
              If set, the full pathname for an alternate pvfs2tab file

FILES
       /etc/pvfs2tab

EXAMPLES
       Display layout of /mnt/foo.

             pvfs2-viewdist -f /mnt/foo
