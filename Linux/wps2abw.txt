NAME
       wps2abw - manual page for wps2abw 0.9.6

SYNOPSIS
       wps2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wps2abw' converts MS Works Documents to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --encoding ENCODING
              set the INPUT encoding. Use --list-encodings to see which encodings can be used.

       --list-encodings
              show the available encodings and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output
