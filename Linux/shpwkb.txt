NAME
       shpwkb - test WKB binary Input / Output

SYNOPSIS
       shpwkb shp_file wkb_file

DESCRIPTION
       Utility to test WKB binary Input / Output.

OPTIONS
       shp_file
              the name of an existing shapefile

       wkb_file
              WKB output file

AUTHOR
       shpwkb is part of shapelib, maintained by Frank Warmerdam.
