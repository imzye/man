NAME
       guile-gnome-alt-2 - invoke guile and load Guile-Gnome-2 librares

SYNOPSIS
       guile-gnome-alt-2 [ ARG ] ...

DESCRIPTION
       Command guile-gnome-alt-2 runs the specified program so that Guile-Gnome libraries are loaded. See the documentation for guile for more information.

SEE ALSO
       guile info pages
