NOM
       swami  - (Sampled Waveforms And Musical Instruments = Formes d'onde échantillonnées et instruments musicaux) est une application pour l'édition et la gestion d'instruments MIDI, comme les fichiers
       SoundFont. Une API de programmation est également fournie pour l'intégration avec d'autres applications.

       Pour davantage d'informations peuvent être trouvées sur le site web du projet Swami http://swami.sourceforge.net.
