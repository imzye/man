NAME
       microhope - Interactive program to learn microcontroller programming.

SYNOPSIS

       microhope

DESCRIPTION
       microhope launches an Integrated Development Environment (IDE) which allows the user to develop several small programs for the ATmega32 microcontroller. This environment comes with useful examples
       and libraries which make the learn process easy.

SEE ALSO
       create-microhope-env(1).
