RKGETSTAT(3)                                                                                                                                                                                   RKGETSTAT(3)

名前
       RkGetStat - カレント候補に関する解析情報の取りだし

形式
       #include <canna/RK.h>
       int RkGetStat(cxnum，stat)
       int cxnum;
       Rkstat *stat;

説明
       RkGetStat は，カレント候補に関する解析情報を求めます．構造体 stat の各フィールドには，以下のような情報が設定されます．

         bunnum  カレント文節番号
         candnum カレント候補番号
         maxcand カレント文節の候補数
         diccand maxcandからRkBgnBun(3)のモード指定分を引いたもの
         ylen    カレント候補の読みがなのバイト数
         klen    カレント候補の漢字候補のバイト数
         tlen    カレント候補の構成単語数

戻り値
       0 を返します． 変換モードでないコンテクストで実行された場合には何もせずに -1 を 返します．
