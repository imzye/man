NAME
       PALETTE - Stores palette information. Allegro game programming library.

SYNOPSIS
       #include <allegro.h>

       typedef PALETTE RGB[PAL_SIZE]

DESCRIPTION
       Allegro palettes are arrays of PAL_SIZE RGB entries.
