SYNOPSIS
       zarchive input_path [output_path]

DESCRIPTION
       zarchive allows you to create and read file archives in the ZArchive format (.zar).

       ZArchive is yet another file archive format. Think of zip, tar, 7z, etc. but with the requirement of allowing random-access reads and supporting compression.

EXIT STATUS
       The zarchive utility exits 0 on success, and >0 if an error occurs.

EXAMPLES
       $ zarchive src
       Outputting to: src.zar
