NOMBRE
       ttv - muestra TV/video en una tty

SYNOPSIS
       ttv [ opciones ] [ nombre cadena ]

DESCRIPTION
       ttv muestra TV/video en un terminal, dibujando las imágenes mediante aalib.

OPCIONES
       -h     muestra un mensaje de ayuda.

       -f     usa la función de dibujado rápido de aalib.

       -c dispositivo
              dispositivo de video4linux (por defecto /dev/video0).

       -D driver
              driver de video4linux (por defecto "libv4l").

       Hay además muchas opciones para aalib, mire la documentación de aalib para más detalles.  El texto de ayuda incluye una lista.

VEA TAMBIÉN
