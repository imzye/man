NAME
       arb-kill - definitely kill all processes started by arb that are started by the user

       If option --all is given try sudo to kill all processes even if they are owned by other users.

SYNOPSIS
       arb-kill [--all]

DESCRIPTION
       Sometimes arb leaves some processes running.  All processes that belong to the user who is calling arb-kill will be killed.

SEE ALSO
       arb_clean(), arb(1), kill(1), sudo(1)
