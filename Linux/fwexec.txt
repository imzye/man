NAME
       fwexec - program to upload and rexecute image file to a connected NXT device

SYNOPSIS
       fwexec file

DESCRIPTION
       This manual page documents briefly the fwexec command.

       fwexec is a program to upload image file to a connected NXT device by the libNXT library and then to execute it.

OPTIONS
       This program has not any options and take file name of the NXT firmaware to upload

SEE ALSO
       nbc(1), fwflash(1), nexttool(1).
