NAME
       foomatic-preferred-driver - set default drivers in Foomatic database

SYNOPSIS
       foomatic-preferred-driver

DESCRIPTION
       This command reads the printer database overview and puts default driver entries (<driver>..</driver>) into every printer's XML database entry.

       It is very unlikely you need to do this.
