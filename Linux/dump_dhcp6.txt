NAME
       dump_dhcp6

SYNOPSIS
       dump_dhcp6 [-V vendorid] interface

OPTIONS
         -V vendorid  send vendorid number,string (e.g. 11,test)
         -N           use fe80:: as source
         -n           use empty mac as source

       DHCPv6 information tool. Dumps the available servers and their setup.
