NAME
       Event::RPC::Message::CBOR - CBOR message formatting

SYNOPSIS
         # Internal module. No documented public interface.

DESCRIPTION
       This module implements the message formatting of Event::RPC using CBOR. Objects of this class are created internally by Event::RPC::Server and Event::RPC::Client and performs message passing over
       the network.
