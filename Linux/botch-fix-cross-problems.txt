NAME
       botch-fix-cross-problems - fix Packages files with the output of botch-cross-problems

SYNOPSIS
       botch-fix-cross-problems [options] dose.yaml buildpackages hostpackages

DESCRIPTION
       fix build and host architecture Packages files according to the result of botch-dose2html

OPTIONS
       -h, --help
           Show help.

       -v, --verbose
           Be verbose.
