NAME
       sss_debuglevel - [DEPRECATED] change debug level while SSSD is running

SYNOPSIS
       sss_debuglevel [options] NEW_DEBUG_LEVEL

DESCRIPTION
       sss_debuglevel is deprecated and replaced by the sssctl debug-level
       command. Please refer to the sssctl man page for more information on
       sssctl usage.
