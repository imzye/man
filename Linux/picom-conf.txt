NAME
       picom-conf - configure the Picom compositor

SYNOPSIS
       picom-conf [options]

DESCRIPTION
       Picom Conf

OPTIONS
       -v, --version
              Displays version information.

       -h, --help
              Displays help on commandline options.

       --help-all
              Displays help including Qt specific options.
