NAME
       pvfs2-server  -  The  PVFS2  file  serving, request processing, dancing
       daemon

SYNOPSIS
       pvfs2-server [-d] [-f] [-h] <global_config_file> <local_config_file>

DESCRIPTION
       pvfs2-server runs on every node that is part of a PVFS2 volume.  A sin‚Äê
       gle  pvfs2-server  can serve as a metadata server or a datafile server,
       or both.

OPTIONS
       -h     print a helpful usage message

       -d     Keeps the server in the foreground.  Useful for debugging.

       -f     Server will create the file system storage space and exit.

       global_config_file
              Contains defaults applied to all servers.

       local_config_file
              Contains settings applied to the local instance of the server.
