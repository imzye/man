NAME
       DHT_bootstrap - A simple DHT bootstrap node

SYNOPSIS
       Usage (connected): DHT_bootstrap [--ipv4|--ipv6] IP PORT KEY

       Usage (unconnected): DHT_bootstrap [--ipv4|--ipv6]

DESCRIPTION
       DHT_bootstrap acts as a simple cross-platform DHT bootstrap node.

OPTIONS
       --ipv4 Use IPv4.

       --ipv6 Use IPv6.

AUTHOR
