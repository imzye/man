NAME
        icewm-set-gnomewm - modify GNOME settings to use IceWM

SYNOPSIS
       icewm-set-gnomewm [OPTIONS]

DESCRIPTION
       icewm-set-gnomewm is a helper script which instruments GNOME to replace
       its own WM with icewm.

OPTIONS
   GENERAL OPTIONS
       -h, --help
           Print a brief usage statement to stdout and exit.

       -V, --version
           Print the program version to stdout and exit.
