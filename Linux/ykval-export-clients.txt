NAME
       ykval-export-clients - Exports Client Info data from the yubikey-val server.

SYNOPSIS
       ykval-export-clients

DESCRIPTION
       Outputs a comma separated value file containing all Client Info from the yubikey-val database. This data can later be imported using ykval-import-clients.

       Database configuration is read from /etc/yubico/val/ykval-config.php

BUGS
       Report ykval-export-clients bugs in the issue tracker ⟨https://github.com/Yubico/yubikey-val/issues⟩

SEE ALSO
       ykval-import-clients(1)

       The Client Info data format ⟨https://developers.yubico.com/yubikey-val/doc/ClientInfoFormat.html⟩

       YubiKeys can be obtained from Yubico ⟨https://www.yubico.com/⟩.
