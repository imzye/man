NAME
     lumina-photo — a graphical utility to open pictures.

SYNOPSIS
     lumina-photo [FILES]

DESCRIPTION
     lumina-photo opens the graphical interface for the image viewer.

     It can open .jpeg, .png, .svg, .bmp, .tiff, and more files.  The user can use the command line to open a list of images.

EXAMPLES
           % lumina-photo Photos/Testing.png
     Opens the interface and shows Testing.png.
           % lumina-photo Photos/Testing.png Testing2.jpeg
     Opens the interface and shows Testing.png. Testing2.jpeg is into a list.

FILES
     /usr/local/bin/lumina-photo
