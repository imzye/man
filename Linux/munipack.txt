NAME
       munipack - command line interface for Munipack

SYNOPSIS
       munipack [options] action file(s)

DESCRIPTION
       Munipack is a general astronomical image processing software.  This utility provides the command line interface for all features of Munipack.

       The detailed description is very extensible and can be found in HTML documentation. There are summarized just help related options.  A short help list is build-in.

OPTIONS
       -h, --help
              Print this help. Try with a selected action.

       --version
              Print version and license.
