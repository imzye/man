NAME
       HTML::WikiConverter::PmWiki - Convert HTML to PmWiki markup

SYNOPSIS
         use HTML::WikiConverter;
         my $wc = new HTML::WikiConverter( dialect => 'PmWiki' );
         print $wc->html2wiki( $html );

DESCRIPTION
       This module contains rules for converting HTML into PmWiki markup. See HTML::WikiConverter for additional usage details.
