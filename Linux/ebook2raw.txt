NAME
       ebook2raw - manual page for ebook2raw 0.1.3

SYNOPSIS
       ebook2raw [OPTION] FILE

DESCRIPTION
       `ebook2text' is used to test libe-book.  Supported formats include FictionBook 2, BBeB and various Palm-based formats.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://sourceforge.net/p/libebook/tickets/> or <https://bugs.documentfoundation.org/>.
