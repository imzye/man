NAME
       centralclock - clock with a station clock look

SYNOPSIS
       centralclock

DESCRIPTION
       centralclock is a clock looking like a station clock. The minute hand has a jump on every full minute. In the digital model railway locking table spdrs60 it is used to show the current system time.

AUTHORS
