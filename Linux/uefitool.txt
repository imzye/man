NAME
       uefitool - UEFI firmware image viewer and editor

SYNOPSIS
       uefitool [image_file]

DESCRIPTION
       uefitool  is  a  cross-platform  C++/Qt program for parsing, extracting and modifying UEFI firmware images. It supports parsing of full BIOS images starting with the flash descriptor or any binary
       files containing UEFI volumes.
