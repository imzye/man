NAME
       git-clear - Rigorously clean up a repository

SYNOPSIS
       git-clear

DESCRIPTION
       Clears  the  repository  to  a state that it looks as if it was freshly
       cloned with the current  HEAD.  Basically  it  is  a  git-reset  --hard
       together  with  deletion  of all untracked files that reside inside the
       working directory, including those in .gitignore.

OPTIONS
       -f, --force

       Force the clean, with no warning to the user

       -h, --help, ?

       Display usage

EXAMPLES
       ·   Clears the repo, with user confirmation required.

           $ git clear

       ·   Clears the repo, without user confirmation.

           $ git clear -f
