NAME
       fh2text - manual page for fh2text 0.1.2

SYNOPSIS
       fh2text [OPTION] INPUT

DESCRIPTION
       `fh2text' converts FreeHand drawings to plain text.

OPTIONS
       --help show this help message

       --version
              show version information
