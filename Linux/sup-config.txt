NAME
       sup-config - interactive configuration tool for Sup

SYNOPSIS
       sup-config [options]

DESCRIPTION
       Interactive configuration tool for Sup.  Won’t destroy existing configuration.

OPTIONS
       -v, --version
              Print versian and exit

       -h, --help
              Show help message

FILES
       $HOME/.sup/config.yaml
              Configuration file for Sup

       $HOME/.sup/sources.yaml
              Configuration file for Sup mail sources

SEE ALSO
       sup(1), sup-add(1)
