NAME
       inifile - OpenLink Virtuoso Opensource ini File Editor

SYNOPSIS
       inifile [-fcnskv] [+inifile arg] [+create] [+nocreate] [+section arg] [+key arg] [+value arg]
       inifile [+help]

DESCRIPTION
       This manual page documents briefly the inifile command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.  Instead, it has documentation in the 'doc' VAD package.

       inifile is the OpenLink Virtuoso server

OPTIONS
       A summary of options is included below. For a complete description, see the 'doc' VAD package.

       +inifile
              use this ini file

       +create
              create the ini file if it does not exist (default)

       +nocreate
              do not create the ini file if it does not exist

       +section
              name of the section

       +key   name of the key

       +value the value you want to write

FILES
       /etc/virtuoso/virtuoso.ini
              The configuration file to control the behaviour of the main instance of inifile.

SEE ALSO
       The programs are documented fully by the 'doc' VAD package.
