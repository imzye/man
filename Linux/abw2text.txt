NAME
       abw2text - manual page for abw2text 0.1.3

SYNOPSIS
       abw2text [OPTION] INPUT

DESCRIPTION
       `abw2text' converts AbiWord documents to plain text.

OPTIONS
       --info display document metadata instead of the text

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
