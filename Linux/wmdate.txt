NAME
       wmdate - A Window Maker window manager Dock app

SYNOPSIS
       wmdate [options]

DESCRIPTION
       This manual page documents briefly the wmdate commands.  This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

       wmdate displays a 64x64 pixel calendar that's suitible for Docking in window manager docking bars.

OPTIONS
       The programs follow the usual GNU command line syntax, with long options starting with two dashes (`-') and short options starting with one.  A summary of options are included below.

       -s, --shape
              Shaped window.

       -t, --test
              Testing Mode.

       -h, -help, --help
              Show summary of options.

       -v, --version
              Show version of program.
