NAME
       sphinx_cepview - View acoustic feature files

SYNOPSIS
       sphinx_cepview [ options ]...

DESCRIPTION
       This program reads acoustic feature files in Sphinx format and displays their contents as text for inspection.

       -b     The beginning frame 0-based.

       -d     Number of displayed coefficients.

       -describe
              Whether description will be shown.

       -e     The ending frame.

       -f     feature file.

       -header
              Whether header is shown.

       -i     Number of coefficients in the feature vector.
