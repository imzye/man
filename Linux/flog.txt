NAME
       flog - dump STDIN to file and reopen on SIGHUP

SYNOPSIS
       flog [ -t ] [ -T format ] [ -l number ] [ -F fifo ] [ -p pidfile ] logfile

DESCRIPTION
       flog (file logger) reads input from STDIN and writes to a file. When a SIGHUP is received, the file will be reopened, allowing for log rotation.

OPTIONS
       -t     prepend each line with "YYYYMMDD;HH:MM:SS: "

       -T format
              prepend each line with specified strftime(3) format

       -l number
              log file length limit (force truncation)

       -F fifo
              read from fifo instead of pipe

       -p pidfile
              create pid file
