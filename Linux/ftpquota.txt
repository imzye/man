NAME
       ftpquota - Perl script for managing limit/tally files for mod_quotatab

SYNOPSIS
       ftpquota [ options ]

DESCRIPTION
       ftpquota  is  a  Perl  script  designed to create and manage limits and
       tally files for the mod_quotatab + mod_quotatab_file module combination
       for proftpd.

       Full     documentation     for    ftpquota    can    be    found    at:
       http://www.proftpd.org/docs/utils/ftpquota.html
