NAME
       foomatic-replaceoldprinterids - Replace references to old printer IDs

SYNOPSIS
       foomatic-replaceoldprinterids

DESCRIPTION
       Replaces  references  to  old numerical or otherwise obsolete printer IDs in arbitrary file. By default it uses the db/oldprinterids file of the current Foomatic database as translation table, but you
       can use arbitrary translation tables.

       It is very unlikely you need to use this.
