NAME
       xpsk31 - a GTK+ PSK31 communication application

SYNOPSIS
              xpsk31 [-hv]

DESCRIPTION
              [-h: print this usage information and exit]
              [-v: print xpsk31 version number and exit]
SEE ALSO
       The full documentation for xpsk31 is maintained as an html document.  If the xpsk31 program is properly installed at your site, the command

              firefox /usr/share/doc/xpsk31/xpsk31.html

       should give you access to the complete manual.
