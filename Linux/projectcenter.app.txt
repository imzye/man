NAME
       ProjectCenter - The Official GNUstep IDE

SYNOPSIS
       ProjectCenter

DESCRIPTION
       This manual page documents briefly the ProjectCenter command.

       ProjectCenter  is  an integrated development environment (IDE) for the GNUstep framework.  It has support for `Aggregate', `Application', `Bundle', `Framework', `Library', `ResourceSet' and `Tool'
       types of projects.

SEE ALSO
       Gorm(1), GNUstep(7).
