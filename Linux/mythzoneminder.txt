NAME
       mythzmserver - MythTV Zoneminder server

DESCRIPTION
       Invalid  argument:  --help Valid options are: -p or --port number        A port number to listen on (default is 6548) -d or --daemon             Runs mythzmserver as a daemon -n or --nodaemon
       Does not run mythzmserver as a daemon (default) -c or --zmconfig           Location of zoneminders default config file (default is /etc/zm/zm.conf)  -o  or  --zmoverrideconfig    Location  of
       zoneminders  override  config  file (default is /etc/zm/conf.d/01-system-paths.conf) -l or --logfile filename   Writes STDERR and STDOUT messages to filename -v or --verbose            Prints
       more debug output
