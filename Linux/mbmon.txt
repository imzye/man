NOM
       mbmon - MotherBoard Monitor

SYNOPSIS
       mbmon [options] <secondes d'attente> (par defaut 5 sec)

       Cette page de manuel décrit rapidement la commande mbmon.

       Elle a été écrite pour la distribution Debian parce que le programme original n'en possédait pas.

       Les  carte-mères  récentes  possèdent des fonctions pour surveiller les températures des processeurs, la vitesse de rotation des ventitaleurs etc ... Bien que des programmes utilisant ces capteurs
       matériels aient été développés pour plateformes Microsoft Windows, il semble qu'aucun n'existe sur les plateformes PC-UNIX et X Window System.

       mbmon utilisé à la ligne de commande affiche les températures, voltages et tours par minutes des processuers et ventilateurs. Il peut également être utilisé à distante via telnet, grâce à l'option
       "-P".

OPTIONS
       -c compte
              répéter <compte> fois puis sortir.

       -d     mode de débogage (toutes les autres options excepté (V|S|I) seront ignorées).

       -e [0-2]
              configure la température [0-2] pour afficher la température supplémentaire (nécéssite -A)

       -f     affiche la température en Fahrenheit.

       -h     affiche un résumé des options disponibles.

       -i     affiche les données sous forme d'entiers (avec l'option -T).

       -n, -N affiche le nom d'hôte (long|court).

       -P port
              fonctionne en mode démon, écoutant les clients sur le port donné.

       -p chip
              pour la détection des chips (chip=winbond|wl784|via686|it87|gl52|lm85|lm80|lm90|lm75)

       -r     affiche au format TAG et Valeur.

       -T|F [1-7]
              affiche la température|vitesse de rotation selon les styles suivants:
                      style1: data1
                      style2: data2
                      style3: data3
                      style4: data1 data2
                      style5: data1 data3
                      style6: data2 data3
                      style7: data1 data2 data3

       -t     affiche la date courante.

       -u     affiche l'uptime.

       -V, -S, -I, -A
              spécifie la méthode d'accès (utilisant "VIA686 HWM directement"|"SMBus"|"port d'E/S ISA"|"toutes ces méthodes").

       -Y     pour les cartes-mères Tyan Tiger MP/MPX.

EXeMPLES
       mbmon -P 11000; telnet localhost 11000; sudo killall mbmon
       mbmon -c1
       mbmon 1 (temps réel ;) )

VOIR AUSSI
