NAME
       django_extensions - autoradio jack daemon

SYNOPSIS
       jackdaemon <action> [options] (action=start|stop|restart|run|version)

DESCRIPTION
       django_extensions is not installed; I do not use it

       jackdaemon daemon for autoradio suite

OPTIONS
       --version
              show program's version number and exit

       -h, --help
              show this help message and exit

       --pidfile=PIDFILE
              PID filename (default /tmp/jackdaemon.lock)

       --stdin=STDIN
              stdin filename (default /dev/null)

       --stdout=STDOUT
              stdout filename (default /tmp/jackdaemon.log)

       --stderr=STDERR
              stderr filename (default /tmp/jackdaemon.err)

       --user=USER
              user name or id (default autoradio)

       --group=GROUP
              group name or id (default audio)

       jackdaemon 3.6
