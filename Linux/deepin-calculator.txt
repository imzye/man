NAME
       deepin-calculator - Calculator for deepin

SYNOPSIS
       deepin-calculator

DESCRIPTION
       deepin-calculator is an easy to use calculator for ordinary users.

OPTIONS
       This program does not accept any command line options.

SEE ALSO
       https://www.deepin.org/en/original/deepin-calculator/
