NAME
       sd2text - manual page for sd2text 0.0.7

SYNOPSIS
       sd2text [OPTION] INPUT

DESCRIPTION
       `sd2text' converts StarOffice documents to plain text.

OPTIONS
       -i     show document metadata instead of the text

       -h     show this help message

       -o OUTPUT
              write ouput to OUTPUT

       -p PASSWORD
              set password to open the file

       -v     show version information

REPORTING BUGS
