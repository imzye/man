NAME
       rpmgrill-update-ruby-advisory-db - maintain a local cache of ruby gem CVEs

SYNOPSIS
       rpmgrill-update-ruby-advisory-db [--verbose] [--cachedir=DIR]

       rpmgrill-update-ruby-advisory-db  --help  |  --version | --man

DESCRIPTION
       rpmgrill-update-ruby-advisory-db maintains a local cache of the Ruby Gem advisory database:

          https://github.com/rubysec/ruby-advisory-db

       It's basically a "git checkout" (first time only) followed by frequent "git update"s.

OPTIONS
       --cachedir=DIR
           Use DIR as the local cache directory. The default is to use the same directory defined in the rpmgrill SecurityPolicy plugin.

       --verbose
           Run git without the "-q" (quiet) option

       --help
           Emit usage hints.

       --version
           Display program version.

       --man
           Display this man page.
