NAME
       linaro-bcb-util - program to read or write the BCB partition

SYNOPSIS
       linaro-bcb-util BCB_partition <command> <field> [data]

DESCRIPTION
       This manual page documents briefly the linaro-bcb-util command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       linaro-bcb-util is a program that read or write the BCB partition.

OPTIONS
       The program has the follows options.

       BCB_partition
              The path of the misc partition device.

       command
              read, write, clear or dump.

       field  command, status, recovery, or stage.

       data   the data to write to the BCB’s field

BUGS
       The upstream BTS can be found at https://gitlab.com/grandpaul/linaro-bcb-util/-/issues.
