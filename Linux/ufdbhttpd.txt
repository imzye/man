NAME
       ufdbhttpd - very simply HTTP daemon for ufdbGuard

SYNOPSIS
       ufdbhttpd

DESCRIPTION
       ufdbhttpd is a very simple HTTP daemon that only knowns how to serve redirection messages produced by ufdbguardd.

       ufdbhttpd should never be run by a user: it is intended to be started by ufdbguardd exclusively.

SEE ALSO
       ufdbguardd(1), ReferenceManual_1_34.pdf, http://www.urlfilterdb.com/en/downloads/documentation.html.
