NAME
       vsd2raw - manual page for vsd2raw 0.1.7

SYNOPSIS
       vsd2raw [OPTION] INPUT

DESCRIPTION
       `vsd2raw' is used to test import of Microsoft Visio documents in libvisio.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
