NAME
       gbi3 - GAMBAS Component Informer

SYNOPSIS
       gbi3 [options] [components]

DESCRIPTION
       Gambas  is  a free development environment based on a Basic interpreter
       with object extensions, a mix of Java(tm) and Visual  Basic(tm).   With
       Gambas,  you can quickly design your program GUI, access MySQL or Post‐
       greSQL databases, pilot KDE applications with DCOP, translate your pro‐
       gram into many languages, and so on...

       gbi3 allows you to generate component description files.

OPTIONS
       -h, --help
              display help and exit

       -L, --license
              display license

       -r, --root <directory>
              gives the gambas installation directory

       -v, --verbose
              verbose output

       -V, --version
              display version

AVAILABILITY
       The latest version of Gambas can always be obtained from gambas.source‐
       forge.net,  documentation  about  the  language   is   at   http://gam‐
       baswiki.org/wiki.
