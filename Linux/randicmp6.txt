NAME
       randicmp6

       Sends all ICMPv6 type and code combinations to destination.  Option -s  sets the source IPv6 address.  Option -p  will not print answers received.

AUTHOR
       thc-ipv6 was written by van Hauser <vh@thc.org> / THC

       The homepage for this toolkit is: http://www.thc.org/thc-ipv6

COPYRIGHT
       Syntax: ./randicmp6 [-p] [-s sourceip] interface destination [type [code]]
