NAME
       icingadb-migrate - IDO to Icinga DB migrator

SYNOPSIS
       icingadb-migrate [-h] [-c PATH] [-t PATH]

DESCRIPTION
       icingadb-migrate migrates IDO history to Icinga DB.

OPTIONS
       Application Options:

       -c=PATH, --config=PATH
              Path to config file

       -t=PATH, --cache=PATH
              Path for caching

       Help Options:

       -h, --help
              Show usage information and exit
