NAME
       sludge-floormaker - edit SLUDGE floor plans

SYNOPSIS
       sludge-floormaker [floorplan.flo]

DESCRIPTION
       This manual page documents briefly the sludge-floormaker command.

       This manual page was written for the Debian distribution because the original program does not have a manual page. Instead, it has documentation in the html
       format; see below.

FILES
       ${HOME}/.config/sludge-devkit/
           This directory is used to store configuration files.

BUGS
       Please report bugs in the SLUDGE forum at http://opensludge.sourceforge.net.

SEE ALSO
       sludge-devkit(1), sludge-engine(1)

       The full SLUDGE documentation is included in the package sludge-doc. It is installed to /usr/share/doc/sludge/SLUDGE_Help.html.
