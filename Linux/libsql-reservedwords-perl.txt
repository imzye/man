NAME
       sqlrw -- Reserved SQL words by standard and vendors.

SYNOPSIS
       sqlrw [options] word

       Options:

               -v, --verbose   Display versions
               -h, --help      Display this help
               -H, --man       Longer manpage
               -V, --version   Display version info
