NAME
       ffinfo - Print Fawkes version

SYNOPSIS
       ffinfo

DESCRIPTION
       This little application prints "Fawkes VERSION", where VERSION will be
       the actual installed version number.

EXAMPLES
       ffinfo
           Get version information.

SEE ALSO
       fawkes(8)
