NAME
       sd2epub - manual page for sd2epub 0.9.6

SYNOPSIS
       sd2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `sd2epub' converts legacy StarOffice documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --password PASSWORD
              set password to open the file
