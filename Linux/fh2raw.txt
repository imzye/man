NAME
       fh2raw - manual page for fh2raw 0.1.2

SYNOPSIS
       fh2raw [OPTION] INPUT

DESCRIPTION
       `fh2raw' is used to test libfreehand.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
