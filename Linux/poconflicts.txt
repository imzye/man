NAME
       poconflicts - Conflict finder for Gettext PO localization files.

SYNOPSIS
       poconflicts  [--version]  [-h|--help] [--manpage] [--progress PROGRESS] [--errorlevel ERRORLEVEL] [-i|--input] INPUT [-x|--exclude EXCLUDE] [-o|--output] OUTPUT [-I|--ignore-case] [-v|--invert] [--ac‚Äê
       celerator ACCELERATORS]

DESCRIPTION
       See: http://docs.translatehouse.org/projects/translate-toolkit/en/latest/commands/poconflicts.html for examples and usage instructions.

OPTIONS
       --version
              show program's version number and exit

       -h/--help
              show this help message and exit

       --manpage
              output a manpage based on the help

       --progress
              show progress as: dots, none, bar, names, verbose

       --errorlevel
              show errorlevel as: none, message, exception, traceback

       -i/--input
              read from INPUT in po format

       -x/--exclude
              exclude names matching EXCLUDE from input paths

       -o/--output
              write to OUTPUT in po format

       -I/--ignore-case
              ignore case distinctions

       -v/--invert
              invert the conflicts thus extracting conflicting destination words

       --accelerator
              ignores the given accelerator characters when matching
