NAME
       showriff - show structure of RIFF files (wav, avi, ...)

SYNOPSIS
       showriff [ options ] filename

DESCRIPTION
       showriff displays the structure of RIFF files.  Common media files which use the RIFF format are wave-files and avi-movies.

OPTIONS
       -h     display help text

       -j     try to decode mjpeg headers

       -e     try to continue on errors.
