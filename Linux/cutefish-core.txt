NAME
       cutefish-screen-brightness - Utility to change the screen brightness in Cutefish DE.

SYNOPSIS
       cutefish-screen-brightness [options]

OPTIONS
       -h, --help
              Displays help on commandline options.

       --help-all
              Displays help including Qt specific options.

       -v, --version
              Displays version information.

       --set value
              Set screen brightness, value ranges [0-100]

       -i     Increase brightness

       -d     Decrease brightness
