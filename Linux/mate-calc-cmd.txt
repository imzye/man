NAME
       mate-calc-cmd - A console calculator for the MATE Desktop Environment.

SYNOPSIS
       mate-calc-cmd

DESCRIPTION
       mate-calc-cmd is the console version of mate-calc, the calculator for the MATE Desktop Environment.
