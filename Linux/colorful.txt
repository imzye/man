NAME

       colorful

CONTROLS
       The game is played solely using the keyboard. Default key bindings:

       Arrow keys
              Movement

       Z/X    Shoot left / right

       -/+    Volume down / up

       P      Pause / unpause

       These key bindings can be fully redefined.

DIRECTORIES
       Game data files are  located  in  /usr/share/suve/colorful.   The  game
       stores its settings and savegames in ~/.suve/colorful/.
