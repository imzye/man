NAME
       fstrandgen - manual page for fstrandgen 1.6.9

DESCRIPTION
       Generates random paths through an FST.

              Usage: fstrandgen [in.fst [out.fst]]

       PROGRAM FLAGS:

       --max_length: type = int32, default = 2147483647

              Maximum  path  length  --npath: type = int32, default = 1 Number of paths to generate --remove_total_weight: type = bool, default = false Remove total weight when output weighted --seed: type =
              int32, default = 1628231198 Random seed --select: type = string, default = "uniform" Selection type: one of:   "uniform",  "log_prob"  (when  appropriate),  "fast_log_prob"  (when  appropriate)
              --weighted: type = bool, default = false Output tree weighted by path count vs. unweighted paths
