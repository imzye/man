NAME
       cslinker  - extend the list of defects by CWE numbers, scan properties,
       etc.

SYNOPSIS
       cslinker [options] proj.err [...], where options are:

DESCRIPTION
       --cwelist arg
              (re)assign CWE numbers to defects by the given CSV list

       --implist arg
              mark reports from the specified list as important

       --inifile arg
              load scan properties from the given INI file

       --reapply-parsing-rules
              canonicalize data originally parsed by an older version  of  the
              parser

       -q [ --quiet ]
              do not report non-fatal errors

       --help produce help message

       --version
              print version
