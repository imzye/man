NAME
       op1svg - normalize SVG files so that the OP-1 understands them

SYNOPSIS
       op1svg [-h] [--debug] [--version] in_path out_path

OPTIONS
   POSITIONAL ARGUMENTS
       in_path
              file path of svg file to fix

       out_path
              file path to write the fixed svg to

   OPTIONAL ARGUMENTS
       -h, --help
              show a help message and exit

       --debug
              print debug messages

       --version
              show programâ€™s version number and exit
