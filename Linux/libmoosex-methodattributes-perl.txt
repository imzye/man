NAME
       MooseX::MethodAttributes::Role::AttrContainer::Inheritable - capture code attributes in the automatically initialized metaclass instance

VERSION
       version 0.32

DESCRIPTION
       This role extends "MooseX::MethodAttributes::Role::AttrContainer" with the functionality of automatically initializing a metaclass for the caller and applying the meta roles relevant for capturing
       method attributes.
