NOM
       sensible-browser - navigateur web pratique

SYNOPSIS
       sensible-browser url

DESCRIPTION
       sensible-browser  prend  des décisions sensées sur le choix du navigateur qu'il faut appeler. Les programmes de la distribution Debian peuvent utiliser ce scripts comme leur navigateur web par dé‐
       faut
       La variable d'environnement BROWSER peut être positionnée et sera utilisé si positionnée. Toute chaîne de caractère acceptable comme chaîne de commande pour la commande sh -c est  valide  (ce  qui
       est le même comportement que la variable EDITOR comme documenté dans environ(7)).

VOIR AUSSI
       environ(7)

STANDARD
       La  documentation du comportement du paquet sensible-utils pour un système debian est disponible au titre 11.4 du manuel de la politique debian qui est usuellement installé sous /usr/share/doc/de‐
       bian-policy (il convient d'installer le paquet debian-policy)
