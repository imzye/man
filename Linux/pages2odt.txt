NAME
       pages2odt - manual page for pages2odt 0.9.6

SYNOPSIS
       pages2odt [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `pages2odt' converts Apple Pages documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
