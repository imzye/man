NAME
       update-sccache-symlinks - update symlinks in /usr/lib/sccache

SYNOPSIS
       update-sccache-symlinks

DESCRIPTION
       update-sccache-symlinks creates and removes symlinks in /usr/lib/sccache. See the sccache man page for more information.

       It is normally not necessary to run update-sccache-symlinks by hand as this is done automatically when compiler packages are removed or installed.

SEE ALSO
       sccache(1)
