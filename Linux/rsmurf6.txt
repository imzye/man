NAME
       rsmurf6

SYNOPSIS
       rsmurf6 interface victim-ip

       Smurfs the local network of the victim. Note: this depends on an implementation error, currently only verified on Linux.  Evil: "ff02::1" as victim will DOS your local LAN completely
