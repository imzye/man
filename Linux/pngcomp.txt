NAME
       pngcomp - PNG (Portable Network Graphics) comparison tool

SYNOPSIS
       pngcomp [-vVh] image1.png image2.png

DESCRIPTION
       pngcomp gives some basic statistics about the difference between two images. It was created as a measure of various color quantization methods.

OPTIONS
       -v     Verbose, does nothing as yet.

       -V     Version, prints version information.

       -h     Help, prints this message.

NOTES
