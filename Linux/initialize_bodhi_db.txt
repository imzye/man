NAME
       initialize_bodhi_db - initialize bodhi's database

SYNOPSIS
       initialize_bodhi_db CONFIG_URI

DESCRIPTION
       initialize_bodhi_db  is  used to create the initial database tables for
       the Bodhi server. It uses the given CONFIG_URI  to  find  the  database
       settings to use.

EXAMPLE
       $ initialize_bodhi_db /etc/bodhi/production.ini
