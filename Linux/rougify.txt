NAME
       rougify - manual page for rougify 3.26.1

DESCRIPTION
       usage: rougify {global options} [command] [args...]

   where <command> is one of:
       highlight
              highlight code

              debug  help            print help info style           print CSS styles list            list available lexers guess           guess the languages of file version         print the rouge version
              number

   global options:
       --require|-r <fname>
              require <fname> after loading rouge

       See `rougify help <command>` for more info.
