NAME
       simul-beacons - Robot dataset simulator

SYNOPSIS
       simul-beacons config_file.ini

DESCRIPTION
       simul-beacons is a command-line application which generates a synthetic rawlog of a simulated robot as it follows a predefined path and observes a set of beacons (through range-only measurements).
