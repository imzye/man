NAME
       Boulder - a reimplementation of 'Boulder Dash'

DESCRIPTION
       Boulder has some new elements and a level editor to create more levels. Local high-score. Currently there are 25 levels.

AUTHOR
       Boulder was written by Ronnie Hedlund <huj56745@gmail.com>.
