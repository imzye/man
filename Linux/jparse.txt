NAME
       jparse - parse JSON files

SYNOPSIS
       jparse [file] ...

DESCRIPTION
       jparse is a small utility based on the jaula library that checks JSON formatted files, detects errors and outputs a single-line representation of them to the standard output.

AUTHOR
