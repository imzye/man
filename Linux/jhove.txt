NAME
       jhove, jhove-gui - identify, validate, and characterize digital objects.

SYNOPSIS
       jhove [options] files...
       jhove-gui

DESCRIPTION
       This manual page documents briefly the jhove and jhove-gui commands.

       jhove-gui  has  a graphical user interface and no significant command line options.  jhove has a command line interface, see full documentation online at http://jhove.openpreservation.org/getting-
       started
