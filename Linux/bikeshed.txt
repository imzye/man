NAME
       bch - open the debian/changelog in sensible editor, appending a line showing the current files that have been modified

SYNOPSIS
       bch

DESCRIPTION
       bch  is a program that determines the files that have been modified in the current bzr tree, and opens debian/changelog for editing, using dch(1), and appending a changelog entry for the cur‚Äê
       rent list of modified files.
