NAME
       MinOZW - a small test application for libopenzwave

SYNOPSIS
       MinOZW [port]

DESCRIPTION

       MinOZW is just a small example to test the openZWAVE library.  The first argument is the port where the ZWAVE controller is attached to. The default value is /dev/ttyUSB0.
