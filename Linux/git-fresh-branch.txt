NAME
       git-fresh-branch - Create fresh branches

SYNOPSIS
       git-fresh-branch <branchname>

DESCRIPTION
       Creates empty local branch named <branchname>.

OPTIONS
       <branchname>

       The name of the branch to create.

EXAMPLES
       $ git fresh-branch docs
