NAME
       pub2epub - manual page for pub2epub 0.9.6

SYNOPSIS
       pub2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `pub2epub' converts MS Publisher documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit
