NAME
       fb_wrapper -- Firebird command line utility wrapper

SYNOPSIS
       firebird command [--fb-ver versiona] [arguments]

DESCRIPTION
       fb_wrapper is a wrapper script that makes it available to choose the version of the real Firebird command to use. Symlinks to it are placed in /usr/bin instead of the real Firebird utilities,
       which are in /usr/lib/firebird/VERSION/bin instead.

OPTIONS
       --fb-ver version
           Choose version as the target Firebird version. Default is the latest available version.
