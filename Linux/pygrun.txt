NAME
       pygrun - manual page for pygrun 4.9.3

SYNOPSIS
       pygrun [options] Grammar_Name Start_Rule

OPTIONS
       -h, --help
              show this help message and exit

       -t, --tree
              Print AST tree

       -k, --tokens
              Show Tokens

       -s, --sll
              Show SLL

       -d, --diagnostics
              Enable diagnostics error listener

       -a, --trace
              Enable Trace
