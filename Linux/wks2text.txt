NAME
       wks2text - manual page for wks2text 0.4.12

SYNOPSIS
       wks2text [OPTION] FILE

DESCRIPTION
       `wks2text' converts various spreadsheets and database formats to plain text.  Supported formats are Microsoft Works Spreadsheet and Database, Lotus Wk1-Wk4 and Quattro Pro Wq1-Wq2.

OPTIONS
       -e ENCODING
              define the file encoding, where encoding can be

              CP037,  CP424,  CP437,  CP737,  CP500,  CP775,  CP850,  CP852, CP855, CP856, CP857, CP860, CP861, CP862, CP863, CP864, CP865, CP866, CP869, CP874, CP875, CP1006, CP1026, CP1250, CP1251, CP1252,
              CP1253, CP1254, CP1255, CP1256, CP1257, CP1258, MacArabic, MacCeltic, MacCEurope, MacCroation, MacCyrillic, MacDevanage, MacFarsi,  MacGaelic,  MacGreek,  MacGujarati,  MacGurmukhi,  MacHebrew,
              MacIceland, MacInuit, MacRoman, MacRomanian, MacThai, MacTurkish.

       -h     show this help message

       -o FILE
              set the output file

       -p PASSWORD
              set password to open the file

       -v     show version information
