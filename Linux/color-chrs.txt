NAME
       color-chrs.pl - Plot output of "bcftools +color-chrs"

SYNOPSIS
       color-chrs.pl [OPTIONS] output.dat

OPTIONS
       -c, --colors file
              File with list of "chr hap color".

       -p, --prefix name
              Prefix of output files

       -h, -?, --help
              Summary of options.
