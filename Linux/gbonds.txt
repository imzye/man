NAME
       gbonds - U.S. Savings Bond inventory program for GNOME

SYNOPSIS
       gbonds [options] files...

DESCRIPTION
       GBonds  is a U.S. Savings Bond inventory program for the GNOME desktop environment.  It allows you to track the current redemption value and performance of your U.S. Savings Bonds and keep a valu‚Äê
       able record of the bonds you own.
