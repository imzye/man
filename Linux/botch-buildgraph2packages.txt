NAME
       botch-buildgraph2packages - convert buildgraph to packages

SYNOPSIS
       botch-buildgraph2packages [-h] [-v] buildgraph.xml Packages

DESCRIPTION
       Given a buildgraph in GraphML format and a Packages file, output all binary packages in deb822 format that the build graph contains in its installation sets.

OPTIONS
       -v, --verbose
           Be verbose.

       -h, --help
           Print help message.
