NAME
       cdr2raw - manual page for cdr2raw 0.1.7

SYNOPSIS
       cdr2raw [OPTION] FILE

DESCRIPTION
       `cdr2raw' is used to test import of CorelDRAW documents in libcdr.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information and exit

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
