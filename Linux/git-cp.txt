NAME
       git-cp - Copy a file keeping its history

SYNOPSIS
       git-cp <current_filename> <destination_filename>

DESCRIPTION
       Copy  a  file  keeping its git history. This allows merge conflict han‐
       dling.

EXAMPLES
       Copy README into README.txt

           $ git cp README README.txt
