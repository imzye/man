NAME
       cdr2xhtml - manual page for cdr2xhtml 0.1.7

SYNOPSIS
       cdr2xhtml [OPTION] FILE

DESCRIPTION
       `cdr2xhtml' converts CorelDRAW documents to SVG.

OPTIONS
       --help show this help message

       --version
              show version information and exit
