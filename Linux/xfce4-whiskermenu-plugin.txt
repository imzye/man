NAME
       xfce4-popup-whiskermenu - shows Whisker Menu

SYNOPSIS
       xfce4-popup-whiskermenu [OPTION...]

DESCRIPTION
       xfce4-popup-whiskermenu shows Whisker Menu if at least one Whisker Menu applet is present on one of your panels.

       The default behavior is to show Whisker Menu near the applet button. If you have several Whisker Menu applets, only the first one you have set up will pop up unless you override it with instance.

OPTIONS
       -h, --help
              Show help options.

       -p, --pointer
              Popup menu at current mouse position.

       -c, --center
              Popup menu at center of screen.

       -l, --list
              Print available menu instance IDs.

       -i, --instance=instance
              Choose which menu to popup by instance ID.

       -V, --version
              Print version information and exit.
