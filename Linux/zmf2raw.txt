NAME
       zmf2raw - manual page for zmf2raw 0.0.2

SYNOPSIS
       zmf2raw [OPTION] INPUT

DESCRIPTION
       `zmf2raw' is used to test libzmf.

       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information and exit
