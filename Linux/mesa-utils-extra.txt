NAME
       es2gears - ``gears'' demo for GLES2

SYNOPSIS
       es2gears

DESCRIPTION
       The es2gears program is a port of the ``gears'' demo to GLES2. It displays a set of rotating gears and prints out the frame rate at regular intervals. It has
       become quite popular as basic benchmarking tool.
