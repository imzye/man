NAME
       flood_unreach6

SYNOPSIS
       flood_unreach6 [-HFD] [-p|-h] [-r|-R] interface target

       Flood  the  target with ICMPv6 unreachable packets.  -F/-D/-H add fragment/destination/hopbyhop header to bypass simple filters -p/-h    send port/host unreachable instead of network unreachable -r/-R
       randomize source from /64 / randomize source completely
