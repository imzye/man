NAME
       mine_detector - a mine-finding game

DESCRIPTION
       Mine  Detector  is a mine-finding game with somewhat different rules from other mine-finding games. Mine Detector rarely requires guessing. Only at the higher levels may guessing sometimes be the only
       way to win.

       For instructions on how to play, start the game and click the Rules button.

AUTHORS
