NAME
       tuned-profiles-atomic - description of profiles provided for the Project Atomic

DESCRIPTION
       These profiles are provided for the Project Atomic. They provides performance optimizations for the Atomic hosts (bare metal) and virtual guests.

PROFILES
       The following profiles are provided:

       atomic-host
              Profile optimized for Atomic hosts (bare metal). It is based on throughput-performance profile. It additionally increases SELinux AVC cache, PID limit and tunes netfilter connections tracking.

       atomic-guest
              Profile optimized for virtual Atomic guests. It is based on virtual-guest profile. It additionally increases SELinux AVC cache, PID limit and tunes netfilter connections tracking.
