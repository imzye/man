NAME
       editcomment - program to edit comments of Ogg vorbis files

SYNOPSIS
       editcomment file

DESCRIPTION
       editcomment is a program that edits comments of Ogg forbis files.

OPTIONS
       This program only takes one argument: the file, the comment of which bill be edited.
