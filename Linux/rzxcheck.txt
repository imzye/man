NAME
       rzxcheck — Check the signature in a ZX Spectrum RZX file

SYNOPSIS
       rzxcheck [OPTION] rzxfile

DESCRIPTION
       rzxcheck verifies the digital signature found in a Sinclair ZX Spectrum
       RZX input recording file.

OPTIONS
       -h, --help
              give brief usage help, listing available options.

       -V, --version
              output version information.

       rzxfile
              specifies an input recording file whose signature is to be veri‐
              fied.

BUGS
       None known.
