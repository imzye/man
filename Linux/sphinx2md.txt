NAME
       sphinx2md - manual page for sphinx2md 0.6

DESCRIPTION
       usage: sphinx2md [-h] [-W] rst_file

       Convert Sphinx ReST file to Markdown

   positional arguments:
       rst_file
              ReST file to convert

   options:
       -h, --help
              show this help message and exit

       -W, --warn-is-error
              turn warnings into errors
