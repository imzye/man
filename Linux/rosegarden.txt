NAME
       rosegarden - MIDI and audio sequencer and musical notation editor.

SYNOPSIS
       rosegarden [options] [File]

DESCRIPTION
       This manual page documents briefly the rosegarden command.

       rosegarden  is  a  MIDI  and audio sequencer, notation editor, and general-purpose music composition and editing application for Unix and Linux. It is a Qt application meaning you need the Qt5 li‐
       braries. However, it can be run within any desktop or window manager environment.

       Most options for rosegarden (such as JACK output channels) are configured through the graphical interface.

OPTIONS
       --help
            Show help about options.

       --version
            Show version information.

       --nosplash
            Don´t show splash screen.

       --nosequencer
            Don´t use an external sequencer.

       File
            File to load at startup. Typically a rosegarden (.rg) file, but it can also be a rosegarden2.1 file or a MIDI file.
