NAME
       bitmask-root - Easy, fast, and secure VPN service from riseup.net

DESCRIPTION
       This is bitmask-root version 19

       This program manipulates the Bitmask firewall. It is *not* intented to be used manually.

       Commands:

       bitmask-root version bitmask-root restart bitmask-root openvpn start <args> bitmask-root openvpn stop bitmask-root firewall start <args> bitmask-root firewall stop bitmask-root firewall isup
