NAME
       zipios - manual page for zipios 2.2.0

SYNOPSIS
       zipios [-opt] [file]

DESCRIPTION
   Where -opt is one or more of:
       --count
              count the number of files in a .zip archive

       --count-directories
              count the number of files in a .zip archive

       --count-files
              count the number of files in a .zip archive

       --help show this help screen

       --version
              print the library version and exit

       --version-tool
              print the tool version and exit
