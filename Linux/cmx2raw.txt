NAME
       cmx2raw - manual page for cmx2raw 0.1.7

SYNOPSIS
       cmx2raw [OPTION] FILE

DESCRIPTION
       `cmx2raw' is used to test import of Corel Binary Metafiles in libcdr.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information and exit

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
