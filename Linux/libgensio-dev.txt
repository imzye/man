NAME
       sergensio - How to use a serial gensio

DESCRIPTION
       NOTE: sergensio is deprecated, the functionality for it is not done with the acontrol() and control() functions in the main gensio type.

       A sergensio object is a type of gensio that provides serial port functions.  It cannot be directly created, instead you create a gensio that has serial port capabilities and cast it using the gen‐
       sio_to_sergensio() function.

       A sergensio object may be a client, meaning that you can set serial port parameters for that gensio.  This is what you would normally expect, and works for all serial port types.

       A  sergensio  object may also be a server, meaning that it can receive requests from the remote end to set serial port parameters.  Currently only the telnet gensio has server capability, so a re‐
       mote end may request that serial port parameters be set.
