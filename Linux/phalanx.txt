NAME
       xphalanx - wrapper for phalanx

SYNOPSIS
       xphalanx

DESCRIPTION
       xphalanx is a very simple shell script serving for launching phalanx with xboard from a window manager menu.

SEE ALSO
       phalanx(6) xboard(6)
