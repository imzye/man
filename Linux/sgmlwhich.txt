NAME
       sgmlwhich - prints location of main sgml configuration file

SYNOPSIS
       sgmlwhich

DESCRIPTION
       Simple script which prints location of main sgml configuration file.  It is provided for consistency with non-LSB platforms.

OPTIONS
       Script doesn't accept any options.

SEE ALSO
       sgml.conf(5)
