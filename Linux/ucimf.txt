NAME
       ucimf.conf - ucimf configuration files

DESCRIPTION
       ucimf.conf can be used either as /etc/ucimf.conf as a system wide configuration or $HOME/.ucimf.conf as a per-user configuration.

       ucimf uses UCIMF_FONTPATH to set unicode font to use. For example:
       UCIMF_FONTPATH = /usr/share/ucimf/unifont.pcf
       and font size can be set by FONT_WIDTH and FONT_HEIGHT:
       FONT_WIDTH = 24
       FONT_HEIGHT = 24

       Input method framework modules of UCIMF search path is set by IMF_MODULE_DIR:
       IMF_MODULE_DIR=/usr/lib/ucimf/
