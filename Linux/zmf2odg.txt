NAME
       zmf2odg - manual page for zmf2odg 0.9.6

SYNOPSIS
       zmf2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `zmf2odg' converts Zoner Draw documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
