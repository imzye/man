NAME
       fs_setcbaddr - Configure IP address used for AFS callbacks

SYNOPSIS
       fs setcbaddr [-addr <address>] [-help]

DESCRIPTION
       fs setcbaddr changes the IP address used by the client for callbacks.

OPTIONS
       -addr <address>]
           The new callback address.

       -help
           Prints the online help for this command. All other valid options are ignored.

OUTPUT
       This command produces no output other than error messages.

EXAMPLES
          % fs setcbaddr 10.2.4.23

PRIVILEGE REQUIRED
       The issuer must be the local superuser.

SEE ALSO
       fs(1)
