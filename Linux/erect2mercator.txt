NAME
       erect2mercator - Extract a mercator projection from an equirectangular image

Synopsis
         erect2mercator equirectangular.tif

DESCRIPTION
       Generates a suitably sized mercator .pto project from a 360 degree equirectangular image.  Input is in any image format understood by hugin, output is in TIFF format with an aspect ratio of 1.618:1 (a
       golden rectangle).

       The output filename is derived automatically from the input filename with '-mercator.pto' appended.  Existing .pto files will not be clobbered but will be updated when input images change size.
