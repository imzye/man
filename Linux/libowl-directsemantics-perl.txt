NAME
       OWL::DirectSemantics::Element::DataAllValuesFrom - represents an OWL DataAllValuesFrom

DESCRIPTION
       This class represents the DataAllValuesFrom element in OWL Direct Semantics.

       This class inherits from OWL::DirectSemantics::Element.

       It does the OWL::DirectSemantics::Writer::FunctionalSyntax and OWL::DirectSemantics::Writer::Dump roles.

   Attributes
       "annotations"
           A list of annotations associated with this object.  The value is a ArrayRef.

       "class"
           The class or datatype.  The value is a RDF::Trine::Node.  This is a required attribute.

       "property"
           The property.  This is a required attribute.
