NAME
       p0rn-grab - automatically download pictures and movies from an URL

SYNOPSIS
       p0rn-grab URL

OVERVIEW
       p0rn-grab downloads all pictures and movies from an URL.

       During download, a temporary directory is created in the current directory.  Downloaded files are then moved to a directory named ./out

SEE ALSO
       p0rn-proxy(1), p0rn-download(1), p0rn-dbdump(1), p0rn-dbrestore(1)

PROGRAMS NEEDED
       lynx: <http://lynx.browser.org/>
       wget: <http://wget.sunsite.dk/>
