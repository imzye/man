NAME
       podofomerge - merge several PDF files

SYNOPSIS
       podofomerge [inputfile1] [inputfile2] [outputfile]

DESCRIPTION
       podofomerge is one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. It can merge several PDF files.
