NAME
       tssrsaencrypt - Runs TPM2 rsaencrypt

DESCRIPTION
       rsaencrypt

       Runs TPM2_RSA_Encrypt

       -hk    key handle

       -id    decrypt file name

       [-oe   encrypt file name (default do not save)]
