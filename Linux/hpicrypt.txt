NAME
       hpigcrypt - An openhpi helper application that encrypts and decrypts a
       config file.

SYNOPSIS
        hpicrypt [-d ] -c config_file

DESCRIPTION
       hpicrypt is an optional helper application that encrypts or decrypts
       config_file.

       The encryption key is machine specific. File encrypted on one machine
       may not decrypt on some other machine.  Please have a backup copy of
       the file before encrypting.

OPTIONS
   Application Options:
       -d
         Decrypt the config_file and output the text to screen with helpful
         message

       -c Config file to encrypt or decrypt
