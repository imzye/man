NAME
       chkboot-desktopalert - notify changes detected in your boot files

SYNOPSIS
       chkboot-desktopalert

DESCRIPTION
       Display a notification with a detailed diff of added, changes and removed files, detected by chkboot.

       It will also display a notification when the MBR has been modified.

       You will need to add this program to your desktop environment's autostart.
