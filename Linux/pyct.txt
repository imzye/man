NAME
       pyct – commands related to versioning

SYNOPSIS
       pyct [-h] [--version] {report} ...

OPTIONS
   OPTIONAL ARGUMENTS
       -h, --help
              Show a help message and exit

       --version
              Show program’s version number and exit

AVAILABLE COMMANDS:
       report Import and print location and version information for specified Python packages

              See pyct-report(1).

SEE ALSO
       pyct-report(1)
