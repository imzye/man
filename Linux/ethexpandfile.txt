NAME
       ethexpandfile

       Expands  a  Intel(R) Ethernet Fabric Suite FastFabric hosts, chassis, or switches file. This tool expands and filter outs blank and commented lines. This can be useful when building other scripts that
       may use these files as input.

Syntax
       ethexpandfile  file

Options
       --help

                 Produces full help text.

       file

                 Specifies the FastFabric file to be processed.

Example
       ethexpandfile allhosts
