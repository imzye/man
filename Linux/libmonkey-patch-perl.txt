NAME
       Monkey::Patch::Handle::Object - Handle for monkey-patching objects

DESCRIPTION
       Monkey::Patch::Handle::Object provides the default Monkey::Patch::Handle for monkey-patching a particular object.

       You should never need to use this directly, so read L(Monkey::Patch) instead.
