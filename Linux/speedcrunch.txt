NAME
       SpeedCrunch - A speed calculator in QT.

SYNOPSIS
       speedcrunch

DESCRIPTION
       SpeedCrunch is a scientific calculator written in QT.

       SpeedCrunch is a high precision calculator, is optimized to be used with keyboard support variables, functions, history, dynamic result displaying and syntax highlighting.
