NAME
       s3dvt - 3d terminal emulator for s3d

SYNOPSIS

       s3dvt [-h|--help] [--s3d-url url]

DESCRIPTION
       s3dvt(1) is a terminal emulator.

OPTIONS
       This program follows the usual GNU command line syntax. A summary of the options supported by s3dvt is below.

       -h, --help
           Show summary of options and exit.

       --s3d-url url
           skip S3D environment and connect to this url
