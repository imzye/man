NAME
       ykval-queue - Sync servers in a yubikey-val sync pool.

SYNOPSIS
       ykval-queue

DESCRIPTION
       Periodically attempts to synchronize the Yubikey counter data with other yubikey-val servers in the sync pool. Reads configuration from /etc/yubico/val/ykval-config.php

       This process is meant to be run as a daemon.
