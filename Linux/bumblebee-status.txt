NAME
       bumblebee-ctl - Send commands to bumblebee-status

SYNOPSIS
       bumblebee-ctl [-h] [-b button] [-i ID] -m module

DESCRIPTION
       bumblebee-ctl can be used to send commands to bumblebee-status.

OPTIONS
       -h, --help
              show this help message and exit

       -b, --button <button>
              Button to emulate. Buttons can be any of left-mouse, right-mouse, middle-mouse, wheel-up, wheel-down, or update.

       -i, --id <ID>
              ID of widget to trigger

       -m, --module <module>
              name of module to trigger
