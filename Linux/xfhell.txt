NAME
       xfhell - a GTK+ Hellshcriber communication application

SYNOPSIS
              xfhell [-hv]

DESCRIPTION
              [-h: print this usage information and exit]
              [-v: print xfhell version number and exit]
SEE ALSO
       The full documentation for xfhell is maintained as an html document.  If the xfhell program is properly installed at your site, the command

              firefox /usr/share/doc/xfhell/xfhell.html

       should give you access to the complete manual.
