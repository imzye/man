NAME
       botch-check-ma-same-versions - check if Multi-Arch:same versions are in sync

SYNOPSIS
       botch-check-ma-same-versions [options] packages1 packages2

DESCRIPTION
       checks if Multi-Arch: Same packages in two package files agree in their versions and also outputs, should there be packages in one but not the other

OPTIONS
       -h, --help
           Show help message

       -v, --verbose
           be verbose

EXIT STATUS
       If the two packages file have the same binary packages and agree on their version, return a zero exit status. Otherwise return a non-zero exit status.
