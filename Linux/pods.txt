NAME
       pods - print out all pod paths

DESCRIPTION
       This program is a front end to print out the paths of all the standard
       podpages and the modules.

SEE ALSO
       faqpods(1), modpods(1), sitepods(1), podpath(1), pminst(1), and
       stdpod(1).
