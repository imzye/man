NAME
       Lingua::Stem::Gl - Stemming algorithm for Galacian

SYNOPSIS
           use Lingua::Stem::Gl;
           my $stems   = Lingua::Stem::Gl::stem({ -words => $word_list_reference,
                                                 -locale => 'gl',
                                             -exceptions => $exceptions_hash,
                                            });

DESCRIPTION
       This routine applies a stemming algorithm to a passed anon array of Galician words, returning the stemmed words as an anon array.

       It is a 'convienence' wrapper for 'Lingua::Stemmer::GL' that provides a standardized interface and caching.

METHODS
       stem({ -words => \@words, -locale => 'gl', -exceptions => \%exceptions });
           Stems a list of passed words using the rules of Galican. Returns an anonymous list reference to the stemmed words.

           Example:

             my $stemmed_words = Lingua::Stem::Gl::stem({ -words => \@words,
                                                         -locale => 'gl',
                                                     -exceptions => \%exceptions,
                                     });

       stem_caching({ -level => 0|1|2 });
           Sets the level of stem caching.

           '0' means 'no caching'. This is the default level.

           '1' means 'cache per run'. This caches stemming results during a single
               call to 'stem'.

           '2' means 'cache indefinitely'. This caches stemming results until
               either the process exits or the 'clear_stem_cache' method is called.

       clear_stem_cache;
           Clears the cache of stemmed words
