NAME
       mm-replayserver - helper program used by mm-webreplay(1).

DESCRIPTION
       mm-replayserver is a program that replies to previously-saved HTTP requests with the corresponding HTTP response. See mm-webreplay(1).

SEE ALSO
       mahimahi(1)

       Project home page: http://mahimahi.mit.edu
