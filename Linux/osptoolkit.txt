       ospenroll â€”  OSP enroll script

SYNOPSIS
       ospenroll ospserver...

DESCRIPTION
       This manual page documents briefly the ospenroll command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       ospenroll is a program that enables the OSP client device to generate its own public-private key pair, get the public key from an OSP peering server, send a certificate request to a peering server
       and receive the resulting signed certificate from the peering server.

OPTIONS
       ospserver OSP server IP or domain name.
