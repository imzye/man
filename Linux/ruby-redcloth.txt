NAME
       redcloth - Ruby Textile formatter

SYNOPSIS
       redcloth [options] [redcloth_formatted.txt]

DESCRIPTION
       Format a Textile document into HTML or other type of output supported by RedCloth. If no file specified, STDIN will be used. If you are typing input, you can send an EOF by pressing ^D.

OPTIONS
       -o, --output STYLE
              Output format (defaults to html).
