NAME
       kmk - framework for writing simple makefiles for complex tasks

SYNOPSIS
       kmk [options]

DESCRIPTION
       kmk and its helper tools are an extension to GNU make to ease writing
       portable Makefile. There is not a lot of documentation though. A
       starting point is <http://svn.netlabs.org/kbuild>.

       On Debian systems the kBuild binaries can be found in /usr/bin, its
       data files in /usr/share/kBuild and its documentation in
       /usr/share/doc/kbuild.

SEE ALSO
       make(1)
