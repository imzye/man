NAME
       pvfs2-set-perf-interval – set performance monitoring parameters

SYNOPSIS
       pvfs2-set-perf-interval

DESCRIPTION
       The pvfs2-set-perf-interval utility is part of the performance monitor‐
       ing system.  It only be used by experienced system administrators.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set the OrangeFS debug mask.  Possible masks are  documented  in
              pvfs2-set-debugmask(1).

BUGS
