NAME
       perditiondb_postgresql_makedb - create perdition database in a PostgreSQL RDMS

SYNOPSIS
       perditiondb_postgresql_makedb

DESCRIPTION
       perditiondb_postgresql_makedb is an interactive programme to help seed a database in a PostgreSQL RDMS to store associations between users and servers for perdition.

OPTIONS
       None - All input is promted for by the programme.

FILES
       None

SEE ALSO
       perditiondb(5), perdition(8)
