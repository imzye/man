NAME
       wps2raw - manual page for wps2raw 0.4.12

SYNOPSIS
       wps2raw [OPTION] FILE

DESCRIPTION
       `wps2raw' is used to test text document import in libwps.  Supported formats are Microsoft Works, Write and Word for DOS.

OPTIONS
       -h, --help
              show this help message

       -v, --version
              show version information

       --callgraph
              display the call graph nesting level

       --password PASSWORD
              set password to open the file
