NAME
       toml-adapt - adapt TOML files

SYNOPSIS
       toml-adapt [-h] [-path PATH] [-a {change,add,remove}] [-dep DEP] [-ver VER]

DESCRIPTION
       This is a very simple package for manipulating toml format files.  This is still a work in progress.  Recent version of this package supports the manipulation of dependencies that are included in toml
       files.

OPTIONS
       -h, --help
              Show a help message and exit

       -path PATH
              Path to the .toml file

       -a {change,add,remove}
              Select action

       -dep DEP
              Dependency name

       -ver VER
              Version of dependency

EXAMPLES
       Change dependency

              toml-adapt -path pyproject.toml -a change -dep niaclass -ver 0.1.0

       Add dependency

              toml-adapt -path pyproject.toml -a add -dep niaclass -ver 0.1.0

       Remove dependency

              toml-adapt -path pyproject.toml -a remove -dep niaclass -ver 0.1.0

   OTHER EXAMPLES
       Change all existing dependencies in toml file

              toml-adapt -path pyproject.toml -a change -dep ALL -ver X
              X represents a *

   HOW TO USE IT IN SPEC FILES?
              %prep
              ...
              # Make dependencies consistent with Fedora dependencies
              toml-adapt -path pyproject.toml -a change -dep ALL -ver X
