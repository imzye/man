affstats(1)  print specific statistics about one or more AFF files affstats(1)

NAME
       affstats - print specific statistics about one or more AFF files

SYNOPSIS
       affstats [options] file
       affstats [options] file1 file2 file3

DESCRIPTION
       Print  specific  statistics  about  one or more AFF files. Ideally, the
       stats can be gotten from the metadata, but this program will  calculate
       it if necessary.

       This file is part of the AFFLIBv3, that provides AFF.

       AFF  is an open and extensible file format to store disk images and as‐
       sociated metadata. It is  useful  in  several  scenaries,  as  computer
       forensics investigations.

OPTIONS
       -m     Print all output in megabytes.

       -V     Just print the version number and exit.
