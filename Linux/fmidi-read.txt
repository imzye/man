SYNOPSIS
       fmidi-read INPUT [INPUT ...]

       Reads each INPUT file and prints a textual s-expression representation.

EXIT STATUS
       0      if all INPUT files were successfully read

       1      if any INPUT file could not be read; the file in question may be
