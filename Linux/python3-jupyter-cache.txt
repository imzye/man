NAME
       jcache - jcache

SYNOPSIS
       jcache [OPTIONS] COMMAND [ARGS]...

DESCRIPTION
              The command line interface of jupyter-cache.

OPTIONS
       -v, --version
              Show the version and exit.

       -p, --print-path
              Print the current cache path and exit.

       -a, --autocomplete
              Print the autocompletion command and exit.

       -h, --help
              Show this message and exit.

   Commands:
       cache  Work with cached execution(s) in a project.

       notebook
              Work with notebook(s) in a project.

       project
              Work with a project.
