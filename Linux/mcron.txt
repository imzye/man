NAME
       mcron - a program to run tasks at regular (or not) intervals

SYNOPSIS
       mcron [OPTIONS] [FILES]

DESCRIPTION
       Run  an  mcron  process  according to the specifications in the FILES (`-' for standard input), or use all the files in ~/.config/cron (or the deprecated ~/.cron) with .guile or .vixie exten‐
       sions.

       -v, --version
              Display version

       -h, --help
              Display this help message

       -sN, --schedule[=]N
              Display the next N jobs that will be run by mcron

       -d, --daemon
              Immediately detach the program from the terminal and run as a daemon process

       -i, --stdin=(guile|vixie) Format of data passed as standard input or
              file arguments (default guile)
