NAME
       kylin-nm - The tool for the UKUI Desktop Environment

SYNOPSIS
       kylin-nm

DESCRIPTION
       The kylin-nm program is a part of the ukui-indicators, It provides network manager for the the UKUI Desktop Environment. Detect the system disk automatically, Can pop up and open the device.

       This manual page documents the kylin-nm command.
