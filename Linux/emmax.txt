NAME
       emmax-kin: - emmax-kin

SYNOPSIS
       emmax-kin [tpedf]

DESCRIPTION
       Required parameters

       [tpedf]
              : tped file

       [outf] : output file name

       Optional parameters

       -d [# digits]
              : precision of the kinship values (default : 10)

       -v : turn on verbose mode
