NAME
       candle - GRBL controller application with G-Code visualizer

SYNOPSIS
       candle [gcode-filename]

DESCRIPTION
       candle is a Qt-based application for controlling GRBL devices such as laser cutters, routers, 3d printers.

COMMAND LINE OPTIONS
       If a filename is passed on the command line, candle will load it at startup.
