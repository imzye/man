NAME
       vvmd - Receive Visual Voicemail

SYNOPSIS
       vvmd [ -d ] [ -v ]

DESCRIPTION
       vvmd This is a daemon that receives Visual Voicemail. It works with modem manager.

OPTIONS
       d      Enable debug mode

       v      Show version information and exit
