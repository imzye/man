NAME
       shelxle - graphical user interface for SHELXL

SYNOPSIS
       shelxle

DESCRIPTION
       ShelXle  combines  an  editor with syntax highlighting for the SHELXL-associated .ins (input) and .res (output) files with an interactive graphical display for visualization of a three-dimensional
       structure including the electron density (Fo) and difference density (Fo-Fc) maps.

OPTIONS
       There are currently no command-line options supported.  to debugging.
