NAME
       botch-graphml2dot - converts graphml to dot

SYNOPSIS
       botch-graphml2dot [options] [g.xml] [h.dot]

DESCRIPTION
       Convert a graph in GraphML format to dot. If either (or both) of the positional arguments is/are omitted, then read graphml from standard input and/or write graphml to standard output, respectively.
       Passing a '-' as either argument, explicitly specifies to read the input from standard input and write the output to standard output, respectively.

OPTIONS
       -h, --help
           Show help.

       -v, --verbose
           Be verbose.

EXAMPLE
       Do some graph operations:

        botch-graph-neighborhood --depth=1 srcgraph_acyclic.xml icewease.xml src:iceweasel
        botch-graphml2dot iceweasel.xml iceweasel.dot
        tred < iceweasel.dot > iceweasel_tred.dot
