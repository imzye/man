NAME
       fvwm-crystal.infoline - gather various information about system

SYNOPSIS

       fvwm-crystal.infoline [directory] [directory] [...]

DESCRIPTION
       fvwm-crystal.infoline is a program for gathering various information about system. random wallpapers. It is shipped with the fvwm-crystal desktop environment and should be used only with fvwm. It
       is just an helper that sets FVWM_INFOLINE environment variable to a sensible value.

       This manual page was written for the Debian distribution because the original program does not have a manual page.
