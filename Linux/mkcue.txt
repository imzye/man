NAME
       mkcue — Generates a CUE sheet from a CD

SYNOPSIS
       mkcue [-t track count]  [device]

DESCRIPTION
       This manual page documents briefly the mkcue command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       mkcue Generates a CUE sheet from a CD.

OPTIONS
       -t track count
                 Exclude tracks above the given number. Useful for CDs with data tracks.

        device   Device path to scan the CDROM drive. mkcue defaults to /dev/cdrom.
