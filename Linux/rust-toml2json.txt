NAME
       toml2json - convert TOML to JSON

SYNOPSIS
       toml2json [OPTIONS] [input]

DESCRIPTION
       Convert TOML to JSON

   Arguments:
       [input]
              the TOML to convert [default: -]

OPTIONS
       -p, --pretty
              pretty print the JSON

       -h, --help
              Print help

       -V, --version
              Print version
