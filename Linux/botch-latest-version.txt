NAME
       botch-latest-version - only keep the latest version

SYNOPSIS
       botch-latest-version [options] inPackages outPackages

DESCRIPTION
       Filters a Packages file so that only the newest version of each package remains.

OPTIONS
       -h, --help
           Show help.

       --verbose
           Be verbose.
