NAME
       afdko - wrapper for afdko utilities

SYNOPSIS
       afdko [utility] [params]

DESCRIPTION
       afdko calls the utilities from AFDKO (Adobe Font Development Kit for OpenType).

       To see available utilities in afdko, run afdko without command.

AUTHOR
       This afdko wrapper is created by Yao Wei <mwei@debian.org>.
       The binaries this wrapper calls are created by Adobe System Incorporated.
