NAME
       versioningit – show the version of a versioningit-enabled project

SYNOPSIS
       versioningit [-h] [--traceback] [-v] [-w] [-V] [project_dir]

OPTIONS
   POSITIONAL ARGUMENTS
       project_dir

   OPTIONAL ARGUMENTS
       -h, --help
              Show a help message and exit

       --traceback
              Show full traceback on library error

       -v, --verbose
              Show more log messages

       -w, --write
              Write version to configured file

       -V, --version
              Show program’s version number and exit
