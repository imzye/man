NAME
       ukui-notebook - A NOTEBOOK PLUGIN IN UKUI

SYNOPSIS
       ukui-notebook [OPTIONS]

DESCRIPTION
       A notebook plugin for UKUI.

OPTIONS
       -v, --version
              Output version information and exit.

BUGS
