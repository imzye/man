NAME
       numbers2ods - manual page for numbers2ods 0.9.6

SYNOPSIS
       numbers2ods [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `numbers2ods' converts Apple Numbers documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
