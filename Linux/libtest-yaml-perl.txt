NAME
       Test::YAML - Testing Module for YAML Implementations

VERSION
       This document describes Test::YAML version 1.07.

SYNOPSIS
           use Test::YAML tests => 1;

           pass;

DESCRIPTION
       Test::YAML is a subclass of Test::Base with YAML specific support.
