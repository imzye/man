NAME
       bibledit-cloud - Bibledit Cloud server

DESCRIPTION
       Bibledit-Cloud  is  an  online Bible editor.  Type "bibledit-cloud" in a terminal to start it.  Point a browser at http://localhost:8080 to open the graphical user interface.  For more information
       see the graphical user interface.
