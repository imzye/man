NAME
       Module::Install::AutoManifest - generate MANIFEST automatically

VERSION
       Version 0.003

SYNOPSIS
       In Makefile.PL:

         auto_manifest;

       In MANIFEST.SKIP:

         # put your list of patterns here
         ^\.git/
         ^your-crazy-files-whatever$

       In MANIFEST:

         Nothing, remove it

METHODS
   auto_manifest
       This extension to Module::Install adds behavior for automatically generating MANIFEST.

       Before "make distdir", "make manifest" will be run for you automatically.  Likewise, "make distclean" will remove your automatically-generated MANIFEST.
