NAME
       catior - Utility for viewing components of a stringified IOR

SYNOPSIS
       catior [-x] <stringified IOR>

DESCRIPTION
       catior displays the components of the stringified object reference supplied to it.

OPTIONS
       -x     Display the object key in hexadecimal

SEE ALSO
       genior(1)
