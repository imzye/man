NAME
       ykval-import - Imports Yubikey Info data into the yubikey-val server.

SYNOPSIS
       ykval-import

DESCRIPTION
       Reads yubikey-val Yubikey Info data from stdin and imports it into the yubikey-val servers database. The data should previously have been exported using ykval-export.

       Database configuration is read from /etc/yubico/val/ykval-config.php

BUGS
       Report ykval-import bugs in the issue tracker ⟨https://github.com/Yubico/yubikey-val/issues⟩

SEE ALSO
       ykval-export(1)

       The YubiKey Info data format ⟨https://developers.yubico.com/yubikey-val/doc/YubiKeyInfoFormat.html⟩

       YubiKeys can be obtained from Yubico ⟨https://www.yubico.com/⟩.
