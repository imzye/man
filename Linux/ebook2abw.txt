NAME
       ebook2abw - manual page for ebook2abw 0.9.6

SYNOPSIS
       ebook2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `ebook2abw' converts e-books to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
