NAME
       HTML::WikiConverter::MoinMoin - Convert HTML to MoinMoin markup

SYNOPSIS
         use HTML::WikiConverter;
         my $wc = new HTML::WikiConverter( dialect => 'MoinMoin' );
         print $wc->html2wiki( $html );

DESCRIPTION
       This module contains rules for converting HTML into MoinMoin markup. See HTML::WikiConverter for additional usage details.

ATTRIBUTES
       In addition to the regular set of attributes recognized by the HTML::WikiConverter constructor, this dialect also accepts the following attributes that can be passed into the "new()" constructor.
       See "ATTRIBUTES" in HTML::WikiConverter for usage details.

   enable_anchor_macro
       Possible values: 0, 1. Enables "[[Anchor(s)]]" formatting. See <http://moinmoin.wikiwikiweb.de/HelpOnMacros> for details.
