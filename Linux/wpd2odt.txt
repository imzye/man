NAME
       wpd2odt - manual page for wpd2odt 0.9.6

SYNOPSIS
       wpd2odt [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wpd2odt' converts WordPerfect documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
