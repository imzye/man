NAME
       fake_router6

SYNOPSIS
       fake_router6 [-HFD] interface network-address/prefix-length [dns-server [router-ip-link-local [mtu [mac-address]]]]

       Announce yourself as a router and try to become the default router.  If a non-existing link-local or mac address is supplied, this results in a DOS.  Option -H adds hop-by-hop, -F fragmentation header
       and -D dst header.
