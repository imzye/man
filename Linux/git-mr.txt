NAME
       git-mr - Checks out a merge request locally

SYNOPSIS
       git-mr <number> [<remote>]
       git-mr <url>
       git-mr clean

DESCRIPTION
       Fetches  merge  request head by its number or URl and check it out in a
       branch named with merge request number.

OPTIONS
       <remote>

       The name of the remote to fetch from. Defaults to origin.

       <url>

       GitLab merge request URL in the format https://gitlab.tld/owner/reposi‐
       tory/merge_requests/453.

EXAMPLES
       This checks out merge request !51 from remote origin to branch mr/51.

           $ git mr 51
           From gitlab.com:owner/repository
            * [new ref]         refs/merge-requests/51/head -> mr/51
           Switched to branch ´mr/51´
