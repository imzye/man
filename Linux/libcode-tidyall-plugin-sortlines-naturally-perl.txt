NAME
       Code::TidyAll::Plugin::SortLines::Naturally - Sort lines of a file using Unicode::Collate::Locale

VERSION
       version 0.000003

SYNOPSIS
          # In configuration:

          [SortLines::Naturally]
          select = .ispell* **/.gitignore

DESCRIPTION
       Sorts the lines of a file; whitespace lines are discarded. Useful for files containing one entry per line, such as ".svnignore", ".gitignore", and ".ispell*".

CONFIGURATION
       locale
           The locale to use for collation. Defaults to "en_US".
