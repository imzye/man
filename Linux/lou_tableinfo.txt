NAME
       lou_tableinfo - A tool to query meta data from a liblouis Braille translation table

SYNOPSIS
       lou_tableinfo [KEY] TABLE

DESCRIPTION
       Print all table metadata defined in TABLE, or a specific metadata field if KEY is specified. Return 0 if the requested metadata could be found, 1 otherwise.

       -h, --help
              display this help and exit

       -v, --version
              display version information and exi
