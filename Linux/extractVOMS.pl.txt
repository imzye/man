NAME
         extractVOMS.pl

SYNOPSIS
         extractVOMS.pl [ -cert /path/to/cert.pem ] [ -out /path/to/save/AC.pem ]

DESCRIPTION
       Extracts the VOMS AC from a GSI Proxy or X509 certificate and saves it in /tmp/vomsAC_u<UID>.  Use -cert to specify the proxy certificate or x509 certificate from which to extract the AC.  Use -out to
       specift the filename to save the PEM encoded Attribute Certificate.
