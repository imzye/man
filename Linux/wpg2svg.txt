NAME
       wpg2svg - manual page for wpg2svg 0.3.3

SYNOPSIS
       wpg2svg [OPTION] FILE

DESCRIPTION
       `wpg2svg' converts WordPerfect Graphics images to SVG.

OPTIONS
       --help show this help message

       --version
              show version information and exit
