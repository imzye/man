NAME
       cstrans-df-run - transform RUN line in a Dockerfile

SYNOPSIS
       cstrans-df-run  [--in-place  Dockerfile]  [--verbose]  cmd  [arg1 [arg2
       [...]]]:

DESCRIPTION
       -i [ --in-place ] arg modify the specified file in-place

       --verbose
              print transformations to standard error output

       --help produce help message

       --version
              print version
