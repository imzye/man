NAME
       kf-slam - SLAM method using Kalman filtering

SYNOPSIS
       kf-slam config_file.ini

DESCRIPTION
       kf-slam is a command-line application which processes a rawlog and create a map of landmarks using a 6D SLAM algorithm.
