NAME
       update-debian-edu-artwork-emerald - Enable/disable Debian Edu Emerald theme system-wide

SYNOPSIS
       update-debian-edu-artwork-emerald { configure | remove }

DESCRIPTION
       update-debian-edu-artwork-emerald  configure  enables  the Debian Edu Emerald theming (as shipped with Debian Edu Bookworm) system-wide (several login managers, KDE theme, background for all desk‚Äê
       tops, GRUB boot loader, etc. pp.).

       update-debian-edu-artwork-emerald remove disables the Debian Edu themes and switches back to Debian's default themes (from package desktop-base).
