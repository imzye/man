NAME
       pdtcompose - manual page for pdtcompose 1.6.9

DESCRIPTION
       Compose a PDT and an FST.

              Usage: pdtcompose in.pdt in.fst [out.pdt]

              in.fst in.pdt [out.pdt]

       PROGRAM FLAGS:

       --compose_filter: type = string, default = "paren"

              Composition  filter,  one  of:  "expand",  "expand_paren",  "paren"  --connect:  type = bool, default = true Trim output?  --left_pdt: type = bool, default = true Is the first argument the PDT?
              --pdt_parentheses: type = string, default = "" PDT parenthesis label pairs

       LIBRARY FLAGS:

              bosity level
