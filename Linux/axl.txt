NAME
       axl - afnix cross librarian

SYNOPSIS
       axl [options] file...

OPTIONS
       [h]
       prints the help message

       [v]
       prints the program version

       [c]
       create a new librarian

       [x]
       extract from a librarian

       [t]
       write a detailed list of the files in the librarian

       [s]
       write a short list of the files in the librarian

       [f] file
       set the librarian file name

       [m] name
       set the the start module

DESCRIPTION
       axlinvokes the cross librarian utility. A librarian is a collection of files that are read by the interpreter. A librarian is similar to an archive. The idea is to store a complete project into a sin‚Äê
       gle file and let the interpreter to find them. A librarian is created with the coption. Extracting a file or all files is performed with the xoption. Both options cand  xare  mutually  exclusive.  The
       start module can be set with the m.
