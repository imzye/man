NAME
       piespy - bot to visualize IRC social networks

SYNOPSIS

       piespy [configuration file]

DESCRIPTION
       This manual page documents briefly the piespy and bar commands.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       piespy is a program that monitors an IRC channel, and graphs the social relationships in that channel.

OPTIONS
       configuration file
           The configuration file to use. If none is specified, config.ini in the current directory will be used.
