NAME
       guix - manual page for guix graph (GNU Guix) 1.3.0.27848-a3264

SYNOPSIS
       guix graph PACKAGE...

DESCRIPTION
       Emit a representation of the dependency graph of PACKAGE...

       -b, --backend=TYPE
              produce a graph with the given backend TYPE

       --list-backends
              list the available graph backends

       -t, --type=TYPE
              represent nodes of the given TYPE

       --list-types
              list the available graph types

       -M, --max-depth=DEPTH
              limit to nodes within distance DEPTH

       --path display the shortest path between the given nodes

       -e, --expression=EXPR
              consider the package EXPR evaluates to

       -L, --load-path=DIR
              prepend DIR to the package module search path

       --help-transform
              list package transformation options not shown here

       -h, --help
              display this help and exit

       -V, --version
              display version information and exit

       --list-systems
              list available systems

       -s, --system=SYSTEM
              attempt to build for SYSTEM--e.g., "i686-linux"
