NAME
       Test::ConsistentVersion - Ensures a CPAN distribution has consistent versioning.

VERSION
       This document describes Test::ConsistentVersion version 0.3.1

SYNOPSIS
       [In a test file]

           use Test::More;

           if ( not $ENV{TEST_AUTHOR} ) {
               my $msg = 'Author test. Set $ENV{TEST_AUTHOR} to a true value to run.';
               plan( skip_all => $msg );
           }

           eval "use Test::ConsistentVersion";
           plan skip_all => "Test::ConsistentVersion required for checking versions" if $@;
           Test::ConsistentVersion::check_consistent_versions();

DESCRIPTION
       The purpose of this module is to make it easy for other distribution authors to have consistent version numbers within the modules (as well as readme file and changelog) of the distribution.

INTERFACE
       check_consistent_versions
               check_consistent_versions()

           Checks the various versions throughout the distribution to ensure they are all consistent.

DIAGNOSTICS
       Nothing so far.

CONFIGURATION AND ENVIRONMENT
       Test::ConsistentVersion requires no configuration files or environment variables.

DEPENDENCIES
       perl 5.6 or greater
       Test::Builder
       autodie

       Optional

       Test::Pod::Content
           For ensuring the module version matches that referenced in the POD.

INCOMPATIBILITIES
       Perl 5.6 or greater is required.
