NAME
       weightwatcher - combine weight-maps, flag-maps and polygon data

SYNOPSIS
       weightwatcher [-c configuration-file] [-parameter1 value1 -parameter2 value2 ...]

       weightwatcher -d

DESCRIPTION
       WeightWatcher is a program that combines weight-maps, flag-maps and polygon data in order to produce control maps which can directly be used in astronomical image-processing packages like Swarp or
       SExtractor.
       See http://astromatic.net/software/weightwatcher for more details.

   Operation modes:
       -h, --help
              print this help, then exit

       -V, --version
              print version number, then exit

       -d, --dump
              dump a default configuration file

MANUAL
       The full documentation for WeightWatcher is maintained as a PDF manual available at http://astromatic.net/software/weightwatcher
