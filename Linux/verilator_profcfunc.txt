NAME
       verilator_profcfunc - Read gprof report created with --prof-cfuncs

SYNOPSIS
         verilator --prof-cfuncs ....
         gcc --ggdb -pg ....
         {run executable}
         gprof
         verilator_profcfuncs gprof.out

DESCRIPTION
       Verilator_profcfunc reads a profile report created by gprof.  The names of the functions are then transformed, assuming the user used Verilator's --prof-cfuncs, and a report printed showing the
       percentage of time, etc, in each Verilog block.

ARGUMENTS
       --help
           Displays this message and program version and exits.

DISTRIBUTION
