NAME
       ykval-checksum-clients - Calculate a checksum of all Client Info data.

SYNOPSIS
       ykval-checksum-clients

DESCRIPTION
       Calculates  a  checksum  using the id, active, and secret fields of all Client Info data in the yubikey-val server database. This checksum can be used to easily compare the state of clients in two yu‐
       bikey-val servers in the same sync pool.

       Database configuration is read from /etc/yubico/val/ykval-config.php
