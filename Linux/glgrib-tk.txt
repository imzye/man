NAME
       glgrib-tk - Perl/Tk interface to glGrib

SYNOPSIS
       Start the interface :

         $ glgrib-tk

DESCRIPTION
       glgrib-tk provides a full Perl/Tk interface to the GLFW based glgrib library.

SEE ALSO
       glgrib
