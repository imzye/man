NAME
       PDF::FDF::Simple::Builder - Module::Build extensions for PDF::FDF::Simple

SYNOPSIS
        perl Build.PL
        ./Build grammar
        ./Build
        ./Build test
        ./Build install

DESCRIPTION
       Provides Module::Build extensions, mainly for precompiling the grammar file.

FUNCTIONS
   ACTION_grammar
       This defines an Build action "grammar" which precompiles the grammar using Parse::RecDescent and moves the file to lib/PDF/FDF/Simple/Grammar.pm. It is usually done by the maintainer, before he
       builds a dist file for CPAN.
