NAME
       deepin-screenshot - Advanced screen shoting tool

SYNOPSIS
       deepin-screenshot

DESCRIPTION
       deepin-screenshot is an andvanced tool to make screenshots on Linux desktop environment. It has a user-friendly UI which eases cropping and saving of screen‐
       shot.

OPTIONS
       This program does not accept any command line options.
