NAME
       sphinx2pxml - manual page for sphinx2pxml 0.6

DESCRIPTION
       usage: sphinx2pxml [-h] [-W] rst_file

       Convert Sphinx ReST file to Pseudo XML

   positional arguments:
       rst_file
              ReST file to convert

   options:
       -h, --help
              show this help message and exit

       -W, --warn-is-error
              turn warnings into errors
