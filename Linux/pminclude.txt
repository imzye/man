NAME
       pminclude - show the include path for modules (@INC)

DESCRIPTION
       This is mostly here for people too lazy to type:

           $ perl -V | tail

EXAMPLES
           $ pminclude
           /etc/perl
           /usr/local/lib/perl/5.14.2
           /usr/local/share/perl/5.14.2
           /usr/lib/perl5
           /usr/share/perl5
           /usr/lib/perl/5.14
           /usr/share/perl/5.14
           /usr/local/lib/site_perl

SEE ALSO
       pmls(1), pmpath(1)
