NAME
       dnsrevenum6

SYNOPSIS
       dnsrevenum6 [-t] dns-server ipv6address

       Performs a fast reverse DNS enumeration and is able to cope with slow servers.  Option -t enables TCP instead of UDP (use this if you get many timeouts)

Examples
         dnsrevenum6 dns.test.com 2001:db8:42a8::/48
         dnsrevenum6 -t dns.test.com 8.a.2.4.8.b.d.0.1.0.0.2.ip6.arpa
