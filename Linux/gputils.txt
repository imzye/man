NOM
       gpvc - Visualiseur GNU de fichiers COD pour PIC

SYNOPSIS
       gpvc [options] fichier

AVERTISSEMENT
       Les  informations  de cette page de manuel sont extraites de la documentation complète de gputils et sont limitées à la signification des options. Pour une documentation complète, reportez-vous au
       fichier « gputils.ps » (en anglais) situé dans le répertoire doc de gputils.

DESCRIPTION
       gpvc est un visualiseur de fichiers COD pour les microcontrôleurs PIC (TM) de Microchip (TM). Ces fichiers contiennent des informations utilisées pour la simulation. gpvc est une  partie  de  gpu‐
       tils. Consultez la page manuel gputils(1) pour avoir des détails sur les autres utilitaires GNU PIC.

OPTIONS
       -a, --all
              Afficher toutes les informations présentes dans le fichier COD.

       -d, --directory
              Afficher le répertoire des en-têtes.

       -l, --listing
              Afficher le listage des sources.

       -m, --message
              Afficher la zone de messages de débogage.

       -h, --help
              Afficher un message d'utilisation et quitter.

       -r, --rom
              Afficher la ROM.

       -s, --symbols
              Afficher les symboles.

       -v, --version
              Afficher les informations de version et quitter.
