NAME
       mfiler3 a console based file manager

SYNOPSIS
       [-c command] [--version] [ filer initial directory or script file ]

DESCRIPTION
       a two pane console based file manager.

       The following options are available -c " string"
       Commands described by string are executed and exit immediately.  -s

USAGE
       run mfiler3 and hit ? key
