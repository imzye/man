WHICH(1)                                                               General Commands Manual                                                              WHICH(1)

名前
       which - コマンドの場所

書式
       which [-a] filename ...

説明
       which  returns  the  pathnames  of  the  files  (or  links) which would be executed in the current environment, had its arguments been given as commands in a
       strictly POSIX-conformant shell.  It does this by searching the PATH for executable files matching the names of the arguments.  It does not canonicalize path
       names.

オプション
       -a     各引数に一致したすべてのパス名を表示します。

終了ステータス
       0      指定したコマンドがすべて見つかり、実行可能な場合

       1      いくつかのコマンドがないか、実行可能でない場合

       2      無効なオプションを指定した場合
