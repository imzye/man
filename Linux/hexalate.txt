NAME
       hexalate - color matching game

SYNOPSIS
       hexalate [options]

DESCRIPTION
       Hexalate  is  a color matching game. The goal of the game is to rotate and position the circles so that each touching line matches in color. You rotate circles by right clicking, and you move cir‐
       cles by dragging them.  The game stores the positions and rotations of the circles across runs.

OPTIONS
       -h, --help
              Displays help on commandline options.

       --help-all
              Displays help including Qt specific options.

       -v, --version
              Displays version information.
