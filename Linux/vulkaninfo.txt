NAME
       vulkaninfo - show Vulkan information

SYNOPSIS
       vulkaninfo

DESCRIPTION
       The vulkaninfo program shows information about the supported Vulkan capabilities. It lists not only device properties and features, but also supported Vulkan layers and extensions.
