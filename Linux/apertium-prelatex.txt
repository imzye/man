NAME
     apertium-prelatex — LaTeX preprocessor for Apertium

SYNOPSIS
     apertium-prelatex [input_file [output_file]]

DESCRIPTION
     apertium-prelatex This filter preprocess LaTeX as input transforming it into a deformatted “XMLish” LaTeX custom format.  The output suitable for preprocess with apertium-deslatex(1) deformatter.

OPTIONS
     -h, --help
             Display this help.

SEE ALSO
     apertium(1), apertium-deslatex(1), apertium-postlatex(1), apertium-postlatex-raw(1), apertium-relatex(1), lt-proc(1)
