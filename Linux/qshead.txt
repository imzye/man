NAME
       qshead - an utility reading from stdin and printing all lines to stdout
       until reaching the defined pattern.

SYNOPSIS
       qshead -p <pattern>

DESCRIPTION
       qshead reads lines from stdin and prints them to stdout  unitl  a  line
       contains the specified pattern (literal string).

OPTIONS
       -p <pattern>
              Search pattern (literal string).
