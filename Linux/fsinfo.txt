NAME
       fsinfo - coordinate site-wide filesystem information

SYNOPSIS
       fsinfo [ -v ] [ -a autodir ] [ -b bootparams ] [ -d dumpsets ] [ -e ex‚Äê
       ports ] [ -f fstabs ] [ -h hostname ] [ -m automounts ] [ -I dir ] [ -D
       string[=string] ] [ -U string[=string] ] config ...

DESCRIPTION
       The fsinfo utility takes a set of system configuration information, and
       generates a coordinated set of amd ,  mount  and  mountd  configuration
       files.

       The  fsinfo command is fully described in the document Amd - The 4.4BSD
       Automounter

SEE ALSO
       amd(8), mount(8), mountd(8).

       ``am-utils'' info(1) entry.

       Linux  NFS  and  Automounter  Administration  by   Erez   Zadok,   ISBN
       0-7821-2739-8, (Sybex, 2001).

       http://www.am-utils.org

       Amd - The 4.4 BSD Automounter

HISTORY
       The fsinfo command first appeared in 4.4BSD.
