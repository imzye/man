NAME
       Syntax::Highlight::Engine::Kate::VHDL - a Plugin for VHDL syntax highlighting

SYNOPSIS
        require Syntax::Highlight::Engine::Kate::VHDL;
        my $sh = new Syntax::Highlight::Engine::Kate::VHDL([
        ]);

DESCRIPTION
       Syntax::Highlight::Engine::Kate::VHDL is a  plugin module that provides syntax highlighting for VHDL to the Syntax::Haghlight::Engine::Kate highlighting engine.

       This code is generated from the syntax definition files used by the Kate project.  It works quite fine, but can use refinement and optimization.

       It inherits Syntax::Higlight::Engine::Kate::Template. See also there.
