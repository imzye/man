NAME
       migrate - Database schema migration for SQLAlchemy

DESCRIPTION
       migrate provides database schema migration for SQLAlchemy. The migrate command is used to work with database schema revisions.

       All documentation for migrate is available in HTML format under /usr/share/doc/python-migrate/html.

SEE ALSO
       migrate's homepage at http://code.google.com/p/sqlalchemy-migrate/
