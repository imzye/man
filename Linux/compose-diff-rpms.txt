NAME
       compose-diff-rpms  - show differences between package lists in two com‐
       poses

SYNOPSIS
       compose-diff-rpms [OPTIONS...]  --old=OLD_COMPOSE --new=NEW_COMPOSE

DESCRIPTION
       compose-diff-rpms shows changes in package lists between two  composes.
       It parses the metadata and checks RPMs mentioned there. The results are
       saved in multiple files in current directory (unless  specified  other‐
       wise).

OPTIONS
       -h, --help
              Print help and exit.

       -o, --outputdir=DIR
              Set directory in which to create the files.

       --name=NAME
              Set custom identifier for the generated files.

EXIT CODE
       This tool always exits with status code of 0.
