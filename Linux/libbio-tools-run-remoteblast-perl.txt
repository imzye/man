NAME
       bp_remote_blast.pl - script for submitting jobs to a remote blast server (ncbi blast queue at this time)

SYNOPSIS
       % bp_remote_blast.pl -p blastp -d ecoli -e 1e-5 -i myseqs.fa

DESCRIPTION
       This module will run a remote blast on a set of sequences by submitting them to the NCBI blast queue and printing the output of the request.
