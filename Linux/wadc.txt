NAME
       wadc - Development Environment for the WadC language

SYNOPSIS
       wadc

DESCRIPTION
       wadc is a self-contained editor for creating doom-style WAD files from a functional programming language.
       A description of this language can be found in the file /usr/share/doc/wadc/reference.adoc.gz and some example programs are available within /usr/share/doc/wadc/examples .

OPTIONS
       None.
