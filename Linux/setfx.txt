NAME
       setfx - load user defined chorus / reverb mode effects

SYNOPSIS
       setfx [-D devidx] [-F name] config-file

DESCRIPTION
       This manual page documents briefly the setfx command.  This manual page was written for the Debian GNU/Linux distribution (but may be used by others), because the original program does not have a man‚Äê
       ual page.

       setfx is a program that will read configuration file, and then use it to set user defined chorus and reverb modes of AWE32 sound driver.

OPTIONS
       The program follows the usual UNIX command line syntax, but doesn't support long options (options starting with two dashes `-').  Here is a summary of the options is accepts:-

       -F [file]
              Specifies device file

       -D [index]
              Sets devices index. Use -1 for autoprobe
