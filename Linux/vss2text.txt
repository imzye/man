NAME
       vss2text - manual page for vss2text 0.1.7

SYNOPSIS
       vss2text [OPTION] INPUT

DESCRIPTION
       `vss2text' converts Microsoft Visio stencils to plain text.

OPTIONS
       --help show this help message

       --version
              show version information
