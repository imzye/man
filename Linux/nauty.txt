NAME
       dreadnaut - command line interface to nauty graph isomorphism library

SYNOPSIS
       dreadnaut [-o options]

DESCRIPTION
              Enter nauty+traces test program.

       -o options  - set initial options.
              The parameter value is a string of

              dreadnaut commands from the following set: a,c,d,m,p,l,G,P,w,y,$,A,V,M The effect is the same as if these commands are entered at the beginning of the standard input.

              For help within dreadnaut, use the h and H commands.
