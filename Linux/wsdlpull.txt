NAME
       wsdlpull - C++ Web Services Library

SYNOPSIS
       #include <wsdlpull/...>

DESCRIPTION
       wsdlpull  is  a C++ web services client library. It includes a WSDL Parser, a XML Schema parser and validator and an XML Parser and serializer. It has an API and command line tool for dynamic WSDL in‚Äê
       spection and invocation. The command line tool is a generic web service client using which you can invoke most web services from command line without writing any code.

       The library covers the following areas,

              WSDL (Web Services Description Language)
              XSD Schema
