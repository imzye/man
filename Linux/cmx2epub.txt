NAME
       cmx2epub - manual page for cmx2epub 0.9.6

SYNOPSIS
       cmx2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `cmx2epub' converts Corel Presentation Exchange documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit
