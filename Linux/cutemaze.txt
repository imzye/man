NAME
       cutemaze - top-down maze game

SYNOPSIS
       cutemaze [options]

DESCRIPTION
       CuteMaze  is  a simple, top-down game in which mazes are randomly generated using one of a choice of several different algorithms. You move the character through the maze while hunting for targets
       (cheese, by default)--the game is won once all of these targets have been picked up.

OPTIONS
       -h, --help
              Displays help on commandline options.

       --help-all
              Displays help including Qt specific options.

       -v, --version
              Displays version information.
