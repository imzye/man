NAME
       mwaw2raw - debug the conversion library

SYNOPSIS
       mwaw2raw [OPTION] <Text Document>

OPTIONS
       --callgraph:
              Display the call graph nesting level

       -h, --help:
              Shows this help message

       -v, --version:
              Output mwaw2raw version
