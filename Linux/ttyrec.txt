NAME
       ttytime - print the time of the recorded session data by ttyrec(1)

SYNOPSIS
       ttytime file...

DESCRIPTION
       Ttytime tells you the time of recorded data in seconds.  For example:

              % ttytime *.tty
                  173 foo.tty
                 1832 bar.tty

SEE ALSO
       script(1), ttyrec(1), ttyplay(1)
