NAME
       esdmon - The Enlightened Sound Daemon

DESCRIPTION
       EsounD monitor.  Opens the esd socket and duplicates the stream being sent to the sound device.

USAGE
       esdmon [OPTIONS] [FILE]

       Options:

       "-s", --server=HOSTNAME set EsounD server "-b", output file should be 8-bit "-m", output file should be mono "-r" RATE, set output rate to RATE

       FILE is the file to write the output stream to.  Default is stdout.
