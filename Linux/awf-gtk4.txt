NAME
       awf-gtk4 — Application d'aperçu de thème pour GTK 4.

SYNOPSIS
       awf-gtk4 [-h] [-v] [-l] [-t theme] [-s filename] [--ltr] [--rtl]

DESCRIPTION
       La fabrique à widgets est une application d'aperçu de thème pour GTK. Elle affiche les différents types de widgets fournis par GTK dans une seule fenêtre permettant de voir l'effet visuel du thème
       appliqué.

OPTIONS
       -h, --help
               Affiche l'aide.

       -v, --version
               Affiche le numéro de version.

       -l      Liste les thèmes disponibles.

       -t theme, --theme theme
               Démarre avec le thème spécifié.

       -s filename, --screenshot filename
               Démarre et enregistre une capture d'écran png lors du SIGHUP.

       --ltr   Démarre avec le texte de gauche à droite (Left-To-Right).

       --rtl   Démarre avec le texte de droite à gauche (Right-To-Left).

VARIABLES D'ENVIRONNEMENT
       GTK_CSD=1                Active la fenêtre CSD.
       GTK_OVERLAY_SCROLLING=0  Définit les barres de défilement comme étant toujours visibles.

FICHIERS
       /home/user/.awf-gtk-accels  Configuration des raccourcis clavier.
