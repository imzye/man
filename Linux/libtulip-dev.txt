NAME
       tulip — program to visualize very large graphs

SYNOPSIS
       tulip

DESCRIPTION
       tulip  is  an information visualization framework dedicated to the analysis and visualization of relational data.  Invoking tulip-config will return a list of arguments you can specify in order to
       query of the installed tulip's configuration.

       This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.  Instead, it has documentation in HTML format; see below.

SEE ALSO
       tulip(1), tulip_perspective(1)

       The programs are documented fully by the Tulip User Manual available on the Tulip website (link to URL https://tulip.labri.fr/Documentation/current/tulip-user/html/index.html)
