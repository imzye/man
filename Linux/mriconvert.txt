NAME
       MRIConvert - interactive MRI conversion software

SYNOPSIS
       MRIConvert

DESCRIPTION
       Interactive utility that reads DICOM MRI files and writes into NifTI, Analyze, MetaImage, or BrainVoyager formats.

SEE ALSO
       MRI files can be converted using the mcverter command-line tool.
