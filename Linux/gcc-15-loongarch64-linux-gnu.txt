NAME
       gcc-ar - a wrapper around ar adding the --plugin option

SYNOPSIS
       gcc-ar [OPTION] ... [ARGS...]

DESCRIPTION
       gcc-ar is a wrapper around ar(1) adding the appropriate --plugin option for the GCC 15 compiler.

OPTIONS
       See ar(1) for a list of options that nm understands.

SEE ALSO
       ar(1)
