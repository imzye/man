NAME
       router - XML packet distributor

SYNOPSIS
       router [-h] [-c config] [-D]

DESCRIPTION
       router is the core component of the jabberd system. It passes XML packets between the components, and manages the entry and exit of components onto the network.

OPTIONS
       -h     Show summary of options.

       -c     Alternate configuration file to use. The compiled-in default is /etc/jabberd/router.xml.

       -D     Print debugging output. You should configure jabberd with the --enable-debug switch to enable this.
