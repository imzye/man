NAME
       NanoVNASaver, NanoVNASaver-gui - save Touchstone files from the NanoVNA device

SYNOPSIS
       NanoVNASaver [-h] [-d] [-D DEBUG_FILE] [-f FILE] [-r REF_FILE] [--version]

DESCRCIPTION
       The NanoVNASaver graphical tool saves Touchstone files from the NanoVNA, sweeps frequency spans in segments to gain more data points, and generally displays and analyzes the resulting data.

       The authors expect most users to use a graphical launcher instead of the command line interface.

OPTIONS
       -h, --help
              Show a summary of options and exit.

       -d, --debug
              Set loglevel to debug.

       -D DEBUG_FILE, --debug-file DEBUG_FILE
              File to write debug logging output to.

       -f FILE, --file FILE
              Touchstone file to load as sweep for off device usage.

       -r REF_FILE, --ref-file REF_FILE
              Touchstone file to load as reference for off device usage.

       --version
              Show program's version number and exit.

SEE ALSO
       The documentation is installed at /usr/share/doc/nanovna-saver/.
