NAME
       Dist::Zilla::Plugin::TemplateFiles - Use files to template a distribution

VERSION
       version 0.03

SYNOPSIS
       In your dist.ini:

         [TemplateFiles]
             filename = README
             filename = path/to/other/file

DESCRIPTION
       Utilize Text::Template to turn certain files into templates.  Each template has available to it the $dist variable that is the instance of Dist::Zilla currently running.  Only those files listed
       in "dist.ini" as "filename = blah" will be considered templates.  Filenames are given relative to the root of the build.
