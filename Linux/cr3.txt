NAME
       cr3 - program to view e-books (epub, fb2, html, txt, rtf...)

SYNOPSIS
       cr3 [options] filename

DESCRIPTION
       This manual page documents briefly the cr3 command.

       cr3 is a GUI for reading e-books in different formats
