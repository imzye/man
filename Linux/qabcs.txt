NAME
       qABCs - learn alphabet application

SYNOPSIS
       qabcs

DESCRIPTION
       qABCs  is Educational Software. An educational game for young children.  qABCs familiarizes children with the alphabet and a keyboard.  Children can play with qABCs to learn about the alphabet and
       words associated with it.

OPTIONS
       Zero options will start qABCs.

AUTHOR
