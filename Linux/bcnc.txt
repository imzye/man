NAME
       bCNC - GRBL CNC command sender, autoleveler and g-code editor

SYNOPSIS
       bCNC [OPTIONS]... [FILENAME]

DESCRIPTION
       GrblHAL (formerly GRBL) CNC command sender, autoleveler, g-code editor, digitizer, CAM and swiss army knife for all your CNC needs.

       An  advanced  fully  featured g-code sender for grblHAL (formerly GRBL). bCNC is a cross platform program (Windows, Linux, Mac) written in python. The sender is robust and fast able to work nicely
       with old or slow hardware like Raspberry Pi (As it was validated by the GRBL maintainer on heavy testing).

       -b, --baud
              set the baud rate

       -d     enable developer features

       -D     disable developer features

       -f, --fullscreen
              enable fullscreen mode

       -g <geometry>
              set the default geometry

       -h, -?, --help
              show a help page

       -i, --ini <file>
              alternative ini file for testing

       -l, --list
              list all recently opened files

       -p, --pendant <port>
              open pendant to specified port

       -P     do not start pendant

       -r, --recent
              load the most recent file opened

       -R <arg>
              load the recent file matching the argument

       -s, --serial <port>
              open serial port specified

       -S     do not open serial port

       --run  directly run the file once loaded
