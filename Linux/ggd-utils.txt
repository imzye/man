NAME
       check-sort-order - programs for use in ggd

SYNOPSIS
       check-sort-order --genome GENOME PATH

DESCRIPTION
   Positional arguments:
              PATH

OPTIONS
       --genome GENOME, -g GENOME
              a genome file of chromosome sizes and order

       --help, -h
              display this help and exit
