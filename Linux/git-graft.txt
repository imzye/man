NAME
       git-graft - Merge and destroy a given branch

SYNOPSIS
       git-graft <src-branch> <dest-branch>

DESCRIPTION
       Merge commits from <src-branch> into <dest-branch>

OPTIONS
       <src-branch>

       <dest-branch>

EXAMPLES
       $ git graft new_feature dev
