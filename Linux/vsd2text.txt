NAME
       vsd2text - manual page for vsd2text 0.1.7

SYNOPSIS
       vsd2text [OPTION] INPUT

DESCRIPTION
       `vsd2text' converts Microsoft Visio documents to plain text.

OPTIONS
       --help show this help message

       --version
              show version information
