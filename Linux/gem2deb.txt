NAME
       dh_ruby_fixdocs - remove unused files from rdoc directories

SYNOPSIS
       dh_ruby_fixdocs

DESCRIPTION
       dh_ruby_fixdocs is called by the ruby dh sequence. You should not call it directly.

SEE ALSO
       gem2deb(1), dh_ruby(1)
