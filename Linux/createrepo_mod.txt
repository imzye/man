NAME
       createrepo_mod

SYNOPSIS
       createrepo_mod [-h] directory_to_index

DESCRIPTION
       A  small wrapper around createrepo_c and modifyrepo_c toprovide an easy
       tool for generating module repositories

OPTIONS
       directory_to_index
              Directory to index

AUTHORS
