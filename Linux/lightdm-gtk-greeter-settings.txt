NAME
       lightdm-gtk-greeter-settings - Settings editor for LightDM GTK+ Greeter

SYNOPSIS
       lightdm-gtk-greeter-settings [OPTIONS]

DESCRIPTION
       lightdm-gtk-greeter-settings is a settings editor for LightDM's GTK+ Greeter.

OPTIONS
       -h, --help
              show this help message and exit

       -s SOCKET_ID, --socket-id SOCKET_ID
              sets the settings manager socket

       --use-gtk-header
              use Gtk header bars for the main window's display

       --test-socket
              enables testing for embedded mode

       lightdm(1)
