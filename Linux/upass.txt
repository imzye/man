NAME
       upass - console UI for pass

SYNOPSIS
       upass

DESCRIPTION
       upass is a curses-based, interactive interface for the pass command (the standard unix password manager).

OPTIONS
       upass has no options. It is only called interactively.

SEE ALSO
       pass(1)
