NAME
       tssnvsetbits - Runs TPM2 nvsetbits

DESCRIPTION
       nvsetbits

       Runs TPM2_NV_SetBits

       -ha    NV index handle

       [-pwdn password for NV index (default empty)]

       [-bit  bit to set, can be specified multiple times]

       -se[0-2] session handle / attributes (default PWAP)

       01     continue
