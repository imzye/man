NAME
       MooseX::ClassAttribute::Meta::Role::Attribute - An attribute metaclass for class attributes in roles

VERSION
       version 0.29

DESCRIPTION
       This class overrides Moose::Meta::Role::Attribute to support class attribute declaration in roles.
