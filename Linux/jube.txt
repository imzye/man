NAME
       JUBE, - manual page for JUBE, version 2.7.1

DESCRIPTION
       usage: jube [-h] [-V] [-v] [--debug] [--force] [--strict] [--devel]

              {run,continue,analyse,result,info,status,tag,output,comment,remove,update,log,complete,help} ...

   positional arguments:
       {run,continue,analyse,result,info,status,tag,output,comment,remove,update,log,complete,help}
              subparsers

       run    processes benchmark

       continue
              continue benchmark

       analyse
              analyse benchmark

       result show benchmark results

       info   benchmark information

       status show benchmark status

       tag    show tag documentation

       output show filename of output

       comment
              comment handling

       remove remove benchmark or workpackages

       update Check if a newer JUBE version is available

       log    show benchmark logs

       complete
              generate shell completion

       help   command help

   options:
       -h, --help
              show this help message and exit

       -V, --version
              show version

       -v, --verbose
              enable verbose console output (use -vv to show stdout during execution and -vvv to show log and stdout)

       --debug
              use debugging mode

       --force
              skip version check

       --strict
              force need for correct version

       --devel
              show development related information
