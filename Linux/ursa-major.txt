NAME
       ursa-major - manual page for ursa-major 0.4.2

DESCRIPTION
       usage: ursa-major [-h]

              {show-config,check-config,remove-module,add-module,add-tag,update-tag} ...

   positional arguments:
       {show-config,check-config,remove-module,add-module,add-tag,update-tag}
              Sub commands

       show-config
              show tag configuration content

       check-config
              validate tag config file

       remove-module
              remove a module config from a tag and also remove that module's koji_tag if it inherits from that tag.

       add-module
              add a module to tag config file

       add-tag
              read module stage change message from an environment variable and add the module's tag to koji tag inheritance depending on the tag configuration

       update-tag
              module in MBS and add its tag to koji tag inheritance.

   options:
       -h, --help
              show this help message and exit
