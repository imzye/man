NAME
     apertium-postlatex-raw — LaTeX code generator for Apertium

SYNOPSIS
     apertium-postlatex-raw [input_file [output_file]]

DESCRIPTION
     apertium-postlatex-raw This filter generates LaTeX code from the output of apertium-relatex(1) command.  Non-ASCII characters are generated into its native encoding, depending on the locale of the run‐
     ning environment.

OPTIONS
     -h, --help
             Display this help.

SEE ALSO
     apertium(1), apertium-deslatex(1), apertium-prelatex(1), apertium-relatex(1), lt-proc(1)
