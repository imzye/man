NAME
       MR::IProto::Error - iproto error

DESCRIPTION
       Instance of this class is returned instead of MR::IProto::Response if error was occurred.

PUBLIC ATTRIBUTES
       error
           Error string.

       errno
           Integer value of $!.

       request
           Instance of MR::IProto::Request.

PUBLIC METHODS
       is_error
           Always returns true.

       error_message
           Error message text.
