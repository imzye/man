NAME
       xgridloc - a GTK+ Grid Locator Calculator Application

SYNOPSIS
              xgridloc [-hv]

DESCRIPTION
              [-h: print this usage information and exit]
              [-v: print xgridloc version number and exit]
SEE ALSO
       The full documentation for xgridloc is maintained as an html document.  If the xgridloc program is properly installed at your site, the command

              firefox /usr/share/doc/xgridloc/xgridloc.html

       should give you access to the complete manual.
