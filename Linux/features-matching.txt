NAME
       features-matching - Demonstration of image feature matching

SYNOPSIS
       features-matching

DESCRIPTION
       features-matching is a command-line application which takes a pair of images and computes the descriptors selected by the user, next showing the potential matchings between the so computed
       descriptors.
