NAME
       podofoincrementalupdates - Provides information about incremental updates in PDF files

SYNOPSIS
       podofoincrementalupdates [-e N out.pdf] file.pdf

DESCRIPTION
       podofoincrementalupdates  is  one  of  the  command line tools from the PoDoFo library that provide several useful operations to work with PDF files. It can print information of incremental updates to
       file.pdf. By default the number of incremental updates will be printed.

OPTIONS
       -e N
              Extract the Nth update

       out.pdf
              Output PDF file.

       file.pdf
              Input PDF file.
