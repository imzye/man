NAME
       test_version - check that all Perl modules in a distribution have version numbers

DESCRIPTION
       This script runs the Test::HasVersion tests on the distribution in the current directory. Test::HasVersion can also be used like normal tests, see the Test::HasVersion man page for how to do that.
