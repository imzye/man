NAME
       kgx - Simple user-friendly terminal emulator for the GNOME desktop

SYNOPSIS
       kgx [OPTION]

DESCRIPTION
       Kings Cross is a simple terminal emulator for GNOME that adjusts nicely to small screen sizes and touch usage.

OPTIONS
   Help Options
       -h, --help
           Show help options

       --help-all
           Show all help options

       --help-gapplication
           Show GApplication options

       --help-gtk
           Show GTK+ options

   Application Options
       --version
           Display Kings Cross version.

       --about
           Display Kings Cross author and copyright information.

       -e, --command
           Execute the argument to this option inside the terminal.

       --working-directory=DIRNAME
           Set the working directory.

       --wait
           Wait until the child exits (TODO).

       --display=DISPLAY
           X display to use.
