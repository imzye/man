NAME
       chasen‐config - Japanese Morphological Analysis System

SYNOPSIS
       chasen‐config [options]

OPTIONS
         [--prefix[=DIR]]
         [--exec-prefix[=DIR]]
         [--chasenrc-path]
         [--libs]
         [--cflags]
         [--dicdir]
         [--sharedir]
         [--mkchadic]  [--help]
         [--version]

DESCRIPTION
       This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

AUTHOR
