NAME
       botch-droppable-diff - difference between two *.droppable files

SYNOPSIS
       botch-droppable-diff [options] droppable1 droppable2

DESCRIPTION
       Display the difference between two files with droppable build dependencies.

OPTIONS
       -h, --help
           Show help message.

       -v, --verbose
           Be verbose.
