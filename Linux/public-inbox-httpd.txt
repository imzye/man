NAME
       public-inbox-httpd - PSGI server optimized for public-inbox

SYNOPSIS
         public-inbox-httpd [OPTIONS] [/path/to/myapp.psgi]

DESCRIPTION
       public-inbox-httpd is a PSGI/Plack server supporting HTTP/1.1 and HTTP/1.0.  It uses options and environment variables common to all public-inbox-daemon(8) implementations in addition to the PSGI
       file.

       If a PSGI file is not specified, PublicInbox::WWW is loaded with a default middleware stack consisting of Plack::Middleware::ReverseProxy, and Plack::Middleware::Head

       This may point to a PSGI file for supporting generic PSGI apps.
