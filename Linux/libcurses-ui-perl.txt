NAME
       Curses::UI::TextEntry - Create and manipulate textentry widgets

CLASS HIERARCHY
        Curses::UI::Widget
        Curses::UI::Searchable
           |
           +----Curses::UI::TextEditor
                   |
                   +----Curses::UI::TextEntry

SYNOPSIS
           use Curses::UI;
           my $cui = new Curses::UI;
           my $win = $cui->add('window_id', 'Window');

           my $textentry = $win->add(
               'mytextentry', 'TextEntry'
           );

           $textentry->focus();
           my $text = $textentry->get();

DESCRIPTION
       Curses::UI::TextEntry is a widget that can be used to create a textentry widget. This class is derived from Curses::UI::TextEditor. The only special thing about this class is that the -singleline
       option is forced to a true value.  So for the usage of Curses::UI::TextEntry see Curses::UI::TextEditor.
