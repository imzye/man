NAME
       pvfs2-getmattr – display file mirroring properties volumes

SYNOPSIS
       pvfs2-getmattr [-cmhf] file
       getmattr [-cmhf] file

DESCRIPTION
       The  pvfs2-getmattr  utility  displays information about file mirroring
       for a given file.  If no options are specified, both the number of mir‐
       ror copies and the mirroring mode will be displayed.

       The options are as follows:

       -c     Retrieve the number of mirror copies.

       -m     Retrieve the mirroring mode.

       -h     Display synopsis.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set  the  OrangeFS debug mask.  Possible masks are documented in
              pvfs2-set-debugmask(1).

       PVFS2TAB_FILE
              If set, the full pathname for an alternate pvfs2tab file

FILES
       /etc/passwd

       /etc/pvfs2tab

EXAMPLES
       Display all mirroring properties for the file /mnt/foo.

             pvfs2-getmattr -f /mnt/foo
