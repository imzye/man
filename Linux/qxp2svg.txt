NAME
       qxp2svg - manual page for qxp2svg 0.0.2

SYNOPSIS
       qxp2svg [OPTION] FILE

DESCRIPTION
       `qxp2svg' converts QuarkXPress documents to SVG.

OPTIONS
       --help show this help message

       --version
              print version and exit
