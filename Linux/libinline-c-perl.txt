NAME
       Inline::C::ParseRegExp - The New and Improved Inline::C Parser

SYNOPSIS
           use Inline C => DATA =>
                      USING => ParseRegExp;

DESCRIPTION
       This module is a much faster version of Inline::C's Parse::RecDescent parser.  It is based on regular expressions instead.
