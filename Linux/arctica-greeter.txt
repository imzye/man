NAME
       arctica-greeter-magnifier - Magnifying glass for Arctica Greeter

SYNOPSIS
       arctica-greeter-magnifier

DESCRIPTION
       Arctica Greeter is a LightDM greeter for the modern desktop. It is run by the LightDM daemon if configured in lightdm.conf.

       arctica-greeter-magnifier is a helper window that can be anabled via the a11y indicator. It shows a magnified view of the screen portion below the mouse pointer.
