NAME
       fshfsmount â€” mounts a Hierarchical File System (HFS) volume

SYNOPSIS
       fshfsmount [-o offset] [-hvV] source

DESCRIPTION
       fshfsmount is a utility to mount a Hierarchical File System (HFS) volume

       fshfsmount is part of the libfshfs package.  libfshfs is a library to access the Hierarchical File System (HFS) format

       source is the source file.

       The options are as follows:

       -h      shows this help

       -o offset
               specify the volume offset in bytes

       -v      verbose output to stderr

       -V      print version

       -X extended_options
               extended options to pass to sub system

ENVIRONMENT
       None

FILES
       None

EXAMPLES
       # fshfsmount /dev/sda1
       fshfsmount 20210729

DIAGNOSTICS
       Errors, verbose and debug output are printed to stderr when verbose output -v is enabled.  Verbose and debug output are only printed when enabled at compilation.
