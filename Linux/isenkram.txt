NAME
       isenkramd - user desktop background process to handle new hardware dongles

SYNOPSIS
       isenkramd

DESCRIPTION
       This  background  desktop process is started by the desktop autostart mechanism and listen for udev events to handle new hardware dongles.  When a new dongle is detected, sources for mapping hard‐
       ware to packages are consulted and a GUI notification is shown to the user to suggest to install the hardware relevant packages.

OPTIONS
       There are no options.

SEE ALSO
       • isenkram-autoinstall-firmware(8), isenkram-pkginstall(8)
