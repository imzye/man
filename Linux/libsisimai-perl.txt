NAME
       Sisimai::Rhost::YahooInc - Detect the bounce reason returned from Yahoo Inc.

SYNOPSIS
           use Sisimai::Rhost::YahooInc;

DESCRIPTION
       "Sisimai::Rhost::YahooInc" detects the bounce reason from the content of "Sisimai::Fact" object as an argument of find() method when the value of "rhost" of the object end with "*.yahoodns.net".
       This class is called only "Sisimai::Fact" class.

CLASS METHODS
   "find(Sisimai::Fact Object)"
       find() method detects the bounce reason.
