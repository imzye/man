NAME
       MPDCon - Music Player Daemon controller

SYNOPSIS
       MPDCon

DESCRIPTION
       This manual page documents briefly the MPDCon command.  This manual page was written for the Debian distribution because the original program does not have a
       manual page.

       MPDCon is a simple GNUstep controller for MPD.

SEE ALSO
       GNUstep(7).
