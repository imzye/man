NAME
       SceneViewer3D - 3D robotic scenes viewer

SYNOPSIS
       SceneViewer3D [file.3Dscene]

DESCRIPTION
       A viewer of 3D scenes generated by many other MRPT applications, in the 3Dscene format: a (possibly gz-compressed) file where a mrpt::opengl::COpenGLScene has been dumped to.

BUGS
