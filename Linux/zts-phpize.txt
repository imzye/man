NAME
       zts-phpize - prepare a PHP extension for compiling

SYNOPSIS
       zts-phpize [options]

DESCRIPTION
       zts-phpize is a shell script to prepare PHP extension for compiling.

OPTIONS
       --clean        Remove all created files
       --help         Prints usage information
       --version
       -v             Prints API version information

SEE ALSO
       zts-php(1)

VERSION INFORMATION
       This manpage describes php, version 8.0.14.
