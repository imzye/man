NAME
       obex_push - Push and Receive Files over the Bluetooth OBEX Push Channel

SYNOPSIS
       obex_push [channel] [bdaddr] [filename]

DESCRIPTION
       obex_push can send and receive files over the Bluetooth OBEX Push Channel.

       This program is designed to be an example application for the openobex library.

       Link your bluetooth device with your computer.

       You can see the SDP services with: "sdptool browse bdaddr"

OPTIONS
       If no options are given, any files sent over bluetooth will be received over channel 10 and stored in /tmp. If bdaddr and filename are given, the file will be send to the device specified with bdaddr.

       channel
           The bluetooth channel to use. Lookup the OBEX Push channel with sdptool browse bdaddr.

       bdaddr
           The bluetooth address bdaddr of the device to send to.

       filename
           The file to send.
