NAME
       unidup - Scan through a sorted .hex file and report duplicate code points

SYNOPSIS
       unidup [input-font.hex]

DESCRIPTION
       unidup  reads  a  sorted GNU Unifont .hex file (sorted with the Unix sort utility) and prints notification of any duplicate code points on stdout.  The input file can be specified on the command line.
       If no file is specified, input will be read from STDIN until end of file.

FILES
       *.hex GNU Unifont font files
