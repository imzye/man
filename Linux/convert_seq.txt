NAME
       convert_seq - conversion of sequence and alignment formats

SYNOPSIS
       convert_seq [OPTION]

DESCRIPTION
       convert_seq is used for sequence and alignment format conversion.  Instead of using command line arguments it starts a conversation with the user where the desired function can be invoked by answering
       questions.

       This program is used by the prof(1) secondary structure, accessibility and transmembrane helix predictor from Burkhard Rost.

OPTIONS
       None.  Answer questions printed on STDOUT by typing your choice into STDIN.
