NAME
       bipmkpw - Password hasher for BIP

SYNOPSIS
       bipmkpw

DESCRIPTION
       bipmkpw converts a password into the double-hash used by bip. Copy the generated output from bipmkpw into bip config file.

       Password cannot contain spaces.

SEE ALSO
       bip.conf(5) bip(1)
