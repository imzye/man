NAME
       Data::Fake::Internet - Fake Internet-related data generators

VERSION
       version 0.006

SYNOPSIS
           use Data::Fake::Internet;

           fake_tld()->();     # .gov, etc.
           fake_domain()->();  # atqueaut.gov, etc.
           fake_email()->();   # john.smith@atqueaut.gov, etc.

DESCRIPTION
       This module provides fake data generators for Internet-related data.

       All functions are exported by default.

FUNCTIONS
   fake_tld
           $generator = fake_tld();

       Returns a generator that randomly selects from a weighted list of about 50 domain suffixes based on a list of the top 500 domains by inbound root-domain links.

   fake_domain
           $generator = fake_domain();

       Returns a generator that concatenates two random words with a random domain suffix.

   fake_email
           $generator = fake_email();

       Returns a generator that constructs an email from a random name and a random domain.
