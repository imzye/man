NAME
       HTTraQt - GUI-program for downloading of internet sites DESCRIPTION

DESCRIPTION
       HTTraQt Program is intended for loading of Internet pages and/or their contents, equipped with flexible system of settings.  HTTraQt - graphical user interface (GUI) for HTTrack library, developed
       in C++ and based on multiplatform Qt library.
