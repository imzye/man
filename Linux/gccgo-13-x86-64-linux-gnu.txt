NAME
       gofmt - format Go code

SYNOPSIS
       gofmt [flags] [path]...

DESCRIPTION
       A tool for formatting Go source code.  For full documentation run gofmt -help or see http://golang.org/cmd/gofmt.

SEE ALSO
       go(1)
