NAME
       mExecTG - Mosaicking executive for 2MASS, SDSS, DSS.

SYNOPSIS
       mExecTG [-l][-k][-c][-o output.fits][-d level][-f region.hdr | -h header] survey band [workspace-dir]

DESCRIPTION
       Mosaicking executive for 2MASS, SDSS, DSS. Includes remote data and metadata access.

OPTIONS
       -l     Level only

       -k     Keep all

       -c     Clean

       -d level
              Debug level
