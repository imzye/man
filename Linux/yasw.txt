NAME
       yasw - an application used to correct images taken with a camera while scanning a book

SYNOPSIS
       yasw

DESCRIPTION
       yasw - Yet Another Scan Wizard (YASW) is an application used to correct images taken with a camera while scanning a book.

OPTIONS
       none
