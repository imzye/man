NAME
       chiro2nallfinetriangs - find the number of fine triangulations of a chirotope

SYNOPSIS
       chiro2nallfinetriangs [OPTION]...

DESCRIPTION
       Compute the number of fine triangulations of a chirotope.  Input is read from stdin; output is written to stdout.  The standard options described in TOPCOM(7) are available.

EXIT STATUS
       0 if the points are read from stdin successfully.

       1 otherwise.
