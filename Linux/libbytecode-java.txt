NAME
       bytecode - java library for the editing of the language's bytecode

DESCRIPTION
       A Java object-model for a Java Bytecode Macro-Assembler.  BioJava requests and distributes it.

SEE ALSO
       http://www.biojava.org /usr/share/doc/libbytecode-java/

AUTHOR
       bytecode was written by the developers of BioJava.  This manual page was written for the Debian project (but may be used by others).
