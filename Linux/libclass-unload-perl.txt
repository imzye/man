NAME
       Class::Unload - Unload a class

VERSION
       version 0.12

SYNOPSIS
           use Class::Unload;
           use Class::Inspector;

           use Some::Class;

           Class::Unload->unload( 'Some::Class' );
           Class::Inspector->loaded( 'Some::Class' ); # Returns false

           require Some::Class; # Reloads the class

METHODS
   unload $class
       Unloads the given class by clearing out its symbol table and removing it from %INC.  If it's a Moose class, the metaclass is also removed.

SEE ALSO
       Class::Inspector
