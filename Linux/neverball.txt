NAME
       mapc - A map compiler for Neverball and Neverputt

SYNOPSIS
       mapc map data [options]

DESCRIPTION
       mapc is a map compiler for Neverball and Neverputt. The output will be saved to map.sol.

OPTIONS
       The following command-line options are available:

       --debug
              Turn off optimizations.

SEE ALSO
       Mapping guide: <http://icculus.org/neverball/mapping/>. The Neverball wiki also contains useful information on how to create new levels: <http://www.nevercorner.net/wiki/>.
