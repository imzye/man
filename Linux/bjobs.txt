NAME
       bjobs - displays and filters information about jobs in familar openlava format.

SYNOPSIS
       bjobs job_id...

DESCRIPTION
       The bjobs displays and filters information about jobs in familar openlava format.

OPTIONS
       -h | --help
           brief help message

       --man
           full documentation

EXIT STATUS
       On success, bjobs will exit with a value of zero. On failure, bkill will exit with a value greater than zero.
