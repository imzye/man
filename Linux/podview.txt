NAME
       podview - graphical pod viewer

   DESCRIPTION
       The small GUI browser for POD format. Given the argument either as a
       file path, or perl module name (f.ex. File::Copy), displays the
       documentation found.

   SEE ALSO
       perlpod - the Plain Old Documentation format

       Prima - perl graphic toolkit the viewer is based on

       Prima::HelpViewer - menu commands explained
