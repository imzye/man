NAME
       biometric-manager - management tool for biometric authentication

SYNOPSIS
       biometric-manager

DESCRIPTION
       biometric-manager is a tool for managing biometric authentication.  It can control the opening or closing of biometric authentication.  It also control the opening or closing of biometric devices'
       drivers.  You can use it to enroll or delete biometric features.  You can only see your own features in a session unless you are root.

BUGS
