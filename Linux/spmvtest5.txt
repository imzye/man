NAME
       spmvtest5 - SpMV kernel benchmark for specified matrix storage format

SYNOPSIS
       spmvtest5 matrix_filename matrix_type iter [block]

DESCRIPTION
       This  program  inputs the matrix data from matrix_filename and compute the multiply of the matrix with matrix_type and a vector (1, ..., 1)^T.  The FLOPS performance is measured as the average of iter
       iterations.  The Matrix Market format is supported for matrix_filename.  If necessary, the block size of the BSR and BSC formats can be specified by block.  See Lis User Guide for more information.

EXIT STATUS
       The following exit values are returned:

       0      The process is normally terminated

       unspecified
              An error occurred
