NAME
       btrfs-sxbackup - a btrfs snapshot backup utility

SYNOPSIS
       btrfs-sxbackup [-h | --help] [-q | --quiet] [-v] [--version]

DESCRIPTION
       Btrfs  snapshot  backup  utility with push/pull support via SSH, reten‐
       tion, email notifications, compression of transferred data, and  syslog
       logging.

OPTIONS
       -h, --help
              show help message and exit

       -q, --quiet
              do not log to stdout

       --version
              show program's version number and exit

       -v     can be specified multiple times to increase verbosity

EXAMPLES
   Initialize
       Initialize a backup job pulling snapshots of subvolume / on remote host
       myhost.org to local subvolume /backup/myhost:

              btrfs-sxbackup init ssh://root@myhost.org:/ /backup/myhost

       Initialize a backup job pushing snapshots of local subvolume /  to  re‐
       mote subvolume /backup/myhost on host mybackupserver.org:

              btrfs-sxbackup init / ssh://root@mybackupserver.org:/backup/myhost

   Run
       Run a backup job:

              btrfs-sxbackup run /backup/myhost

   Cron
       Cronjob performing a pull backup job:

              # /etc/cron.d/btrfs-sxbackup
              PATH="/usr/sbin:/usr/bin:/sbin:/bin"
              30 2    * * *     root     btrfs-sxbackup run /backup/myhost
