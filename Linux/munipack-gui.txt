NAME
       xmunipack - graphical user interface for Munipack

SYNOPSIS
       xmunipack [fits]

DESCRIPTION
       Munipack is a general astronomical image processing software.  Xmunipack provides a graphical user interface for selected features.

       With no parameters, browser is opened.

       Viewer is opened when a FITS file is provided.

OPTIONS
       -h, --help
              Print this help.

       --version
              Print version and license.
