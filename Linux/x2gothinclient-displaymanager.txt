NAME
       x2gothinclientd - Session Control Daemon running on X2Go Thin Clients.

SYNOPSIS
       x2gothinclientd

DESCRIPTION
       x2gothinclientd is a session control daemon that runs on X2Go thin clients.

OPTIONS
       x2gothinclientd has no known options.
