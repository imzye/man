NAME
       chime2jmol - convert chime into jmol scripts

SYNOPSIS
       chime2jmol [ -s sourcedir | -d destdir | -c | -a archive | -b path | -v ]

       chime2jmol
              usage

DESCRIPTION
       chime2jmol is a Perl program for converting Chime into JMol scripts.

OPTIONS
       The following options are currently supported:

       usage  Show summary of options.

       -s sourcedir
              Specify source directory.

       -d destdir
              Specify destination directory.

       -c     Clear destination directory.

       -a archive
              Specify alternate archive name.

       -b path
              Specify codebase.

       -v     Be verbose.
