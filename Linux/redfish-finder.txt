NAME
       redfish-finder - discover and provide canonical access to local redfish bmc services

SYNOPSIS
       redfish-finder [options]

DESCRIPTION
       redfish-finder automatically discovers redfish based BMC services on a system, configures the appropriate network interface to access it, and adds a host entry to reach the service via the BMC config‚Äê
       ured name, and the canonical name 'redfish-localhost'

OPTIONS
       -h | --help
              Display Usage synopsis and options description

       -s | --shutdown
              Remove OS Host and Interface configuration

FILES
       /etc/hosts
              redfish-finder will read and write /etc/hosts to add the redfish-localhost name

       https://github.com/nhorman/redfish-finder
