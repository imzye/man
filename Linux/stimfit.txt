NAME
       stimfit - stimfit

SYNOPSIS
       stimfit [-h] [--verbose] [-d <str>] [File to open]

DESCRIPTION
       -h, --help
              show this help message

       --verbose
              generate verbose log messages

       -d, --dir=<str>
              Working directory to change to
