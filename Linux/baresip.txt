NAME
       baresip - portable and modular command-line SIP user-agent

SYNOPSIS
       baresip [options]

DESCRIPTION
       baresip v0.6.1 Copyright (C) 2010 - 2019 Alfred E. Heggestad et al.

   options:
       -6     Prefer IPv6

       -d     Daemon

       -e <commands>
              Execute commands (repeat)

       -f <path>
              Config path

       -m <module>
              Pre-load modules (repeat)

       -p <path>
              Audio files

       -h -?  Help

       -t     Test and exit

       -n <net_if>
              Specify network interface

       -u <parameters>
              Extra UA parameters

       -v     Verbose debug
