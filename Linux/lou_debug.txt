NAME
       lou_debug - A debugger for liblouis Braille translation tables

SYNOPSIS
       lou_debug [OPTIONS] TABLE[,TABLE,...]

DESCRIPTION
       Examine  and  debug Braille translation tables. This program allows you to inspect liblouis translation tables and gather information about them, such as forward and backward rules, characters and dot
       patterns, specific opcodes, the size of a table, whether a hyphenation table is used, how many passes the translation takes and much more.

       -h, --help
              display this help and exit

       -v, --version
              display version information and exit
