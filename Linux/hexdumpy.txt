NAME
       hexdumpy - manual page for hexdumpy 3.4

DESCRIPTION
   Usage:
              hexdumpy [binfile|-] hexdumpy -r hexfile hexdumpy --test [logfile]

OPTIONS
       --version
              show program's version number and exit

       -h, --help
              show this help message and exit

       -r, --restore
              restore binary from hex dump

       --test run hexdump sanity checks

SEE ALSO
       The full documentation for hexdumpy is maintained as a Texinfo manual.  If the info and hexdumpy programs are properly installed at your site, the command

              info hexdumpy

       should give you access to the complete manual.
