NAME
     apertium-reodt — OpenDocument Text format processor for Apertium

SYNOPSIS
     apertium-reodt [input_file [output_file]]

DESCRIPTION
     apertium-reodt is an ODT format processor.  It restores the original ODT formatting the text had before being passed through the apertium-desodt(1) deformatter.

OPTIONS
     -h, --help
             Display this help.

SEE ALSO
     apertium(1), apertium-rertf(1), apertium-retxt(1), lt-proc(1)
