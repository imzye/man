NAME
       vrml2oogl - convert VRML to OOGL

SYNOPSIS
       vrml2oogl [handle]

DESCRIPTION
       This manual page documents briefly the vrml2oogl command.

       vrml2oogl  Convert VRML file (possibly compressed/gzipped) into an OOGL
       file.  If a handle is given, return GCL not OOGL, where the OOGL geome‐
       try  is  defined  as the handle.  The handle generally corresponds to a
       URL.
