NAME
       pyqso - amateur radio logbook application

SYNOPSIS
       pyqso [--debug|-d] [--logbook logbook-file ]

DESCRIPTION
       pyqso is a contact logging tool for amateur radio operators.

OPTIONS
       -d, --debug
              All debugging-related messages are written to a file called pyqso.debug located in the current working directory.

       -l, --logbook
              Open the specified log book file when starting pyqso. If the file does not exist, it will be created.
