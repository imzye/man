NAME
       PTroller - Merge several images into a single one

SYNOPSIS
       PTroller [options] tiff_file(s)

DESCRIPTION
       PTroller takes a set of images and merges them into a single one.

OPTIONS
       -o filename
           Output filename (defaults to merged.tif).

       -f  Force processing (do not stop on warnings).

       -x  Delete source files (use with care).

       -q  Quiet run

       -h  Show this message
