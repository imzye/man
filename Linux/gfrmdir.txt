NAME
       gfrmdir - remove Gfarm directories

SYNOPSIS
       gfrmdir path...

DESCRIPTION
       gfrmdir removes directories specified in the arguments.

OPTIONS
       -?
           displays a list of command options.

SEE ALSO
       gfmkdir(1)
