NAME
       pkgjs-install-minimal - pkgjs-install-minimal

DESCRIPTION
       Link or copy all available dependencies of a JS project using Debian dependencies.

OPTIONS

       -h, --help: print this

       --copy: copy modules instead of link them

       --prod: don't install dev dependencies
