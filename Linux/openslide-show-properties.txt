NAME
       openslide-show-properties - Print OpenSlide properties for a slide

SYNOPSIS
       openslide-show-properties [--help] [--version] slide...

DESCRIPTION
       Print OpenSlide properties for a virtual slide file.

OPTIONS
       --help Display usage summary.

       --version
              Display version and copyright information.

EXIT STATUS
       openslide-show-properties returns 0 on success, 1 if a slide file could
       not be read, or 2 if the arguments are invalid.
