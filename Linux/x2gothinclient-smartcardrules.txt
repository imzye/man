NAME
       x2gognupgccid - udev Script handling GnuPG smart card readers on X2Go Thin Clients.

SYNOPSIS
       x2gognupgccid

DESCRIPTION
       x2gognupgccid is a udev script that handles the usage of GnuPG smart card readers that are connected to X2Go Thin Clients.

OPTIONS
       x2gognupgccid has no known options.
