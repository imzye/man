NAME
       ruby_parse - ruby_parse checks syntax of ruby files

SYNOPSIS
       ruby_parse [FILE]...

DESCRIPTION
       ruby_parse checks syntax of ruby files and prints if they are good or bad.

USAGE
       EXAMPLES:

       $ ruby_parse hello.rb
