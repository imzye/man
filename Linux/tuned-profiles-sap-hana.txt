NAME
       tuned-profiles-sap-hana - description of profiles provided for the SAP HANA

DESCRIPTION
       These profiles provides performance optimizations for the SAP HANA applications.

PROFILES
       The following profiles are provided:

       sap-hana
              A  performance  optimized  profile for the SAP HANA applications. It disables power saving mechanisms and enables sysctl settings that improve throughput performance of disk and network IO. CPU
              governor is set to performance and CPU energy performance bias is set to performance. It also disables transparent hugepages, locks CPU to the low C states (by PM QoS) and tunes sysctl  regard‐
              ing semaphores.
