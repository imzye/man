NAME
       wks2ods - manual page for wks2ods 0.9.6

SYNOPSIS
       wks2ods [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wks2ods' converts MS Works databases/spreadsheets to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --encoding ENCODING
              set the INPUT encoding. Use --list-encodings to see which encodings can be used.

       --list-encodings
              show the available encodings and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output
