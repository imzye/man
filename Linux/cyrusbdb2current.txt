NAME
       cyrusbdb2current - command-line utility converting the SASLDB database from BerkeleyDB to the database format currently used bys sasldb.

SYNOPSIS
       cyrusbdb2current <sasldb_old_path> <sasldb_new_path>

DESCRIPTION
       cyrusbdb2current converts the current sasldb database from BerkeleyDB format to the currently used database format. It is STRONGLY RECOMMENDED to make a backup of the current database before the
       conversion.

       We expect that the old path is /etc/sasldb2 and the new one is /etc/sasl2/sasldb2

SEE ALSO
       saslpasswd2(8)

       rfc4422 - Simple Authentication and Security Layer (SASL)
