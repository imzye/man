NAME
       mysofa2json - convert SOFA files to JSON.

SYNOPSIS
       mysofa2json <sofafile>

DESCRIPTION
       mysofa2json is a utility to convert HRTFs stored in the AES69-2015 SOFA format to JSON.  The JSON data is output to stdout.

       mysofa2json is part of libmysofa.
