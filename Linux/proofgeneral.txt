NAME
       coqtags - tags generator for Coq files for Emacs

SYNOPSIS
       coqtags <filenames>

DESCRIPTION
       Generates the TAGS file used by Emacs for a Coq project.

       <filenames>
              the names of the files for which to generate the TAGS file
