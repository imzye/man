NAME
       Mu::Role - Moo::Role but with less typing

VERSION
       version 1.191300

SYNOPSIS
           use Mu::Role;

           ro "hro";
           lazy hlazy => sub { 2 };
           rwp "hrwp";
           rw "hrw";

DESCRIPTION
       Mu imports both Moo::Role and MooX::ShortHas, making it even less work in typing and reading to set up a role.
