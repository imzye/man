NAME
       apertium-validate-dictionary — dictionary file validator for Apertium

SYNOPSIS
       apertium-validate-dictionary input_file

DESCRIPTION
       apertium-validate-dictionary is a script that validates a dictionary file against the apertium DTD file for dictionaries using the xmllint(1) utility.
