NAME
       xcluster3 — Eisen-clustering of microarray data

SYNOPSIS
       xcluster3

DESCRIPTION
       xcluster3  is  the  Motif-based X-interface of the Linux-variant of the program cluster by Michael Eisen. It is used to cluster microarray expression data. The presentation of the data is performed by
       TreeView. This is a Windows program of his, for which the Linux variants TreeView and MapleView exist as Java rewrites.

SEE ALSO
       mapletree (1), treeview (1), smile (1).
