NAME
       nauty-blisstog - convert files of graphs in bliss format to stdout in sparse6 format

SYNOPSIS
       blisstog [-n#:#] [infile]*

DESCRIPTION
              Read files of graphs in Bliss (Dimacs) format and write

              them to stdout in sparse6 format.

       -n#:#  Specify a range of n values for output

              Input files with name *.gz are ungzipped
