NAME
       mailman-wrapper - wrapper around mailman that runs it as user 'list'.

SYNOPSIS
       mailman-wrapper <arguments>

DESCRIPTION
       mailman-wrapper is a wrapper around mailman that takes care of exectuing mailman as user list. When invoking mailman manually, you should always use mailman-wrapper instead.

SEE ALSO
       mailman(1)
