NAME
       assa-hexdump - Dump the contents of the file in hex to standard output

SYNOPSIS
       assa-hexdump [OPTIONS] <INPUT-FILE>

OPTIONS
       -h, --help          print help message

       -v, --version       print version number

AUTHOR
