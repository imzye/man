NAME
       speakupconf - script to load/save all the vars in speakup

SYNOPSIS
       speakupconf save|load

DESCRIPTION
       Speakupconf is a script that saves and restores speakup's customizable variables.

       If it is run as root, it saves in /etc/speakup, or, if it is run as a user, it saves in $HOME/.speakup.  This means that each user can have unique speakup settings.
