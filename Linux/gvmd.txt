NAME
       greenbone-feed-sync - Greenbone Vulnerability Manager data sync script

DESCRIPTION
       Update the data (CERT, SCAP or GVMD_DATA) from the feed server.

       --describe
              display current feed info

       --feedversion
              display version of this feed

       --help display this help

       --identify
              display information

       --selftest
              perform self-test

       --type <TYPE>
              choose type of data to sync (CERT, SCAP or GVMD_DATA)

       --version
              display version
