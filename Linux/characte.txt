NAME
       charactermanaj - program to edit character avatars

SYNOPSIS
       charactermanaj

DESCRIPTION
       This manual page documents briefly the charactermanaj command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       charactermanaj is a program that edit the character avatars through a GUI.

OPTIONS
       The program doesn't have any options.
