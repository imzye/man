NAME
       Graph::Maker::Empty - Creates an graph with no nodes or edges

VERSION
       Version 0.01

SYNOPSIS
       Creates a graph with no nodes or edges.

               use strict;
               use warnings;
               use Graph;
               use Graph::Maker;
               use Graph::Maker::Empty;

               my $g = new Graph::Maker('empty', undirected => 1);
               # work with the graph

FUNCTIONS
   new %params
       Creates a graph with no nodes or edges, the only recognized parameter is graph_maker all parameters are passed onto Graph's constructor.  If graph_maker is specified it will be called to create
       the Graph class as desired (for example if you have a subclass of Graph), this defaults to create a Graph with the parameters specified.
