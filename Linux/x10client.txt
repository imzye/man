NAME
       x10client - use to send a list of x10 events to an x10 server

USAGE
       Usage:

           x10client event1 event2 event3 ...

       Where event1, event2, event3, etc. each represent an x10 event

DESCRIPTION
       Sends x10 events to an x10 server
