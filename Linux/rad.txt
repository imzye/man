NAME
       rad - manual page for rad 1.0

DESCRIPTION
       NAME

              rad - Radical series shared_meataxe 1.0

       SYNTAX

              rad [<Options>] [-l <Length>] [-H <Num>] <Name>

       FILES

       <Name>.1 ... <Name>.nbgen
              i  generators of a representation

       <Name>.cfinfo
              i  info-file after running PWKOND

       <Name>.rad
              o  matrix for basischange

              or the vectors for hom

       <Name><S>.h<Num>
              o  ???

       OPTIONS

       -Q ...................... Quiet, no messages

       -V ...................... Verbose, more messages

       -T <MaxTime> ............ Set CPU time limit [s]

       -l <Length> ............. calculate the first <length> heads

       -H <Num> ................ calculate generators for the <num>th head

       DESCRIPTION

              This  program  calculates the radical series of an arbitrary module given by <gens> or the homomorphisms from the projective modules corresponding to the composition factors of the given module
              to the module.  In case of option 'h', the vectors generating submodules with head isomorphic to <cfs>, are stored in gens<Name>.h<num>
