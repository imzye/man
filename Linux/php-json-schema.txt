NAME
       validate-json - JSON Schema command line interface

SYNOPSIS
       validate-json data.json
       validate-json data.json schema.json

DESCRIPTION
       Validate schema

OPTIONS
       --dump-schema
              Output full schema and exit

       --dump-schema-url Output URL of schema

       --verbose
              Show additional output

       --quiet
              Suppress all output

       -h --help
              Show this help
