NAME
       chkservice - tool for managing systemd units in terminal

SYNOPSIS
       chkservice

DESCRIPTION
       This manual page documents briefly the chkservice utility

       chkservice is a tool that helps manage systemd unit in terminal.

OPTIONS
       ?      Show summary of options.

       space  Enable/disable service.

       q      Exits chkservice

       s      Start/stop unit

       r      Reloads the service.

       Up/k   Moves cursor up

       Down/j Moves cursor down

       PgUp/b Moves cursor down

       PgDown/f
              Moves page down
