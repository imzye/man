NAME
       MCE::Mutex::Channel - Mutex locking via a pipe or socket

VERSION
       This document describes MCE::Mutex::Channel version 1.902

DESCRIPTION
       A pipe-socket implementation for "MCE::Mutex".

       The API is described in MCE::Mutex.

       new
       lock
       lock_exclusive
       lock_shared
       guard_lock
       unlock
       synchronize
       enter
       timedwait
