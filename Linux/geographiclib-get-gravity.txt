NAME
       geographiclib-get-gravity -- a GeographicLib administrative tool

DESCRIPTION
       geographiclib-get-gravity  downloads  and installs the gravity datasets
       used  by  GeographicLib,  <https://geographiclib.sourceforge.io>.   For
       documentation, supply the -h option:

           geographiclib-get-gravity -h

       By   default,  the  datasets  are  installed  in  /usr/share/Geographi‚Äê
       cLib/gravity and the script requires write access to this directory.
