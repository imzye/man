NAME
       qdelaunay - compute the Delaunay triangulation

SYNOPSIS
       qdelaunay [<options>]

DESCRIPTION
       This manual page was written for the Debian distribution because the original program does not have a manual page.

       This program is documented fully in HTML format ⟨file:///usr/share/doc/libqhull-dev/qhull/html/qdelaun.htm⟩, available in the libqhull-doc package.

SEE ALSO
       /usr/share/doc/libqhull-dev/qhull/html/qdelaun.htm
