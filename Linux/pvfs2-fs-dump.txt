NAME
       pvfs2-fs-dump  â€“ display lower-level information about an OrangeFS file
       system

SYNOPSIS
       pvfs2-fs-dump [-v] [-m mount_point]

DESCRIPTION
       The pvfs2-fs-dump utility displays low level information about the file
       system database.

       The options are as follows:

       -v     Print version number and exit.

       -m     Specify the file system to operate on.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set  the  OrangeFS debug mask.  Possible masks are documented in
              pvfs2-set-debugmask(1).

       PVFS2TAB_FILE
              If set, the full pathname for an alternate pvfs2tab file

FILES
       /etc/pvfs2tab
