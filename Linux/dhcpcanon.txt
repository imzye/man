NAME
       dhcpcanon - DHCP Client Anonymity profile

SYNOPSIS
       dhcpcanon [-h] [-d] [-l LEASE] [-v] [interface]

DESCRIPTION
       dhcpcanon is a DCHP client implementation of the DHCP Anonymity Profiles (RFC7844).  using Scapy Automaton.

OPTIONS
       The options which apply to the dhcpcanon command are:

       -h, --help
           Show help message and exit.

       -d, --debug
           Show debug messages.

       -l, --lease LEASE
           Custom lease time.

       -v, --version
           Show version.
