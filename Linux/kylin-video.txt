NAME
       kylin-video - The best GUI frontend for MPlayer

SYNOPSIS
       kylin-video

DESCRIPTION
       Kylin Video is a GUI media player based on Qt 5, using mplayer(1) as its backend.

SEE ALSO
       mplayer(1)
