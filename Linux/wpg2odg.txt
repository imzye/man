NAME
       wpg2odg - manual page for wpg2odg 0.9.6

SYNOPSIS
       wpg2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wpg2odg' converts WordPerfect Graphics images to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
