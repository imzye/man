NAME
       Wx::GLCanvas - interface to wxWidgets' OpenGL canvas

SYNOPSIS
           use OpenGL; # or any other module providing OpenGL API
           use Wx::GLCanvas;

DESCRIPTION
       The documentation for this module is included in the main wxPerl distribution (wxGLCanvas).
