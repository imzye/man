NAME
       shove - test tool for shell scripts with TAP outputs

SYNOPSYS
       shove

OPTIONS
       -h, --help
           show help.

       -v, --version
           show version.

       -s, --shell
           SHELL to execute tests. Default is $SHELL.

       --verbose
           verbose output.
