NAME
       Ticket::Simple - A basic ticket system.

VERSION
       version v0.0.2

SYNOPSIS
        my $ts=Ticket::Simple->new();

       or

        my $ts=Ticket::Simple->new({ttl=>600});

DESCRIPTION
       Provides a simple ticket system for  creating, storing, fetching, comparing user assigned tickets.

SUBROUTINES/METHODS
   create_ticket
   wipe_ticket
   destroy_ticket
   fetch_ticket
   is_ticket_equal_stored
   is_ticket_valid
   is_ticket_valid_now
       Test if the ticket was issued

   now
   seed
   store_ticket
   log_cfg
   set_ttl
   get_ttl
DIAGNOSTICS
CONFIGURATION AND ENVIRONMENT
       No external configuration needed.
