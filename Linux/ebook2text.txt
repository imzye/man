NAME
       ebook2text - manual page for ebook2text 0.1.3

SYNOPSIS
       ebook2text [OPTION] FILE

DESCRIPTION
       `ebook2text' converts e-books to plain text.  Supported formats include FictionBook 2, BBeB and various Palm-based formats.

OPTIONS
       --info display document metadata instead of the text

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://sourceforge.net/p/libebook/tickets/> or <https://bugs.documentfoundation.org/>.
