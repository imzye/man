NAME
       osmo-stp - Osmocom SIGTRAN STP (Signaling Transfer Point)

DESCRIPTION
       -h --help
              This text.

       -D --daemonize
              Fork the process into a background daemon

       -c --config-file filename
              The config file to use. Default: ./osmo-stp.cfg

       -V --version
              Print the version of OsmoSTP
