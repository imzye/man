NAME
       cinnamon-looking-glass - Debugging interface for Cinnamon

SYNOPSIS
       cinnamon-looking-glass

DESCRIPTION
       cinnamon-looking-glass runs the graphical debugger interface for cinnamon.

AUTHOR
