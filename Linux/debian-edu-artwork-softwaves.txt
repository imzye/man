NAME
       update-debian-edu-artwork-softwaves - Enable/disable Debian Edu softWaves theme system-wide

SYNOPSIS
       update-debian-edu-artwork-softwaves { configure | remove }

DESCRIPTION
       update-debian-edu-artwork-softwaves configure enables the Debian Edu softwaves theming (as shipped with Debian Edu Stretch) system-wide (several login managers, KDE theme, background for all desk‚Äê
       tops, GRUB boot loader, etc.pp.).

       update-debian-edu-artwork-softwaves remove disables the Debian Edu themes and switches back to Debian's default themes (from package desktop-base).
