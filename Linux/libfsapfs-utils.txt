NAME
       fsapfsmount â€” mounts an Apple File System (APFS) container

SYNOPSIS
       fsapfsmount [-f file_system_index] [-o offset] [-p password] [-r password] [-hvV] source

DESCRIPTION
       fsapfsmount is a utility to mount an Apple File System (APFS) container

       fsapfsmount is part of the libfsapfs package.  libfsapfs is a library to access the Apple File System (APFS) format

       source is the source file.

       The options are as follows:

       -f file_system_index
               mounts a specific file system or "all"

       -h      shows this help

       -o offset
               specify the container offset in bytes

       -p password
               specify the password

       -r password
               specify the recovery password

       -v      verbose output to stderr

       -V      print version

       -X extended_options
               extended options to pass to sub system

ENVIRONMENT
       None

FILES
       None

EXAMPLES
       # fsapfsmount -f 1 -p Password /dev/sda1
       fsapfsmount 20181214

DIAGNOSTICS
       Errors, verbose and debug output are printed to stderr when verbose output -v is enabled.  Verbose and debug output are only printed when enabled at compilation.
