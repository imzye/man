NAME
       Web::Machine::I18N - The I18N support for HTTP information

VERSION
       version 0.17

SYNOPSIS
         use Web::Machine::I18N;

DESCRIPTION
       This is basic support for internationalization of HTTP information. Currently it just provides response bodies for HTTP errors.

SUPPORT
