NAME
       odcs-frontend - manual page for odcs-frontend 0.3.4

SYNOPSIS
       odcs-frontend [OPTIONS]

DESCRIPTION
              Runs the Flask app with the HTTPS settings configured in config.py

OPTIONS
       -h, --host TEXT
              Bind to this address

       -p, --port INTEGER
              Listen on this port

       -d, --debug
              Debug mode

       --help Show this message and exit.
