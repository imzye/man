NAME
       tsspolicycphash - Runs TPM2 policycphash

DESCRIPTION
       policycphash

       Runs TPM2_PolicyCpHash

       -ha    policy session handle

       -cp    cpHash file

       -se[0-2] session handle / attributes (default NULL)

       01     continue

       20     command decrypt
