NAME
       lprint.conf - printer configuration file.

DESCRIPTION
       The  ~/.lprint.conf  (per-user  server)  and  /etc/lprint.conf (root server) files contain a list of the printer queues configured using the lprint-add(1), lprint-default(1), lprint-delete(1), and
       lprint-modify(1) commands along with a history of print jobs submitted using the lprint-submit(1) command.  The files are not intended to be hand-edited and cannot be changed while the  server  is
       running as those changes will be lost.

SEE ALSO
       lprint(1), lprint-add(1), lprint-default(1), lprint-delete(1), lprint-modify(1), lprint-server(1), and lprint-submit(1).
