NAME
       slugify - Python slugify application that handles unicode

SYNOPSIS
       slugify TEXTTOSLUGIFY

DESCRIPTION
       slugify is a Python slugifying application that handles unicode.

EXAMPLES
       "slugify This is a test ---" returns "this-is-a-test"
