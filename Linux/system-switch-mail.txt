NAME
       system-switch-mail - The Mail Transport Agent Switcher

SYNOPSIS
       system-switch-mail

DESCRIPTION
       system-switch-mail is the Mail Transport Agent Switcher.  It enables users to easily switch between various Mail Transport Agent that they have installed.
