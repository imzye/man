NAME
       gyrus - Cyrus-IMAP administration program for GNOME

SYNOPSIS
       gyrus [OPTIONS]...

DESCRIPTION
       This manual page documents briefly the gyrus program, a GTK/GNOME tool to administer Cyrus-IMAP servers.

OPTIONS
       gyrus accepts only the standard GApplication/GTK+ options.  Try “gyrus --help-all” to obtain more information about these options.
