NAME
       imv-msg - Utility for sending commands to a running imv instance

DESCRIPTION
       imv-msg is a tool to simplify the sending of commands to a running instance of imv. Given an instanceâ€™s pid it opens the corresponding unix socket and sends the provided command.

SYNOPSIS
       imv-msg <pid> <command>
