NAME
       chronicle-ping - Send outgoing PING requests for new pages.

SYNOPSIS
         Help Options

           --help        Show a brief help overview.
           --version     Show the version of this script.

         Mandatory Options

           --url         The URL of the entry to ping.
           --service     The URL of the service to ping.

         Misc Options

           --name        The name of the blog.

ABOUT
       This script is designed to receive an URL as its argument, and make an outgoing ping for it.

       If a service URL (pointing to an XML-RPC end-point) is specified it will be used, otherwise the ping will be made to a number of default sites.
