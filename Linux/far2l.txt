FAR2L(1)                                                                                Linux fork of FAR Manager v2                                                                               FAR2L(1)

НАИМЕНОВАНИЕ
       far2l - Двухпанельный файловый менеджер, работающий в текстовом режиме, со встроенным терминалом и другими полезными функциями.

СИНТАКСИС
       far2l [ОПЦИИ] [-cd АПУТЬ -cd ППУТЬ]]
       far2ledit [ОПЦИИ] [ФАЙЛ]

ОПИСАНИЕ
       far2l является Linux-форком файлового менеджера FAR Manager v2 для Unix-подобных операционных систем.

ОПЦИИ
       -h     Показать краткую справку и закончить работу.

       -a     Отключить отображение символов с кодами 0 - 31 и 255.

       -ag    Отключить отображение псевдографики с кодами > 127.

       -an    Полностью отключить отображение псевдографических символов.

       -co    Заставляет FAR2L загружать плагины только из кеша.

       -cd АПУТЬ|ППУТЬ
              Изменить директорию панели на указанный путь.
               АПУТЬ - путь к директории (или файлу, или архиву, или команде с префиксом) для активной панели.
               ППУТЬ - путь к директории (или файлу, или архиву, или команде с префиксом) для пассивной панели.

       -m     Не загружать макросы.

       -ma    Не выполнять макросы автозапуска.

       -u ПОЛЬЗОВАТЕЛЬ OR /ПОЛНЫЙ/ПУТЬ
              Позволяет использовать раздельные настройки для различных ПОЛЬЗОВАТЕЛей или расположений в файловой системе по /ПОЛНОМУ/ПУТИ (перекрывает значение переменной среды FARSETTINGS).

       -v ФАЙЛ
              Просмотреть указанный ФАЙЛ.

       -v - КОМАНДА
              Выполнить указанную КОМАНДУ и открыть просмотрщик для её вывода.

       -e[СТРОКА[:ПОЗИЦИЯ]] [ФАЙЛ]
              Указанный ФАЙЛ (или новый пустой файл) открыть в редакторе.  Дополнительно можно указать СТРОКУ и ПОЗИЦИЮ в строке, устанавливаемые после запуска редактора.

       -e[СТРОКА[:ПОЗИЦИЯ]] - КОМАНДА
              Выполнить указанную КОМАНДУ и открыть просмотрщик для её вывода.  Дополнительно можно указать СТРОКУ и ПОЗИЦИЮ в строке, устанавливаемые после запуска редактора.

       -set:PARAMETER=VALUE
              Изменение значения конфигурационного параметра. См. far:config
              Пример: far2l -set:Language.Main=English -set:Screen.Clock=0 -set:XLat.Flags=0xff -set:System.FindFolders=false

ПАРАМЕТРЫ РЕЖИМОВ ИНТЕРФЕЙСА (BACKEND-SPECIFIC OPTIONS)
       --tty  Запускать far2l сразу в режиме TTY вместо автоматического определения возможности работы в режиме GUI.

       --notty
              Не использовать терминальный режим при невозможности использовать GUI режим.

       --norgb
              Не использовать полноцветную (24-битную) палитру цветов

       --nodetect, --nodetect=[x|xi][f][w][a][k]
              По  умолчанию far2l пытается на запуске определить, не работает ли он в терминале другого far2l. В этом случае far2l автоматически использует режим TTY с расширениями терминала far2l (f). В
              случае отсутствия таких расширений терминала, far2l проверяет наличие доступа к X11 сессии или Xi расширению и использует их для улучшения возможностей интерфейса, если был собран с  опцией
              TTYX.   Кроме  того,  по  умолчанию  используются  и  доступны  для  отключения  w=win32,  a=apple  iTerm2,  k=kovidgoyal's  kitty  input modes.  Данный ключ выключает всю или перечисленную
              функциональность, предотвращая авто определение наличия расширений и разрешает к использованию лишь базовые возможности обычного терминала в TTY режиме.

       --mortal
              Завершить работу вместо перехода в фоновый режим при получении сигнала SIGHUP (по умолчанию, если в Linux TTY).

       --immortal
              Перейти в фоновый режим при получении сигнала SIGHUP (по умолчанию, если не в Linux TTY).

       --x11  force GUI backend to run on X11

       --wayland
              force GUI backend to run on Wayland

       --ee=N Срок действия ESC в мсек (по умолчанию 100, укажите 0 для отключения), чтобы избежать необходимости двойного нажатия ESC (действительно только в режиме TTY без расширений FAR2L).

       --primary-selection
              Использовать PRIMARY selection вместо CLIPBOARD X11 selection. Этот аргумент применим только для far2l, который работает в режиме WX.

       --maximize
              force maximize window upon launch (only for GUI backend)

       --nomaximize
              dont maximize window upon launch even if its has saved maximized state (only for GUI backend)

       --clipboard=SCRIPT
              use external clipboard handler script that implements get/set text clipboard data via its stdin/stdout

ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ
       FARSETTINGS
              Переопределяет расположение пользовательских настроек и конфигурационных файлов (см. параметр -u).

       FAR2L_ARGS
              Все параметры (кроме -h и -u) также могут быть установлены через переменную окружения FAR2L_ARGS
              (например: export FAR2L_ARGS="--tty --nodetect" и затем просто far2l для принудительного запуска только режима TTY).

ПОЛНАЯ СПРАВКА
