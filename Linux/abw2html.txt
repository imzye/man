NAME
       abw2html - manual page for abw2html 0.1.3

SYNOPSIS
       abw2html [OPTION] INPUT

DESCRIPTION
       `abw2html' converts AbiWord documents to HTML.

OPTIONS
       --help show this help message

       --version
              show version information
