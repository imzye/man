NAME
       awardeco - Decompress flashfiles equipped with an AWARD BIOS

SYNOPSIS
       awardeco filename [OPTIONS]

DESCRIPTION
       awardeco is a program which can decompress BIOS images which contain an AWARD BIOS.

OPTIONS
       -l List BIOS structure.

       -la List all BIOS structure elements.

       -x Extract BIOS modules.

       -xa Try to find all module hooks and unpack them.

       -xr This option is intended to cut and paste the packed module as is into a file.

       -h Show help text and exit.
