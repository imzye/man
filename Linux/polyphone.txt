NOM
       polyphone - éditeur de banque de sons

SYNOPSIS
       polyphone
       polyphone [FICHIER]
       polyphone [FICHIER_1] [FICHIER_2] ...
       polyphone -1 [-i CHEMIN_DE_FICHIER_EN_ENTRÉE] [-d RÉPERTOIRE_EN_SORTIE] [-o NOM_EN_SORTIE]
       polyphone -2 [-i CHEMIN_DE_FICHIER_EN_ENTRÉE] [-d RÉPERTOIRE_EN_SORTIE] [-o NOM_EN_SORTIE] [-c CONFIG]
       polyphone -3 [-i CHEMIN_DE_FICHIER_EN_ENTRÉE] [-d RÉPERTOIRE_EN_SORTIE] [-o NOM_EN_SORTIE] [-c CONFIG]

DESCRIPTION
       polyphone  fournit une interface simple et efficace pour créer et éditer des fichiers .sf2 . Ce logiciel incorpore des outils pour faciliter et automatiser l'édition de différents paramètres, ren‐
       dant possible la gestion de grandes quantités de données.

       Les formats sf2, sf3, sfz (import / export) et sfArk (import uniquement) sont supportés. Les conversions sont possibles en ligne de commande.

       Davantage d'informations peuvent être trouvées à https://www.polyphone-soundfonts.com .

OPTIONS
       FICHIER
              Spécifie le fichier à ouvrir quand polyphone démarre. Plusieurs fichiers peuvent être chargés s'ils sont séparés par des espaces. Les formats sf2 (version 2.01 ou 2.04), sf3, sfz  et  sfArk
              (version 1 ou 2) sont supportés.

       -1     Utilisez polyphone pour convertir un fichier dans le format sf2.

       -2     Utilisez polyphone pour convertir un fichier dans le format sf3.

       -3     Utilisez polyphone pour convertir un fichier dans le format sfz.

       [-i CHEMIN_DU_FICHIER_EN_ENTRÉE]
              Chemin d'entrée à convertir. Le format du fichier d'entrée doit être sf2, sf3, sfz ou sfArk.

       [-d RÉPERTOIRE_DE_SORTIE]
              Le répertoire de sortie dans lequel le fichier d'entrée sera converti. Par défaut, c'est le même répertoire que celui du fichier d'entrée.

       [-o NOM_DE_SORTIE]
              Le nom de sortie du fichier converti. L'extension sera ajoutée automatiquement suivant la conversion. Par défaut, c'est le même nom que le fichier d'entrée.

       [-c CONFIG]
              Configuration de la conversion, le contenu étant dépendant du type de conversion. -r Supprime la configuration existante.
               * conversion sf2
              Pas de configuration pour cette conversion.
               * conversion sf3
              La configuration est faite avec un caractère indiquant la qualité de la compression. '0' pour basse, '1' pour moyenne, et '2' pour haute qualité. Le défaut est de '1'.
               * conversion sfz
              La  configuration  est faite de trois caractères. Le premier caractère est '1' si chaque ensemble doit être préfixé par son numéro d'ensemble, '0' sinon. Le deuxième caractère est '1' si un
              répertoire par banque doit être créé, '0' sinon. Le troisième caractère est '1' si la classification General MIDI doit être utilisée pour trier les ensembles, '0' sinon. Par  défaut,  c'est
              '000'.

EXEMPLES
        * Conversion depuis sfArk vers sf2 :
       polyphone -1 -i /chemin/vers/le/fichier.sfArk
        * Conversion depuis sf2 vers sf3 (bonne qualité) :
       polyphone -2 -i /chemin/vers/le/fichier.sf2 -c 2
        * Conversion depuis sf3 vers sfz (sous-répertoire pour la banque et classification GM) :
       polyphone -3 -i /chemin/vers/le/fichier.sf3 -c 011
