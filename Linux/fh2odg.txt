NAME
       fh2odg - manual page for fh2odg 0.9.6

SYNOPSIS
       fh2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `fh2odg' converts FreeHand documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
