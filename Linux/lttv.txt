NAME
       lttv - The Linux Trace Toolkit Viewer

DESCRIPTION
       lttv is a simple script that starts lttv.real(1) with no modules. This is useful for running lttv in batch mode.

SEE ALSO
       lttv-gui(1), lttv.real(1).
