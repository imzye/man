NAME
       asonq - Interactive ASON evaluation

SYNOPSIS
       asonq

DESCRIPTION
       asonq is an interactive shell for evaluating ASON expressions. It supports standard ASON expressions as well as assignment to variables with the := operator.

COMMANDS
       Commands begin with \ to differentiate them from ASON expressions.

       \quit  Exit the asonq shell.
