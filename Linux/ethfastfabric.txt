NAME
       ethfastfabric

       Starts the top-level Intel(R) Ethernet Fabric Suite FastFabric Text-based User Interface (TUI) menu to enable setup and configuration.

       Refer to the Intel(R) Ethernet Fabric Suite FastFabric User Guide

       for details on this tool.

Syntax
       ethfastfabric

Options
       None.

Example
       #ethfastfabric

        Intel Ethernet FastFabric Tools

        Version:  X.X.X.X.X

            1) Host Setup

            2) Host Verification/Admin

            X) Exit (or Q)
