NAME
       Dist::Zilla::Plugin::InstallGuide - Build an INSTALL file

VERSION
       version 1.200014

SYNOPSIS
       In "dist.ini":

           [InstallGuide]

DESCRIPTION
       This plugin adds a very simple INSTALL file to the distribution, telling the user how to install this distribution.

       You should use this plugin in your Dist::Zilla configuration after "[MakeMaker]" or "[ModuleBuild]" so that it can determine what kind of distribution you are building and which installation
       instructions are appropriate.

METHODS
   gather_files
       Creates the INSTALL file.

   munge_files
       Inserts the appropriate installation instructions into INSTALL.
