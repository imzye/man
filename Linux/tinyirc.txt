NAME
       tinyirc - A tiny IRC client

SYNOPSIS
         tinyirc [-d] nick [server [port]]

DESCRIPTION
       A stripped down IRC client, providing basic IRC commands. Please note that this is bare bones IRC client which has no editing capabilities other than backspace to correct typing. No command
       history (cursor up, down) or word editing (cursor left, left).

OPTIONS
       -h, --help
           Display short help.

       -d, --dumb
           Dumb mode. Do not use the curses interface; simply write to stdout and read commands from stdin.

EXAMPLES
       Join Freenode:

           tinyirc mynick irc.freenode.org

ENVIRONMENT
       IRCSERVER
           Sets the default server to connect to. Format is HOST[:PORT].

       IRCNICK
           Sets the default nickname to use.

FILES
       None.

STANDARDS
       RFC 1459
