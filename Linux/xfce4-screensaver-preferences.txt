NAME
       xfce4-screensaver-preferences - Configure Xfce Screensaver

SYNOPSIS
       xfce4-screensaver-preferences [ OPTIONS]

DESCRIPTION
       Manages preferences for the Xfce Screensaver.

OPTIONS
       This program only accepts the standard GTK options.
