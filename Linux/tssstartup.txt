NAME
       tssstartup - Runs TPM2 startup

DESCRIPTION
       startup

       Runs TPM2_Startup

       [-c    startup clear (default)]

       [-s    startup state]

       [-st   run TPM2_SelfTest]

       [-sto  run only TPM2_SelfTest (no startup)]

       [-loc  locality]
