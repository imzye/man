NAME
       webcamd - Take a image from your webcam

SYNOPSIS
       webcamd [ start | stop | refresh ]

DESCRIPTION
       This manual page documents briefly the webcamd daemon.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       webcamd is a program that take images from your webcam and put it on a ftp server on a local (web) directory

OPTIONS
       start  Start the daemon.

       stop   Stop the daemon.

       refresh
              Take a shot !
