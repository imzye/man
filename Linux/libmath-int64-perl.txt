NAME
       Math::UInt64 - Manipulate 64 bit unsigned integers from Perl

DESCRIPTION
       Math::UInt64 is just an alias for Math::Int64, the only reason it exists as an independent package is to allow Storable to load it on demand.

SEE ALSO
       Math::Int64.
