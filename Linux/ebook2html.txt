NAME
       ebook2html - manual page for ebook2html 0.1.3

SYNOPSIS
       ebook2html [OPTION] FILE

DESCRIPTION
       `ebook2html' converts e-books to HTML.  Supported formats include FictionBook 2, BBeB and various Palm-based formats.

OPTIONS
       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://sourceforge.net/p/libebook/tickets/> or <https://bugs.documentfoundation.org/>.
