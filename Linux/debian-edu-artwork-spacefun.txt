NAME
       update-debian-edu-artwork-spacefun - Enable/disable Debian Edu Spacefun theme system-wide

SYNOPSIS
       update-debian-edu-artwork-spacefun { configure | remove }

DESCRIPTION
       update-debian-edu-artwork-spacefun  configure  enables  the Debian Edu Spacefun theming (as shipped with Debian Edu Trixie) system-wide (several login managers, KDE theme, background for all desk‚Äê
       tops, GRUB boot loader, etc. pp.).

       update-debian-edu-artwork-spacefun remove disables the Debian Edu themes and switches back to Debian's default themes (from package desktop-base).
