NAME
       sd2odf - manual page for sd2odf 0.9.6

SYNOPSIS
       sd2odf [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `sd2odf' converts binary StarOffice documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output

       -x, --extension
              print extension for output (odt, ods, odp, odg)
