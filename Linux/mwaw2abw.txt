NAME
       mwaw2abw - manual page for mwaw2abw 0.9.6

SYNOPSIS
       mwaw2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `mwaw2abw' converts legacy Mac Documents to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
