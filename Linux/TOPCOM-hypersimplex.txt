NAME
       TOPCOM-hypersimplex - compute the symmetry generators of a hypersimplex

SYNOPSIS
       hypersimplex K D

DESCRIPTION
       Compute the vertices and symmetry generators of the Kth hypersimplex in dimension D.  The output is written to stdout.

EXIT STATUS
       0 if the correct number of command line arguments are given.

       1 otherwise.
