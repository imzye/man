NAME MouseX::Getopt::Meta::Attribute::NoGetopt - Optional meta attribute for ignoring params
SYNOPSIS
         package App;
         use Mouse;

         with 'MouseX::Getopt';

         has 'data' => (
             metaclass => 'NoGetopt',  # do not attempt to capture this param
             is        => 'ro',
             isa       => 'Str',
             default   => 'file.dat',
         );

DESCRIPTION
       This is a custom attribute metaclass which can be used to specify that a specific attribute should not be processed by "MouseX::Getopt". All you need to do is specify the "NoGetopt" metaclass.

         has 'foo' => (metaclass => 'NoGetopt', ... );
