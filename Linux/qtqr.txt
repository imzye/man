NAME
       qtqr - QtQR

SYNOPSIS
       qtqr [file list...]

DESCRIPTION
       QtQR is a Qt based software that helps you to generate QR Codes easily, scan an image file for QR Codes and decode them or use your webcam to scan a printed one.

        This command will launch QtQR.

BUGS
       If you find any bugs, please report them at http://bugs.launchpad.net/qr-tools or use reportbug tool to report them on Debian Bug Tracking System.

DEVELOPMENT
       The development of qtqr is managed at http://launchpad.net/qr-tools, where you can find the source code and report bugs.
