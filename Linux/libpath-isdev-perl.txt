NAME
       Path::IsDev::Heuristic::VCS::Git - Determine if a path contains a .git repository

VERSION
       version 1.001003

METHODS
   "dirs"
       Directories relevant to this heuristic:

           .git

   "matches"
       Return a match if any children of "path" exist called ".git" and are directories
