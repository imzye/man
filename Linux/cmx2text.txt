NAME
       cmx2text - manual page for cmx2text 0.1.7

SYNOPSIS
       cmx2text [OPTION] FILE

DESCRIPTION
       `cmx2text' converts Corel Binary Metafiles to plain text.

OPTIONS
       --help show this help message

       --version
              show version information and exit
