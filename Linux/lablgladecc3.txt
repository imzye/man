NAME
       lablgladecc3 - manual page for lablgladecc3 3.1.1

DESCRIPTION
       lablgladecc2 [<options>] [<file.glade>]

       -test  check lablgladecc (takes no input)

       -embed embed input file into generated program

       -debug add debug code

       -root <widget>
              generate only a wrapper for <widget> and its children

       -hide-default
              hide widgets with default names like 'label23'

       -help  Display this list of options

       --help Display this list of options
