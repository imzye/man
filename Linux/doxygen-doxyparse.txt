NAME
       doxyparse - parse and dumps information about the code

SYNOPSIS
       doxyparse [source file...]

DESCRIPTION
       Parses source code and dumps the dependencies between the code elements.

SEE ALSO
       doxygen(1), doxytag(1), doxywizard(1).
