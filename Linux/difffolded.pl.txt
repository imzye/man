NAME
       difffolded.pl - manual page for difffolded.pl 1.0

DESCRIPTION
       USAGE:   ./difffolded.pl  [-hns]  folded1  folded2  |  flamegraph.pl  >
       diff2.svg

       -h     # help message

       -n     # normalize sample counts

       -s     # strip hex numbers (addresses)

       See stackcollapse scripts for generating folded files.   Also  consider
       flipping the files and hues to highlight reduced paths: ./difffolded.pl
       folded2 folded1 | ./flamegraph.pl --negate > diff1.svg
