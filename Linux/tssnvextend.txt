NAME
       tssnvextend - Runs TPM2 nvextend

DESCRIPTION
       nvextend

       Runs TPM2_NV_Extend

       -ha    NV index handle

       -pwdn  password for NV index (default empty)

       -ic    data string

       -if    data file

       -se[0-2] session handle / attributes (default PWAP)

       20     command decrypt

       01     continue
