NAME
       expeyes-progman-jr-doc - program to display the user manual of expeyes and expeyes-junior.

SYNOPSIS

       expeyes-progman-jr-doc [lang]

       expeyes-junior-doc [lang]

DESCRIPTION
       expeyes-progman-jr-doc launches a viewer for the programmer's manual of expEYES Jr.

       The preferred language is detected automatically.

OPTIONS
       lang
           Enforces the language lang as preferred. For example, lang may be 'en' or 'fr'.

ENVIRONMENT
       LANGUAGE, LANG, LC_ALL
           If defined, these variables are taken in account to find automatically the preferred language. These variables are taken in account only when the optional argument lang is not given.
