NAME
       esgenerate - ECMAScript code generator

SYNOPSIS
       esgenerate [options] file...

DESCRIPTION
       esgenerate is an ECMAScript (also popularly known as JavaScript) code generator from Mozilla's Parser API AST.

OPTIONS
       -c, --config String configuration json for escodgen

SEEALSO
       escodgen(1)
