NAME
       unmo3 - uncompress and extract samples from MO3 modules

SYNOPSIS
       unmo3 [-d debuglevel] [-a parselevel] [-h outheaderfile]

       [-s <all|sample_number>] [-v patternchannel] [-o ] filename.mo3

DESCRIPTION
       This manual page documents briefly the unmo3 command.

       unmo3 is a program that uncompresses, and extracts samples from mo3 files.

OPTIONS
       A summary of options is included below.

       -a     Set parselevel.

       -d     Set debuglevel.

       -h     Output filename for header.

       -s     Select sample(s).

       -v     Display voice.

       -o     Output track.

SEE ALSO
       opencubicplayer(1), schism(1).
