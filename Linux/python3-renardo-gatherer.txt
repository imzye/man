NAME
       renardo_gatherer - download assets (samples, synthdefs, etc) for Renardo

SYNOPSIS
         renardo_gatherer

DESCRIPTION
       renardo_gatherer is the asset collector for Renardo Python livecoding environment, it collects (download) assets files (samples packs, synthdefs, etc) for Renardo.

SEE ALSO
       renardo(1), FoxDotEditor(1), sclang(1), https://foxdot.org, https://renardo.org
