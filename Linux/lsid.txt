NAME
       lsid - displays the current Slurm version number, the cluster name, and the master host name

SYNOPSIS
       lsid

DESCRIPTION
       The lsid command displays the current Slurm version number, the cluster name, and the master host name.

OPTIONS
       -h | --help
           brief help message

       --man
           full documentation

EXIT STATUS
       On success, lsid will exit with a value of zero. On failure, lsid will exit with a value greater than zero.
