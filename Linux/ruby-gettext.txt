NAME
       rxgettext - tool to extract translatable messages from Ruby scripts.

SYNOPSIS
       rxgettext input.rb -o output.pot

DESCRIPTION
       Extract translatable string from given input Ruby script files.

SEE ALSO
       gettext(1), /usr/share/doc/ruby-gettext/README.doc, http://www.yotabanana.com/hiki/ruby-gettext-translate.html
