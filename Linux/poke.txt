NAME
       pokefmt - The poke templating system

SYNOPSIS
       pokefmt [OPTION]... [POKE-CODE]...

DESCRIPTION
       Template system to process embedded code in files.

       -h, --help
              print a help message and exit

       -v, --version
              show version and exit
