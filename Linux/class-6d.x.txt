NAME
       class-6d.x - manual page for class-6d.x 2.11

SYNOPSIS
       class-6d.x [options] [ascii-input-file [ascii-output-file]]

DESCRIPTION
       This is  `bin/class-6d.x', a program for classifying reflexive polytopes

       Options:   -h          print this information

       -f or -
              use as filter; otherwise parameters denote I/O files

       -m*    various types of minimality checks (* ... lvra)

       -p* NAME
              specification of a binary I/O file (* ... ioas)

       -d* NAME
              specification of a binary I/O database (DB) (* ... ios)

       -r     recover: file=po-file.aux, use same pi-file

       -o[#]  original lattice [omit up to # points] only

       -s*    subpolytopes on various sublattices (* ... vphmbq)

       -k     keep some of the vertices

       -c, -C check consistency of binary file or DB

       -M[M]  print missing mirrors to ascii-output

       -a[2b], -A
              create binary file from ascii-input

       -b[2a], -B
              ascii-output from binary file or DB

       -H*    applications related to Hodge number DBs (* ...cstfe)
