NAME
       mwaw2text - convert Mac text document into plain text

SYNOPSIS
       mwaw2text [OPTION] <Mac Document>

OPTIONS
       -i     Display document metadata instead of the text

       -h     Shows this help message

       -o file.txt
              Define the output[default stdout]

       -v:    Output mwaw2text version
