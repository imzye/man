NAME
       sd2-nfs2ac - Car model converter from 'Need for Speed' to Ac3d

SYNOPSIS
       sd2-nfs2ac

DESCRIPTION
       This manual describes the nfs2ac application that is shipped with Speed Dreams 2.

OPTIONS
       This program does not use arguments.
