NAME
       slconsole - utility to connect to a sooperlooper engine

SYNOPSIS
       slconsole [options...]

OPTIONS
       -H host,  --connect-host=host
              connect to sooperlooper engine on given host (default is localhost)

       -P <num>, --connect-port=<num>
              connect to sooperlooper engine on given port (default is 9951)

       -h , --help
              this usage output

       By default, the gui will try to connect to an engine running at the given (or default) host and port.

SEE ALSO
       sooperlooper(1), slregister(1)
