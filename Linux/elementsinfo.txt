NAME
       elementsinfo - Periodic table with Atomic Constants used by PyMca

SYNOPSIS
       elementsinfo

DESCRIPTION
       Periodic table displaying the shell constants and X-ray emission branching ratios as a function of photon excitation energy used by PyMca.

SEE ALSO
       xraylib
