NAME
       tk707 - a drum machine

SYNOPSIS
       tk707 [options]

DESCRIPTION
       tk707  emulates  the  Roland TR-707 Rhythm Composer.  It can be used with external MIDI hardware, a capable sound card or Timidity plus any sound card supported by ALSA.  Output can also be saved to a
       file.  The complete documentation in info and html formats are available in tk707.info and tk707.html, respectively.

EXAMPLES
       tk707 -l
       tk707 -p 64:0
       tk707 -p 128:0 -display eve.imag.fr:0.0

OPTIONS
       -l     List the available ALSA sound output ports.

       -p port
              Use this port

       -small | -medium | -normalsize
                      Set the size of the window, since some screens are too small
                      for the whole window.

       -display
              Display to use (example: myhost:0.0).

       -use Window id
              Id of window in which to embed application. The Window id can be obtained e.g. by using xwininfo(1).

       -geometry
              Initial geometry for window

       -name  Name to use for application.

       -sync  Use synchronous mode for display server.

       -visual
              Visual for main window.

       -colormap
              Colormap for main window.

       -h | -help
              Print very short help.

SEE ALSO
