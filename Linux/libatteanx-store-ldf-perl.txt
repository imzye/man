NAME
       AtteanX::Query::AccessPlan::LDF - An access plan for Linked Data Fragments

DESCRIPTION
       This provides the implementation of a Moo::Role that serves to wrap any "access_plan" in query planning. An access plan introduces a plan object for a triple or quad pattern, in this case a
       AtteanX::Plan::LDF::Triple object.
