NAME
       mp2eps - Convert METAPOST to encapsulated PostScript

SYNOPSIS
       mp2eps file1 ...

DESCRIPTION
       mp2eps converts METAPOST output to encapsulated PostScript. It uses TeX and epsf.tex.

SEE ALSO
       tex(1).
