NAME
       qxp2epub - manual page for qxp2epub 0.9.6

SYNOPSIS
       qxp2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `qxp2epub' converts QuarkXPress documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit
