NAME
       onak-mail.pl - Process OpenPGP keyserver emails.

SYNOPSIS
       onak-mail.pl

DESCRIPTION
       onak-mail.pl  is  the mail processing component of the onak OpenPGP keyserver. It processing incoming mails from other keyservers and users using the email interface to the keyserver. It should be
       run from an aliases file, a .forward or via procmail and takes no options.

FILES
       /etc/onak.ini              default configuration file
