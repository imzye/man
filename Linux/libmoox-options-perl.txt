NAME
       MooX::Options::Manual::NamespaceClean - Use namespace::clean with MooX::Options

DESCRIPTION
       MooX::Options export private methods to keep track of all your options.  You need to keep some of them to make it work. In addition to the advises of Moo, some of the imported methods needs to be
       visible ...

USAGE
         use MooX::Options;
         use namespace::clean -except => [qw/_options_data _options_config/];

SEE ALSO
       "CLEANING UP IMPORTS" in Moo
