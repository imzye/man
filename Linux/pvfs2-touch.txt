NAME
       pvfs2-touch – create files

SYNOPSIS
       pvfs2-touch [-lr] pvfs2_filename[s]

DESCRIPTION
       The  pvfs2-touch utility creates empty files.  It does not update modi‐
       fication and access times on extant files as touch(1) does.

       The options are as follows:

       -l     Use list layout.

       -r     Use random layout.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set the OrangeFS debug mask.  Possible masks are  documented  in
              pvfs2-set-debugmask(1).

       PVFS2TAB_FILE
              If set, the full pathname for an alternate pvfs2tab file

FILES
       /etc/pvfs2tab

EXAMPLES
       Create the empty file /mnt/foo.

             pvfs2-touch /mnt/foo

BUGS
