NAME
       tssunseal - Runs TPM2 unseal

DESCRIPTION
       unseal

       Runs TPM2_Unseal

       -ha    sealed data item handle

       [-pwd  password sealed data item (default empty)]

       [-of   output data (default do not save)]

       -se[0-2] session handle / attributes (default PWAP)

       01     continue

       40     response encrypt
