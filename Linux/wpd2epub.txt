NAME
       wpd2epub - manual page for wpd2epub 0.9.6

SYNOPSIS
       wpd2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `wpd2epub' converts WordPerfect documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --password PASSWORD
              set password to open the file

REPORTING BUGS
