NAME
       matekbd-keyboard-display - Preview keyboard layouts on MATE desktop

SYNOPSIS
       matekbd-keyboard-display [OPTION]

DESCRIPTION
       matekbd-keyboard-display is an application that allows you to preview keyboard layouts on MATE desktop.

       It uses the libmatekbd library, similarly to gkbd-keyboard-display and libgnomekbd.

OPTIONS
       -g     Specify the layout group you wish to preview.  If you omit this, the currently selected layout will be used.

EXIT VALUES
       0 The command was invoked successfully.

       1 There was an initialisation problem.

       2 An invalid group number was supplied.
