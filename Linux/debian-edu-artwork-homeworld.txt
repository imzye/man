NAME
       update-debian-edu-artwork-homeworld - Enable/disable Debian Edu Homeworld theme system-wide

SYNOPSIS
       update-debian-edu-artwork-homeworld { configure | remove }

DESCRIPTION
       update-debian-edu-artwork-homeworld  configure  enables  the  Debian  Edu Homeworld theming (as shipped with Debian Edu Bullseye) system-wide (several login managers, KDE theme, background for all
       desktops, GRUB boot loader, etc. pp.).

       update-debian-edu-artwork-homeworld remove disables the Debian Edu themes and switches back to Debian's default themes (from package desktop-base).
