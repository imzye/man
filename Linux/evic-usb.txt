NAME
       evic-usb - manual page for evic-usb 0.1

SYNOPSIS
       evic-usb [OPTIONS] COMMAND [ARGS]...

DESCRIPTION
              A USB programmer for devices based on the Joyetech Evic VTC Mini.

OPTIONS
       --help Show this message and exit.

   Commands:
       dump-dataflash
              Write device data flash to a file.

       fmcread
              Write device flash memory to a file.

       hidcmd Send a HID command to the device.

       reset  Resets the device.

       reset-dataflash
              Reset device data flash.

       screenshot
              Take a screenshot.

       time   Sets the device date/time to now.

       upload Upload an APROM image to the device.

       upload-ldrom
              Upload an LDROM image to the device.

       upload-logo
              Upload a logo to the device.
