NAME
       compose-print-essentials - print basic information about a compose

SYNOPSIS
       compose-print-essentials [OPTIONS...]  COMPOSE_PATH

DESCRIPTION
       compose-print-essentials  loads  metadata  from  a compose and prints a
       summary of basic information: which kernel version is used,  what  con‚Äê
       tainers were built and to which registries they were pushed, and a list
       of other images that were created in the compose.

OPTIONS
       -h, --help
              Print help and exit.

EXIT CODE
       Exit code is always 0 on success and non-zero if there  was  a  problem
       loading the compose.
