NAME
       xprintidle - utility printing user's idle time in X

SYNOPSIS
       xprintidle [OPTION...]

DESCRIPTION
       xprintidle is a utility that queries the X server for the user's idle time and prints it to stdout (in milliseconds).

OPTIONS
   Generic Program Information
       -h ,  --help
              Output a usage message and exit.

       -H ,  --human-readable
              Output the idle time in a human-readable format.

       -v ,  --version
              Output the version number and exit.
