NAME
       s3d_x11gate - 3d gateway for x11 desktops

SYNOPSIS

       s3d_x11gate [-h|--help] [--s3d-url url]

DESCRIPTION
       s3d_x11gate(1) is a 3D gateway for x11 desktops.

OPTIONS
       This program follows the usual GNU command line syntax. A summary of the options supported by s3d_x11gate is below.

       -h, --help
           Show summary of options and exit.

       --s3d-url url
           skip S3D environment and connect to this url
