NAME
       orthorobot - Ortho Robot v 1.1.1

SYNOPSIS
       A perspective based puzzle game

DESCRIPTION
       Literally bridging the gap between 2D and 3D games, Ortho Robot is a perspective based puzzle game, where you flatten the view to move across gaps. This game is made with LOVE. You can find more games
       and information about the author(s) at http://stabyourself.net

OPTIONS
       -h, --help Show manpage.

       love(6)
