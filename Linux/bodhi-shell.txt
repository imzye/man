NAME
       bodhi-shell - run python shell initialized with Bodhi models

SYNOPSIS
       bodhi-shell

DESCRIPTION
       bodhi-shell  is  used to run a Python shell initialized with Bodhi mod‐
       els.

OPTIONS
       --help
          Show help text and exit.

       --version
          Show version and exit.
