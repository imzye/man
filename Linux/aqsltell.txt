NAME
       aqsltell - shader inspection tool from Aqsis Renderer, a 3D rendering solution adhering to the RenderMan(R) standard

SYNOPSIS
       aqsltell <shadername>

DESCRIPTION
       -h, -help
              Print this help and exit

       -version
              Print version information and exit

       -shaders=string
              Override the default shader searchpath(s) [/usr/local/aqsis/share/shaders]
