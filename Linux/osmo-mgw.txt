NAME
       OsmoMGW - osmo-mgw

DESCRIPTION
       Osmocom  Media  Gateway,  to  manage, connect and optionally transcode voice streams between different network elements such as BTSs and external entities like SIP. It is typically co-located with
       both OsmoBSC and OsmoMSC and controlled by them via IETF MGCP (Media Gateway Control Protocol).

       https://osmocom.org/projects/osmo-mgw

OPTIONS

       -h --help is printing this text.

       -c --config-file filename The config file to use.

       -s --disable-color

       -D --daemonize Fork the process into a background daemon

       -V --version Print the version number

   VTY reference generation:
       --vty-ref-mode MODE
              VTY reference generation mode (e.g. 'expert').

       --vty-ref-xml
              Generate the VTY reference XML output and exit.
