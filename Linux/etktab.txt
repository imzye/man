NAME
       fileconvert-v1-to-v2 - Converts eTktab saved files from v1 to v2

SYNTAX
       fileconvert-v1-to-v2 <filename>

DESCRIPTION
       This program is used to convert saved files from eTktab v1 to the newer v2 format
