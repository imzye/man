NAME
       Log::Any::Adapter::Stderr - Simple adapter for logging to STDERR

VERSION
       version 1.718

SYNOPSIS
           use Log::Any::Adapter ('Stderr');

           # or

           use Log::Any::Adapter;
           ...
           Log::Any::Adapter->set('Stderr');

           # with minimum level 'warn'

           use Log::Any::Adapter ('Stderr', log_level => 'warn' );

DESCRIPTION
       This simple built-in Log::Any adapter logs each message to STDERR with a newline appended. Category is ignored.

       The "log_level" attribute may be set to define a minimum level to log.
