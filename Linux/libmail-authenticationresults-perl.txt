NAME
       Mail::AuthenticationResults::Header::Comment - Class modelling Comment parts of the Authentication Results Header

VERSION
       version 2.20250709

DESCRIPTION
       Comments may be associated with many parts of the Authentication Results set, this class represents a comment.
