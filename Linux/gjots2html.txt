NAME
       gjots2html - Convert a gjotsfile to HTML (on stdout)

SYNOPSIS
       gjots2html { -lc ] [ -t title ] [ -i level } [ gjotsfile ]

DESCRIPTION
       gjots2html converts a gjotsfile into HTML with a table of contents.

       Presently up to 8 levels of table of contents subject tree are support‐
       ed.

       gjotsfile uses m4(1) and awk(1)

OPTIONS
       -l     adds <HR> lines after each section.

       -c     adds a link to the TABLE OF CONTENTS after each section.

       -t title
              adds a title to the page

       -i level
              Creates the table of contents down to that level of heading. The
              default is to include all headings.
