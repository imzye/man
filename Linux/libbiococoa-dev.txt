NAME
       BioCocoa - Bioinformatics framework for GNUstep and Cocoa

SYNOPSIS
       #include <BioCocoa/BCFoundation.h>

DESCRIPTION
       This  manual  page  documents briefly the BioCocoa framework.  This manual page was written for the Debian distribution because the original program does not
       have a manual page.

       BioCocoa is an open source framework for bioinformatics written in Objective-C. It provides Cocoa and GNUstep programmers with a full suite of APIs for  han‐
       dling and manipulating biological sequences.

SEE ALSO
       GNUstep(7), SequenceConverter.app(1)
