NAME
       address6

SYNOPSIS
            address6 mac-address [ipv6-prefix]      address6 ipv4-address [ipv6-prefix]      address6 ipv6-address

       Converts  a  mac or IPv4 address to an IPv6 address (link local if no prefix is given as 2nd option) or, when given an IPv6 address, prints the mac or IPv4 address. Prints all possible variations. Re‚Äê
