NAME
       sd2raw - manual page for sd2raw 0.0.7

SYNOPSIS
       sd2raw [OPTION] INPUT

DESCRIPTION
       `sd2raw' is used to test libstaroffice.

OPTIONS
       -c     display the call graph nesting level

       -h     show this help message

       -p PASSWORD
              set password to open the file

       -v     show version information

REPORTING BUGS
       Report bugs to <https://github.com/fosnola/libstaroffice/issues>.
