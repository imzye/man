NAME
       WrapperFactory - Application wrappers configuration tool

SYNOPSIS
       WrapperFactory

DESCRIPTION
       This  manual  page  documents briefly the WrapperFactory command.  This manual page was written for the Debian distribution because the original program does
       not have a manual page.

       WrapperFactory provides an easy way to create GNUstep app-wrappers of non-GNUstep  applications. It is the most useful in conjunction with GWorkspace   envi‐
       ronment.

SEE ALSO
       GNUstep(7).
