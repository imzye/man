NAME
       dmtcp_srun_helper -- This is a helper program that should not be run by the user.

SYNOPSIS
       dmtcp_srun_helper [OPTIONS] command [command...]

DESCRIPTION
       dmtcp_srun_helper shouldn't be used directly.

OPTIONS
       no options
               This is a helper program. Please do not try to run this program directly.
