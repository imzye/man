NAME
       fstreplace - manual page for fstreplace 1.6.9

DESCRIPTION
       Recursively replaces FST arcs with other FST(s).

              Usage: fstreplace root.fst rootlabel [rule1.fst label1 ...] [out.fst]

       PROGRAM FLAGS:

       --call_arc_labeling: type = string, default = "input"

              Which  labels  to  make non-epsilon on the call arc. One of: "input" (default), "output", "both", "neither" --epsilon_on_replace: type = bool, default = false Call/return arcs are epsilon arcs?
              --return_arc_labeling: type = string, default = "neither" Which labels to make non-epsilon on the return arc. One of: "input", "output",  "both",  "neither"  (default)  --return_label:  type  =
              int64, default = 0 Label to put on return arc
