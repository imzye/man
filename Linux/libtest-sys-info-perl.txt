NAME
       Test::Sys::Info - Centralized test suite for Sys::Info.

VERSION
       version 0.23

SYNOPSIS
           use Test::Sys::Info;
           driver_ok('Windows'); # or Linux, etc.

DESCRIPTION
       This is a centralized test suite for Sys::Info Drivers.

TESTS
   driver_ok OSID
       Tests the driver.
