NAME
       rebuilderd - rebuilderd

SYNOPSIS
       rebuilderd [OPTIONS]

OPTIONS
       -v, --verbose...
              Verbose logging

       --check-config
              Load and print a config

       -c, --config <CONFIG>
              Configuration file path

       -h, --help
              Print help

       -V, --version
              Print version
