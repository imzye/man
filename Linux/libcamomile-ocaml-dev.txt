NAME
       camomilecharmap.de - eine Übersetzungsdatenbank der Zeichentabelle für camomile

SYNOPSIS

       camomilecharmap [-d { Verzeichnis }] [-help | --help] [Datei]

BESCHREIBUNG
       Diese Handbuchseite dokumentiert kurz den Befehl camomilecharmap.

       Diese Handbuchseite wurde für die Debian-Distribution geschrieben, da das Originalprogramm keine Handbuchseite hat.

       camomilecharmap wurde erstellt, um eine Zeichentabellendatei in eine .mar-Datei umzuwandeln. Dies wandelt eine reguläre Zeichentabellendatei in das interne Datenbankformat von camomile um.

       -d Verzeichnis
           Ausgabeverzeichnis

       -help, --help
           zeigt eine Hilfenachricht.

       Datei
           Zeichentabellendatei
