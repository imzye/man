NAME
       PRICE - image filtering and manipulation

SYNOPSIS
       PRICE

DESCRIPTION
       This manual page documents briefly the PRICE command.  This manual page was written for the Debian distribution because the original program does not have a manual page.

       PRICE can open several image file formats and apply high-quality filters and other enhancements.
