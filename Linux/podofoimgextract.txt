NAME
       podofoimgextract - Extract all images from a PDF file

SYNOPSIS
       podofoimgextract [inputfile] [outputdirectory]

DESCRIPTION
       podofoimgextract  is  one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. Itw can extract all images from a PDF file into the specified
       output directory.
