NAME
       git-remote-bzr - Git remote support for Bazaar repositories

SYNOPSIS
       git clone bzr::<source-repo> [<destination>]
       git fetch bzr::<source-repo> [<destination>]

DESCRIPTION
       This command provides support for using bzr repositories as Git remotes, through the breezy-git plugin. At the moment it supports cloning from, fetching from and pushing into Bazaar repositories.
       Fetch support is still experimental, and may be slow.
