NAME
       Extract - Extract a text tag from an ICC profile.

DESCRIPTION
       Extract a text tag from an ICC profile

SYNOPSIS
       extractttag [-v] infile.icc outfile

       -v     Verbose

       -t tag Extract this tag rather than default 'targ'

       -c     Extract calibration file from 'targ' tag
