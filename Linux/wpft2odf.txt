NAME
       wpft2odf - manual page for wpft2odf 0.9.6

SYNOPSIS
       wpft2odf [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wpft2odf' converts documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --encoding ENCODING
              set the INPUT encoding. Use --list-encodings to see which encodings can be used.

       --list-encodings
              show the available encodings and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output

       -x, --extension
              print extension for output (odt, ods, odp, odg)
