NAME
       ebsnvme-id - Reads and prints EBS information from NVMe devices

SYNOPSIS
       ebsnvme-id [options] DEVICE

DESCRIPTION
       ebsnvme-id  uses  the NVME Management Interface to read various metadata about NVME-attached EBS storage volumes.  It can be used to retrive the EBS Volume ID or EC2 API device mapping value given
       the local block device name.

OPTIONS
       -v, --volume
              Return volume-id

       -b, --block-dev
              Return block device mapping

       -u, --udev
              Output data in format suitable for udev rules

       -h, --help
              Show summary of options.

SEE ALSO
       nvme(1), ec2-metadata(1).
