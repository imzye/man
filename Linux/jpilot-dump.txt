NAME
       jpilot-dump - A command line tool for dumping jpilot databases.

SYNOPSIS
       jpilot-dump [ options ]

DESCRIPTION
       Dump jpilot databases.

OPTIONS
       +B +M +A +T
              format like date +format.

       -v     displays version and exits.

       -h     displays help and exits.

       -f     displays help for format codes.

       -D     dump DateBook.

       -N     dump appts for today in DateBook.

       -NYYYY/MM/DD
              dump appts on YYYY/MM/DD in DateBook.

       -A     dump Address book.

       -T     dump Todo list as CSV.

       -M     dump Memos.
