NAME
       wpd2raw - manual page for wpd2raw 0.10.3

SYNOPSIS
       wpd2raw [OPTION] FILE

DESCRIPTION
       `wpd2raw' is used to test libwpd.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --password PASSWORD
              try to decrypt password protected document

       --version
              print version and exit

REPORTING BUGS
       Report bugs to <https://sourceforge.net/p/libwpd/tickets/> or <https://bugs.documentfoundation.org/>.
