NAME
       xmonad-sesssion - Starts the XMonad window manager as a session

SYNOPSIS
       xmonad-session arguments

DESCRIPTION
       xmonad-session is a simple shell script that starts sources the content of ~/.xmonad/xmonad-session-rc (if that file exists) and then starts XMonad.

       Any command line arguments supplied will be passed on to XMonad.
