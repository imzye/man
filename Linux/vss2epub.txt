NAME
       vss2epub - manual page for vss2epub 0.9.6

SYNOPSIS
       vss2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `vss2epub' converts MS Visio stencils to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit
