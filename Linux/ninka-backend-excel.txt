NAME
       ninka-excel - source file license identification tool (Excel backend)

SYNOPSYS
       ninka-excel package filename

DESCRIPTION
       Scans a package and returns the found licenses in an Excel spreadsheet.

EXAMPLES
       ninka-excel package.zip pacakge.xls
           Determine the licenses in package package.zip.

AUTHOR
       ninka was written by Daniel M. German <dmg@uvic.ca> and Yuki Manabe <y-manabe@ist.osaka-u.ac.jp>.

SEE ALSO
       Daniel M. German, Yuki Manabe and Katsuro Inoue. A sentence-matching method for automatic license identification of source code files. In 25nd IEEE/ACM
       International Conference on Automated Software Engineering (ASE 2010).

       You can download it from http://turingmachine.org/~dmg/papers/dmg2010ninka.pdf.
