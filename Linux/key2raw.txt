NAME
       key2raw - manual page for key2raw 0.1.10

SYNOPSIS
       key2raw [OPTION] INPUT

DESCRIPTION
       `key2raw' is used to test Apple Keynote import in libetonyek.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
