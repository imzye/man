NAME
       srecord - library to manipulate EPROM load files

SYNOPSIS
       #include <srecord/name.h>

       cc ... -lsrecord

DESCRIPTION
       The srecord sjared library may be used to add all of the EPROM file formats and filters to your own projects.

       The full documentation for the shared library is generated by Doxygen from the source files, and is available on the Internet at
       http://srecord.sourceforge.net/srecord/index.html
