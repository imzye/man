NAME
       neobio - Pairwise Amino acid sequence and nucleotide sequence alignment in Java

SYNOPSIS
       neobio

DESCRIPTION
       A graphical user interface is opened. Two amino acid or nucleotide sequences are entered by the user.  The program aligns them automatically.
