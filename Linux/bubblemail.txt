NAME
       bubblemail - run the bubblemail settings window

SYNOPSIS
       bubblemail

DESCRIPTION
       Bubblemail  is  a  DBus  service providing a list of the new and unread
       user's mail from local mailboxes, pop, imap, and gnome online accounts.
       It  includes  a  libnotify  frontend to create notifications and can be
       used by other frontends as well.

OPTIONS
       The bubblemail does not take any options.

SEE ALSO
       bubblemaild(1)

BUGS
       No known bugs.
