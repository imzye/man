NAME
       dimacs-solver - manual page for dimacs-solver 1.3.1

DESCRIPTION
   Usage:
              /builddir/build/BUILD‐
              ROOT/coin-or-lemon-1.3.1-23.fc35.x86_64/usr/bin/dimacs-solver

              [-int|-long|-double|-ldouble]  [--help|-h|-help]  [-infcap  str]
              [-q] [INFILE [OUTFILE]]

   Where:
              [INFILE [OUTFILE]]

              If either the INFILE or OUTFILE file is missing the standard in‐
              put/output will be used instead.

       --help|-h|-help

              Print a short help message

       -double

              Use 'double' for capacities, costs etc.

       -infcap str

              Value used for 'very high' capacities

       -int

              Use 'int' for capacities, costs etc. (default)

       -ldouble

              Use 'long double' for capacities, costs etc.

       -long

              Use 'long long' for capacities, costs etc.

       -q

              Do not print any report
