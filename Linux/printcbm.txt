NAME
       printcbm - list a Commodore BASIC file

SYNOPSIS
       printcbm FILE

DESCRIPTION
       printcbm lists all lines of the specified Commodore BASIC program.

OPTIONS
       --help Show summary of options.

       --version
              Show version of program.
