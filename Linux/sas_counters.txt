NAME
       sas_counters - show Serial Attached SCSI (SAS) counters

SYNOPSIS
       sas_counters [-h] [--prefix PREFIX]

DESCRIPTION
       sas_counters  reports  Serial Attached SCSI (SAS), SCSI Enclosure Services (SES), SCSI devices I/O and phy error counters organized in an output suitable for Carbon/Graphite with SAS topology informa‚Äê
       tion.

OPTIONS
       optional arguments:

              -h, --help
                     show this help message and exit

              --prefix PREFIX
                     carbon prefix (example: "datacenter.cluster", default is "sasutils.sas_counters")

EXIT STATUS
       An exit status of zero indicates success of the command, and failure otherwise.

       sas_devices(1), sas_discover(1), ses_report(1)
