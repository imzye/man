NAME
       DDNet-Server - Starts the DDNet server.

SYNOPSIS
       DDNet-Server [OPTION]... [-f CONFIG_FILE]...

DESCRIPTION
       DDraceNetwork (DDNet) is an actively maintained version of DDRace, a Teeworlds modification with a unique cooperative gameplay.

EXAMPLES
       DDNet-Server -f myfile.cfg

       DDNet-Server -f myfile.cfg -f myfile2.cfg

       DDNet-Server 'sv_register 0' 'sv_name "Nameless Server"'

OPTIONS
   Configuration file
       -f CONFIGURATION_FILE
           Load CONFIGURATION_FILE instead of default autoexec_server.cfg

   Other options
       All options can be found here: https://ddnet.org/settingscommands/
