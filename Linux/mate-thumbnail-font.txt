NAME
       mate-thumbnail-font - create font thumbnails

SYNOPSIS
       mate-thumbnail-font [-h|-t <TEXT> -s <SIZE>] <FONT_FILE> <OUTPUT_FILE>

DESCRIPTION
       This executable is part of the package 'mate-control-center': The MATE Control Center.

       mate-thumbnail-font is a command-line tool for creating thumbnails of PCF fonts.

OPTIONS
       -h|--help
              Show some help text.

       -t <TEXT>, --text <TEXT>
              Text to thumbnail (default: Aa).

       -s <SIZE>, --size <SIZE>
              Font size (default: 64).
