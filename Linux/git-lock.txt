NAME
       git-lock - Lock a file excluded from version control

SYNOPSIS
       git-lock <filename>

DESCRIPTION
       Lock local files from version control.

OPTIONS
       <filename>

       The name of the filename.

EXAMPLES
       $ git lock config/database.yml
