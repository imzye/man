NAME
       wpg2raw - manual page for wpg2raw 0.3.3

SYNOPSIS
       wpg2raw [OPTION] FILE

DESCRIPTION
       `wpg2raw' is used to test libwpg.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information and exit

REPORTING BUGS
       Report bugs to <https://sourceforge.net/p/libwpg/tickets/> or <https://bugs.documentfoundation.org/>.
