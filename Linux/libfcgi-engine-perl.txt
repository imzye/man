NAME
       Plack::Handler::FCGI::Engine - A Plack::Handler backend for FCGI::Engine

SYNOPSIS
         use Plack::Handler::FCGI::Engine;

         my $handler = Plack::Handler::FCGI::Engine->new(
             nproc  => $num_proc,
             listen => $listen,
             detach => 1,
         );

         $handler->run($app);

DESCRIPTION
       This is a subclass of Plack::Handler::FCGI which will use the Plack::Handler::FCGI::Engine::ProcManager process manager by default, instead of FCGI::ProcManager.
