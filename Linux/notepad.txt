NAME
       notepad - Wine text editor

SYNOPSIS
       notepad [text file]

DESCRIPTION
       notepad is the Wine text editor, designed to be compatible with its Microsoft Windows counterpart.  It supports ANSI, UTF-8 and UTF-16 files, and can read (but not write) files with Unix line endings.

BUGS
       Bugs can be reported on the Wine bug tracker ⟨https://bugs.winehq.org⟩.

AVAILABILITY
       notepad is part of the Wine distribution, which is available through WineHQ, the Wine development headquarters ⟨https://www.winehq.org/⟩.
