NAME
       Code::TidyAll::Plugin::UniqueLines - Remove duplicate lines from a file

VERSION
       version 0.000003

SYNOPSIS
          # In configuration:

          [UniqueLines]
          select = .ispell* **/.gitignore

DESCRIPTION
       Discards duplicate lines from a file.  Useful for files containing one entry per line, such as ".svnignore", ".gitignore", and ".ispell*".
