NAME
       sendpeesmp6 modified sendpeesMP by Marcin Pohl <marcinpohl@gmail.com> Code based on thc-ipv6

SYNOPSIS
       sendpeesmp6 interface key_length prefix victim

       Send SEND neighbor solicitation messages and make target to verify a lota CGA and RSA signatures Example: sendpeesmp6 eth0 2048 fe80:: fe80::1
