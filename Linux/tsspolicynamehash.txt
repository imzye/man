NAME
       tsspolicynamehash - Runs TPM2 policynamehash

DESCRIPTION
       policynamehash

       Runs TPM2_PolicyNameHash

       -ha    policy session handle

       -nh    NameHash file - TPM2B_DIGEST

       -se[0-2] session handle / attributes (default NULL)

       01     continue

       20     command decrypt
