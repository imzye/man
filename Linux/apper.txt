NAME
       apper - Ein Verwaltungsprogramm für Anwendungen und Pakete, das PackageKit verwendet

ÜBERSICHT

       apper [--updates] [--settings] [--backend-details] [--type mime-type] [--name name] [--file file] [--resource font] [--catalog file]

BESCHREIBUNG
       Apper ist eine Bedienungsoberfläche zu PackageKit

       Apper kann Pakete, Anwendungen und Aktualisierungen für beliebige Distributionen verwalten, indem es PackageKit verwendet.

       Es stellt Linux®benutzern eine einzige Anwendung zur Verfügung, deren Bedienung Sie lernen und dann für jede beliebige Distribution verwenden können,

OPTIONEN
       --updates
           Zeigt verfügbare Aktualisierungen an

       --settings
           Öffnet den Einrichtungsdialog von Apper

       --backend-details
           Zeigt Informationen über das zurzeit von PackageKit verwendete Backend.

       --type mime-type
           Startet das Installationsprogramm für MIME-Typen

       --name name
           Installiert das Paket name.

       --file file
           Installiert das Paket, das die Datei file bereitgestellt.

       --resource font
           Installiert eine Ressource für eine Schrift.

       --catalog file
           Installiert den Paketkatalog in file.
