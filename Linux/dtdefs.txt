NAME
       dtdefs - Displays defaults defined for DNSSEC-Tools

SYNOPSIS
         dtdefs

DESCRIPTION
       The dtdefs program displays defaults defined for DNSSEC-Tools.

OPTIONS
       The following options are handled by dtdefs.

       -Version
           Displays the version information for dtdefs and the DNSSEC-Tools
           package.
