NAME
       gwaei - Japanese-English dictionary

SYNOPSIS
       gwaei [options]

DESCRIPTION
       A dictionary program for Japanese-English translation.

       -d, --dictionary name
              Search using a chosen dictionary

       -n, --new-instance
              Open a new instance of gWaei

       -v, --version
              Check the gWaei version info
