NAME
       MySQL::Sandbox::Scripts - Script templates for MySQL Sandbox

PURPOSE
       This package is a collection of data and scripts templates for MySQL::Sandbox.  It is not supposed to be used stand alone. It is an ancillary package.  For a reference manual, see MySQL::Sandbox.
       For a cookbook, see MySQL::Sandbox::Recipes.
