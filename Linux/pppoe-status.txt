NAME
       pppoe-status - Shell script to report on status of PPPoE link

SYNOPSIS
       pppoe-status [config_file]

DESCRIPTION
       pppoe-status  is  a shell script which checks the status of the PPPoE link established by the Roaring Penguin user-space PPPoE client.  If you omit config_file, the default file /etc/ppp/pppoe.conf is
       used.

AUTHOR
       pppoe-status was written by Dianne Skoll <dfs@roaringpenguin.com>.

       The pppoe home page is http://www.roaringpenguin.com/pppoe/.
