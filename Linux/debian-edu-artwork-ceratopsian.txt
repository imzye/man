NAME
       update-debian-edu-artwork-ceratopsian - Enable/disable Debian Edu Ceratopsian theme system-wide

SYNOPSIS
       update-debian-edu-artwork-ceratopsian { configure | remove }

DESCRIPTION
       update-debian-edu-artwork-ceratopsian  configure  enables  the Debian Edu Ceratopsian theming (as shipped with Debian Edu Trixie) system-wide (several login managers, KDE theme, background for all
       desktops, GRUB boot loader, etc. pp.).

       update-debian-edu-artwork-ceratopsian remove disables the Debian Edu themes and switches back to Debian's default themes (from package desktop-base).
