NAME
       pvfs2-setmattr â€“ set file mirroring properties volumes

SYNOPSIS
       pvfs2-setmattr [-c copies] [-m mode] [-h] -f file
       setmattr [-c copies] [-m mode] [-h] -f file

DESCRIPTION
       The pvfs2-setmattr utility changes information about file mirroring for
       a given file.

       The options are as follows:

       -c     Set the number of mirror copies.

       -m     Set the mirroring mode.  The mode 100 means no mirroring and 200
              means create mirror when immutable is set.

       -h     Display synopsis.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set  the  OrangeFS debug mask.  Possible masks are documented in
              pvfs2-set-debugmask(1).

       PVFS2TAB_FILE
              If set, the full pathname for an alternate pvfs2tab file

FILES
       /etc/passwd

       /etc/pvfs2tab

EXAMPLES
       Set mirroring for the file /mnt/foo.

             pvfs2-setmattr -c 2 -m 200 -f /mnt/foo
