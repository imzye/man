NAME
       tetzle - tetromino jigsaw puzzle game

SYNOPSIS
       tetzle [options] [files]

DESCRIPTION
       Tetzle is a jigsaw puzzle game that uses tetrominoes for the pieces. Any image can be imported and used to create puzzles with a wide range of sizes. Games are saved automatically, and you can se‐
       lect between currently in progress games.

OPTIONS
       -h, --help
              Displays help on commandline options.

       --help-all
              Displays help including Qt specific options.

       -v, --version
              Displays version information.

       files  Images to add to the choose game dialog.

SEE ALSO
       qt6options(7)
