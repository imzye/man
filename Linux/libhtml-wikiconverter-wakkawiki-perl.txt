NAME
       HTML::WikiConverter::WakkaWiki - Convert HTML to WakkaWiki markup

SYNOPSIS
         use HTML::WikiConverter;
         my $wc = new HTML::WikiConverter( dialect => 'WakkaWiki' );
         print $wc->html2wiki( $html );

DESCRIPTION
       This module contains rules for converting HTML into WakkaWiki markup. See HTML::WikiConverter for additional usage details.
