NAME
       sightrun - launch a Sight application

SYNOPSIS
       sightrun [FILE]

DESCRIPTION
       This program allows you to run applications and tutorials built with Sight (Surgical Image Guidance and Healthcare).

EXAMPLES
       sightrun /usr/share/sight/SightViewer/profile.xml
              Start the Application
