NAME
       pages2abw - manual page for pages2abw 0.9.6

SYNOPSIS
       pages2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `pages2abw' converts Apple Pages Documents to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
