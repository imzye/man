NAME
       selinux-policy-upgrade - upgrade the modules in the SE Linux policy database from a policy package

SYNOPSIS
       selinux-policy-upgrade

DESCRIPTION
       selinux-policy-upgrade this command will upgrade all policy modules (including the base modules) for to the versions that are under /usr/share/selinux/SELINUXTYPE
