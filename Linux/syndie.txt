NAME
       syrthes-gui - GUI for SYRTHES

SYNOPSIS
       syrthes-gui [-a] [-d DATAPATH]

DESCRIPTION
       syrthes-gui is a GUI for handling SYRTHES cases and theirs results.

       A detailed documentation about SYRTHES and its tools is available at <http://rd.edf.com/syrthes>

OPTIONS
       -a          Activate the advanced mode of the GUI.

       -d datafile Load the specified datafile.

AUTHOR
