NAME
       presage_demo_text - presage demo program (text)

SYNOPSIS
       presage_demo_text [OPTION]...

DESCRIPTION
       At the prompt, type in some text. Hit enter to generate a prediction.  Any text input is valid, including no text, a single character, or a long string.

       -c, --config CONFIG
              use config file CONFIG

       -s, --suggestions N
              set prediction size to N suggestions

       -h, --help
              display this help and exit

       -v, --version
              output version information and exit
