NAME
       pskc_global_done - API function

SYNOPSIS
       #include <pskc.h>

       void pskc_global_done( void);

ARGUMENTS
        void

DESCRIPTION
       This  function deinitializes the PSKC library, which were initialized using pskc_global_init().  After calling this function, no other PSKC library function may be called except for to re-initial‐
       ize the library using pskc_global_init().
