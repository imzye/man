NAME
       mate-settings-daemon - Handles the MATE session settings

SYNOPSIS
       mate-settings-daemon [--debug] [--replace] [--display=DISPLAY]

DESCRIPTION
       mate-settings-daemon is responsible for setting the various preference parameters of a MATE session and the applications that run under it.

OPTIONS
       Help options

       -?, --help
              Show this help message

       Application options

       --debug
              Enable debugging code

       --replace
              Replaces the current running daemon

       --display=DISPLAY
              X display to use
