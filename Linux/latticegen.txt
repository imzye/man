NAME
       latticegen - manual page for latticegen 5.4.1

SYNOPSIS
       latticegen [-randseed [<int> | 'time']] options

DESCRIPTION
   Options :
              r  <d>  <b> : gen_intrel s <d> <b> <b2> : gen_simdioph u <d> <b> : gen_uniform n <d> <b> <c> : gen_ntrulike N <d> <b> <c> : gen_ntrulike2 q <d> <k> <b> <c>: gen_qary t <d> <f> : gen_trg T <d> :
              gen_trg2

       See README.md for option descriptions: https://github.com/fplll/fplll/blob/master/README.md#latticegen
