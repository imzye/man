NAME
       botch-graph-info - show info about a buildgraph or sourcegraph

SYNOPSIS
       botch-graph-info [options] g.xml

DESCRIPTION
       Print type and number of vertices, edges and SCC of a graph in GraphML or dot format.

OPTIONS
       -h, --help
           Show help.

       -v, --verbose
           Be verbose.
