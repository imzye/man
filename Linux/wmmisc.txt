NAME
       wmmisc - Dock app that monitors your system

SYNOPSIS
       wmmisc { options }

DESCRIPTION
       wmmisc  is  a  simple dockapp that monitors the following: the amount of users logged in, the total number of processes, the number of running (or 'alive') processes, the total number of forks and
       the system load average.

OPTIONS
       -v     Output version information and exit

       -h     Display help and exit

       -display DISPLAY
              Set display

       -geometry +x+y
              Set position
