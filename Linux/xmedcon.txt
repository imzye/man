NAME
       xmedcon - (X)MedCon's GUI - Medical Image Conversion Utility

SYNOPSIS

       xmedcon [options] [-f <file>]

        or

       xmedcon [--help | <file>]

DESCRIPTION
  XMedCon  is  the graphical user interface for (X)MedCon's medical image conversion library. The program is capable of reading a few medical image formats containing multiple images in a single file. It
  is built around the Gtk widget toolkit and GdkPixbuf image loading library.

  For more help type 'xmedcon --help', read the man-page medcon(1) or check out the documentation on the project's website.
