NAME
       ntsc-cc - closed caption decoder

SYNOPSIS
       ntsc-cc [ options ]

DESCRIPTION
       ntsc-cc reads vbi data from /dev/vbi0 and decodes the enclosed cc data.  Start it with '-h' to get a list of cmd line options.

SEE ALSO
       alevt(1)
