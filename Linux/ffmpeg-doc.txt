NAME
       libavformat - multimedia muxing and demuxing library

DESCRIPTION
       The libavformat library provides a generic framework for multiplexing and demultiplexing (muxing and demuxing) audio, video and subtitle streams. It encompasses multiple muxers and demuxers for
       multimedia container formats.

       It also supports several input and output protocols to access a media resource.

SEE ALSO
