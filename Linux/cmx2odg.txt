NAME
       cmx2odg - manual page for cmx2odg 0.9.6

SYNOPSIS
       cmx2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `cmx2odg' converts Corel Presentation Exchange documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
