NAME
       pub2raw - manual page for pub2raw 0.1.4

SYNOPSIS
       pub2raw [OPTION] FILE

DESCRIPTION
       `pub2raw' is used to test libmspub.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
