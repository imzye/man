NAME
       tssecephemeral - Runs TPM2 ecephemeral

DESCRIPTION
       ecephmeral

       Runs TPM2_EC_Ephemeral

       -ecc   curve bnp256 nistp256 nistp384

       [-oq   output Q ephemeral public key file name (default do not save)]

       [-cf   output counter file name (default do not save)]
