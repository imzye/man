NAME
       vsd2odg - manual page for vsd2odg 0.9.6

SYNOPSIS
       vsd2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `vsd2odg' converts MS Visio documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
