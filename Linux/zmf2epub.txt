NAME
       zmf2epub - manual page for zmf2epub 0.9.6

SYNOPSIS
       zmf2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `zmf2epub' converts Zoner Draw documents to ePub.

OPTIONS

       --version
              print version and exit
