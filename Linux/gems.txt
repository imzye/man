NOMBRE
       gems-client - Mostrar datos transmitidos por el servidor gems-server (1)

SINOPSIS
       gems-client [-i] host [puerto]

       gems-client -h | -v

DESCRIPCION
       Permite conectarse a una computadora donde se está ejecutando gems-server(1) para mostrar en pantalla los datos transmitidos, en tiempo real.

OPCIONES
       host   Dominio o IP en donde se está ejecutando el servidor.

       puerto Puerto TCP usado por el servidor. Por omisión: 6666.

       -h     Mostrar un breve texto de ayuda.

       -v     Mostrar información acerca de la versión.

TECLAS
       Mientras gems-client está en funcionamiento, las siguientes teclas ingresadas mediante el teclado tienen funciones asignadas:

       q      Salir.

       a      Activar/desactivar  el  bloqueo de la alarma. Cuando la opción está activa, si el servidor transmite una alarma (caracter '\a'), el cliente la ignora,
              por lo que la alarma no suena. Esto es especialmente útil cuando varios clientes se están ejecutando en una misma sala, para evitar que suenen  varios
              'beeps' simultáneos.
