NAME
       git-setup - Set up a git repository

SYNOPSIS
       git-setup [<directory>]

DESCRIPTION
       Set up a git repository if one doesnÂ´t exist, add all files and make an
       initial commit.

OPTIONS
       <directory>

       The name of a new directory to setup. By default, the  current  working
       directory.

EXAMPLES
       $ git setup path/to/repository
