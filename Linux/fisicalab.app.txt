NAME
       FisicaLab - Educational program for Physics

SYNOPSIS
       FisicaLab

DESCRIPTION
       This manual page documents briefly the FisicaLab command.

       FisicaLab is a graphical Physics educational program, designed to solve physics problems.  Although it is intuitive to use, there is a comprehensive manual available in English and Spanish.

OPTIONS
       Like most GNUstep applications, FisicaLab does not support any command-line options (except the standard GNUstep ones).
