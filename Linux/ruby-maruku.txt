NAME
       marutex - converts markdown files to PDF using pdfLaTeX

SYNOPSIS
       marutex [files]

DESCRIPTION
       The marutex is a small utiliy that converts files in the maruku mardown format into PDF using pdflatex.

       This program will not run if you don't have pdflatex installed !

SEE ALSO
       maruku(1).  pdflatex(1).
