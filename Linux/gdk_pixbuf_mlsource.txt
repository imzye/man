NAME
       gdk_pixbuf_mlsource - manual page for gdk_pixbuf_mlsource 2.18.11

DESCRIPTION
       usage: gdk_pixbuf_mlsource [options] [image]

              gdk_pixbuf_mlsource [options] --build-list [ [name] [image] ...]

       --rle Enables run-length encoding for the generated pixel data (default)

       --raw Disables run-length encoding for the generated pixel data

       --build-list Enables (name, image) pair parsing mode

       -help  Display this list of options

       --help Display this list of options
