NAME
       msXpertSuite-data          —             The msxpertsuite package is a metadata package that depends on the msxpertsuite-massxpert and msxpertsuite-minexpert
       packages.

DESCRIPTION
       This manual page documents briefly the msxpertsuite package of the msXpertSuite software suite.

       If you install this package, the two msxpertsuite-massxpert and msxpertsuite-minexpert packages will be installed.

Bibliographic reference to be cited
       F. Rusconi (2009) massXpert 2: a cross-platform software environment for polymer chemistry modelling and  simulation/analysis  of  mass  spectrometric  data.
       Bioinformatics, 25:2741-2742.  doi:10.1093/bioinformatics/btp504.
