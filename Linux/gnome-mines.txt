NAME
       Mines - The popular logic puzzle minesweeper

SYNOPSIS
       gnome-mines [OPTION...]

DESCRIPTION
       gnome-mines is a puzzle game where you locate mines floating in an ocean using only your brain and a little bit of luck.

OPTIONS
       -v, --version
              Print release version and exit

       --small
       --medium
       --big
              Start the game with a small (8 by 8), medium (16 by 16) or big (30 by 16) board.  If more than one of these options is given, the biggest size is selected.

       This program also accepts the standard GNOME and GTK options.
