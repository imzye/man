NAME
       xtrx_fft - Spectrum browser for XTRX SDR

SYNOPSIS
       xtrx_fft [-h|-v]

DESCRIPTION
       This manual page documents briefly the xtrx_fft command.

       xtrx_fft is an example spectrum browser for XTRX SDR which uses libxtrx API.

OPTIONS
       -h, --help
              display help and exit

       -h, --help
              display version

AUTHOR
