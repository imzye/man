NAME
       tssnvchangeauth - Runs TPM2 nvchangeauth

DESCRIPTION
       nvchangeauth

       Runs TPM2_NV_ChangeAuth

       -ha    NV index handle

       -pwdo  password (default empty)

       -pwdn  new password (default empty)

       -se[0-2] session handle / attributes (default PWAP)

       01     continue

       20     command decrypt
