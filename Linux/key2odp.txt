NAME
       key2odp - manual page for key2odp 0.9.6

SYNOPSIS
       key2odp [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `key2odp' converts Apple Keynote documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
