NAME
       urlout - stream output of url.

SYNOPSIS
       urlout [options] url

DESCRIPTION
       This  utility  will stream a url address to stdout.  It can be used to test usecure ssl/tls streaming operations, as well as supporting generic http urls.  Various options will be added to aide debug‐
       ging of tls streams in the future.

OPTIONS
       --port=numnber
              Specify alternate port number.

       --help Outputs help screen for the user.
