NAME
       Xeus-GP - a xeus-based jupyter kernel for PARI/GP

DESCRIPTION
       Xeus-GP is a jupyter kernel for PARI/GP based on xeus.  The interface has been designed to mirror the behaviour of gp.

       Start jupyter notebook and select PARI/GP (xeus-gp).
