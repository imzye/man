NAME
       mock-parse-buildlog - parse build.log

SYNOPSIS
       mock-parse-buildlog --path PATH

DESCRIPTION
       mock-parse-buildlog parses a build.log file saved in logs directory and tries to recognize what caused that a build failed and prints to stdout the cause.

OPTIONS
       -p PATH, --path PATH
              set the path where is build.log located.
