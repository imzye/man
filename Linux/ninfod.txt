NAME
       ninfod - Respond to IPv6 Node Information Queries

SYNOPSIS
       ninfod [-dhv] [-p pidfile] [-u user]

DESCRIPTION
       Responds to IPv6 Node Information Queries (RFC4620) from clients.
       Queries can be sent by various implementations of ping6 command.

OPTIONS
       -a
           Debug mode. Do not go background.

       -h
           Show help.

       -v
           Verbose mode.

       -u user
           Run as another user.  user can either be username or user ID.

       -p pidfile
           File for process-id storage.  user is required to be able to create
           the file.

SEE ALSO
       ping(8).
