NAME
       trovacap - tool to find Italian ZIP codes (CAP - Codice Avviamento Postale)

SYNOPSIS
       trovacap

DESCRIPTION
       This manual page documents briefly the trovacap command.

       trovacap is a program to find Italian ZIP codes, given an address (city, locality, street, ...)
