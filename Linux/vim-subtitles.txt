NAME
       vim-subtitles - Syntax highlighting for subtitle files

DESCRIPTION
       vim-subtitles is a package that provides filetype detection and syntax highlighting
       for subtitle files in the following formats:

       MicroDVD format (files ending with ".sub")
       Subrip format   (files ending with ".srt")
       Mplayer format  (files ending with ".mpsub")

       If you want to enable them for your user account, add packadd subtitles to ~/.vimrc (vim) or ~/.config/nvim/init.vim (neovim)

EXAMPLES
        /usr/share/doc/vim-subtitles/examples/subtitle.srt
        /usr/share/doc/vim-subtitles/examples/subtitle.mpsub
        /usr/share/doc/vim-subtitles/examples/subtitle.sub
