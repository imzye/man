NAME
       epoptes-client - Client side daemon for epoptes

SYNOPSIS
       epoptes-client [OPTION] [SERVER] [PORT]

DESCRIPTION
       epoptes-client is the client side daemon for epoptes, a computer lab administration tool.

OPTIONS
       -c        Fetch the /etc/epoptes/server.crt certificate from the server.

       -h        Print a help message.

       -v        Display version information.

SEE ALSO
       epoptes(1).
