NAME
       "Commandable::Finder::SubAttributes::Attrs" - subroutine attribute definitions for "Commandable::Finder::SubAttributes"

DESCRIPTION
       This module contains the attribute definitions to apply to subroutines when using Commandable::Finder::SubAttributes. It should not be used directly.
