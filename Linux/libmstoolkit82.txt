NAME
       libmstoolkit —  The libmstoolkit package ships light-weight C++ libraries for reading, writing, and manipulating mass spectrometry data. The libMSToolKit li‐
       braries are easily linked to virtually any C++ algorithm for simple, fast file reading and analysis.

SYNOPSIS
       libmstoolkit

DESCRIPTION
       This manual page documents briefly the libmstoolkit source package.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       The libmstoolkit source package is used to create the following packages:

          •  libmstoolkit82 runtime binary package;

          •  libmstoolkit82-dbg; runtime binary package (debugging symbols);

          •  libmstoolkit-dev development binary package.

          •  libmstoolkit-tools tools binary package.

       The libmstoolkit82 runtime binary package ships the following libraries

          •  libmstoolkit.so

          •  libmstoolkitlite.so

       The libmstoolkit-dev development binary package ships the following files:

          •  Include files, shipped in /usr/include/libmstoolkit

       The libmstoolkit-tools binary package ships the following file:

          •  msSingleScan shipped in /usr/bin; DESCRIPTION: Reads an MS/MS spectrum from any MSToolkit supported file type and outputs to screen in MS2 format.  US‐
             AGE: MSSingleScan [scan number] [file]
