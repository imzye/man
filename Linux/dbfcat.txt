NAME
       dbfcat - appends the records of a source xBase file

SYNOPSIS
       dbfcat [-v] [-f] from_DBFfile to_DBFfile

DESCRIPTION
       Appends the records of a source xBase file into a destiny xBase file. Both files must have the same number of fields.

OPTIONS
       -v     verbose mode.

       -f     forces data conversion if data field types is not the same at both files or if is there any null value into from_DBFfile.

       from_DBFfile
              source xBase file.

       to_DBFfile
              destiny xBase file.

EXAMPLE
       dbfcat -v testbase1 testbase2
