NAME
       vss2raw - manual page for vss2raw 0.1.7

SYNOPSIS
       vss2raw [OPTION] INPUT

DESCRIPTION
       `vss2raw' is used to test import of Microsoft Visio stencils in libvisio.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              show version information

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
