NAME
       bcm43xx-fwcutter - manual page for bcm43xx-fwcutter

SYNOPSIS
       bcm43xx-fwcutter [OPTION] [DRIVER]

DESCRIPTION
       Bcm43xx-fwcutter  can extract the firmware for your Broadcom 43xx hard‚Äê
       ware from different closed source drivers. The bcm43xx  driver  depends
       on this firmware files and can't work without them.

       Currently  bcm43xx-fwcutter  supports  Apple MacOS X, Microsoft Windows
       98/ME/2000/XP and Linux drivers, but keep in mind that bcm43xx-fwcutter
       doesn't support all driver versions.

       Example:

              bcm43xx-fwcutter bcmwl5.sys

       to cut the firmware out of bcmwl5.sys

OPTIONS
       -l|--list
              List supported driver versions

       -i|--identify
              Only identify the driver file (don't extract)

       -w|--target-dir DIR
              Extract and write firmware to DIR

       -a|--alt-iv
              Extract alternative initvals (only 3.10.x.x)

       -p|--postfix ".FOO"
              Postfix for firmware filenames (.FOO.fw)

       -v|--version
              Print bcm43xx-fwcutter version

       -h|--help
              Print help information

DRIVERS
       AppleAirPort2
              Apple MacOS 10.x driver file (ppc, x86)

       AppleAirPort3
              Apple MacOS 10.x driver file (ppc)

       AppleAirPortBrcm4311
              Apple MacOS 10.4 driver file (x86)

       bcmwl5.sys
              Microsoft Windows driver file, 32bit (x86)

       bcmwl564.sys
              Microsoft Windows driver file, 64bit (x86-64)

       bcmwl5a.sys
              Microsoft Windows driver file, 32bit (x86)

       mn720-50.sys
              This  is  a  renamed  bcmwl5.sys  driver  for Microsoft Wireless
              MN-720 Notebook Adapter and MN-730 PCI Adapter

       n54gnds.sys
              This is a renamed bcmwl5.sys driver for Planex  GW-NS54G  54Mbps
              Wireless LAN Cards

       wmp11v27.sys
              This  is  a renamed bcmwl5.sys driver for Linksys WMP-11 802.11b
              PCI Cards

       wn511b.sys
              This is a renamed bcmwl5.sys driver for Netgear WN511B Cards

       ap_d11ucode.o
              Linux 2.4.x driver file (mips)

       apsta_d11ucode.o
              Linux 2.4.x driver file (mips)

       sta_d11ucode.o
              Linux 2.4.x driver file (mips)

       wl.o   Linux 2.4.x driver file (mips)

       wl_ap.o
              Linux 2.4.x driver file (mips)

       wl_apsta.o
              Linux 2.4.x driver file (mips)

       wl_sta.o
              Linux 2.4.x driver file (mips)

       wl96345GW.o_save
              Linux 2.6.x driver file (mips, bcm6345)

       wl96348GW.o_save
              Linux 2.6.x driver file (mips, bcm6348)

       wl96348GWV_DT.o_save
              Linux 2.6.x driver file (mips, bcm6348)
