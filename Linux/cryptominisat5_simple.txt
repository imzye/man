NAME
       cryptominisat5_simple - SAT solver

SYNOPSIS
       cryptominisat5_simple  [options]  <input-file> where input is plain DI‐
       MACS.

OPTIONS
       --verb = [0...]  Sets verbosity level. Anything higher than 2 will give
              debug log

       --drat = {fname} DRAT dumped to file

       --sls  = {walksat,yalsat} Which SLS solver to use

       --threads
              = [1...]  Sets number of threads
