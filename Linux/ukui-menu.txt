NAME
       ukui-menu - The tool for the UKUI Desktop Environment

SYNOPSIS
       ukui-menu

DESCRIPTION
       The ukui-menu program is a part of the ukui-menu, it provides users with graphical interface tools.

       This manual page documents the ukui-menu command.
