NAME
       guild-gnome-2 - invoke guild and load Guile-Gnome-2 libraries

SYNOPSIS
       guild-gnome-2 [ ARG ] ...

DESCRIPTION
       Command guild-gnome-2 runs program guild so that Guile-Gnome libraries are loaded. See the documentation for guile for more information.

       guile info pages
