       ngramsplit - manual page for ngramsplit 1.3.4

DESCRIPTION
       ngramsplit.

              Usage: ngramsplit [--options] in_fst [out_fsts_prefix]

       PROGRAM FLAGS:

       --backoff_label: type = int64, default = 0

              Backoff label --complete: type = bool, default = false Complete partial models --contexts: type = string, default = "" Context patterns file --method: type = string, default = "count_split" One
              of: "count_split", "histogram_split" --norm_eps: type = double, default = 0.001 Normalization check epsilon

              Cache byte size that triggers garbage collection --fst_read_mode: type = string, default = "read" Default file reading mode for mappable files --fst_verify_properties: type =  bool,  default  =
