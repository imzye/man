NAME
       restic-migrate - Apply migrations

SYNOPSIS
       restic migrate [flags] [migration name] [...]

DESCRIPTION
       The  "migrate"  command checks which migrations can be applied for a repository and prints a list with available migration names. If one or more migration names are specified, these migrations are
       applied.

EXIT STATUS
       Exit status is 0 if the command was successful.  Exit status is 1 if there was any error.  Exit status is 10 if the repository does not exist.  Exit status is  11  if  the  repository  is  already
       locked.  Exit status is 12 if the password is incorrect.

OPTIONS
       -f, --force[=false]      apply a migration a second time

       -h, --help[=false]      help for migrate
