NAME
       ofgctester - test game controllers using ObjFWHID

SYNOPSIS
       ofgctester

DESCRIPTION
       ofgctester is an interactive program to test game controllers using ObjFWHID.

       It takes no options and it can be quit using Ctrl-C.
