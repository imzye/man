NAME
       ykval-export - Exports Yubikey Info data from the yubikey-val server.

SYNOPSIS
       ykval-export

DESCRIPTION
       Outputs comma separated values containing YubiKey Info formatted data from the yubikey-val database. This data can later be imported using ykval-import.

       Database configuration is read from /etc/yubico/val/ykval-config.php

BUGS
       Report ykval-export bugs in the issue tracker ⟨https://github.com/Yubico/yubikey-val/issues⟩

SEE ALSO
       ykval-import(1)

       The YubiKey Info data format ⟨https://developers.yubico.com/yubikey-val/doc/YubiKeyInfoFormat.html⟩

       YubiKeys can be obtained from Yubico ⟨https://www.yubico.com/⟩.
