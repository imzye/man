NAME
       autojack - a jackdbus manager

SYNOPSIS
       autojack

DESCRIPTION
       Autojack is run at session start and using a config file generated by studio-controls to run jackdbus and other audio utilities.

       Autojack is meant to be run from /etc/xdg/autorun/ and will change configuration according to dbus signals received from studio-controls.

OPTIONS
       Autojack does not take any options.

SEE ALSO
       studio-controls(1), studio-system(2)

BUGS
