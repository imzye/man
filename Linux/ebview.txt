DESCRIPTION
       This manual page documents briefly the ebview commands.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       ebview  is  an  EPWING  CD-ROM  dictionary viewer using libeb. It uses libeb and supports EPWING format. It can read multiple dictionary at same time and can
       search words on X11 cut-buffer automatically.
