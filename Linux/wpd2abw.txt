NAME
       wpd2abw - manual page for wpd2abw 0.9.6

SYNOPSIS
       wpd2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wpd2abw' converts WordPerfect Documents to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output
