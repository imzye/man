NAME
       syrthes-ppfunc - preliminary traitement of data file functions for SYRTHES

SYNOPSIS
       syrthes-ppfunc -d data_file
       syrthes-ppfunc -h

OPTIONS
       -d data_file
                   design the SYRTHES data file to process.

       -h          display help message.
