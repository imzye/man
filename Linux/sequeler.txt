NAME
       com.github.alecaddd.sequeler - SQL client built in Vala and Gtk

SYNOPSIS
       com.github.alecaddd.sequeler [OPTION?]

DESCRIPTION
       Sequeler  is a native Linux SQL client built in Vala and Gtk. It allows you to connect to your local and remote databases, write SQL in a handy text editor with language recognition, and visualize
       SELECT results in a Gtk.Grid Widget.

       Supported databases:

       •      PostgreSQL

       •      MariaDB/MySQL

       •      SQLite

OPTIONS
       -h, --help
              Show help options

AUTHOR
