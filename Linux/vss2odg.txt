NAME
       vss2odg - manual page for vss2odg 0.9.6

SYNOPSIS
       vss2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `vss2odg' converts MS Visio stencils to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
       Report bugs to <https://sourceforge.net/p/libwpd/tickets/>.
