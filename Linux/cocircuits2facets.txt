NAME
       cocircuits2facets - compute the facets of a set of cocircuits

SYNOPSIS
       cocircuits2facets [OPTION]...

DESCRIPTION
       Compute the facets of a set of cocircuits.  Input is read from stdin; output is written to stdout.  The standard options described in TOPCOM(7) are available.

EXIT STATUS
       0 if the points are read from stdin successfully.

       1 otherwise.
