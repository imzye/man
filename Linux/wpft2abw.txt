NAME
       wpft2abw - manual page for wpft2abw 0.9.6

SYNOPSIS
       wpft2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `wpft2abw' converts documents to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --encoding ENCODING
              set the INPUT encoding. Use --list-encodings to see which encodings can be used.

       --list-encodings
              show the available encodings and exit

       --password PASSWORD
              set password to open the file

       --stdout
              print the result as flat XML to standard output
