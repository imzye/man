NAME
       lxqt-leave - Dialog to easily select logout, reboot, shutdown, suspend, hibernate, and lock screen

SYNOPSIS
       lxqt-leave [argument]

DESCRIPTION
       lxqt-leave  is a graphical Qt tool for easily triggering leave session actions: logout, reboot, shutdown, hibernate, and lock screen. If run with an argument, the dialog is not shown and the action is
       executed directly.

OPTIONS
       The options which apply to the lxqt-leave command are:

       --logout

       --lockscreen

       --suspend

       --hibernate

       --shutdown

       --reboot

       --help
