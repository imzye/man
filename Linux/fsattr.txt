NAME
       fsattr - manual page for fsattr 0.9.4

SYNOPSIS
       fsattr --files={normal|prealloc} DEVICE

DESCRIPTION
       Modify file-system internal data structures Options:

       --     end of options. treat subsequent parameters as arguments even if
              they start with '-'

       --files=normal
              set files to normal mode, i.e. show their contents.

       --files=prealloc set files to 'preallocated',  i.e.  clear  their  con‚Äê
              tents.

       --fstype=FSTYPE
              specify file system inside device.

       --help display this help and exit

       --version
              output version information and exit
