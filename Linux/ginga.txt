       ggrc - Ginga Remote Control client

DESCRIPTION
       usage: Ginga Remote Control program [-h] [--host HOST] [-p PORT]

   options:

              show this help message and exit

       --host HOST
              Connect to server at HOST

       -p, --port PORT
              Connect to server at PORT
