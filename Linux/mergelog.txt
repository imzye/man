NAME
       zmergelog - a fast tool to merge gzipped http log files by date

SYNOPSIS
       zmergelog logfile1 logfile2 ...

DESCRIPTION
       The zmergelog program is a tool which merges by date http log files in ´Common Log Format´. The result is sent to the standard output. It is useful if you want to create a single log file for
       multiple hosts using round-robin DNS. It can process plain text or gzipped log files.
