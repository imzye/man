NAME
       mwaw2odf - manual page for mwaw2odf 0.9.6

SYNOPSIS
       mwaw2odf [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `mwaw2odf' converts legacy Mac documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

       -x, --extension
              print extension for output (odt, ods, odp, odg)
