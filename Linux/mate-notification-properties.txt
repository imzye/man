NAME
       mate-notification-properties - Set up the options for desktop notifications

SYNOPSIS
       mate-notification-properties

DESCRIPTION
       This manual page documents briefly the mate-notification-properties command.

       mate-notification-properties is a utility to set up various options related to desktop notifications.
