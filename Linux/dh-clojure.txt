NAME
       dh-clojure - debhelper infrastructure for Clojure packages

DESCRIPTION
       dh-clojure is intended to help with the packaging of Clojure projects.

       See /usr/share/doc/dh-clojure/example/rules for a sample debian/rules file, and dh-clojure-lein(7) for additional information.

FILES
       /usr/share/doc/dh-clojure/example/rules
       /usr/share/doc/debhelper/PROGRAMMING.gz
