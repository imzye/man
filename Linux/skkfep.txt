NAME
       skkfep - Japanese SKK input method on text terminal

SYNOPSIS
       skkfep

DESCRIPTION
       skkfep is a tool that allows to use Japanese input method called SKK (Simple Kana to Kanji conversion program) on ANSI compliant text terminals.

       More information about the SKK input method itself can be found at: http://openlab.jp/skk/
