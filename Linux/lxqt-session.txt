NAME
       lxqt-config-session - Application settings of LXQt: The Lightweight Qt Desktop Environment

SYNOPSIS
       lxqt-config-session

DESCRIPTION
       This  application  handles  LXQt  X11 window manager selection, autostart of LXQt modules and other applications, settings for Qt scaling, confirmation for session exit, screenlocking, besides XDG
       user directories paths and advanced environment settings.

       If the lxqt-wayland-session module is installed, Wayland-related configurations for selecting the compositor and the screenlock command will be enabled.

REPORTING BUGS
       Report bugs to https://github.com/lxqt/lxqt-session/issues
