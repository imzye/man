SYNOPSIS
       pvfs2-get-uid [-s server] [-t history] [-f fs_id]

DESCRIPTION
       The pvfs2-get-uid performs the pvfs2-get-uid function.

       The options are as follows:

       -s     Specify the s option.  Specify the file system to operate on.

       -t     Specify the t option.

       -f     Specify the f option.

       -h     Display synopsis.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set the OrangeFS debug mask.  Possible masks are  documented  in
              pvfs2-set-debugmask(1).

       PVFS2TAB_FILE
              If set, the full pathname for an alternate pvfs2tab file

FILES
       /etc/pvfs2tab
