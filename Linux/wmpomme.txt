NAME
       wmpomme - WindowMaker dockapp for use with pommed

SYNOPSIS
       wmpomme [-r time] [-v]

DESCRIPTION
       wmpomme displays, as a WindowMaker dockapp, the state of the devices controlled by pommed

OPTIONS
       -v     Display the version

       -display display
              This option specifies the X server to contact; see X(1).

AUTHOR
