NAME
       pmd2epub - manual page for pmd2epub 0.9.6

SYNOPSIS
       pmd2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `pmd2epub' converts Adobe PageMaker documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit
