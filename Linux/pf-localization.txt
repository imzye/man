NAME
       pf-localization - MonteCarlo mobile robot localization

SYNOPSIS
       pf-localization config_file.ini

DESCRIPTION
       pf-localization is a command-line application which processes a rawlog and tracks a mobile robot pose from the sequence of odometry and observations. It can also perform global localization.
