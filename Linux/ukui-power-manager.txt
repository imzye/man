NAME
       ukui-power-manager-tray - The power-manager UI for the UKUI Desktop Environment

DESCRIPTION
       This  module  show power-manager widget This process is mainly for displaying the power management tray and the main interface after opening the tray icon.  Currently, the main interface will show
       the user the current remaining power and remaining charging time.  And provide users with an interface to enter the control panel to set related power operations.
