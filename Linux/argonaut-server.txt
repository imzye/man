NAME
       argonaut-server - Dispatching action received from FusionDirectory to the clients

SYNOPSIS
       argonaut-server

DESCRIPTION
       argonaut-server - argonaut-server dispatch actions received from FusionDirectory and send it to the clients. It is modular and can load various modules at
       run time.

BUGS
       Please report any bugs, or post any suggestions, to the fusiondirectory mailing list fusiondirectory-users or to
       <https://gitlab.fusiondirectory.org/argonaut/argonaut/issues/new>
