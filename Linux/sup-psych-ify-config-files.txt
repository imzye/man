NAME
       sup-psych-ify-config-files - migrate Sup configuration

SYNOPSIS
       sup-psych-ify-config-files

DESCRIPTION
       YAML migration is deprecated by Ruby 2.1 and newer.

       If sup-psych-ify-config-files is executed by Ruby <= 2.0 it migrates the $HOME/.sup/sources.yaml configuration file by reading it with the SYCK YAML parser and emitting the results with the Psych YAML
       emitter.

       Read more on the Sup wiki

       https://github.com/sup-heliotrope/sup/wiki/Migration-0.13-to-0.14

FILES
       $HOME/.sup/config.yaml
              Sup configuration file

       $HOME/.sup/sources.yaml
              Configuration file for Sup mail sources
