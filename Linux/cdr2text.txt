NAME
       cdr2text - manual page for cdr2text 0.1.7

SYNOPSIS
       cdr2text [OPTION] FILE

DESCRIPTION
       `cdr2text' converts CorelDRAW documents to plain text.

OPTIONS
       --help show this help message

       --version
              show version information and exit

REPORTING BUGS
       Report bugs to <https://bugs.documentfoundation.org/>.
