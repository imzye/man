NAME
       xyscan - data thief for scientist

SYNOPSIS

       xyscan

DESCRIPTION
       xyscan is a tool for scientist in the need to extract data points, i.e. numeric values, from a plot. It allows to scan the plots and extract data points including the size of the error bars (both
       in x and y). It can handle plots with linear and logarithmic scales. In fact xyscan can be used for extracting numeric values from any kind of 2D technical drawings.
