NAME
       fmidi-play – MIDI player with a terminal interface

SYNOPSIS
       fmidi-play [OPTION ...]  INPUT [INPUT ...]

       Plays the specified MIDI file or files with an interactive terminal in‐
       terface.

OPTIONS
       -r     Play INPUT files in random order.

       -n NAME
              Specify MIDI client name (default “fmidi”).

       -M [alsa|jack]
              Choose a particular MIDI API for playback; default  is  unspeci‐
              fied, allowing the implementation to choose.

       -L LOG Write a playback log to the given LOG file path.

SEE ALSO
       fmidi-convert(1), fmidi-grep(1), fmidi-read(1), fmidi-seq(1)
