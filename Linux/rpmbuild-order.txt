NAME
       rpmbuild-order - manual page for rpmbuild-order 0.4.5

SYNOPSIS
       rpmbuild-order [--version] COMMAND

DESCRIPTION
       Order packages by build dependencies

              Sort package sources (spec files) in build dependency order

   Available options:
       -h,--help
              Show this help text

       --version
              Show version

   Available commands:
       sort   sort packages

       deps   sort dependencies

       rdeps  sort dependents

       layers ordered output suitable for a chain-build

       chain  ordered output suitable for a chain-build

       leaves List of the top leaves of package graph

       roots  List lowest root packages

       render Show graph with graphviz
