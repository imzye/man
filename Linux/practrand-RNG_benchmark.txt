NAME
       practrand-RNG_benchmark  - manual page for practrand-RNG_benchmark ver‐
       sion 0.95

DESCRIPTION
       usage: practrand-RNG_benchmark    (benchmarks  all  recommended  PRNGs)
       or: practrand-RNG_benchmark --help    (displays this message) or: prac‐
       trand-RNG_benchmark --version    (displays version information)

       Benchmarks all current recommended PRNGs in PractRand.

SEE ALSO
       The full documentation for practrand-RNG_benchmark is maintained  as  a
       Texinfo  manual.   If the info and practrand-RNG_benchmark programs are
       properly installed at your site, the command

              info practrand-RNG_benchmark

       should give you access to the complete manual.
