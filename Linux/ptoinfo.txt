NAME
       ptoinfo - Prints human readable reports for .pto files

Synopsis
         ptoinfo project1.pto [project2.pto ...]

DESCRIPTION
       Takes one or more hugin .pto projects and outputs a text report on STDOUT, the report is incidentally in YAML format.

Calling syntax
         ptoinfo <pto-file> <pto-file> ...
