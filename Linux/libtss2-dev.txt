NAME
       tcti-cmd - TPM2 Suprocess Command TCTI library

SYNOPSIS
       A TPM Command Transmission Interface (TCTI) module for interaction with a subprocess.

DESCRIPTION
       tcti-cmd  is a library that abstracts the details of direct communication with the interface and protocol exposed by a subprocess that can recieve and trans‐
       mit raw TPM2 command and response buffers. The interface exposed by this library is defined in the “TSS System Level API and TPM Command Transmission  Inter‐
       face Specification” specification.
