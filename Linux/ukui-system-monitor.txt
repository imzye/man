NAME
       ukui-system-monitor - The tool for the UKUI Desktop Environment

SYNOPSIS
       ukui-system-monitor

DESCRIPTION
       The ukui-system-monitor program is a part of the ukui-system-monitor, it provides users with graphical interface tools.

       This manual page documents the ukui-system-monitor command.
