NAME
         myproxy-get.pl

SYNOPSIS
         This needs writing, ignore the following...

         myproxy-get.pl username@myproxy.server[:port]
                        [ -out /path/to/save/proxy ] \
                        [ -lifetime N (hours, default 12 hours) ]

DESCRIPTION
       Retrieves a proxy certificate from a MyProxy Server.
