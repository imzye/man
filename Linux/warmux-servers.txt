NAME
       warmux-index-server - WarMUX index server

SYNOPSIS
       warmux-index-server [-d] [-f configfile]

DESCRIPTION
       warmux-index-server is the index server for the WarMUX game.

OPTIONS
       -d
           Run as daemon.

       -f configfile
           Set an alternative configuration file.
