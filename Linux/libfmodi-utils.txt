NAME
       modiinfo â€” determines information about a Mac OS disk image

SYNOPSIS
       modiinfo [-hvV] source

DESCRIPTION
       modiinfo is a utility to determine information about a Mac OS disk image

       modiinfo is part of the libmodi package.  libmodi is a library to access Mac OS disk image formats

       source is the source file.

       The options are as follows:

       -h      shows this help

       -v      verbose output to stderr

       -V      print version

ENVIRONMENT
       None

FILES
       None

EXAMPLES
       # modiinfo folder-ufbi.dmg
       modiinfo 20210403

       Mac OS disk image information:

               Image type:             UDIF uncompressed
               Media size:             4.0 MiB (4194304 bytes)

DIAGNOSTICS
       Errors, verbose and debug output are printed to stderr when verbose output -v is enabled.  Verbose and debug output are only printed when enabled at compilation.
