NAME
       Config::Find::Win98 - Win98 idiosyncrasies for Config::Find

SYNOPSIS
         # don't use Config::Find::Win98 directly
         use Config::Find;

ABSTRACT
       Implements Win98 specific features for Config::Find

DESCRIPTION
       Contains any idiosyncrasies found within Win98, that do not apply to the standard Win32 base.

SEE ALSO
       Config::Find, Config::Find::WinAny, Config::Find::Any
