NAME
       cachedel - drop pagecache for a file

SYNOPSIS
       cachedel [-n <n>] file

OPTIONS
       -n <n> "Repeat system call"
              Will call posix_fadvise(POSIX_FADV_DONTNEED) <n> in a row.

DESCRIPTION
       Call posix_fadvise(POSIX_FADV_DONTNEED) on file.
