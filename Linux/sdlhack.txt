NAME
       sdlhack - Force SDL fullscreen games to minimize

SYNOPSIS
       sdlhack [options] game [arguments]

DESCRIPTION
       SDLHack is a wrapper for SDL which lets you force full-screen games to minimize.  It also allows you to disable joystick detection.

OPTIONS
       -h     show brief help

       -d     disable joystick
