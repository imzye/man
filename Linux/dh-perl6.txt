NAME
       dh_perl6_maintscript -- create maintainer scripts for perl 6 module packages

SYNOPSIS
       dh_perl6_maintscript

DESCRIPTION
       This script will create postinst and prerm snippets that call rakudo-helper.pl from the rakudo package, which is required by perl 6 module packages to create and remove precompilation files.

SEE ALSO
       dh_perl6_depsfile(1), dh_perl6_test(1), debhelper(7), dh(1)
