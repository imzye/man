NAME
       tsspolicypcr - Runs TPM2 policypcr

DESCRIPTION
       policypcr

       Runs TPM2_PolicyPCR

       -ha    policy session handle

       [-halg (sha1, sha256, sha384, sha512) (default sha256)]

       -bm    pcr mask in hex e.g., -bm 10000 is PCR 16, 000001 is PCR 0
