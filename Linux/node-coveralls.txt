NAME
       node-coveralls - report coverage to coveralls.io service

SYNOPSIS
       node-coveralls

DESCRIPTION
       This  program  takes standard input from any tool that emits the lcov data format (including mocha's) and send it to coveralls.io to report your code coverage there.  Once your app is instrumented
       for coverage, and building, you need to pipe the lcov output to this command.

SEE ALSO
       mocha(1)
