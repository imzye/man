NAME
       pdi2iso - convert a Instant Copy image file into standard ISO image file

SYNOPSIS
       pdi2iso [OPTION] [BASENAME.pdi]

DESCRIPTION
       PDI2ISO is a very simple utility to convert a Instant Copy BIN image to the standard ISO-9660 format.

OPTIONS
       --cue Generate cue file
