NAME
       WebService::Validator::HTML::W3C::Warning - Warning messages from the W3Cs online Validator

DESCRIPTION
       This is a wee internal module for WebService::Validator::HTML::W3C. It has three methods: line, col and msg which return the line number, column number and the warning that occured at that
       location in a validated page.
