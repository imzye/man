NAME
       lomiri-upload-manager - The Lomiri Upload Manager

SYNOPSIS
       lomiri-upload-manager

DESCRIPTION
       lomiri-upload-manager is a user space DBus daemon of the Lomiri Operating Environment centrally managing uploads.
