NAME
     apertium-utils-fixlatex — LaTeX format fixer for Apertium

SYNOPSIS
     apertium-utils-fixlatex [input_file [output_file]]

DESCRIPTION
     apertium-utils-fixlatex gawk-based script to fix some constructions in “XMLish” apertium LaTeX format to get better translation results.

OPTIONS
     -h, --help
             Display this help.

SEE ALSO
     apertium(1), apertium-deslatex(1), apertium-destxt(1), apertium-postlatex(1), apertium-postlatex-raw(1), apertium-prelatex(1), apertium-relatex(1), lt-proc(1)
