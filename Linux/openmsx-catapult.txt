NAME
       openmsx-catapult — perfectly emulate the MSX standard and more

SYNOPSIS
       openmsx-catapult

DESCRIPTION
       This manual page documents briefly the openmsx-catapult command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.  Instead, it has documentation in the HTML format; see below.

       openmsx-catapult emulates the MSX standard.

SEE ALSO
       openmsx (1).

       The programs are documented fully by openMSX Catapult User Manual available in HTML at the location /usr/share/doc/openmsx-catapult/manual on the system.
