NAME
       shpdxf - creates a DXF from an existing shapefield

SYNOPSIS
       shpdxf shapefile idfield

OPTIONS
       shapefile
              the name of an existing shapefile.

       idfield
              attribute that will be used as id

EXAMPLE
       shpdxf shapefile name
