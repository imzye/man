NAME
       termtime - determine the length of a ttyrec file

SYNOPSIS
       termtime file...

DESCRIPTION
       Termtime tells you the length of recorded data in a ttyrec file(s).  All formats and compressions supported by termplay(1) are available.

SEE ALSO
       termrec(1), termplay(1), termcat(1).
