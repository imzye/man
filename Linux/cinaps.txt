NAME
       cinaps - manual page for cinaps 0.15.1

DESCRIPTION
       /builddir/build/BUILDROOT/ocaml-cinaps-0.15.1-3.fc35.x86_64/usr/bin/cinaps <options> <files>

       -syntax {auto|c|ocaml}

              Syntax to use (default: auto)

       -staged FILE Staged mode: write a .ml file that must be built and executed

       -i     Update the file in-place

       -diff-cmd
              Diff command when using code expectations

       -no-color
              Don't use colors when printing errors

       -styler
              Code styler

       -help  Display this list of options

       --help Display this list of options
