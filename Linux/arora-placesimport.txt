NAME
       arora-placesimport  - a tool for importing browser history from Firefox
       3 and up

SYNOPSIS
       arora-placesinfo ~/.mozilla/firefox/[profile-dir]/places.sqlite

DESCRIPTION
       htmlToXBel is a tool for importing browser history from Firefox  3  and
       up.
