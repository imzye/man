NAME
       acetoneiso - feature-rich graphical application to mount and manage CD/DVD images

SYNOPSIS
       acetoneiso [OPTION] [FILE]

DESCRIPTION
       Opens a graphical filemanager which lets you mount typical proprietary images formats of the Windows world such as ISO BIN NRG MDF IMG and do plenty of other things.

       If FILE is specified, then the program mounts the image before opening the GUI.

OPTIONS
       --help
              Print a help message and exit.

SEE ALSO
       A detailed manual for acetoneiso can be found from the menu of the graphical filemanger in Help -> Manual.
