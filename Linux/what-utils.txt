NAME
       what-repo - determine what archive a source package is in

SYNOPSIS
       what-repo SOME_SOURCE_PACKAGE

DESCRIPTION
       what-repo(1) is a simple program that determines what archive a source package resides in.

       It's a simple wrapper for 'apt-cache(1) show | grep'.

EXAMPLE
        $ what-repo sl
        universe

SEE ALSO
       apt-cache(1)
