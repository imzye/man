NAME
       Data::Perl::String::MooseLike - data::Perl::String subclass that simulates Moose's native traits.

VERSION
       version 0.001009

SYNOPSIS
         use Data::Perl::Collection::String::MooseLike;

         my $string = Data::Perl::Collection::String::MooseLike->new("this is a string\n");

         $string->chomp;

         $string->substr(); # etc

DESCRIPTION
       This class provides a wrapper and methods for interacting with a string. All methods are written to emulate/match existing behavior that exists with Moose's native traits. You should probably be
       looking at Data::Perl's documentation instead.
