NAME
       zt_test_case_func â€” type of pointer to a test case function

SYNOPSIS
       typedef void
       (*zt_test_case_func)(zt_t);

DESCRIPTION
       zt_test_case_func is a pointer to a function implementing a single test case.

IMPLEMENTATION NOTES
       The zt_test pointer (aka zt_t) argument is meant to be passed to zt_check() and zt_assert() to alter the outcome of the test.

       zt_test(3), zt_visit_test_case(3), zt_visit_test_suite(3)
