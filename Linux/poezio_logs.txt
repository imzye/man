NAME
       poezio_logs - Display a Poezio log in a human-readable format

SYNOPSIS
       poezio_logs [options] {LOG_FILE}

DESCRIPTION
       Poezio is a console-based XMPP client.

       poezio_logs displays a log stored in Poezio format in a more human-readable way.

OPTIONS
       -i, --hide-info
           Hide info lines

       -t, --hide-time
           Hide timestamps

       -c, --no-color
           Remove color

       poezio(1)
