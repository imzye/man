NAME
       VolumeControl - audio mixer for GNUstep

SYNOPSIS
       VolumeControl

DESCRIPTION
       This manual page documents briefly the VolumeControl command.  This manual page was written for the Debian distribution because the original program does not
       have a manual page.

       VolumeControl is a GNUstep program for adjusting the audio mixer on systems that use the ALSA or OSS APIs. It allows the sound  level,  left/right  speakers,
       muting for master, PCM, bass, and treble levels to be controlled.

SEE ALSO
       GNUstep(7).
