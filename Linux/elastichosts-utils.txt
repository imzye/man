NAME
       elastichosts - command line interface for ElasticHosts' API

SYNOPSIS
       elastichosts [ -c | -f FILENAME ] CMD [ARG]...

OPTIONS
       -c     read input data for API call from stdin

       -f FILENAME
              read input data for API call from FILENAME

       -j     input and output are in JSON format

       -v     show full headers sent and received during API call
