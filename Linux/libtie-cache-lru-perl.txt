NAME
       Tie::Cache::LRU::LinkedList - Tie::Cache::LRU implemented using a linked list

SYNOPSIS
         use Tie::Cache::LRU::LinkedList;

         tie %cache, 'Tie::Cache::LRU', 500;

         ...the rest is as Tie::Cache::LRU...

DESCRIPTION
       This is an implementation of Tie::Cache::LRU using a linked list structure.  Theoretically, this is an efficient algorithm, however it may be lose out in smaller cache sizes (where small <= ??)
       due to its relatively high constant.
