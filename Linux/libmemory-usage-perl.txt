NAME
       module-size -- Spit out estimated size of module code.

SYNOPSIS
           module-size Some::Module::Name [ Other::Module ]

DESCRIPTION
       Reports the process size after 'use'ing a module.  If more than one module name is provided, the modules are included in order, allowing you to compare (for example) the incremental effect of
       adding other modules to an existing application.
