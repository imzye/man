NAME
       dfarc - An integrated frontend and .dmod installer.

SYNOPSIS
       dfarc [file.dmod]

DESCRIPTION
       This manual page documents briefly the dfarc command.

       DFArc  is  a frontend to run the Dink game and its D-Mods from a conve‐
       nient graphical user interface. If file.dmod is given, it will  be  in‐
       stalled in the D-Mods directory.
