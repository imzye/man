NAME
       uphpmvault - upload recovery images to HP MediaVault2 via Ethernet

SYNOPSIS
       uphpmvault [OPTION...] IMAGE_FILE

DESCRIPTION
       Transmit the recovery file IMAGE to an HP MediaVault2 waiting for recovery.

       -q, --quiet
              suppress program output

       The  HP  MediaVault--generation  2  a.k.a MV2--implements a custom recovery protocol for rescuing a device when the operating system image is corrupt.  The uphpmvault application listens for broadcast
       Ethernet packets from an MV2 devices that is waiting for recovery.  On receipt of one of these beacon packets, uphpmvault transmits the recovery file IMAGE to the waiting device which  will  automati‐
       cally boot the recovery image.
