NAME
       stshell - StepTalk shell

SYNOPSIS
       stshell [options]

DESCRIPTION
       Interactive shell for the StepTalk environment.  The shell allows you to manipulate GNUstep system objects, run scripts, and many other things.

       -help  print help text.
              -language LANG use language LANG as the scripting language.
              -environment ENV use scripting environment with name ENV
