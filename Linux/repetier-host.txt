NAME
       repetier-host - program to control 3D printers

SYNOPSIS

       repetier-host

DESCRIPTION
       This manual page documents briefly the repetier-host command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       repetier-host is a host controller for RepRap style 3D printers like Mendel, Prusa, and Huxley.

OPTIONS
       The program doesn't have any options yet.

FILES
       ${HOME}/.mono/registry/CurrentUser/software/repetier/*
           The per-user configuration files to control the behaviour of repetier-host.

BUGS
       The upstreams BTS can be found at https://github.com/repetier/Repetier-Host/issues.
