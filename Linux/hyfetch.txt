NAME
       HyFetch - Command-line Tool that Presents System Info

SYNOPSIS
       hyfetch [-h] [-c] [-C CONFIG_FILE]

DESCRIPTION
       hyfetch - neofetch with flags <3

       HyFetch is a command line script to display information about your Linux system, such as amount of installed packages, OS and kernel version, active GTK theme, CPU info, and used/available memory.
       It is a fork of neofetch, and adds pride flag coloration to the OS logo.
