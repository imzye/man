NAME
       shield-purge - This program is part of the PAM-shield package.

SYNOPSIS
       shield-purge <option>

DESCRIPTION
       This  program  is  part  of the PAM-shield package.  PAM-shield comes with ABSOLUTELY NO WARRANTY.  This is free software, and you are welcome to redistribute it under certain conditions.  See the GNU
       General Public Licence for details.

OPTIONS
       -h, --help
              Display this information

       -c, --conf=file
              Specify config file (default: /etc/security/shield.conf)

       -d, --debug
              Verbose output for debugging purposes

       -n, --dry-run
              Do not perform any updates

       -l, --list
              List all database entries

       -f, --force
              Delete all entries, even if unexpired
