NAME
       qxp2raw - manual page for qxp2raw 0.0.2

SYNOPSIS
       qxp2raw [OPTION] FILE

DESCRIPTION
       `qxp2raw' is used to test libqxp.

OPTIONS
       --callgraph
              display the call graph nesting level

       --help show this help message

       --version
              print version and exit

REPORTING BUGS
       Report bugs to <http://bugs.documentfoundation.org/>.
