NAME
       mednafen - multi-system emulator

SYNOPSIS
       mednafen [options] romfile

DESCRIPTION
       Mednafen  is a command-line driven emulator for many different systems. It has full support for OpenGL and SDL graphics, network play, sound output via various methods (Esound, Jack, etc.), remap‚Äê
       pable input configuration, joystick and keyboard support, save states, game rewinding, and screenshots.
