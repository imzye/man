NAME
     apertium-rertf — Rich Fext Format processor for Apertium

SYNOPSIS
     apertium-rertf [input_file [output_file]]

DESCRIPTION
     apertium-rertf is an RTF format processor.  It restores the original RTF formatting the text had before being passed through the apertium-desrtf(1) deformatter.

OPTIONS
     -h, --help
             Display this help.

SEE ALSO
     apertium(1), apertium-rehtml(1), apertium-retxt(1), lt-proc(1)
