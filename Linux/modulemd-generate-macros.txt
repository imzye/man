NAME
       modulemd-generate-macros

SYNOPSIS
       modulemd-generate-macros  [-h]  [--disttag  DISTTAG] [--conflicts-from-
       file CONFLICTS_FROM_FILE] yaml

DESCRIPTION
       Generate `module-build-macros` SRPM package, which is a  central  piece
       for  building modules. It should be present in the buildroot before any
       other module packages are submitted to be built.

OPTIONS
       yaml   Path to modulemd YAML file

       --disttag DISTTAG
              Disttag

       --conflicts-from-file CONFLICTS_FROM_FILE
              Path to a file containing conflicts definitions and  their  rea‐
              soning.  Content  of this file gets simply pasted into the spec‐
              file
