NAME
       Plack::Middleware::Debug::Parameters - Parameters Panel

SYNOPSIS
           builder {
               enable 'Debug'; # load defaults
               enable 'Debug::Parameters', elements => [qw/headers cookies/];
               $app;
           };

DESCRIPTION
       return info about:

       request headers
       query parameters
       body parameters
       cookies
       session

       Plack::Middleware::Debug
