NAME
       airr-tools - Data Representation Standard library for antibody and TCR sequences

DESCRIPTION
       usage: airr-tools [-h] [--version]  ...

       AIRR Community Standards utility commands.

   help:
       -h, --help
              show this help message and exit

       --version
              show program's version number and exit

   subcommands:
              Database operation

       merge  Merge AIRR rearrangement files.

       validate
              Validate AIRR rearrangement files.

AUTHOR
