NAME
       numbers2text - manual page for numbers2text 0.1.10

SYNOPSIS
       numbers2text [OPTION] INPUT

DESCRIPTION
       `numbers2text' converts Apple Numbers spreadsheets to plain text.

OPTIONS
       --help show this help message

       --version
              show version information
