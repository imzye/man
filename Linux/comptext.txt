NAME
       comptext - GUI-based tool to compare two text streams

DESCRIPTION
       Comptext  is  used  to compare two text streams.  Its comparison function uses a linear-programming technique.  It computes total number of errors, the character error rate (CER) and the bit error
       rate (BER).

GUI OPTIONS
       Expected
              Use button to select the file which contains the text stream for comparison

       Received
              Use button to select the second text stream file to compare

       Compare
              Click to perform the comparison between the two text streams. It displays the total number of errors, CER and BER
