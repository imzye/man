NAME
       mbs-frontend - manual page for mbs-frontend 3.6.1

SYNOPSIS
       mbs-frontend [OPTIONS]

DESCRIPTION
              Runs the Flask app, locally.

OPTIONS
       -h, --host HOST
              Bind to this host.

       -p, --port PORT
              Bind to this port along with --host.

       -d, --debug
              Run frontend in debug mode.

       --help Show this message and exit.
