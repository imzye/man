NAME
       key2text - manual page for key2text 0.1.10

SYNOPSIS
       key2text [OPTION] INPUT

DESCRIPTION
       `key2text' converts Apple Keynote presentations to plain text.

OPTIONS
       --help show this help message

       --version
              show version information
