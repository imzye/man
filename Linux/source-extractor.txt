NAME
       source-extractor - extract a source catalogue from an astronomical FITS image

SYNOPSIS
       source-extractor image [-c configuration-file]
              [-parameter1 value1 -parameter2 value2 ...]

       source-extractor image_detect image_measure [-c configuration-file]
              [-parameter1 value1 -parameter2 value2 ...]

       source-extractor -d

DESCRIPTION
       SExtractor  is a program that builds a catalogue of objects from an astronomical image. Although it is particularly oriented towards reduction of large scale galaxy-survey data, it performs rather
       well on moderately crowded star fields.
       See http://astromatic.net/software/sextractor for more details.

   Operation modes:
       -h, --help
              print this help, then exit

       -V, --version
              print version number, then exit

       -d, --dump
              dump a default configuration file
