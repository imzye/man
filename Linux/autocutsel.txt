NAME
       cutsel - show information about the X clipboard and the cutbuffer in sync

SYNOPSIS
       cutsel [-selection selection_name] [-cutbuffer number] [-debug] [-verbose] {cut|sel}

DESCRIPTION
       This manual page documents briefly the cutsel command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       cutsel [options] cut
              shows the contents of the cut buffer and

       cutsel [options] sel
              shows the owner of the selection.

OPTIONS
       -selection, -select, -sel, -s {CLIPBOARD|PRIMARY}
              Specify which selection to use.  Default is CLIPBOARD.

       -cutbuffer <number>
              (Details of this parameter unknown.)

       -debug, -d
              Run with debugging mode.

       -verbose, -v
              Be more verbose.  Reports the version and announces synchronizations.
