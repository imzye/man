NAME
       bootcd - use bootcd utils

SYNOPSIS
       bootcd-tool options

       bootcd-tool = bootcdwrite | bootcd2disk | bootcdflopcp

DESCRIPTION
       bootcd-tools  can copy a running or mounted Debian system.  Different boot configuration are supportet (efi, old bios, secure boot).  bootcdwrite creates an ISO life-cd file called bootcd using an
       overlayfs.  bootcdflopcp writes changes from overlayfs to floppy disk, when running from bootcd.  bootcd2disk writes bootable disks (with our without lvm, ext2, ext3, ext4, vfat, swap).

SEE ALSO
       bootcdwrite(1), bootcd2disk(1), bootcdflopcp(1), bootcdmk2diskconf(1), bootcdwrite.conf(5), bootcd2disk.conf(5)
