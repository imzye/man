NAME
       watch_maildirs - watch for and display Maildir/Maildir++ changes

SYNOPSIS
       watch_maildirs [options ...]

DESCRIPTION
       watch_maildirs  is  a  command  line  application  which  watches  a  given  Maildir and displays change notifications. The Maildir may contain sub-mailboxes
       (Maildir++). Once watch_maildirs is watching, a newline is printed. Then, for each change the mailbox's name and a newline are printed.

OPTIONS
       -m dir, --maildir=dir
              Override the setting of the $MAILDIR environment variable (or if $MAILDIR is not defined, ~/Maildir) with dir.

       -V, --version
              Display version information.

       -?, --help
              Display a summary of command line options.

EXIT CODES
       1 upon error
       2 upon folder creation, deletion, and rename that requires a watch_maildirs restart

SEE ALSO
