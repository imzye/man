NAME
       tsssignapp - Runs TPM2 signapp

DESCRIPTION
       signapp

       Runs a TPM2_Sign application, including creating a primary storage key and creating and loading a signing key

       -ic    input message to hash and sign

       [-pwsess
              Use a password session, no HMAC or parameter encryption]
