NAME
       expand-solitaire-multi-cards-moves -- a command-line tool for expanding multi-card moves.

VERSION
       version 0.2403

SYNOPSIS
           $ fc-solve -p -t -sam 24.board | expand-solitaire-multi-cards-moves -
           $ fc-solve -g seahaven_towers -p -t -sam 24.seahaven.board \
               | expand-solitaire-multi-cards-moves -g seahaven_towers

OVERVIEW
       expand-solitaire-multi-cards-moves is a command-line interface to Games::Solitaire::Verify::Solution::ExpandMultiCardsMoves.

COMMAND LINE OPTIONS
   -g [variant]
       Picks up a variant.

   --decks-num [1|2]
       Number of decks - 1 or 2.

   --freecells-num [n]
       Number of freecells.

   --stacks-num [n]
       Number of columns.

   --sequences-are-built-by [alternate_color|alt_color|suit|rank]
       How the sequences are built by.

   --empty-stacks-filled-by [any|kings|none]
       What empty stacks can be filled by.

   --sequence-move [limited|unlimited]
       Whether the sequence move is limited or unlimited.
