NAME
       git-delete-submodule - Delete submodules

SYNOPSIS
       git-delete-submodule <path>

DESCRIPTION
OPTIONS
       <path>

       The path of the submodule to delete.

EXAMPLES
       $ git delete-submodule lib/foo
