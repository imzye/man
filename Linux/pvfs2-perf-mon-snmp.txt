NAME
       pvfs2-perf-mon-example – performance monitor example

SYNOPSIS
       pvfs2-perf-mon-example

DESCRIPTION
       The  pvfs2-perf-mon-example utility is part of the performance monitor‐
       ing system.  It only be used by experienced system administrators.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set the OrangeFS debug mask.  Possible masks are  documented  in
              pvfs2-set-debugmask(1).

BUGS
       Please submit bug reports to pvfs2-developers@beowulf-underground.org
