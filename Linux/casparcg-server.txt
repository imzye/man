NAME
       casparcg - layered real-time video compositor to multiple outputs

SYNOPSIS
       casparcg [path-to-config]

DESCRIPTION
       Play out professional graphics, audio and video to multiple outputs as a layerbased real-time compositor.

       If no path-to-config is provided, the default path is 'casparcg.config'.

OPTIONS
       There are no options.
