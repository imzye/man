NAME
       srt-ffplay - live transmit a URL via SRT and ffplay

SYNOPSIS
       srt-ffplay < source-URL >

DESCRIPTION
       str-ffplay will call srt-live-transmit to transmit the source URL to ffplay.

       This is mostly used as a testbed for SRT.

SEE ALSO
       srt-live-transmit(1), ffplay(1)
