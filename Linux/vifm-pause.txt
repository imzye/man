NAME
       vifm-pause - a helper script for vifm

SYNOPSIS
       Only called automagically from within vifm

       vifm-pause

DESCRIPTION
       A helper script for vifm, not for standalone use.

       Prints "Press return ..." message and waits until Enter key is pressed.

       vifm(1)
