NAME
       qxp2text - manual page for qxp2text 0.0.2

SYNOPSIS
       qxp2text [OPTION] FILE

DESCRIPTION
       `qxp2text' converts QuarkXPress documents to plain text.

OPTIONS
       --help show this help message

       --version
              print version and exit

REPORTING BUGS
