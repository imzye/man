NAME
       wks2raw - manual page for wks2raw 0.4.12

SYNOPSIS
       wks2raw [OPTION] FILE

DESCRIPTION
       `wks2raw' is used to test spreadsheet import in libwps.  Supported formats are Microsoft Works Spreadsheet and Database, Lotus Wk1-Wk4 and Quattro Pro Wq1-Wq2.

OPTIONS
       -h, --help
              show this help message

       -v, --version
              show version information

       --callgraph
              display the call graph nesting level

       --password PASSWORD
              set password to open the file
