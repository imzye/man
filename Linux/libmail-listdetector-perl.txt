NAME
       Mail::ListDetector::Detector::CommuniGate - CommuniGate message detector

SYNOPSIS
         use Mail::ListDetector::Detector::CommuniGate;

DESCRIPTION
       An implementation of a mailing list detector, for CommuniGate mailing lists, CommuniGate is a legacy MacOS messaging application, see http://www.stalker.com/mac/default.html for details about
       CommuniGate.

METHODS
   new()
       Inherited from Mail::ListDetector::Detector::Base.

   match()
       Accepts a Mail::Internet object and returns either a Mail::ListDetector::List object if it is a post to a CommuniGate mailing list, or "undef".

BUGS
       No known bugs.
