NAME
       scl2trd â€” Convert .scl disk images to .trd.

SYNOPSIS
       scl2trd [OPTION] sclfile trdfile

DESCRIPTION
       scl2trd  converts a .scl disk file to a 80 track, double sided (640 KB)
       .trd disk file.

OPTIONS
       -h, --help
              give brief usage help, listing available options.

       -V, --version
              output version information.

       sclfile
              specifies a .scl file whose contents are to be converted.

       trdfile
              specifies a .trd file to create.

BUGS
       None known.
