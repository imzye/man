NAME
       cshtml - format the input list of defects as a human-readable HTML doc‚Äê
       ument

SYNOPSIS
       cshtml [options] proj.js, where options are:

DESCRIPTION
       --cwe-names arg
              CSV file mapping CWE numbers to names

       --defect-url-template arg
              e.g. http://localhost/index.php?proj=%d&defect=%d

       --diff-base arg
              use the given list of defects as diff base

       --diff-base-ignore-checkers arg
              do not diff base for checkers matching the given regex

       --plain-text-url arg
              generate a link to plain-text version

       --scan-props-placement arg (=bottom) placement of the table  with  scan
              properties: top, bottom, none

       -q [ --quiet ]
              do not report any parsing errors

       --help produce help message

       --version
              print version
