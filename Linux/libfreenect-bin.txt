NAME
       freenect-glpcview - manual page for freenect camera test

DESCRIPTION
       This is a simple demo program that illustrates the use of the C synchronous libfreenect library. It also provides a simple way to test the communication with the freenect device.

   Shortcuts
       ESC    Exit the program

       Key W  Increase zoom

       Key S  Decrease zoom

       Key C  Switch on/off the texture image
