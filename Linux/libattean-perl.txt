NAME
       AtteanX::Parser::SPARQLLex - SPARQL Lexer

VERSION
       This document describes AtteanX::Parser::SPARQLLex version 0.035

SYNOPSIS
        use Attean;

DESCRIPTION
       ...

ATTRIBUTES
       "canonical_media_type"
       "media_types"
       "file_extensions"
       "handled_type"
       "extend"

METHODS
       "file_extensions"
           Returns a list of file extensions that may be parsed with the parser.

       parse_iter_from_bytes( $data )
           Returns an iterator of SPARQL tokens that result from parsing the SPARQL query/update read from the UTF-8 encoded byte string $data.

       parse_iter_from_io( $fh )
           Returns an iterator of SPARQL tokens that result from parsing the SPARQL query/update read from the IO::Handle object $fh.
