NAME
       sphinx2py - manual page for sphinx2py 0.6

DESCRIPTION
       usage: sphinx2py [-h] [-W] rst_file

       Convert Sphinx ReST file to Python code

   positional arguments:
       rst_file
              ReST file to convert

   options:
       -h, --help
              show this help message and exit

       -W, --warn-is-error
              turn warnings into errors
