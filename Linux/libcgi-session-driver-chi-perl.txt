NAME
       CGI::Session::Driver::chi - This driver allows CGI::Session to use CHI as a session store

SYNOPSIS
       $s = CGI::Session->new(
           "driver:chi",
           $sid, {
               driver => 'File',
               root_dir => '/path/to/root'
           });

       $s = CGI::Session->new(
           "driver:chi",
           $sid, {
               chi_class => 'My::CHI',
               namespace => 'cgi',
           });

DESCRIPTION
       This driver allows CGI::Session to use CHI as a session store

   DRIVER ARGUMENTS
       The init sub accepts a hash ref with the same arguments as would be passed to CHI, plus an additional argument, chi_class.
