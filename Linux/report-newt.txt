NAME
       report-newt - Reports ABRT problem to various bug tracking tools.

SYNOPSIS
       report-newt [-d] PROBLEM_DIR

DESCRIPTION
       report-newt is a command line tool that reports application crashes and
       other problems caught by abrtd daemon, or created by other programs
       using libreport.

OPTIONS
       -d, --delete
           Remove PROBLEM_DIR after reporting

       -V, --version
           Display version and exit
