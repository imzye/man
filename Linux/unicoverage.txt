NAME
       unicoverage - Print coverage of each Unicode Script

SYNOPSIS
       unicoverage [-iinput-file] [-ooutput-file]

DESCRIPTION
       unicoverage  reads  a  GNU  Unifont  .hex font and uses data in coverage.dat (which must reside in the current directory).  The output is the percent coverage of each script listed in the coverage.dat
       file.  The Unicode code points in the input .hex file must be in ascending order.

OPTIONS
       -i          Specify the input file. The default is STDIN.

       -o          Specify the output file. The default is STDOUT.

EXAMPLE
       Sample usage:

              unicoverage < unifont.hex >coverage.txt

FILES
       coverage.dat, *.hex GNU Unifont files.
