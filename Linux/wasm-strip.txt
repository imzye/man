     wasm-strip â€” remove sections of a WebAssembly binary file

SYNOPSIS
     wasm-strip [options] file

DESCRIPTION
     wasm-strip removes sections of a WebAssembly binary file.

     The options are as follows:

     --help  Print a help message

EXAMPLES
     Remove all custom sections from test.wasm

           $ wasm-strip test.wasm
