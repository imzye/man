NAME
       JSON::RPC::Common::Procedure::Return::Version_2_0::Error - JSON-RPC 2.0 error class.

VERSION
       version 0.11

SYNOPSIS
               my $return_with_error = $call->return_error("foo");

DESCRIPTION
       This class implements 2.0 error objects.

       "code" and "message" are mandatory.
