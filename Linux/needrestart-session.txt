NAME
       needrestart-session - helper application for needrestart.

DESCRIPTION
       needrestart-session checks the user's sessions for processes which need to be restarted after library upgrades.  It is launched by needrestart via DBUS within the user's session automatically.

SYNOPSIS
       needrestart-session
