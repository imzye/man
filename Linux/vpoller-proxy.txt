NAME
       vpoller-proxy - manual page for vpoller-proxy 0.7.3

SYNOPSIS
       vpoller-proxy [-d] [-f <config>] start

DESCRIPTION
              vpoller-proxy [-d] -e <endpoint> stop vpoller-proxy [-d] -e <endpoint> status vpoller-proxy --help vpoller-proxy --version

   Arguments:
       start  Start the VPoller Proxy

       stop   Stop the VPoller Proxy

       status Get status information

OPTIONS
       -h, --help
              Display this usage info

       -v, --version
              Display version and exit

       -d, --debug
              Debug mode, be more verbose

       -f <config>, --file <config>
              Specify config file to use [default: /etc/vpoller/vpoller.conf]

       -e <endpoint>, --endpoint <endpoint>
              Specify the endpoint we connect to

SEE ALSO
       The full documentation for vpoller-proxy is maintained as a Texinfo manual.  If the info and vpoller-proxy programs are properly installed at your site, the command

              info vpoller-proxy

       should give you access to the complete manual.
