NAME
       vdoreadonly - puts a VDO device into read-only mode

SYNOPSIS
       vdoreadonly filename

DESCRIPTION
       vdoreadonly forces an existing VDO device into read-only mode.

OPTIONS
       --help Print this help message and exit.

       --version
              Show the version of vdoreadonly.

SEE ALSO
       vdo(8).
