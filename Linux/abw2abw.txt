NAME
       abw2abw - manual page for abw2abw 0.9.6

SYNOPSIS
       abw2abw [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `abw2abw' converts AbiWord documents back to AbiWord.  If OUTPUT is missing, the result is printed to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output
