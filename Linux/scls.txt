NAME
       scls - The scls program is used to list objects in checkpoint files.

Running scls
       scls takes the following command line options followed by a list of files.

       -memorygrp val
           Which memory group to use.

       -threadgrp val
           Which thread group to use.

       -messagegrp val
           Which message group to use.

       -W val
           Set the working directory.

       -d  Turn on debugging.

       -v  Print the version.

       -w  Print the warranty.

       -l  Detailed list of objects.

       -L  Print the license.

       -h  Print this help.
