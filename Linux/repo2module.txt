NAME
       repo2module

SYNOPSIS
       repo2module  [-h]  [--debug  |  --nodebug]  [-n  MODULE_NAME]  [-s MOD‐
       ULE_STREAM] [-v  MODULE_VERSION]  [-c  MODULE_CONTEXT]  [-O]  repo_path
       [modules_yaml]

DESCRIPTION
       Generates  modules.yaml file with a module, that provides all RPM pack‐
       ages that are available within a repository.

OPTIONS
       repo_path

       modules_yaml

       --debug

       --nodebug

       -n MODULE_NAME, --module-name MODULE_NAME
              Default is the current directory name

       -s MODULE_STREAM, --module-stream MODULE_STREAM

       -v MODULE_VERSION, --module-version MODULE_VERSION

       -c MODULE_CONTEXT, --module-context MODULE_CONTEXT

       -O, --to-stdout

AUTHORS
