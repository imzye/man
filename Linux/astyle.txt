NAME
       astyle - multi-language indentation and reformatting filters

SYNOPSIS
       astyle

       astyle [options] File1 [File2] [...]

DESCRIPTION
       Artistic Style (or astyle) is a source code indenter, formatter and beautifier, written in C++. It handles the following programming languages:

              •  C

              •  C++

              •  C++/CLI

              •  Objective-C

              •  C#

              •  Java

USAGE
       For the full usage, please refer to the HTML documentation provided with this distribution in /usr/share/doc/astyle/html

       Please  note that when indenting a specific file the newly indented file retains the original filename, while a copy of the original file is created, with a suffix of ".orig" added to the original
       filename.
