NAME
       lou_checktable - Test a liblouis Braille translation table

SYNOPSIS
       lou_checktable [OPTIONS] TABLE[,TABLE,...]

DESCRIPTION
       Test  a  Braille  translation table. If the table contains errors, appropriate messages are displayed. If there are no errors the message "no errors found." is shown unless you specify the --quiet op‐
       tion.

       -h, --help
              display this help and exit

       -v, --version
              display version information and exit

       -q, --quiet
              do not write to standard error if there are no errors.
