NAME
       info2www - read info files with a WWW browser

SYNOPSIS
       http://localhost/cgi-bin/info2www

DESCRIPTION
       This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

       info2www is a CGI script which allows you to use a Web-browser to view any GNU Info manuals installed on your system.
