NAME
       fsmount_kernel - manual page for fsmount_kernel 0.9.4

SYNOPSIS
       fsmount_kernel [-o OPTIONS] -t FILESYSTEM_TYPE SOURCE TARGET

DESCRIPTION
       Call the kernel to mount a filesystem, ignoring any userspace helper or
       wrapper (fuse...)  Options:

       --     end of options. treat subsequent parameters as arguments even if
              they start with '-'

       -t <type>
              set the filesystem type. MANDATORY.

       -o <list>
              comma-separated list of mount options.

       -h, --help
              display this help and exit

       --version
              output version information and exit
