NAME
       subliminal - Subtitles, faster than your thoughts.

SYNOPSIS
       subliminal [OPTIONS] <video files>

DESCRIPTION
       Subliminal is a command-line tool to search, download, and manage subtitles for video files from various subtitle providers. It's fast and efficient, ensuring you get subtitles quickly.

OPTIONS
       -l, --languages <languages>
              Select subtitles language. You can specify the languages using standard language codes (e.g., "en" for English).

       -d, --download
              Download subtitles for the specified video files.

       -u, --update
              Update subtitles for the specified video files.

       -s, --search
              Search for subtitles for the specified video files without downloading.

EXAMPLES
       To download English subtitles for a specific video:

       subliminal download -l en The.Big.Bang.Theory.S05E18.HDTV.x264-LOL.mp4
