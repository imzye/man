NAME
       freegish - A physics based arcade game

SYNOPSIS
       freegish [options]

DESCRIPTION
       freegish is a physics based arcade game

OPTIONS
       -nosound
              Disable sound and music

       -sound Enable sound

       -nomusic
              Disable  music

FILES
       ${HOME}/.freegish/level/ - directory containing custom levels

AUTHOR
