NAME
       botch-packages-intersection - calculate the set intersection

SYNOPSIS
       botch-packages-intersection [options] inPackages1 inPackages2 outPackages

DESCRIPTION
       calculate the set intersection of two packages files

OPTIONS
       -h, --help
           Show help.

       --verbose
           Be verbose.
