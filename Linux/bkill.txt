NAME
       bkill - deletes jobs in a familiar openlava format

SYNOPSIS
       bkill job_id...

DESCRIPTION
       The bkill command cancels the specified jobs.

OPTIONS
       -h | --help
           brief help message

       --man
           full documentation

EXIT STATUS
       On success, bkill will exit with a value of zero. On failure, bkill will exit with a value greater than zero.
