NAME
       screenbin - create a temporary, throwaway shared screen session on a remote Amazon EC2 instance

DESCRIPTION
       screenbin  is like pastebin, but for screen.  It uses Amazon's EC2 cloud to dynamically instantiate an Ubuntu instance, install the specified ssh keys, and launch a shared screen session as a
       guest user.

       To see an up-to-date list of parameters, please refer to:
        $ screenbin --help

SEE ALSO
       screen(1), screen-profiles(1), pastebin(1)

       http://launchpad.net/screenbin
