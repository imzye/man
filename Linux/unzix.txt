NAME
       unzix - WinZix archive extractor

SYNOPSIS
       unzix <zix-file>

DESCRIPTION
       unzix is a small command-line program for extracting files from the new WinZix archive format. Pass the file to be extracted as an argument to unzix.

BUGS
       To report bugs in Unzix, please visit http://banu.com/unzix/.
