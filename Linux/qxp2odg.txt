NAME
       qxp2odg - manual page for qxp2odg 0.9.6

SYNOPSIS
       qxp2odg [OPTIONS] INPUT [OUTPUT]

DESCRIPTION
       `qxp2odg' converts QuarkXPress documents to ODF.  If OUTPUT is omitted, the result is printed as Flat ODF to standard output.

OPTIONS
       --help show this help message

       --version
              print version and exit

       --stdout
              print the result as flat XML to standard output

REPORTING BUGS
