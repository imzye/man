       pvfs2-fsck â€“ check and correct file system errors

SYNOPSIS
       pvfs2-fsck

DESCRIPTION
       The  pvfs2-fsck  utility  checks  for  and  corrects  some file systtem
       errors.  It should only be used by experienced system administrators.

ENVIRONMENT
       PVFS2_DEBUGFILE
              If set to the path of a local file, redirect debug output to it.

       PVFS2_DEBUGMASK
              Set the OrangeFS debug mask.  Possible masks are  documented  in
              pvfs2-set-debugmask(1).
