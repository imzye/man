NAME
       geographiclib-get-geoids -- a GeographicLib administrative tool

DESCRIPTION
       geographiclib-get-geoids  downloads  and  installs  the geoids datasets
       used  by  GeographicLib,  <https://geographiclib.sourceforge.io>.   For
       documentation, supply the -h option:

           geographiclib-get-geoids -h

       By   default,  the  datasets  are  installed  in  /usr/share/Geographi‚Äê
       cLib/geoids and the script requires write access to this directory.
