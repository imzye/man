NAME
       libpwiz-doc - library to perform proteomics data analyses - documentation

DESCRIPTION
       This manual page documents briefly the libpwiz-doc package that brings the documentation of the libpwiz library as packaged in libpwiz. The documentation is available as a set of HTML files, auto‐
       matically extracted from the source code using Doxygen.

EXAMPLES OF BINARIES
       The libpwizX package ships examples of programs built with the library.
