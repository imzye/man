NAME
       loompy - access loom formatted files for bioinformatics

SYNOPSIS
       loompy [OPTIONS] COMMAND [ARGS]...

OPTIONS

       --show-message / --hide-message

       --verbosity [error|warning|info|debug]

       --help
              Show this message and exit.

   Commands:
              fromfq
