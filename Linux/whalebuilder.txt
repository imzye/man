NAME
       whalebuilder - Debian package builder using Docker

SYNOPSIS
       whalebuilder [globalopts] <command> [options]

DESCRIPTION
       Whalebuilder is a tool for building Debian packages in a minimal environment using Docker to manage the build environment.  For more information, see /usr/share/doc/whalebuilder/README.md.gz.
