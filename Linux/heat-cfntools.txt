NAME
       cfn-hup - Implements cfn-hup CloudFormation functionality

SYNOPSIS
       cfn-hup

DESCRIPTION
       Implements cfn-hup CloudFormation functionality

OPTIONS
       -c, --config
              Hook Config Directory, defaults to /etc/cfn/hooks.d

       -f, --no-daemon
              Do not run as a daemon

       -v, --verbose
              Verbose logging
