NAME
       sdw2html - manual page for sdw2html 0.0.7

SYNOPSIS
       sdw2html [OPTION] INPUT

DESCRIPTION
       `sdw2html' converts StarOffice documents to HTML.

OPTIONS
       -h     show this help message

       -p PASSWORD
              set password to open the file

       -v     show version information

REPORTING BUGS
       Report bugs to <https://github.com/fosnola/libstaroffice/issues>.
