NAME
       a2j_control - utility to control a2jmidid daemon

SYNOPSIS
       a2j_control [OPTIONS]

OPTIONS
       start  starts the a2jmidid bridging daemon

       stop   stops the a2jmidid bridging daemon

       status queries the a2jmidid bridge status

       exit   deactivates the a2jmidid D-Bus service
