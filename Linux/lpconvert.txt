NAME
       lpconvert - converter between gringo's aspif and smodels format

SYNOPSIS
       lpconvert [OPTION]... [FILE]...

DESCRIPTION
       This manual page documents briefly the lpconvert command.

       lpconvert is a converter between gringo's aspif and smodels format.

OPTIONS
       These  programs follow the usual GNU command line syntax, with long options starting with two dashes (`-').  A summary of options is included below.  For a complete description, see the potassco-guide
       https://sourceforge.net/projects/potassco/files/guide/2.0/.

       -h, --help
              Show summary of options.

       -v, --version
              Show version of program.
