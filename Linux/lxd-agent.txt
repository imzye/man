NAME
       lxd-agent - LXD virtual machine guest agent

DESCRIPTION
   Description:
              LXD virtual machine agent

              This daemon is to be run inside virtual machines managed by LXD.  It will normally be started through init scripts present or injected into the virtual machine.

   Usage:
              lxd-agent [--debug] [flags]

   Flags:
       -d, --debug
              Show all debug messages

       -h, --help
              Print help

       -v, --verbose
              Show all information messages

       --version
              Print version number
