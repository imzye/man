NAME
       pydoc3.13 - the Python documentation tool

SYNOPSIS
       pydoc3.13 name

       pydoc3.13 -k keyword

       pydoc3.13 -p port

       pydoc3.13 -g

       pydoc3.13 -w module [...]

DESCRIPTION
       pydoc3.13 name Show text documentation on something.  name may be the name of a Python keyword, topic, function, module, or package, or a dotted reference to a class or function within a module or
       module in a package.  If name contains a '/', it is used as the path to a Python source file to document. If name is 'keywords', 'topics', or 'modules', a listing of these things is displayed.

       pydoc3.13 -k keyword Search for a keyword in the synopsis lines of all available modules.

       pydoc3.13 -p port Start an HTTP server on the given port on the local machine.

       pydoc3.13 -g Pop up a graphical interface for finding and serving documentation.

       pydoc3.13 -w name [...]  Write out the HTML documentation for a module to a file in the current directory.  If name contains a '/', it is treated as a filename; if it names a directory, documenta‚Äê
       tion is written for all the contents.
