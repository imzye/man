NAME
       fmidi-grep – search for pattern matches in one or more input files

SYNOPSIS
       fmidi-grep [OPTION ...]  PATTERN INPUT [INPUT ...]

       Searches  for pattern matches in one or more INPUT files and writes re‐
       sults on stdout.

OPTIONS
       -r,-R  recursive

       -E     extended pattern

       -F     fixed string pattern

SEE ALSO
       fmidi-convert(1), fmidi-play(1), fmidi-read(1), fmidi-seq(1)
