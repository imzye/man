NAME
       utop-full - Universal toplevel for OCaml

SYNOPSIS
       utop [ options ] [ object-files ] [ script-file ]

DESCRIPTION
       utop-full is the same as utop (1) except that compiler libraries are available.

OPTIONS
       Same as utop(1).

FILES
       Same as utop(1).
