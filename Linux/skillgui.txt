NAME
       skillgui - Monitor and instruct the skiller remotely

SYNOPSIS
       skillgui

DESCRIPTION
       This GUI application allows for monitoring the skiller and luaagent
       plugins, and instructig the skiller plugin. It reads data from the
       blackboard and uses Graphviz to visualize the finite state machines
       graphs of active skills.
