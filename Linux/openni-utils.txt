NAME
       niLicense - register a license key for OpenNI middleware modules

SYNOPSIS
       niLicense [FILE]

DESCRIPTION
       This manual page documents briefly the niLicense command.

       niLicense  is  a  utility for registering and unregistering license keys for the OpenNI frameworks.  It is used both for 'middleware' modules like the PrimeSense NITE skeleton and gesture tracking
       module.
