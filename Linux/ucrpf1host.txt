NAME
       ucrpf1host - host program for Panowin F1 3D printer

SYNOPSIS

       ucrpf1host

DESCRIPTION
       This manual page documents briefly the ucrpf1host command.

       This manual page was written for the Debian distribution because the original program does not have a manual page.

       ucrpf1host is a program that can control Panowin F1 3D printer.

BUGS
       The upstreams BTS can be found at https://github.com/grandpaul/ucrpf1host/issues.
