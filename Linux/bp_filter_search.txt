NAME
       bp_filter_search - filters searchio results, outputting a tab delimited summary

SYNOPSIS
         #bp_filter_search -format blast -score 200 < search.bl > search.tab

DESCRIPTION
       This script filters searchio results allowing a number of different filters to be applied before outputting to stdout in a tab delimited format.
