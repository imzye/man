NAME
       cdr2epub - manual page for cdr2epub 0.9.6

SYNOPSIS
       cdr2epub [OPTIONS] INPUT OUTPUT

DESCRIPTION
       `cdr2epub' converts CorelDRAW documents to ePub.

OPTIONS
       --help show this help message

       --version
              print version and exit
