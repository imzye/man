       ecpmulti - manual page for ecpmulti 1.0

       Project Trellis - Open Source Tools for ECP5 FPGAs Version 1.0-0.21.20210904git03e0070.fc35 ./ecpmulti: ECP5 multiboot bitstream assembler

   Allowed options:
       -h [ --help ]
              show help

       -v [ --verbose ]
              verbose output

       --db arg
              Trellis database folder location

       --input arg
              input bitstream file 0..N

       --address arg
              address to place next bitstream at [1..N]

       --flashsize arg
              Flash size in Mbits, e.g. 2, 4, 8, ..., 128

       --input-idcode arg
              IDCODE override for input file

       --output-idcode arg
              IDCODE override in output bitstreams

       --output arg
              output bitstream file
