NAME
       ykval-import-clients - Imports Yubikey client data into the yubikey-val server.

SYNOPSIS
       ykval-import-clients

DESCRIPTION
       Reads yubikey-val Client Info data from stdin and imports it into the yubikey-val servers database. Use ykval-export-clients to export data into a suitable format.

       Database configuration is read from /etc/yubico/val/ykval-config.php

BUGS
       Report ykval-import-clients bugs in the issue tracker ⟨https://github.com/Yubico/yubikey-val/issues⟩

SEE ALSO
       ykval-export-clients(1)

       The Client Info data format ⟨https://developers.yubico.com/yubikey-val/doc/ClientInfoFormat.html⟩

       YubiKeys can be obtained from Yubico ⟨https://www.yubico.com/⟩.
