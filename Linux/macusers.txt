NAME
       macusers - List the users connecting via AFP

SYNOPSIS
       macusers

       macusers -v | -version | --version | -h | -help | --help

DESCRIPTION
       macusers list the users connecting via AFP.

OPTIONS
       -v, -version, --version
           Show version and exit

       -h, -help, --help
           Display the help and exit

SEE ALSO
       afpd(8)
