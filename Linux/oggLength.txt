NAME
       oggLength - gives the length of an ogg video/audio file

SYNOPSIS
       oggLength [options] inputfile.ogv

DESCRIPTION
       oggLength  analyses  an  ogg  audio/video file and gives the duration of a file. If there is more than one stream available, the end time position of the longest stream is printed. The return value is
       given by milliseconds.

       oggLength does not care for skeleton information.

OPTIONS
       -v     prints out the length of the vorbis stream.

       -t     prints out the length of the theora stream.
