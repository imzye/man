NAME
       tsstpm2pem - Runs TPM2 tpm2pem

DESCRIPTION
       tpm2pem

       Converts an RSA or EC TPM2B_PUBLIC to PEM

       -ipu   public key input file in TPM format

       -opem  public key output file in PEM format
