NAME
       qtvr2erect - Utility to convert a QTVR file to equirectangular

Synopsis
         qtvr2erect myqtvr.mov

DESCRIPTION
       Given one or more QTVR .mov files, extracts cube-faces with qtvr2img (a tool shipped with freepv) and assembles them into a single equirectangular image using nona (a tool shipped with hugin).

Calling syntax
         qtvr2erect <MOV-file> [<MOV-file>] ...
