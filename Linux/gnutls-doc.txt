NAME
       gnutls_get_system_config_file - API function

SYNOPSIS
       #include <gnutls/gnutls.h>

       const char * gnutls_get_system_config_file( void);

ARGUMENTS
        void

DESCRIPTION
       Returns the filename of the system wide configuration file to be loaded by the library.

RETURNS
       a constant pointer to the config file path

SINCE
       3.6.9
