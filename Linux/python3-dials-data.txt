NAME
       dials.data - DIALS regression data manager

DESCRIPTION
       usage: dials.data <command> [<args>]

       DIALS regression data manager v2.4.0-dev

   The most commonly used commands are:
       list   List available datasets

       get    Download datasets

       Each command has its own set of parameters, and you can get more information by running dials.data <command> --help

   options:
       -h, --help
              show this help message and exit
