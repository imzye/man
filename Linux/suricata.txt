NAME
       suricatactl - Suricata Control

SYNOPSIS
       suricatactl [-h] <command> [<args>]

DESCRIPTION
       This tool helps control Suricata's features.

OPTIONS
       -h

       Get help about the available commands.

COMMANDS
       suricatactl-filestore(1)

BUGS
