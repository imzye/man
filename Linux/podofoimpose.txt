NAME
       podofoimpose - A powerful PDF imposition tool

SYNOPSIS
       podofoimpose Input Output Plan [Interpreter]

DESCRIPTION
       podofoimpose is one of the command line tools from the PoDoFo library that provide several useful operations to work with PDF files. It can do imposition of the final output according to the specified
       imposition plan.

OPTIONS
       Input
              PDF file or a file which contains a list of PDF file paths.

       Output
              Resulting PDF file.

       Plan
              Imposition plan file.

       [Interpreter]
              Imposition interpreter. It can be "native" (default value) or "lua".
