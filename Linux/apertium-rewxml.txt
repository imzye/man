NAME
     apertium-rewxml — WXML format processor for Apertium

SYNOPSIS
     apertium-rewxml [input_file [output_file]]

DESCRIPTION
     apertium-rewxml is an WXML format processor.  It restores the original WXML formatting the text had before being passed through the apertium-deswxml(1) deformatter.

OPTIONS
     -h, --help
             Display this help.

SEE ALSO
     apertium(1), apertium-rertf(1), apertium-retxt(1), lt-proc(1)
