NAME
       bdfimplode - Convert a BDF font into GNU Unifont .hex format

SYNOPSIS
       bdfimplode < input-font.bdf > output-font.hex

DESCRIPTION
       bdfimplode reads a BDF font from STDOUT and writes GNU Unifont .hex conversion of the font to STDOUT.

FILES
       *.bdf font files
