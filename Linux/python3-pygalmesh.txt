NAME
       pygalmesh - create high-quality meshes with ease

DESCRIPTION
       usage: pygalmesh [-h] [--version]

              {from-inr,remesh-surface,volume-from-surface} ...

       pygalmesh command-line tools

   optional arguments:
       -h, --help
              show this help message and exit

       --version, -v
              display version information

   subcommands:
              {from-inr,remesh-surface,volume-from-surface}

       from-inr
              Create volume mesh from INR voxel files

       remesh-surface
              Remesh surface mesh

       volume-from-surface
              Generate volume mesh from surface mesh

       pygalmesh is a Python frontend to CGAL's 2D and 3D mesh generation capabilities.  pygalmesh makes it easy to create high-quality 2D, 3D volume meshes, periodic volume meshes, and surface meshes.
