NAME
       xineplayer - Plays videos via vdr-plugin-xine

SYNOPSIS
       xineplayer [ --vdr-xine-instance=N ] [ options ] mrl

DESCRIPTION
       xineplayer  plays videos via the VDR xine plugin. Vide playback can then be controlled via VDR's OSD.  For integration of xineplayer with vdr-plugin-mplayer,
       please refer to /usr/share/doc/vdr-plugin-xine/MANUAL.gz.

OPTIONS
       --vdr-xine-instance
              Instance number as used by vdr-plugin-xine via the '-i' parameter in etc/vdr/plugins/plugin.xine.conf
