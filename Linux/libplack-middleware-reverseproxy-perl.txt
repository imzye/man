NAME
       Plack::Middleware::ReverseProxy - Supports app to run as a reverse proxy backend

SYNOPSIS
         builder {
             enable_if { $_[0]->{REMOTE_ADDR} eq '127.0.0.1' }
                 "Plack::Middleware::ReverseProxy";
             $app;
         };

DESCRIPTION
       Plack::Middleware::ReverseProxy resets some HTTP headers, which changed by reverse-proxy. You can specify the reverse proxy address and stop fake requests using 'enable_if' directive in your
       app.psgi.
