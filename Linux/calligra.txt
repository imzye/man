NAME
       Calligra - Calligra Document Opener

SYNOPSIS
          calligra [options] FILES

DESCRIPTION
       Calligra is an integrated office suite for KDE. It offers a word processor, a spreadsheet, a presentation program, graphics tools, and more.

OPTIONS
       file  File to open

       Generic options:

       --help Show help about options

       --author
              Show author information

       -v, --version
              Show version information

       --license
              Show license information

       Options:

       --apps Lists names of all available Calligra applications

SEE ALSO
       More detailed user documentation is available from help:/calligra (either enter this URL into Konqueror, or run khelpcenter help:/calligra).
