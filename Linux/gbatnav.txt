NAME
       gbnserver - GNOME server for Batalla Naval

SYNOPSIS
       gbnserver [options]

DESCRIPTION
       gbnserver is a GNOME server for Batalla Naval

OPTIONS
       -p, --port=ARG
              Port number. Default is 1995

USAGE
       Just run it
