NAME
       pikchr-cli - PIC-like diagramming language to SVG converter

SYNOPSIS
       pikchr-cli <PIKCHR>

DESCRIPTION
       PIC-like diagramming language to SVG converter

   Arguments:
       <PIKCHR>
              Pikchr file to convert to SVG

OPTIONS
       -h, --help
              Print help

       -V, --version
              Print version
