NAME
       Catalyst::TraitFor::Request::BrowserDetect - Browser detection for
       Catalyst::Requests

VERSION
       version 0.02

SYNOPSIS
           package MyApp;

           use Moose;
           use namespace::autoclean;

           use Catalyst;
           use CatalystX::RoleApplicator;

           extends 'Catalyst';

           __PACKAGE__->apply_request_class_roles(qw/
               Catalyst::TraitFor::Request::BrowserDetect
           /);

           __PACKAGE__->setup;

DESCRIPTION
       Extend request objects with a method for browser detection.

ATTRIBUTES
   browser
           my $browser = $ctx->request->browser;

       Returns an "HTTP::BrowserDetect" instance for the request. This allows
       you to get information about the client's user agent.
