NAME
       selectwm - select a window manager at X startup

SYNOPSIS
       selectwm [options]

DESCRIPTION
       This manual page documents briefly the selectwm command.  This manual
       page was written for the Debian GNU/Linux distribution

       selectwm is a program that allows you to pick a window manager upon X
       startup, optionally set a default windowmanager with a timer, and
       optionally return you to the selection menu after the windowmanager
       exits. The menu can be edited from within selectwm, through a right-
       mouse click on the listbox.

OPTIONS
       This program follow the usual GNU command line syntax, with long
       options starting with two dashes (`-').  A summary of options is
       included below.

       -h, --help
              Show summary of options.

       -v, --version
              Show version of program.

       -d, --DontSaveOnExit
              Do not save configuration file on exit.

       -c, --config configuration file
              Specify a configuration file instead of ~/.selectwmrc.

FILES
       ~/.selectwmrc
              Default configuration file.

SEE ALSO
