NAME
       gnmsgid - standalone msgid generator

SYNOPSIS
       gnmsgid [-h] [-v] [-seqdir <path>] [-seqoutrun <outrun>] [<num>]

DESCRIPTION
       gnmsgid print new msgid to stdout.

OPTIONS
       [-h] print usage information and exit
       [-v] print version and exit
       [-seqdir <path>] specify msgid sequencer directory (see husky docs)
       [-seqoutrun <outrun>] specify msgid outrun value (see husky docs)
       [<num>] number msgid to generate
