NAME
       mlkit-mllex - lexer generator for Standard ML.


SYNOPSIS
       mlkit-mllex file.lex


DESCRIPTION
       When invoked, mlkit-mllex reads the file file.lex and outputs a lexer
       (written in Standard ML) in the file file.lex.sml

       To learn more about using mlkit-mllex , see the manual "A Lexical
       Analyzer Generator for Standard ML" available as
       /usr/local/share/doc/mlkit/mllex.pdf
