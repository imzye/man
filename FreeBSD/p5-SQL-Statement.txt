SQL::Dialects::ANSI(3)                     User Contributed Perl Documentation


NAME
       SQL::Dialects::ANSI

SYNOPSIS
         use SQL::Dialects::ANSI;
         $config = SQL::Dialects::ANSI->get_config();

DESCRIPTION
       This package provides the necessary configuration for ANSI SQL, as far
       it's supported.

FUNCTIONS
   get_config
       Returns the configuration for ANSI SQL. The configuration is delivered
       in ini-style:

         [VALID COMMANDS]
         ...

         [VALID OPTIONS]
         ...

         [VALID COMPARISON OPERATORS]
         ...

         [VALID DATA TYPES]
         ...

         [RESERVED WORDS]
         ...
