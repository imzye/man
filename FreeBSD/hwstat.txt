NAME
       HWSTAT - command line tool to display CPU temp and battery infos

SYNOPSIS
       hwstat [-hvdcf]

DESCRIPTION
       HWSTAT is a command line utility to display coretemp(4) and acpi infos.

   OPTIONS
       -h     show help usage

       -v     show version information

       -f     display the temperature in Fahrenheit

       -k     display the temperature in Kelvins

       -d     also display non working devices
