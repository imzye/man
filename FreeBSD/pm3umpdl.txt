NAME
       pm3umpdl - Loads a pls file into mpd, musicpd.

SYNOPSIS
       pm3umpdl -f <file> [-c] [-l] [-p]

FLAGS
       -c  If set, it will clear the playlist first.

       -f  This arguement is required and is the file to load.

       -l  If set, along with -p, this will play the last entry.

       -p  If set, it

REQUIREMENTS
       mpc http://www.musicpd.org/mpc.shtml
