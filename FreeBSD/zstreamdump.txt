NAME
     zstreamdump – filter data in zfs send stream

SYNOPSIS
     zstreamdump [-C] [-d] [-v]

DESCRIPTION
     The zstreamdump utility reads from the output of the "zfs send" command,
     then displays headers and some statistics from that output. See zfs(8).

     The following options are supported:

     -C      Suppress the validation of checksums.

     -d      Dump contents of blocks modified, implies verbose.

     -v      Verbose. Dump all headers, not only begin and end headers.

SEE ALSO
     zfs(8)

HISTORY
     The zstreamdump utility first appeared in FreeBSD 7.0.
