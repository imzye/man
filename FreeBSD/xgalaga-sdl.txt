NAME
       xgalaga-sdl - SDL port of XGalaga, version of the famous Galaga game

SYNOPSIS
       xgalaga [options]

       xgalaga-hyperspace [options]

INSTRUCTIONS
       Use the arrows on a keyboard, or a joystick to move the ship, and the
       space bar or a button to fire. Destroy the alien wave and move on to
       the next.

       The xgalaga-hyperspace version features some new weapons and different
       gameplay.

OPTIONS
       -scores Prints out the high score file and exits

       -nosound
               Turn sound OFF

       -level lvl
               Choose starting level (>= 1)

       -window Run in a window, not full screen. Full screen is the default.

       -winsize widthxheight
               Set window size.

KEYBOARD COMMANDS
       p      - Pause

       ESC, q - End game and exit xgalaga

       space  - Start game, fire

       s      - Toggle sound

       x      - Use your shields (xgalaga-hyperspace only)

       alt-enter - Toggle full screen node.
