NAME
       ishido - X11 tile game

SYNOPSIS
       ishido [-bonehead] [-silent] [-h]

DESCRIPTION
       A tile game which challenges you to place tiles of six colors and six
       shapes onto a 12x8 grid.

OPTIONS
       -bonehead
              Shows possible moves.

       -silent
              Turns off bell.

       -h     Usage.

FILES
       /usr/local/share/misc/ishido_scores
              The high scores file.

DIAGNOSTICS
       (none)

BUGS
       (none)
