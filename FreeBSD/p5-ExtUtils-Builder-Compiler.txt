ExtUtils::Builder::ArgumentCollector(3)    User Contributed Perl Documentation


NAME
       ExtUtils::Builder::ArgumentCollector - Helper role for argument
       collecting classes

VERSION
       version 0.032

DESCRIPTION
       This is a helper role for classes that collect arguments for their
       command. Classes that use this include ExtUtils::Builder::Compiler and
       ExtUtils::Builder::Linker
