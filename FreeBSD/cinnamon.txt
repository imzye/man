NAME
       cinnamon-launcher - Start or restart a cinnamon session

SYNOPSIS
       cinnamon-launcher

DESCRIPTION
       cinnamon-launcher runs the graphical user interface allowing to start
       or restart a session of cinnamon desktop.

       cinnamon(1)
