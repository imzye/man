G_PROVIDER_BY_NAME(9)  FreeBSD Kernel Developer's Manual G_PROVIDER_BY_NAME(9)

NAME
     g_provider_by_name – find GEOM provider with given name

SYNOPSIS
     #include <geom/geom.h>

     struct g_provider *
     g_provider_by_name(const char *name);

DESCRIPTION
     The g_provider_by_name() function searches for a provider called name and
     returns the structure g_provider bound to it.  Argument name should be a
     name, not a full path (i.e., “da0”, instead of “/dev/da0”).

RESTRICTIONS/CONDITIONS
     The topology lock has to be held.

RETURN VALUES
     The g_provider_by_name() function returns a pointer to the provider
     called name or NULL if there is no such provider.
