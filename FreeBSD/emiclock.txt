NAME
       emiclock - hyper animated face analog clock for X11

SYNOPSIS
       emiclock [-toolkitoption ...] [-nossound] [-nosave]
            [-noscache] [-noshape] [-perfect] [-grayscale]
            [-gray4] [-mono] [-help] [-version]

DESCRIPTION
       Emi Clock is a hyper animated face analog clock with pretty characters.

       Click the left button to display menu.

FILES
       $HOME/.emiclockrc - system environment file

SEE ALSO
       X(1), X Toolkit Intrinsics, Athena Widget Set
