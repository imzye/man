NAME
       Sys::Group::GIDhelper - Helps for locating free GIDs.

VERSION
       Version 0.0.2

SYNOPSIS
           use Sys::Group::GIDhelper;

           #implements it with the default values
           my $foo = Sys::Group::GIDhelper->new();

           #sets the min to 0 and the max to 4000
           my $foo = Sys::Group::GIDhelper->new({max=>'0', min=>'4000'});

           #finds the first free one
           my $first = $foo->firstfree();
           if(defined($first)){
               print $first."\n";
           }else{
               print "not found\n";
           }

           #finds the first last one
           my $last = $foo->lastfree();
           if(defined($last)){
               print $last."\n";
           }else{
               print "not found\n";
           }

FUNCTIONS
   new
       This initiates the module. It accepts one arguement, a hash. Please See
       below for accepted values.

       min

       The minimum GID.

       max

       The maximum GID.

   firstfree
       This finds the first free GID. If it returns undef, no free ones were
       found.

   lastfree
       This finds the first last UID. If it returns undef, no free ones were
       found.
