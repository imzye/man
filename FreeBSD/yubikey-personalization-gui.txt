yubikey-personalization-gui(1)                 FreeBSD General Commands Manual

NAME
       yubikey-personalization-gui - personalize YubiKey OTP tokens

SYNOPSIS
       yubikey-personalization-gui

DESCRIPTION
       Graphical interface for configuration fo YubiKey OTP tokens. Help is
       available from within the program.

BUGS
       Report yubikey-personalization-gui bugs in the issue tracker ⟨URL:
       https://github.com/Yubico/yubikey-personalization-gui/issues ⟩

SEE ALSO
       YubiKeys can be obtained from Yubico ⟨URL: https://www.yubico.com/ ⟩.
