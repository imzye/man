NAME
       Crypt::OFB - Encrypt Data using OFB Mode

SYNOPSIS
       Use Crypt::OFB OO style

         use Crypt::OFB;

         $crypt = Crypt::OFB->new;
         $crypt->padding(PADDING_AUTO);
         $crypt->cipher("Blowfish") || die $crypt->errstring;
         $crypt->key("some_key");

         $enc = $crypt->encrypt("Some data.");
         print $crypt->decrypt($enc);

DESCRIPTION
       This module is a Perl-only implementation of the OFB mode.  It is a
       inheritance class of Crypt::ECB.  Please read Crypt::ECB(3) for the
       default function description.
