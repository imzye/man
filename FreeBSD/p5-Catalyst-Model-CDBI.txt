NAME
       Catalyst::Helper::Model::CDBI - Helper for CDBI Models

SYNOPSIS
           script/create.pl model CDBI CDBI dsn user password

DESCRIPTION
       Helper for CDBI Model.

   METHODS
       mk_compclass
           Reads the database and makes a main model class as well as
           placeholders for each table.

       mk_comptest
           Makes tests for the CDBI Model.

       Catalyst::Manual, Catalyst::Test, Catalyst::Request,
       Catalyst::Response, Catalyst::Helper
