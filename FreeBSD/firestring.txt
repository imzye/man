NAME
       firestring_chomp - trim excess whitespace from the end of a string

SYNOPSIS
       #include <firestring.h>
       -lfirestring

       char *firestring_chomp(char *s)

DESCRIPTION
       firestring_chomp() removes any whitespace at the end of the string s.
       The string is modified in place and returned as well.

RETURN VALUE
       s after having been modified.
