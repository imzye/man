NAME
       SMS::Send::TW::emome - SMS::Send driver for www.emome.net

SYNOPSIS
         use SMS::send;

         my $sender = SMS::Send->new('TW::emome',
                         _username   => 'UserName',
                         _password   => 'Password',
                         _language   => '1',           # 1:Chinese  2:English
                       );

         my $sent = $sender->send_sms(
                         text => 'My very urgent message',
                         to   => '0912345678',
                    );

DESCRIPTION
       SMS::Send::TW::emome is a SMS::Send driver which allows you to send
       messages through <http://www.emome.net/>.

METHODS
   new
       The "new" method takes a few parameters. "_username" , "_password" ,
       and "_language" > are mandatory.

   send_sms
       Takes "to" as recipient phonenumber, and "text" as the text that's
       supposed to be delivered.
