NAME
       CouchDB::View - handle and create CouchDB views in Perl

VERSION
       Version 0.003

DESCRIPTION
       See CouchDB::View::Document for a convenient interface to defining
       CouchDB views in Perl.

       See CouchDB::View::Server for instructions on how to use Perl as a
       CouchDB view server.

       You may also want to see the CouchDB home page
       <http://incubator.apache.org/couchdb/> or the CouchDB wiki
       <http://wiki.apache.org/couchdb/>.
