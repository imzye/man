NAME
       Crypt::OICQ - cryptographic algorithm used by OICQ protocol

SYNOPSIS
         use Crypt::OICQ;
         $oicq = new Crypt::OICQ;

DESCRIPTION
   EXPORT
       None by default.

       encrypt and decrypt may be exported.
