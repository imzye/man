NAME
       X11::IdleTime - Get the idle time of X11

SYNOPSIS
          use X11::IdleTime;

          $idle = GetIdleTime();

          print "Your mouse and keyboard have been idle for $idle seconds.\n";

DESCRIPTION
       The X11::IdleTime module is useful for checking how long the user has
       been idle.
