NAME
       Scalar::Util::LooksLikeNumber - Access to looks_like_number() perl API
       function

SYNOPSIS
           use Scalar::Util::LooksLikeNumber;
           print Scalar::Util::LooksLikeNumber::looks_like_number(1); # -> 4352

DESCRIPTION
       "Scalar::Util::LooksLikeNumber" contains looks_like_number() like
       "Scalar::Util"'s looks_like_number(), except it returns the raw value
       from the C function. Scalar::Util used to do this also, but it returns
       a booleanized value since 1.39.

FUNCTIONS
   $res = looks_like_number( $var )
       Returns a non-zero if perl thinks $var is a number. See
       "looks_like_number" in perlapi.
