NAME
       synthpod - a lightweight nonlinear LV2 plugin container


SYNOPSIS
       synthpod_jack [options] [bundle-path]


DESCRIPTION
       synthpod is a lightweight nonlinear LV2 plugin container, aka host.

       It is a pure stand-alone JACK client with JACK/NSM session support and
       JACK AUDIO/MIDI.

       It is also a first-class LV2 plugin (http://open-music-
       kontrollers.ch/lv2/synthpod#stereo).


OPTIONS
       -v

              Print version and license information


       -h

              Print usage information


       -g

              Load GUI


       -G

              Do NOT load GUI (default)


       -k

              Kill DSP with GUI


       -K

              Do NOT kill DSP with GUI (default)


       -t

              Run GUI in threaded mode


       -T

              Run GUI in separate process (default)


       -b

              Enable bad plugins


       -B

              Disable bad plugins (default)


       -a

              Enable CPU affinity


       -A

              Disable CPU affinity (default)


       -u

              Show alternate UI


       -l link-path

              Socket link path (shm:///synthpod), e.g. tcp://*:9090


       -n server-name

              Connect to named JACK daemon


       -s sequence-size

              Minimal byte size of event sequence buffers (8192)


       -c slave-cores

              Number of slave cores for parallel audio processing (auto)


       -f update-rate

              Update rate in frames per second of GUI


FILES
       $HOME/.lv2/Synthpod_default.preset.lv2
              Default bundle state directory

       $HOME/.lv2
              Default LV2 preset directory
