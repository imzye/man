NAME
       MojoX::Renderer::Haml - Mojolicious renderer for HAML templates.

SYNOPSIS
          my $haml = MojoX::Renderer::Haml->build(%$args, mojo => $app);

          # Add "haml" handler
          $app->renderer->add_handler(haml => $haml);

DESCRIPTION
       This module is a renderer for HTML::Haml templates. normally, you just
       want to use Mojolicious::Plugin::HamlRenderer.
