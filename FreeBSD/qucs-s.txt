NAME
       Qucs-S - An universal GUI for SPICE and not-SPICE circuit simulators

SYNOPSIS
       qucs-s [OPTION] ...

DESCRIPTION
       Qucs-S is an unversal tool that provides GUI for different SPICE and
       not-SPICE circuit simulators. The following SPICE simulators are
       supported: Ngspice, XYCE, and SpiceOpus. Also it allows to use not-
       SPICE simulation kernel Qucsator as simulation backends.



AVAILABILITY
       The latest version of Qucs-S can always be obtained from
       www.sourceforge.net or www.freshmeat.net
