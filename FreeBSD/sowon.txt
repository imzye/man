NAME
     sowon – Starting soon timer

SYNOPSIS
     sowon [-p] [seconds | clock]

DESCRIPTION
     sowon is a graphical countdown/timer program.  With no arguments
     provided, it starts in ascending mode. With an amount of seconds
     provided, it starts in descending mode. Clock starts it in clock mode,
     that displays the current time.

OPTIONS
     -p      start in paused state

KEY BINDINGS
     SPACE   Toggle pause

     =       Zoom in

     -       Zoom out

     0       Zoom 100%

FILES
     /usr/local/bin/sowon
