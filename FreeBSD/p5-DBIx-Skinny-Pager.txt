NAME
       DBIx::Skinny::Mixin::Pager

SYNOPSIS
         package Proj::DB;
         use DBIx::Skinny;
         use DBIx::Skinny::Mixin modules => ['Pager'];

         package main;
         use Proj::DB;

         my $rs = Proj::DB->resultset_with_pager('MySQLFoundRows');
         # $rs is DBIx::Skinny::Pager::Logic::MySQLFoundRows

       or

         my ($iter, $pager) = Proj::DB->search_with_pager({ foo => "bar" }, { pager_logic => "PlusOne" });

DESCRIPTION
       DBIx::Skinny::Mixin::Pager is a interface for mixin
       resultset_with_pager method to DBIx::Skinny.  resultset_with_pager
       return DBIx::Skinny::Pager object.
