NAME
       Mahjongg - Match tiles and clear the board

SYNOPSIS
       mahjongg [OPTION...]

DESCRIPTION
       Single-player tile-matching game, commonly known as Mahjong solitaire.

OPTIONS
       -v, --version
              Print release version and exit

       This program also accepts the standard GNOME and GTK options.

AUTHORS
       mahjongg was written by Francisco Bustamante, Max Watson and many
       others.

       This manual page was written by Sven Arvidsson <sa@whiz.se>, for the
       Debian project (but may be used by others).
