SYNOPSIS
     lscpu [-h|--help]

DESCRIPTION
     lscpu is a utility that displays CPU information for the system.

     The options are as follows:

     -h|--help
             Print usage information and exit.

EXIT STATUS
     The lscpu utility exits 0 on success, and >0 if an error occurs.
