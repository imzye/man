NAME
       Template::Plugin::Filter::Minify::CSS::XS - CSS::Minifier::XS filter
       for Template Toolkit

VERSION
       version 0.96

SYNOPSIS
         [% USE Filter.Minify.CSS.XS %]

         [% FILTER minify_css %]
           .foo {
               color: #aabbcc;
               margin: 0 10px 0 10px;
           }
         [% END %]

DESCRIPTION
       This module is a Template Toolkit filter, which uses CSS::Minifier::XS
       to minify css code from filtered content during template processing.

SEE ALSO
       CSS::Minifier::XS, Template::Plugin::Filter, Template
