NAME
       AnyEvent::Graphite - A non-blocking Graphite
       (http://graphite.wikidot.com/) client

VERSION
       Version 0.08

SYNOPSIS
           my $graphite = AnyEvent::Graphite->new(
               host => '127.0.0.1',
               port => '2003',
           );
           $graphite->send("a.b.c.d", $value, $timestamp); #timestamp is optional
