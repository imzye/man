NAME
     VOP_BWRITE â€“ write a file system buffer

SYNOPSIS
     #include <sys/param.h>
     #include <sys/vnode.h>

     int
     VOP_BWRITE(struct vnode *vp, struct buf *bp);

DESCRIPTION
     The arguments are:

     vp  The vnode of the file being written to.

     bp  The buffer to be written.

RETURN VALUES
     Zero is returned on success, otherwise an error is returned.
