NAME
       App::SD - a distributed ticket tracking system

DESCRIPTION
       SD is a peer to peer ticket tracking system built on the Prophet
       distributed database. SD is designed to make it easy to work with
       tickets and to share ticket databases with your collaborators.

       Please run "sd help" for documentation and "sd help intro" for an
       introduction to SD.
