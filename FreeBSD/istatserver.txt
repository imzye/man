NAME
     istatserver â€“ system monitoring daemon

SYNOPSIS
     istatserver [-c config] [-a address] [-p port] [-d]



DESCRIPTION
     istatserver is a system monitoring daemon that collects and stores
     performance monitoring data.  You can then use iStat for iOS and iStat
     for macOS to remotely monitor your computer.


OPTIONS
     -v             Print version number

     -d             Run in the process in background

     -a address     Listen for network connections on this address

     -p port        Listen for network connections on this port

     -u user        User to run daemon as

     -g group       Group to run daemon as

     --pid path     Custom pid file location

     --clear-sessions
                    Reset list of authorized devices.

     --verify       Verify sqlite history database.

FILES
     /usr/local/etc/istatserver/istatserver.conf

     Configuration for passcode, server port and more.
