     acdplay â€“ simple console CD player using autocd

SYNOPSIS
     acdplay [-h] [-d device] [-s socket]

DESCRIPTION
     acdplay is a simple console CD player using autocd(8) daemon.

OPTIONS
     The following options are available:

     -h      display the acdplay help information.

     -d device
             choose device for control.

     -s socket
             path to a control socket.

FILES
     /tmp/autocd
               default location of a control socket.

SEE ALSO
     autocd(8), cdctl(1)
