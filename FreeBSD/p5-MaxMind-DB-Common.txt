NAME
       MaxMind::DB::Common - Code shared by the MaxMind DB reader and writer
       modules

VERSION
       version 0.040001

DESCRIPTION
       This distribution provides some shared code for use by both the MaxMind
       DB reader and writer Perl modules.

       For now, the only piece documented for public consumption is
       MaxMind::DB::Metadata.

VERSIONING POLICY
       This module uses semantic versioning as described by
       <http://semver.org/>. Version numbers can be read as X.YYYZZZ, where X
       is the major number, YYY is the minor number, and ZZZ is the patch
       number.

SUPPORT
       Please report all issues with this code using the GitHub issue tracker
       at <https://github.com/maxmind/MaxMind-DB-Common-perl/issues>.
