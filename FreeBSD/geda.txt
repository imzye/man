NAME
       convert_sym - convert a Viewlogic symbol/schematic to gEDA gschem
       format


SYNOPSIS
       convert_sym viewlogic_file


DESCRIPTION
       convert_sym converts a Viewlogic symbol/schematic to gEDA gschem
       format. It accepts one argument, the name of the file to convert;
       converted output is sent to stdout.
