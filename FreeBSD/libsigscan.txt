NAME
     sigscan â€“ scans a file for binary signatures

SYNOPSIS
     sigscan [-hvV] source

DESCRIPTION
     sigscan is a utility to scan a file for binary signatures

     sigscan is part of the libsigscan package.  libsigscan is a library to
     scan a file for binary signatures

     source is the source file.

     The options are as follows:

     -h      shows this help

     -v      verbose output to stderr

     -V      print version

ENVIRONMENT
     None

FILES
     None

EXAMPLES
     # sigscan ...


DIAGNOSTICS
     Errors, verbose and debug output are printed to stderr when verbose
     output -v is enabled.  Verbose and debug output are only printed when
     enabled at compilation.
