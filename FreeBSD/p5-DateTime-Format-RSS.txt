NAME
       DateTime::Format::RSS - Format DateTime For RSS

SYNOPSIS
         use DateTime::Format::RSS;
         my $fmt = DateTime::Format::RSS->new;
         my $dt  = $fmt->parse_datetime($str);
         my $str = $fmt->format_datetime($dt);

         my $fmt = DateTime::Format::RSS->new(version => 2.0);

DESCRIPTION
       DateTime::Format::RSS attempts to deal with those nasty RSS date/time
       strings used in fields (such as <issued>, <modified>, <pubDate>) that
       never ever seems to be right.

METHODS
   new
       Creates a new DateTime::Format::RSS object

   parse_datetime SCALAR
       Internally, it just attempts to parse the string using
       DateTime::Format::Mail, DateTime::Format::ISO8601, and then finally
       with DateTime::Format::DateParse.

   format_datetime OBJECT
       Formats the given DateTime object using DateTime::Format::ISO8601

CREDITS
       This module was based on Plagger's LiberalDateTime DateTime parser.
