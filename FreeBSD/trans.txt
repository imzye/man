checkiso  --  8-bit Character Encoding Proofreader V1.20
1997-07-27

                                 checkiso

Function
        checkiso is a program that reads a Character Encoding Description
        File and checks the validity of the ISO 10646 names used in there.
        It also checks for duplicate names.  :)

Type
        (External) tool

Format
        checkiso src_code

              Positional parameters:

              src_code
                          the name of the source Encoding Description File

Example
        # check the description for ISO 8859-1
        checkiso iso8859.1
