NAME
       Dancer::Debug - Extend Plack::Middleware::Debug with some specific
       panels for Dancer

VERSION
       version 0.04

SYNOPSIS
       You can activate the panels in your development configuration file:

           plack_middlewares:
             Debug:
               - panels
               -
                 - Parameters
                 - Dancer::Version
                 - Dancer::Settings
                 - Dancer::Logger
                 - Dancer::Routes

       or in your app.psgi:

           $handler = builder {
               enable "Debug",
                   panels => [qw/Dancer::Settings Dancer::Logger Parameters Dancer::Version/];
               $handler;
           };

DESCRIPTION
       Dancer::Debug extends Plack::Middleware::Debug with some specific
       panels for Dancer.
