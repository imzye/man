NAME
       wmappl - a scrolling application launcher dockapp

SYNOPSIS
       wmappl [options]

DESCRIPTION
       wmappl is an application launcher dockapp which displays an small icon
       for each application it is configured to launch. Any number of
       applications may be configured.  Scroll arrows and optionally the
       scroll-wheel may be used to scroll through the list of configured
       applications.



OPTIONS
       -f, --rcfile <filename>
              absolute path of configuration file [default: ~/.wmapplrc]

       -i, --iconpath <path>
              a colon separated path to search for icons

       -n, --name <name>
              window name (specify different names to run multiple instances)
              [default: wmappl]

       -w, --window
              run in windowed mode, default is to run in withdrawn

       -d, --debug
              dump debug information to stderr (commands are not actually
              executed)

       -V, --version
              show program version and exit

       -h, --help
              show usage information
