NAME
       pudconvert - Tool to convert Warcraft II map pud files to Stratagus map
       format

SYNOPSIS
       pudconvert inputfile [outputdir]

OPTIONS
       inputfile
              Input Warcraft II pud file

       outputdir
              Output directory where will be created map files

SEE ALSO
       stratagus(6), wargus(6), wartool(6)
