NAME
       img2dev - Write a disk image file to a physical disk




SYNOPSIS
       img2dev image-file device



DESCRIPTION
       The img2dev command writes a disk image file to a disk device.  It is
       primarily intended as a convenience for writing FreeBSD install images
       to an external disk such as a USB flash drive.
