NOM
       ghostwriter - Editor de text lliure de distraccions per al Markdown

SINOPSI
       ghostwriter [FITXER]

DESCRIPCIÓ
       El ghostwriter és una aplicació Qt d'edició de text per al Markdown que
       proporciona un entorn d'escriptura estètic i lliure de distraccions.

       Creeu temes propis personalitzats, escriviu en mode de pantalla
       completa o utilitzeu el mode de focus per a concentrar-vos només en una
       o més línies.

       Una vista prèvia HTML en directe us permet veure el format final mentre
       escriviu. Les opcions de previsualització i exportació d'HTML són
       proporcionades pel processador Markdown integrat «cmark-gfm»; però
       podeu afegir-ne més instal⋅lant els processadors Pandoc, MultiMarkdown
       o «cmark». L'aplicació detectarà automàticament la seva instal⋅lació si
       les seves ubicacions es troben a la variable d'entorn PATH del sistema.

OPCIONS
       --disable-gpu
           Desactiva l'acceleració de la GPU.
