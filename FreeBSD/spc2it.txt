NAME
       spc2it - converts SPC700 sound files to the Impulse Tracker format

SYNOPSIS
       spc2it [options] filename

       Where <filename> is any .spc or .sp# file

DESCRIPTION
       This manual describes the spc2it command. It converts SPC files to IT
       (Impulse Tracker) files.

       Options:

       -t x   Specify a time limit in seconds        [60 default]

       -d xxxxxxxx Voices to disable (1-8)           [none default]

       -r xxx Specify IT rows per pattern            [200 default]

SEE ALSO
       adplay(1), schismtracker(1).
