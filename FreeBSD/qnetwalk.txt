NAME
       qnetwalk - The game for system administrators

SYNOPSIS
       qnetwalk [-novice | -amateur | -expert | -master] [-nosound] [-help]

DESCRIPTION
       You are the system administrator and your goal is to connect each
       computer to the central server. Click the right mouse's button for
       turning the cable in a clockwise direction, and left mouse's button for
       turning cable in a counter-clockwise direction. Start the LAN with less
       turns!

OPTIONS
       -novice
              Set the skill Novice.

       -amateur
              Set the skill Amateur.

       -expert
              Set the skill Expert.

       -master
              Set the skill Master.

       -nosound
              Disable all sound effects.

       -help  Show help about options.

ENVIRONMENT
       BROWSER
              Default browser

       USER   Default user name

FILES
       ~/.config/QNetWalk.conf
