NAME
       Template::Plugin::XML::Escape - Escape variables to suit being placed
       into XML

SYNOPSIS
         [% USE XML.Escape %]
         ...
         <foo bar="[% c.variable | xml_escape %]" />

DESCRIPTION
       Escapes XML entities from text, so that you don't fall prey to people
       putting quotes, less-than/greater-than, and ampersands, into variables
       that end up in TT templates.
