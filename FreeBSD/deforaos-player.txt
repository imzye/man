NAME
       player - Media player

SYNOPSIS
       player [file...]

DESCRIPTION
       player is a media player, able to read audio and video files alike.

OPTIONS
       A list of files to play and queue can be provided directly on the
       command line.

BUGS
       Issues can be listed and reported at
       http://www.defora.org/os/project/bug_list/336/Player.
