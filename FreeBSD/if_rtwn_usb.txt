     rtwn_usb â€“ Realtek USB device glue

SYNOPSIS
     To compile this driver into the kernel, place the following lines in your
     kernel configuration file:

           device xhci
           device ehci
           device uhci
           device ohci
           device usb
           device rtwn
           device rtwn_usb
           device wlan

DESCRIPTION
     This module provides the USB bus glue needed for the devices supported by
     the rtwn(4) driver.

HARDWARE

     RTL8188CUS/RTL8188RU/RTL8188EU/RTL8192CU/RTL8192EU/RTL8812AU/RTL8821AU

BUGS
     The rtwn_usb driver does not support any of the 802.11ac capabilities
     offered by the adapters.  Additional work is required in ieee80211(9)
     before those features can be supported.
