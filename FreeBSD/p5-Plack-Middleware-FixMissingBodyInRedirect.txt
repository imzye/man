Plack::Middleware::FixMissingBodyInRedirect(3)
                                           User Contributed Perl Documentation


NAME
       Plack::Middleware::FixMissingBodyInRedirect - Plack::Middleware which
       sets body for redirect response, if it's not already set

VERSION
       version 0.12

SYNOPSIS
          use strict;
          use warnings;

          use Plack::Builder;

          my $app = sub { ...  };

          builder {
              enable "FixMissingBodyInRedirect";
              $app;
          };

DESCRIPTION
       This module sets body in redirect response, if it's not already set.
