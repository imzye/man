NAME
       tleenx2 - tlen.pl client for X Window System.

SYNOPSIS
       tleenx2

DESCRIPTION
       TleenX2 is a tlen.pl (Polish IM protocol based on Jabber) client for X
       Window System with Gtk+-2.0 library. You will also need libtlen
       library, on which this program is based.

AVAILABILITY
       The latest version of tleenx2 can be obtained online from:

            http://tleenx.sourceforge.net/
