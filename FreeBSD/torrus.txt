NAME
       torrus - Commandline wrapper for Torrus utilities

SYNOPSIS
       torrus command [options...]

DESCRIPTION
       All Torrus utilities use this common command-line wrapper, which simply
       passes all arguments to the appropriate utility.

       Accepted commands:

       acledit
       cleanup
       clearcache
       collector
       compilexml
       configinfo
       configsnapshot
       devdiscover
       genddx
       genlist
       install_plugin
       monitor
       rrddir2xml
       schedulerinfo
       srvderive

OPTIONS
       Refer to approproate manpages for options for each Torrus command.

       torrus_rrddir2xml(1), torrus_schedulerinfo(1) torrus_srvderive(1)
