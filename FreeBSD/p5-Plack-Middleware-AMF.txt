NAME
       Plack::Middleware::AMF - The great new Plack::Middleware::AMF!

VERSION
       Version 0.01

SYNOPSIS
           use Plack::Builder;

           builder {
                   enable "AMF", path => qr/^\/amf\/gateway/;
                   $app
           };

DESCRIPTION
       Enable this middleware to allow your Plack-based application to handle
       Flash Remoting and Flex RPC.

CONFIGURATIONS
   path
   headers_handler
   message_handler

METHOD
   prepare_app
   call
