NAME
       Mozo - MATE Menu Editor

SYNOPSIS
       mozo

DESCRIPTION
       Mozo is a menu editor for MATE using the freedesktop.org menu
       specification. It is a fork of Alacarte.

BUGS
   Should you encounter any bugs, they may be reported at:
       http://github.com/mate-desktop/mozo/issues
