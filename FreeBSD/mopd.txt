NAME
     mopa.out â€“ Create MOP image from a a.out file

SYNOPSIS
     mopa.out infile outfile

DESCRIPTION
     mopa.out is used to convert an a.out file to a MOP-image.

     This program will check if a.out(5) machine-id is recognized. If not it
     will try to swap the a.out(5) header, and try again.

BUGS
     This program just supports the VAX machine-id for now.
