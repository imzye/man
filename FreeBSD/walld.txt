NAME
       walld - a wallpaper daemon

SYNOPSIS
       walld [-c config] [-t time] [-s source] [-m monitors] [ -v]

DESCRIPTION
       walld automatically changes the wallpaper. It sets the wallpaper
       through feh(1).  It changes the wallpaper by default every 30 minutes.

OPTIONS
       -c config
              Set a custom config

       -t time
              Set a custom interval to change wallpapers

       -s source
              Set a custom source to get wallpapers from in the config

       -m monitors
              Set the number of monitors your system has to display a diffrent
              wallpaper on each one

       -v     Get the program version

FILES
       ~/.walld
              A directory for walld files
       ~/.walld/.walldrc
              The default config file for walld see walldrc(5) for formatting.
       ~/.walld/images
              The default image directory for walld(1)
