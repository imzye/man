     a800 â€“ webcamd driver for AVerMedia AverTV A800 DVB-T receiver

SYNOPSIS
     a800

DESCRIPTION
     The a800 driver supports the AVerMedia AverTV A800 USB 2.0 DVB-T
     receiver.

     Devices using the a800 driver become available to other applications once
     webcamd(8) is loaded.

HARDWARE
     The a800 driver should work with the following USB devices. The
     usbconfig(8) command can be used to determine the VendorID:ProductID of a
     plugged in USB device.

     VendorID:ProductID
                     Device
     07ca:a800                               AverTV DVB-T
     07ca:a801                               AverTV DVB-T
