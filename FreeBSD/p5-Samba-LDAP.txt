NAME
       Samba::LDAP - Manage a Samba PDC with an LDAP Backend

VERSION
       This document describes Samba::LDAP version 0.05

SYNOPSIS
           use Carp;
           use Samba::LDAP;

           my $samba = Samba::LDAP->new()
           or croak "Can't create object\n";
           my $domain = $samba->get_local_sid();

DESCRIPTION
       Main functions.

       DEVELOPER RELEASE!

       BE WARNED - Not yet complete and neither are the docs!

INTERFACE
   new
       Creates a new Samba::LDAP object

   get_local_sid
       Tries to find Samba Domain SID, if not, returns undef.

   does_sid_exist
   get_dn_from_line
   do_ldapadd
   do_ldapmodify
   get_user_dn2
   connect_ldap_master
   connect_ldap_slave
   group_type_by_name
   subst_configvar
   read_config
   read_parameter
   split_arg_comma
   list_union
   list_minus
   get_next_id
   print_banner
   get_domain_name

DIAGNOSTICS
       None yet.

CONFIGURATION AND ENVIRONMENT
       Samba::LDAP requires no configuration files or environment variables.
