NAME
       qcc - ncurses frontend for qico ftn mailer.

SYNOPSIS
       qcc [-nhvPaw]

DESCRIPTION
       qcc is a graphical ncurses-based frontend for qico ftn mailer.

OPTIONS
       -Pport Connect to port.  default port is service 'qicoui' or 60178.

       -ahost Connect to host.  default host is localhost (127.0.0.1).

       -wpassword
              Use password for connect to server. also may request password
              from user, if password is '-' (-w-).

       -n     Start qcc in silent mode (without stupid beeps).

       -h     Print help message.

       -v     Print version message.
