NAME
       wmping - WindowMaker dockapp


SYNOPSIS
       wmping


DESCRIPTION
       wmping
        is a simple but very usefull program that allows to check host status
       in your network environment. It show "up" status for host that is
       avaliable (pings packets are retyurns to you) and "down" status for
       host that is not pinged.  Can be used for monitoring servers in the
       network.


OPTIONS
       -v     Displays program version.

       -geometry
              The standard X geometry specifications. Only X and Y position
              requests  are  honoured.  Any  height and width specifications
              will be ignored.

       -display
              Specify an X server other than the local display.
