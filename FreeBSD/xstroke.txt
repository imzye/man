NAME
       xstroke - full-screen gesture recognition for X

SYNOPSIS
       xstroke [-h, -help] [-d, -display displayname] [-g, -geometry geomspec]
       [-fg, -foreground fg_color] [-bg, -background bg_color] [-fn, -font
       font] [-a, -alphabet configuration file (default
       /etc/xstroke/alphabet)]

DESCRIPTION
       Xstroke is a gesture recognition program.

       Tap in the small docked window to toggle stroke recognition.

       When stroke recognition is enabled, stroke anywhere on the screen.

ENVIRONMENT
       DISPLAY To get the default host, display number, and screen.

AUTHOR
