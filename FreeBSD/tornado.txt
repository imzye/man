NAME
       tornado - Klon des original C64 Spiels.

SYNOPSIS
       tornado [--connect IP|--demo|--help|--server|--version]

DESCRIPTION
       Das Ziel des Spiels tornado ist es das Haus des Gegners mit den Kräften
       des Regens, Schnees, Hagels, Blitzes und des Tornados zu zerstören.

       Nachdem das Spiel gestartet wurde geben die beiden Spieler ihre Namen
       ein.  Wird ein Feld leer gelassen, so übernimmt der Computer diesen
       Spieler. Nach dem Start sind zwei Häuser mit den jeweiligen
       Spielernamen unter ihnen und eine Wolke am oberen Bildschirmrand zu
       sehen. Der Spieler, der am Zug is, hat ein Sternchen * neben seinem
       Namen.

       Aus dem Menü in der Bildschirmmitte wählt jeder Spieler sein Wetter
       Phänomen aus. Eine zweite Box erscheint wo man die Abweichung angeben
       kann. Vielleicht hast du ja den Wert Windgeschwindigkeit in der Mitte
       des unteren Bildschirmrands bemerkt mittlerweile. Idealerweise würde
       man einen Wert von 2 nehmen, wenn die Windgeschwindigkeit -2 beträgt.
       Der Regen/Hagel/Schnee/Tornado würde dann senkrecht nach unten fallen
       von der Wolke aus. Allerdings wird zufällig ausgewählt ob der Regen nun
       mehr links oder rechts and der Wolke erscheint. Zielen ist also nicht
       immer unbedingt so einfach wie es aussieht.

       Wenn du das Haus des Gegners (oder dein eigenes) triffst wird die
       Prozentanzeige darunter reduziert abhängig davon wieviel Schaden
       angerichtet wurde. Wenn der Zähler auf 0% ist, verliert derjenige
       Spieler.

       Die Tasten, die im Spiel benutzt werden sind:

       s  wählt Schnee
              Schnee ist eine defensive Waffe. Wenn du damit ein Haus triffst,
              entsteht darauf eine Schicht, die von Hagel nicht durchdrungen
              werden kann. Nur Regen kann diese Schicht entfernen. Blitz und
              Tornado sind nicht davon betroffen.

       r  wählt Regen
              Zerstört eine Reihe des Hauses. Kann verwendet werden um eine
              Reihe Schnee zu schmelzen.

       h  wählt Hagel
              Dies zerstört zwei Reihen des Hauses, kann aber nicht durch eine
              Schneeschicht dringen.

       l  wählt Blitz
              Spaltet das Haus in zwei Hälften. Nicht vom Wind beeinflusst.

       t  wählt Tornado
              Nicht so effektiv wie Blitz, aber der Tornado dringt auch bis
              zum Boden des Hauses vor.

       q  beendet das Spiel

       o  zeigt die Highscoreliste an

       c  zeigt den aktuellen Punktestand

OPTIONS
       Das Spiel kann mit den folgenden Kommandozeilenoptionen gestartet
       werden:

       --connect IP, Das Spiel connected mit dem angegebenen server (IP oder
       Hostname)

       --demo, Selbst laufender Demo Modus.

       --help, Kurzer Hilfe Bildschirm.

       --server, Startet im Server Modus und wartet auf Verbindungen von
       clients.

       --version, Zeigt die Versionsnummer an und beendet sich wieder.

FILES

NOTES
       Wenn das Spiel im Netzwerkmodus läuft sollte auf beiden Seiten die
       selbe Version laufen. Versionen vor 1.0pre4 funktionieren damit
       überhaupt nicht, weil dort die Netzwerkfunktionalität noch nicht
       vorhanden ist. Das Spiel startet mit unterschiedlichen Versionen, kann
       aber jederzeit abstürzen oder out-of-sync geraten.

       Der Tornado server läuft auf port 3000 TCP. (definiert in main.c)

BUGS
       Sein eigenes Haus zu zerstören erhöht auch die Punktezahl. :^)
