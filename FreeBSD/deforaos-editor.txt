NAME
       editor - Text editor

SYNOPSIS
       editor [-F] [filename]

DESCRIPTION
       editor is a text editor with a graphical user interface. It can be used
       as a filter as well: reading data from the standard input and writing
       it back to the standard output, together with any modification made.

OPTIONS
       The filename of a file to edit can be supplied directly on the command
       line. The following option is supported:

       -F
           Enables filter mode.
