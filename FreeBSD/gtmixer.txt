NAME
     gtmixer – graphical set/display soundcard mixer values


DESCRIPTION
     The gtmixer utility is used to set and display soundcard mixer device
     levels.  gtmixer based on standart mixer(8) utility.

FILES
     /usr/local/sbin/gtmixer

SEE ALSO
     mixer(8)
