NAME
       dontspace - an X solitaire programm

SYNOPSIS
       dontspace

DESCRIPTION
       Dontspace is an X11/Tk-based implementation of the classic soltare
       game.

       See the ``Help'' menu for a complete description of how to play and
       scoring.

BUGS
       If you have trouble with colors, try removing generic X resources such
       as ``*.Background''.

       For other bugs and desired features, see the ``Help/Release Notes''
       menu.
