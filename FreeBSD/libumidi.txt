NAME
     umidi20 – Universal MIDI library

LIBRARY
     Universal MIDI Library (libumidi20, -lumidi20)

SYNOPSIS
     #include <umidi20.h>

DESCRIPTION
     The universal MIDI library is a multithreaded MIDI processing toolkit
     which can be used by realtime and non-realtime applications to load,
     save, modify and playback MIDI files.  The umidi20 library supports
     direct playback and recording through JACK MIDI, FIFOs and character
     device nodes.

SEE ALSO
     MIDI (Musical Instrument Digital Interface), see
     http://en.wikipedia.org/wiki/MIDI_1.0.

STANDARDS
     The umidi20 library is fully compliant to the MIDI v1.0 standard.
