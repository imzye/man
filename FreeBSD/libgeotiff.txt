NAME
       applygeo - apply georeferencing to an existing file


SYNOPSIS
       applygeo file.geo file.tiff

DESCRIPTION
       applygeo is a utility for applying georeferencing to an existing file.


OPTIONS
       file.geo
             file containing projection (eg. from listgeo)

       file.tiff
             TIFF file into which the projection is written
