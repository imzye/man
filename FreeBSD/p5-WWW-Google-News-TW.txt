METHODS
       get_news()
           Scrapes <http://news.google.com.tw/news?ned=ttw> and returns a
           reference to a hash keyed on News Section, which points to an array
           of hashes keyed on URL and Headline.

       get_news_for_topic( $topic )
           Queries <http://news.google.com.tw/news?ned=tw> for results on a
           particular topic, and returns a pointer to an array of hashes
           containing result data.
