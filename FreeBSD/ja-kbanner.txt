NAME
       kbanner - display kanji files in large letters

SYNOPSYS
       kbanner [ -l | -n ] [ -s ] [ -m ] [ filename...  ]

DESCRIPTION
       Kbanner reads each filename in sequence and displays it in large
       letters on the standard output.  JIS, EUC, and Shifted-JIS codes are
       available for kanji code.

OPTIONS
       -l      Display in landscape.

       -n      n-column output.

       -s      Display in small banner.

       -m      Translate MIME encodings.

BUGS
       Shifted-JIS is often misjudged as EUC.  Only ISO-2022-JP is available
       for MIME.

SEE ALSO
       banner(6).
