NAME
     rndpassw â€“ generate random passwords

SYNOPSIS
     rndpassw [-dulpsvh] [password length] [password count]

DESCRIPTION
     The rndpassw utility generates random passwords

     The following options are available:

     -d      password contains numeric characters [0-9]

     -u      password contains upper alpha characters [A-Z]

     -l      password contains lower alpha characters [a-z]

     -p      password contains printable charcaters

     -s      password contains all previous characters groups

     -v      print version

     -h      print short usage

EXIT STATUS
     The rndpassw utility exits 0 on success, and >0 if an error occurs.
