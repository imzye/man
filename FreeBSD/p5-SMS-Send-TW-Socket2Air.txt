NAME
       SMS::Send::TW::Socket2Air - SMS::Send driver for HiNet Socket2Air

SYNOPSIS
       use SMS::send;

       my $sender = SMS::Send->new('TW::Socket2Air',     _username   =>
       'UserName',    _password   => 'Password',    _proxy_host =>
       'proxy.to.specified',    _proxy_port => 3128,     _proxy_type =>
       'http',   );

       my $sent = $sender->send_sms(      text => 'My very urgent message',
            to   => '0912345678',    );

DESCRIPTION
       SMS::Send::TW::Socket2Air is a SMS::Send driver which allows you to
       send messages through <http://sms.hinet.net/new/>.

METHODS
   new
       The "new" method takes a few parameters. "_username" and "_password" >
       are mandatory.

   send_sms
       Takes "to" as recipient phonenumber, and "text" as the utf-8 encoded
       text that's supposed to be delivered.
