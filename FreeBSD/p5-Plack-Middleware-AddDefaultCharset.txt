Plack::Middleware::AddDefaultCharset(3)    User Contributed Perl Documentation


NAME
       Plack::Middleware::AddDefaultCharset - a port of Apache2's
       AddDefaultCharset

SYNOPSIS
       use Plack::Builder;

       builder {
           enable 'AddDefaultCharset', charset => $charset_to_add;
           $app; }

DESCRIPTION
       Plack::Middleware::AddDefaultCharset is a port of the AddDefaultCharset
       configuration directive of the Apache HTTP server.
