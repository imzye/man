HTTP::Session::State::MobileAgentID(3)     User Contributed Perl Documentation


NAME
       HTTP::Session::State::MobileAgentID - Maintain session IDs using mobile
       phone's unique id

SYNOPSIS
           HTTP::Session->new(
               state => HTTP::Session::State::MobileAgentID->new(
                   mobile_agent => HTTP::MobileAgent->new($r),
               ),
               store => ...,
               request => ...,
           );

DESCRIPTION
       Maintain session IDs using mobile phone's unique id

CONFIGURATION
       mobile_agent
           instance of HTTP::MobileAgent

       check_ip
           check the IP address in the carrier's cidr/ or not?  see also
           Net::CIDR::MobileJP

       cidr
           The object have get_carrier($ip) method like Net::CIDR::MobileJP.

           If not provided to constructor, this class create new instance of
           Net::CIDR::MobileJP automatically.
