NAME
       radgrep - find a user by regular expression


SYNOPSIS
       radgrep [radwho-options][-- grep-options] regexp


DESCRIPTION
       GNU Radius tools are fully documented in the texinfo documentation.  To
       access the help from your command line, type

       info radgrep

       The online copy of the documentation is available at the following
       address:

       http://www.gnu.org/software/radius/manual.
