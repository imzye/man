NAME
       makeweb - create user web directory

SYNOPSIS
       makeweb [ -d webdir ]

DESCRIPTION
       This program allows users to create their own web subdirectories off of
       the main web directory.

       -d     Specify the root directory for new user directories. The default
              directory is /usr/local/www/thttpd

SEE ALSO
       thttpd(8)
