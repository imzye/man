NAME
       vxmount - Mount a VxFS filesystem


SYNOPSIS
       vxmount [-dh] [-p subpart] device


DESCRIPTION
       The vxmount command "mounts" a VxFS filesystem and stores mount state
       in ~/.vxcwd file.


OPTIONS
       -d     Set vxlib debugging mode.

       -h     Display help.

       -p subpart
              Select UnixWare partition number. Use this option if your system
              doesn't support UnixWare VTOC. UnixWare VTOC can be displayed by
              prtvtoc(8) command.


EXAMPLES
       vxmount /dev/hda21
       vxmount -p 1 /dev/hda


BUGS
       Only VxFS version 2 is supported for now.
