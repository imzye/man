NAME
       pdf-extract-outline - extract outlines (aka bookmarks) from PDF files

SYNOPSIS
       pdf-extract-outline input.pdf outlines.txt

DESCRIPTION
       pdf-extract-outline reads input file given as first argument and
       extracts outlines into the file given as second argument.  The output
       format is compatible with one accepted by pdfoutline.

OPTIONS
       pdf-extract-outline accepts no options.

SEE ALSO
       pdfoutline(1)
