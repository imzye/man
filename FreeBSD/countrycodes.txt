NAME
       iso3166 - an ISO 3166 country code finder.

SYNOPSIS
       iso3166 [-a] [[code]|[country]|[number]...] [-d <domain name>]

DESCRIPTION
       This program is an ISO 3166 country code finder. You could find to what
       country an ISO 3166 two letter code belongs and viceversa.

   OPTIONS
       -a     Show all the ISO 3166 table.

       [code] Search the ISO 3166 table looking for that country code. It
              could be a two or three letters code.

       [country]
              Search the table looking for the country name.

       [number]
              Search the table looking for the country number.

       -d <domain name>
              Show the domain name country (if possible) and try to explain
              it.
