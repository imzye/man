NAME
       inline2test - The Test::Inline 2 Test Compiler

VERSION
       version 2.214

DESCRIPTION
       "inline2test" is the Test::Inline 2 test compiler.

       It's job is to scan through an arbitrary tree of Perl source code
       files, locate inline test sections, extract them, convert them to test
       scripts, and write them to an output path.

SYNOPIS
         > inline2test ./inline2test.conf

         # In your inline2test.conf
         input=lib
         output=t
         execute=0
         verbose=1
         readonly=1
         header=inline2text.txt
