NAME
       rdma_get_dst_port - Returns the remote port number of a bound
       rdma_cm_id.

SYNOPSIS
       #include <rdma/rdma_cma.h>

       uint16_t rdma_get_dst_port (struct rdma_cm_id *id);

ARGUMENTS
       id          RDMA identifier.

DESCRIPTION
       Returns the remote port number for an rdma_cm_id that has been bound to
       a remote address.

RETURN VALUE
       Returns the 16-bit port identifier associated with the peer endpoint.
       If the rdma_cm_id is not connected, the returned value is 0.
