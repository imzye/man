NAME
       Shell::Command - Cross-platform functions emulating common shell
       commands

SYNOPSIS
         use Shell::Command;

         mv $old_file, $new_file;
         cp $old_file, $new_file;
         touch @files;

DESCRIPTION
       Thin wrapper around "ExtUtils::Command". See ExtUtils::Command for a
       description of available commands.
