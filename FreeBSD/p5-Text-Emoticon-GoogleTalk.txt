Text::Emoticon::GoogleTalk(3)              User Contributed Perl Documentation


NAME
       Text::Emoticon::GoogleTalk - Emoticon filter of GoogleTalk

SYNOPSIS
         use Text::Emoticon::GoogleTalk;
         my $emoticon = Text::Emoticon::GoogleTalk->new;

         my $text = "I <3 You :(|)";

DESCRIPTION
       Text::Emoticon::GoogleTalk is a text filter that replace text emoticons
       like ":-)", "<3", etc. with the icons of Google Talk (or Gmail Chat),
       detailed in <http://mail.google.com/support/bin/answer.py?answer=34056>
