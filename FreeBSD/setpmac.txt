NAME
     setpmac – run a command with a different MAC process label

SYNOPSIS
     setpmac label command [arg ...]

DESCRIPTION
     The setpmac utility forks a new process, attempts to set the label to
     label and if successful, runs command.

SEE ALSO
     mac(4), maclabel(7), getfmac(8), getpmac(8), setfmac(8)
