NAME
       bsdebfetch - Prints short info about *BSD.

SYNOPSIS
        bsdebfetch [OPTIONS]

DESCRIPTION
       bsdebfetch - Prints short info about *BSD.

OPTIONS
       -d     No *BSD logo

       -l     No line

       -p     No package info

       -h     This help
