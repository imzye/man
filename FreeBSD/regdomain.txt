DESCRIPTION
     The regdomain.xml file describes regulations for the operation of IEEE
     802.11 wireless radios.

     This information is used by the ifconfig(8) program to construct
     regulatory state for download to the system.  This file should be changed
     only to reflect changes in regulations.

FILES
     /etc/regdomain.xml  XML database of 802.11 regulatory constraints

SEE ALSO
     wlan(4), ifconfig(8)
