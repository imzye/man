NAME
       getdpi - get the DPI for the current monitor setup

SYNOPSIS
       getdpi Display the average DPI of all connected monitors.


DESCRIPTION
       getdpi can fetch information about all connected monitors, for X or
       Wayland, and return the average DPI (dots per inch).


OPTIONS
       -a or --all
              Output DPI information for all available monitors.

       -b or --both
              Output both the horizontal and vertical average DPI.

       -h or --help
              Show usage information.

       -V or --version
              Show the current version.
