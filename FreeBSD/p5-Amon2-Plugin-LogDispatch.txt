NAME
       Amon2::Plugin::LogDispatch - Log::Dispatch glue for Amon2

SYNOPSIS
           __PACKAGE__->load_plugin('LogDispatch');

           # in your config.pl
           'Log::Dispatch' => {
               outputs => [
                   [Screen::Color',
                       min_level => 'debug',
                       name      => 'debug',
                       stderr    => 1,
                       color     => {
                           debug => {
                               text => 'green',
                           }
                       }
                   ],
               ],
           },

           # in your controller
           $c->log->emerg('help me');

DESCRIPTION
       This is a Log::Dispatch glue for Amon2. This module still maintained,
       but I don't use this module on production environment today.  I'm a
       user of Log::Minimal.
