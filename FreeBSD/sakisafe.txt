SYPNOSIS
     sakisafecli [https]://sakisafe.tld FILE


DESCRIPTION
     sakisafecli is a file uploader. Intended for sakisafe, but also works for
     0x0.st, 0xff.i2p, i.kalli.st and probably others. It also supports
     transfer via scp.

OPTIONS
     --http_proxy Use an HTTP Proxy to make the request to the server.

     --socks_proxy Use a SOCKS Proxy to make the request to the server.

     --server Specify which server to use to upload the given file.

     -4 Force an IPv4 connection.

     -6 Force an IPv6 connection.

     -x Paste mode. Reads from stdin and publish the given content to the
     server.

     -C Print the current settings.

     --key path to the private SSH key to use. Only useful when using the
     scp:// protocol.
