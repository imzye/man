GSS_RELEASE_NAME(3)                     FreeBSD Library Functions Manual (prm)

NAME
     gss_release_name â€“ Discard an internal-form name

SYNOPSIS
     #include <gssapi/gssapi.h>

     OM_uint32
     gss_release_name(OM_uint32 *minor_status, gss_name_t *name);

DESCRIPTION
     Free GSS-API allocated storage associated with an internal-form name.
     Implementations are encouraged to set the name to GSS_C_NO_NAME on
     successful completion of this call.

PARAMETERS
     minor_status      Mechanism specific status code.

     name              The name to be deleted.

RETURN VALUES
     GSS_S_COMPLETE      Successful completion

     GSS_S_BAD_NAME      The name parameter did not contain a valid name

STANDARDS
     RFC 2743      Generic Security Service Application Program Interface
                   Version 2, Update 1

     RFC 2744      Generic Security Service API Version 2 : C-bindings

HISTORY
     The gss_release_name function first appeared in FreeBSD 7.0.
