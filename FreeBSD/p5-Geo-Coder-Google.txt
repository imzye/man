NAME
       Geo::Coder::Google - Google Maps Geocoding API

DESCRIPTION
       Geo::Coder::Google provides a geocoding functionality using Google Maps
       API.

       See Geo::Coder::Google::V2 for V2 API usage.

       See Geo::Coder::Google::V3 for V3 API usage.

       Note that Google no longer supports the V2 API. Geo::Coder::Google
       defaults to the V3 API. The V2 interface is still here but any attempts
       to use it will fail since the V2 API service is no longer reachable.
