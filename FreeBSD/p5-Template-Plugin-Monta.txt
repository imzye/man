NAME
       Template::Plugin::Monta - TT2 Filter, Acme::Monta adaptor

SYNOPSIS
         [% USE Monta %]

         [% FILTER $Monta close_font => '#0f0', close_back => '#0f0' %]
         <monta>Let's Montaize!</monta>
         [% END %]

         [% FILTER monta %]
         <monta>some string</monta>
         [% END %]

         [% some_string = '<monta>some string</monta>' %]
         [% some_string | monta %]

DESCRIPTION
       This is a Filter for Template-Toolkit.  This modules allows you to use
       monta-method on your template.  See details in Acme::Monta's documents.

SEE ALSO
       Acme::Monta
