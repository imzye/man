NAME
       SVG::DOM2 - SVG extention to the popular XML::DOM2

   VERSION
       Version 1.00, 08 May, 2006

DESCRIPTION
       An SVG Extention of XML::DOM2, this should provide for all features of
       the svg specification upto 1.1.

METHODS
   new
       $svg = SVG::DOM2->new(   -file = [svgfilename],   -data = [svgdata],
            %options );

       Create a new svg object, it will parse a file or data if required or
       will await creation of nodes.

   _element_handle
       This is the XML::DOM2 handel for all the elements that svg documents
       can handle.
