NAME
       precice-tools - command line utility for preCICE


SYNOPSIS
       precice-tools xml | dtd | md | [ version | --version ] | check file
       [participant [size]]


OPTIONS
       xml    Prints the configuration reference as XML with inlined help to
              the standard output.


       dtd    Prints the DTD file to the standard output, which may be used to
              check XML configuration files.


       md     Prints the configuration reference as markdown to the standard
              output.


       version | --version
              Prints the preCICE version string (
              precice::getVersionInformation() ) to the standard output.


       check file [participant [size]]
              Checks given preCICE configuration file and prints any errors.
              Pass the name of a participant to fully check its configuration.
              The optional size indicates how many ranks run in parallel,
              which checks custom <intra-comm: ... /> tags.


EXIT STATUS
       precice-tools returns a zero exit status if it succeeds to run the
       provided command.  Non zero is returned in case of failure.
