     pmap_page_init â€“ initialize machine-dependent fields of a VM page

SYNOPSIS
     #include <sys/param.h>
     #include <vm/vm.h>
     #include <vm/pmap.h>

     void
     pmap_page_init(vm_page_t m);

DESCRIPTION
     The pmap_page_init() function initializes the machine-dependent fields of
     a VM page structure.  This procedure is normally used when adding new
     pages to the VM page queue management lists.
