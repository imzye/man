NAME
       DateTime::Cron::Simple - Parse a cron entry and check against current
       time

SYNOPSIS
         use DateTime::Cron::Simple;

         $c = DateTime::Cron::Simple->new($cron);

         $boolean = $c->validate_time;

         $c->new_cron($cron);

DESCRIPTION
       This module is a quick and dirty way to determine if a cron time format
       is valid for the current date and time.

       A cron entry follows the cron format from crontab(5).

       The validate_time function uses the current date and time for
       comparison, but will also accept a valid DateTime object as a
       parameter.

EXAMPLE
         use DateTime::Cron::Simple;

         $c = DateTime::Cron::Simple->new('0-59/2 10,12 * * 5');

         if($c->validate_time) { ... }

         $c->new_cron('* * 1 * 0');

         if($c->validate_time) { ... }
