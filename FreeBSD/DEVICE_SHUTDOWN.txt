NAME
     DEVICE_SHUTDOWN â€“ called during system shutdown

SYNOPSIS
     #include <sys/param.h>
     #include <sys/bus.h>

     int
     DEVICE_SHUTDOWN(device_t dev);

DESCRIPTION
     The DEVICE_SHUTDOWN() method is called during system shutdown to allow
     the driver to put the hardware into a consistent state for rebooting the
     computer.

RETURN VALUES
     Zero is returned on success, otherwise an error is returned.
