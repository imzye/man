NAME
       Filesys::Virtual::DAAP - present a DAAP share as a VFS

SYNOPSIS
        use Filesys::Virtual::DAAP;
        my $fs = Filesys::Virtual::DAAP->new({
            host      => 'localhost',
            port      => 3689, # the default
            cwd       => '/',
            root_path => '/',
            home_path => '/home',
        });
        my @albums    = $fs->list("/Library");
        my @playlists = $fs->list("/Playlists");

DESCRIPTION
       The module can be used to republish a DAAP share.  You'll probably want
       to use Net::DAV::Server or POE::Component::Server::FTP to re-export it
       in a browseable form.
