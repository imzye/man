NAME
       helper - Help browser

SYNOPSIS
       helper [-c] [-p package] [command]

       helper -d package

       helper -s section

DESCRIPTION
       helper is a help browser for the desktop.

OPTIONS
       The name of a command, package or manual page can be specified on the
       command line.

       The following options are available:

       -c
           Obtain documentation for the command specified from a specific
           package.

       -p
           Set the package for the command specified.

       -d
           Obtain documentation for the package specified.

       -s
           The section from which to obtain the manual page specified.
