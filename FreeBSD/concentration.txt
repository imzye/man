NAME
       concentration - A matching game.

SYNOPSIS
       concentration

DESCRIPTION
       The classic memory game has some new life in this recreation for the
       Linux platform.  Your object is to find all matching pairs of icons in
       the fastest time possible.

OPTIONS
       -f/--fullscreen
              used to start the game in full screen mode
              http://concentration.shiftygames.com
