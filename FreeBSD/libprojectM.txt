NAME
       projectM-pulseaudio - ProjectM module for PulseAudio

DESCRIPTION
       This manual page documents the projectM-pulseaudio command.

       projectM-pulseaudio provides direct visualisation of PulseAudio streams
       using the projectM audio visualisation library.

       projectM-pulseaudio can be run from the Applications menu after it was
       installed.
