NAME
       HTML::FormFu::Imager - Imager.pm helpers for HTML::FormFu file uploads

DESCRIPTION
       HTML::FormFu::Inflator::Imager - inflate file uploads to Imager
       objects.

       HTML::FormFu::Transformer::Imager - perform image transformations, such
       as scale, crop and rotate.

       HTML::FormFu::Validator::Imager::Size - ensure the image doesn't exceed
       a given pixel size.
