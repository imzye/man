NAME
       Template::Plugin::Markdown - TT plugin for Text::Markdown

SYNOPSIS
         [% USE Markdown -%]
         [% FILTER markdown %]
         #Foo
         Bar
         ---
         *Italic* blah blah
         **Bold** foo bar baz
         [%- END %]

DESCRIPTION
       Template::Plugin::Markdown is a plugin for TT, which format your text
       with Markdown Style.

SEE ALSO
       Template, Text::Markdown
