NAME
       Alien::LibGumbo - Gumbo parser library

DESCRIPTION
       This distribution installs
       <libgumbo:https://github.com/google/gumbo-parser> on your system for
       use by perl modules like HTML::Gumbo.

       If you're interested in parsing HTML then you want HTML::Gumbo module,
       not this.
