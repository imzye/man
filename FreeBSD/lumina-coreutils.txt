LUMINA-CONFIG(1)        FreeBSD General Commands Manual       LUMINA-CONFIG(1)



NAME
     lumina-config – modifies different desktop settings.


SYNOPSIS
     lumina-config



DESCRIPTION
     Appearance, desktop defaults, interface configuration, system settings,
     and user settings are adjustable.  Different configuration options for
     each category are listed in the GUI configuration section below.


GUI CONFIGURABLES
     Each category contains different configurables.

     Appearance:

           Theme           Change the interface of fonts and colors.

           Window Effects  Adjust transparency levels and window effects.

           Wallpaper       Change background image.

           Window Manager  Change window settings and appearances.

     Desktop Defaults:

           Applications        Change default applications.

           Keyboard Shortcuts  Change keyboard shortcuts.

           Autostart           Automatically start applications or services.

     Interface Configuration:

           Desktop  Change what icons or tools are embedded on the desktop.

           Panels   Change any floating panels and what they show.

           Menu     Change what options are shown on the desktop context menu.

     System Settings:

           AppCafe                      Manage system applications

           Fcitx Configuration          Change Fcitx configuration.

           Lumina Screen Configuration  Setup monitors and screens.

           Mouse Configuration          Change mouse sensitivty and
                                        orientation.

           Screensaver                  Change screensaver properties.

           System Updates               Check and apply updates or change
                                        update settings.

           User Configuration           Change user details and groups.

           XScreenSaver Lock            Lock the screen with XScreenSaver.

           XScreenSaver Stop            Stop XscreenSaver daemon.

           Device Manager               Manage devices connected to the
                                        system.

           PulseAudio Preferences       View and modify the configuration of
                                        the local sound server.

           Service Manager              Run or enable start on boot for
                                        various services.

           Task Manager                 View or stop a currently running
                                        process.

           XScreenSaver Blank           Blank the screen with (activate)
                                        XScreenSaver.

           XScreenSaver Start           Start XScreenSaver daemon.

     User Settings:

           General Options  Change basic user settings such as time and date
                            formats.

           Localization     Change the default locale settings for this user.


FILES
     /usr/local/bin/lumina-config
