NAME
       LWP::UserAgent::WithCache - LWP::UserAgent extension with local cache

SYNOPSIS
         use LWP::UserAgent::WithCache;
         my %cache_opt = (
           'namespace' => 'lwp-cache',
           'cache_root' => File::Spec->catfile(File::HomeDir->my_home, '.cache'),
           'default_expires_in' => 600 );
         my $ua = LWP::UserAgent::WithCache->new(\%cache_opt);
         my $response = $ua->get('http://search.cpan.org/');

DESCRIPTION
       LWP::UserAgent::WithCache is a LWP::UserAgent extention.  It handle
       'If-Modified-Since' request header with local cache file.  local cache
       files are implemented by Cache::FileCache.
