NAME
       UUID::Random - Generate random uuid strings

SYNOPSIS
         use UUID::Random;
         my $uuid = UUID::Random::generate;

DESCRIPTION
       This module generates random uuid strings. It does not satisfy any of
       the points listed in RFC 4122 (<http://tools.ietf.org/html/rfc4122>)
       but the default format.

SEE ALSO
       If you need RFC compliant UUID strings have a look at Data::UUID
