NAME
       btree-dump - unformatted dump of a B-Tree DB file

SYNOPSIS
       btree-dump btree-file.db

DESCRIPTION
       btree-dump reads a B-Tree DB 2.x file, such as those produced by
       dns-terror or getdominfo.  It prints the key-value mappings.

SEE ALSO
       convert-dom-db(1), convert-ip-db(1), dns-terror(1), getdominfo(1)
