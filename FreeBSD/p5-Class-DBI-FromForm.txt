NAME
       Class::DBI::FromForm - Update Class::DBI data using Data::FormValidator
       or HTML Widget

SYNOPSIS
         package Film;
         use Class::DBI::FromForm;
         use base 'Class::DBI';

         my $results = Data::FormValidator->check( ... );
         my $film = Film->retrieve('Fahrenheit 911');
         $film->update_from_form($results);

         my $new_film = Film->create_from_form($results);

DESCRIPTION
       Create and update Class::DBI objects from Data::FormValidator or
       HTML::Widget.

   METHODS
       create_from_form

       Create a new object.

       update_from_form

       Update object.

fill_widget <widget>
       This only applies to HTML::Widget>.  Fills the form from a CDBI object.

SEE ALSO
       Class::DBI, Class::DBI::FromCGI, Data::FormValidator
