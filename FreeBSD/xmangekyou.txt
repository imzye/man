NAME
       XMangekyou - a kaleidoscope for X

SYNOPSIS
       xmangekyou [ -options ... ]

DESCRIPTION
       XMangekyou is a kaleidoscope. You can see many, many, many patterns,
       but, care for yourself from 'trip' filling!

OPTIONS
       You can use all of the standard X-Toolkit options.  (x) is default
       value.
       -display [displayname]
            This option specifies the name of the X server to use.
       -geometry [geometry]
            This option specifies the placement of the window on the screen.
       -background [color], -bg [color]
            This option specifies the background color.
       -h, -help
            Output this message.
       -root
            Draw a mangekyou on the root window.
       -noroot
            Open a new window and draw a mangekyou on it. (Default)
       -quit
            Draw a mangekyou once and quit.
       -noquit
            Draw a mangekyou and repeat it. (Default)
       -lines [x]
            This option specifies the number of lines to draw.     (Default
       value is 50)
       -circles [x]
            This option specifies the number of circles to draw.   (Default
       value is 20)
       -line-thickness [x]
            This option specifies the maximum thickness of lines.
            (Default value is 5)
       -circle-size [x]
            This option specifies the maximum size of circles.     (Default
       value is 10)
       -size [x]
            This option specifies the size of the center of a mangekyou. (80)
       -speed [x]
            This option specifies the speed of the rotation. (128)
       -change [x]
            This option specifies the speed of the change. (1000)
       -wait [x]
            This option specifies the wait. (0)
       -animation-file [filename]
            This option specifies the filename for the animation.
       (animation.dat)
