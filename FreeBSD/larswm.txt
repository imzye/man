NAME
       larsclock - Status bar clock for larswm

SYNOPSIS
       larsclock [ -display dpy ] [ -format fmt ]

DESCRIPTION
       larsclock is a minimal status bar clock application for the larswm
       window manager.

OPTIONS
       The following command line options are supported:

       -display dpy      lets you specify which display you want larsclock to
                         open.

       -format fmt       lets you format how the time is displayed. The
                         default format is "%c". See strftime(3) for details.

SEE ALSO
       larsmenu(1x), larsremote(1x), larswm(1x), strftime(3x)
