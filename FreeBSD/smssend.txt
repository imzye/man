NOM
       email2smssend - Recevez votre courrier electronique sur votre portable


SYNOPSYS
       email2smssend [options] -- [smssend options]


DESCRIPTION
       email2smssend vous permet de rediriger votre courrier electronique sur
       votre telephone portable. C'est un script perl qui lance le programme
       smssend(1). Vous pouvez controler la taille maximale des mini-messages
       ainsi que le nombre maximum des mini-messages envoyes par mail.

       Ce programme est concu pour s'integrer facilement dans un fichier
       .forward


OPTIONS
   Options pour email2smssend
       -ms, --msg-max n
              Positionne le nombre maximum de carateres par mini-message a n.


       -mm, --max-msg n
              Positionne le nombre maximum de mini-messages correspondant a un
              meme mail a n (n doit etre inferieur a 10).


       -i, --index
              Si cette option est presente, chaque mini-message (d'un meme
              mail) sera prefixe par un numero de sequence.


       -r, --reverse
              Cette option permet d'envoyer les mini-messages en ordre
              inverse. Ceci est parfois necessaire.


       -h, --help
              Affiche un resume de l'aide puis quitte.


       -V, --version
              Affiche le numero de version ainsi que le copyright puis quitte.


   Options de smssend
       Voir la page de manuel de smssend pour plus d'informations.


EXAMPLES
       Ce script est prevu pour etre facile a integrer dans votre .forward.
       Vous n'avez qu'a y ajouter une ligne du type :


       | email2smssend -i -ms 130 -mm 4 -- [provider name] [provider options]


       Dans ce cas, les mails seront envoyes par troncons de 130 caracteres,
       avec un nombre de mini-messages limite a 4 par mail. Lisez la page de
       manuel de smssend(1) pour plus de details sur provider name et provider
       options.
