NAME
       wbxmlc - WBXML Compiler

SYNOPSIS
       wbxmlc [-p path] file

OPTIONS
       -h      Display help.

       -p      Specify the path of rules (the default is
               WAP/wbxml/wap.wbrules.xml).

       -v      Display version.

DESCRIPTION
       wbxmlc parses the given input XML file and generates a binarized file
       according the specification :

       WAP - Wireless Application Protocol / Binary XML Content Format
       Specification / Version 1.3 WBXML (15th May 2000 Approved)

       The file WAP/wbrules.xml configures this tool for all known DTD.

       WAP Specifications, including Binary XML Content Format (WBXML)
        are available on <http://www.wapforum.org/>.

SEE ALSO
        wbxmld, WAP::SAXDriver::wbxml
