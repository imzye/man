NAME
       Bundle::PlRPC - A bundle to install PlRPC-Server, Client and
       prerequisites.

SYNOPSIS
       "perl -MCPAN -e 'install Bundle::PlRPC'"

       Storable

       Net::Daemon

       RPC::PlServer

DESCRIPTION
       This bundle includes all that's needed to run PlRPC-Server and Client.
