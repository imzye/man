NAME
       orepan2-audit - 02packages auditor

SYNOPSIS
           % orepan2-audit --darkpan path/to/darkpan/02packages --cpan http://cpan.metacpan.org/modules/02packages.details.txt --show outdated-modules --verbose

DESCRIPTION
       OrePAN2 auditor.  This script finds differences between your DarkPAN
       and a CPAN mirror.

OPTIONS
       " --darkpan "
           The full path or URL to a DarkPAN 02packages file in .txt or
           .txt.gz format.

       " --cpan "
           The full path or URL to a CPAN 02packages file in .txt or .txt.gz
           format.

       " --verbose "
           Print out some version information.  Currently only applies to
           outdated modules

       " --show "
           Possible values are "outdated-modules", "cpan-only-modules" and
           "darkpan-only-modules".
