NAME
     jkill â€“ Shutdowns a running jail and all its processes.

SYNOPSIS
     jkill [lj] [long_jails]

DESCRIPTION
     This is a simple perl script that shutdowns a running jail and all its
     processes.

     No additional software or tools like jailer(8) needed.

OPTIONS
     One or more of the following options to jkill may be combined
     arbitrarily.

     lj             Use full jail hostnames
     long_jails     Same as lj
     <jail name>    Hostname of the jail to kill

SEE ALSO
     jail(8)
