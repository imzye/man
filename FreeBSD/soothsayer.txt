NAME
       soothsayerDemo - soothsayer demo program (ncurses)

SYNOPSIS
       soothsayerDemo [OPTION]...

DESCRIPTION
       Begin typing. soothsayer will attempt to predict the desired word.
       After each keystroke, soothsayer will return a number of predictions.
       If the desired word appears in the prediction list, select it by
       pressing the corresponding function key.

       -c, --config CONFIG
              use config file CONFIG

       -s, --suggestions N
              set prediction size to N suggestions

       -h, --help
              display this help and exit

       -v, --version
              output version information and exit
