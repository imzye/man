PDF::Template::Element::Weblink(3)         User Contributed Perl Documentation


NAME
       PDF::Template::Element::WebLink

PURPOSE
       To provide a clickable web-link

NODE NAME
       WEBLINK

INHERITANCE
       PDF::Template::Element

ATTRIBUTES
       •   URL The URL to go to, when clicked

       •   X1 / X2 / Y1 / Y2

           The dimensions of the clickable area

CHILDREN
       None

AFFECTS
       Nothing

DEPENDENCIES
       None

USAGE
       This node is currently under review as to whether it should be removed
       and a URL attribute should be added to various nodes, such as IMAGE,
       TEXTBOX, and ROW.

   USE AT YOUR OWN RISK
