NAME
       ebug - A simple, extensible console Perl debugger

VERSION
       version 0.64

SYNOPSIS
         % ebug calc.pl
         % ebug "add.pl 3 4"

DESCRIPTION
       ebug is an interactive command-line front end to Devel::ebug. It is a
       simple Perl debugger, much like perl5db.pl.

SEE ALSO
       Devel::ebug
