NAME
       utfstringinfo - Analyze a UTF-8 encoded string

SYNOPSIS
       utfstring [string]

DESCRIPTION
       utfstringinfo reads arguments or, if no arguments are provided, stdin
       and for every single character it finds it shows the UTF-8 character
       sequence, the unicode code-point and, if applicable, the utftex command
       that can be used to generate the symbol.
