NAME
       ne - Text editor


SYNOPSIS
       ne [-ejor] [-D name=val] [file ...] [+line]


DESCRIPTION
       ne is full screen text editor.


options
       -j
               japanese title bar

       -e
               english title bar

       -o
               with opening messeges

       -r
               resume a edit

       -D
               set inside parameter

       +line
               direction edit point vi
               like format
