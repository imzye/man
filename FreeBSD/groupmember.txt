NAME
     groupmember â€“ checks group set for a group ID

SYNOPSIS
     #include <sys/param.h>
     #include <sys/ucred.h>

     int
     groupmember(gid_t gid, struct ucred *cred);

DESCRIPTION
     The groupmember() function checks to see if the given gid is in the group
     set of the credentials.

     Its arguments are:

     gid       The group ID to check for.

     cred      The credentials to search for the group in.

RETURN VALUES
     If the gid is found, 1 is returned; otherwise, 0 is returned.
