NAME
       Catalyst::Plugin::Session::Store::Cache - Store sessions using a
       Catalyst::Plugin::Cache

SYNOPSIS
           use Catalyst qw/Cache::YourFavoriteCache Session Session::Store::Cache/;

DESCRIPTION
       This plugin will store your session data in whatever cache module you
       have configured.

METHODS
       See Catalyst::Plugin::Session::Store.

       get_session_data
       store_session_data
       delete_session_data
       delete_expired_sessions
