NAME
       ocr - perform ocr on an image and output text to stdout

DESCRIPTION
       This is just an interface to make it quick an easy to get ocr output
       from an image file.  No matter what image type you provide, imagemagick
       convert is called to turn it into the format for tesseract.

USAGE
       ocr [OPTION]... FILE...

          -b          blowup (useful for small images)
          -d          debug
          -h          help
          -l string   language
          -v          version

   EXAMPLE USAGE
          ocr ./image.jpg > ./savetext.txt
          ocr ./image.png
          ocr ./image1.jpg ./image2.jpg > ./alltext.txt
          ocr ./image*jpg

SEE ALSO
       Image::OCR::Tesseract - parent package tesseract - look up google ocr
       project convert - from ImageMagick suite.
