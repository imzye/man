NAME
       browser - File manager and image viewer

SYNOPSIS
       browser [-D | -I | -L | -T] [directory...]

DESCRIPTION
       browser is a program that lists and manages files.

OPTIONS
       The following options are available:

       -D
           Start the file manager in detailed view.

       -I
           Start the file manager in icon view.

       -L
           Start the file manager in list view.

       -T
           Start the file manager in thumbnail view.

       A list of directories to browse can also be provided directly on the
       command line.
