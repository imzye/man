NAME
       BSb_back_solve - Backward triangular matrix solution on a block of
       right hand sides


INPUT PARAMETERS
       A      - The sparse matrix
       x      - The contiguous block of right hand sides
       comm   - The communication structure for A
       block_size
              - the number of rhs's
       procinfo
              - the usual processor information


OUTPUT PARAMETERS
       x      - on exit these vectors contain the solution vectors


RETURNS
       void


SYNOPSIS
       void BSb_back_solve(BSpar_mat *A, FLOAT *x, BScomm *comm,
         int block_size, BSprocinfo *procinfo)



LOCATION
        BlockSolve95/src/BSb_back_solve.c
