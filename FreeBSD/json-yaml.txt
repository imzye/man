     json-yaml â€“ convert JSON to YAML

SYNOPSIS
     json-yaml [file]

DESCRIPTION
     Convert JSON to YAML.  Uses standard input if no filename is supplied.

EXAMPLES
     Convert a file:

           $ json-yaml sample.json

     Piping from a web API call:

           $ curl -s http://api.icndb.com/jokes/random | json-yaml
