NAME
       gWaoN - gtk+ GUI frontend for WaoN

SYNOPSIS
       gwaon

DESCRIPTION
       gWaoN is a gtk+ GUI frontend for WaoN.  Right now, gWaoN can do the
       following: 1) read an audio file (wav or flac) and analyze its
       spectrogram, 2) play the file through libao, 3) change the playback
       speed and pitch by phase vocoder in real-time!

KEY BINDINGS
       Space            : toggle play and stop.

       p, P             : change the analysis mode.

       w, W             : change the window function.

       Left, Right      : change the FFT size by factor 2.

       q, Q             : toggle PV mode (no phase lock/loose phase lock).

       h, H             : change hop size for the Phase-Vocoder correction.

       o, O             : change the upper octave limit.

       l, L             : change the lower octave limit.

       Up, Down         : change the upper limit for the power spectrum.

       PageUp, PageDown : change the lower limit for the power spectrum.
