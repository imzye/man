NAME
       CQL::AndNode - represents an AND node in a CQL parse tree

SYNOPSIS
           use CQL::AndNode;
           my $node = CQL::AndNode->new( left => $left, right => $right );

DESCRIPTION
       CQL::AndNode inherits from CQL::BooleanNode, where you will find most
       of the functionality.
