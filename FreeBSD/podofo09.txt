NAME
       podofobox - set the media, crop, bleed, trim, and art box on pages of a
       PDF file.

SYNOPSIS
       podofobox [inputfile] [outpufile] [box] [left] [bottom] [width]
       [height]

DESCRIPTION
       podofobox is one of the command line tools from the PoDoFo library that
       provide several useful operations to work with PDF files. It can set
       the media, crop, bleed, trim, and art box on pages of a PDF file.
