     BUF_LOCKINIT â€“ initializes a buffer lock

SYNOPSIS
     #include <sys/param.h>
     #include <sys/systm.h>
     #include <sys/uio.h>
     #include <sys/bio.h>
     #include <sys/buf.h>

     void
     BUF_LOCKINIT(struct buf *bp);

DESCRIPTION
     The BUF_LOCKINIT() macro initializes a buffer lock.

     Its argument is:

     bp      The buffer whose lock it to be initialized.
