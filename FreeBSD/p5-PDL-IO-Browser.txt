NAME
       PDL::IO::Browser -- 2D data browser for PDL

DESCRIPTION
       cursor terminal browser for ndarrays.

SYNOPSIS
        use PDL::IO::Browser;
        browse sequence 3,2;

FUNCTIONS
   browse
        Signature: (a(n,m))
        Types: (byte short ushort long longlong float double)

   browse
       browse a 2D array using terminal cursor keys

        browse $data

       This uses the CURSES library to allow one to scroll around a PDL array
       using the cursor keys.

       Broadcasts over its inputs.

       "browse" does not process bad values.  It will set the bad-value flag
       of all output ndarrays if the flag is set for any of the input
       ndarrays.
