NAME
       Perl::Lint - Yet Another Perl Source Code Linter

SYNOPSIS
           use Perl::Lint;

           my $linter = Perl::Lint->new;
           my $target_files = [qw(foo/bar.pl buz.pm)];
           my $violations   = $linter->lint($target_files);

DESCRIPTION
       Perl::Lint is the yet another source code linter for perl.

AIMS
       Development of this module aims to create a fast and flexible static
       analyzer for Perl5 that has compatibility with Perl::Critic

       Please see also
       <http://news.perlfoundation.org/2014/03/grant-proposal-perllint---yet.html>.

METHODS

           default, this function checks the all of policies that are in
           "Perl::Lint::Policy::*".
