NAME
       2048 - play the game 2048 in your terminal


SYNOPSIS
       2048 [-hrcCaA] [-s SIZE] [-b RATE]


DESCRIPTION
       2048 is an implementation of the popular game, 2048, designed to be run
       on a terminal. It is deisgned to be easy to understand and extend.


CONTROLS
       hjkl and wasd
              Default movement keys. Depending on the graphics implementation,
              there may be extra alternatives.

       q      Quit the current game.


OPTIONS
       -h     Print the program usage.

       -H     Print the current highscore.

       -c     Enable color support if supported.

       -C     Disable color support (default).

       -a     Enable animations (default).

       -A     Disable animations.

       -i     Enable ai without displaying game.

       -I     Enable ai and display game.

       -r     Reset highscore. Will prompt user.

       -s SIZE
              Set the size of the playing field. Default is 4. Maximum value
              is 16, minimum is 4.

       -b RATE
              Set the rate at which blocks are spawned. Default is 1.
