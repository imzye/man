NAME
       XBlood - make the root window bloody

SYNOPSIS
       xblood [ -options ... ]

DESCRIPTION
       XBlood fills the root window with blood.  Red, blue or green bloody
       window is very bad taste!

OPTIONS
       -display [displayname]
            The name of the X server to use.
       -h, -help
            Output help message.
       -stay, -noflow
            Blood stays the bottom of the root window.
       -nostay, -flow
            Blood flows down through the root window. (Default)
       -quit
            Draw blood and quit.
       -noquit
            Draw blood and they flow. (Default)
       -number [x]
            The number of bloody drops. Default value is 30.
       -size [x]
            The maximum size of bloody drops' width. Default value is 20.
       -speed [x]
            The speed of droping. Default value is 100.
       -scatter-number [x]
            The maximum number of scatters. Default value is 150.
       -scatter-radius [x]
            The maximum radius of scatters. Default value is 200.
       -scatter-size [x]
            The maximum size of scatters. Default value is 60.
       -color [color]
            Bloody drops' color. Default is red.
       -wait [x]
            Wait time. Default value is 10000.
