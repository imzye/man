NAME
       base64url - Base64Url encode or decode FILE, or standard input, to
       standard output.

SYNOPSIS
       base64url [OPTIONS]

DESCRIPTION
       base64url - Base64Url encode or decode FILE, or standard input, to
       standard output.

       Version 0.9

       Command-line options:

       -d, --decode

              Decode data

       -i, --ignore-garbage

              when decoding, ignore non-alphabet characters

       -w, --wrap <COLS>

       wrap encoded lines after COLS character (default 76).
              Use 0 to disable line wrapping

       -f, --file <FILE_NAME>

              Encode or decode FILE_NAME content - if this option is missing,
              red from stdin

       -h, --help

              display this help and exit

       -v, --version

              output version information and exit
