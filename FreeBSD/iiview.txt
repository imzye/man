SYNOPSIS
     iiview [file]

DESCRIPTION
     iiview IIViewer is an application for X. It allows you to display
     pictures in a directory as thumbnails. Selected images can then be shown
     in full size. It reads the most common graphic formats.

OPTIONS
     The following command line arguments are supported:

     file    the picture to show directly (without thumb-screen)

AUTHOR
     nakal

BUGS
     IIView had some problems with resolutions and bpp-values.  Please keep an
     eye open. Any help is welcome.  You can always complain about
     incompatibilities on my project site https://github.com/nakal/iiview
