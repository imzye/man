NAME
       "Plack::Handler::FCGI::Async" - FastCGI handler for Plack using
       FCGI::Async

DESCRIPTION
       This module allows Plack to run a PSGI application as a standalone
       FastCGI daemon under IO::Async, by using FCGI::Async.

        plackup -s FCGI::Async --listen ":2000" application.psgi

       This is internally implemented using FCGI::Async::PSGI; further
       information on environment etc.. is documented there.
