NAME
       drg2sbg - convert I-Doser drg files to sbagen


SYNOPSIS
       drg2sbg [OPTION] drgfile


DESCRIPTION
       drg2sbg converts drg file to sbagen format


   Options
       -o, --output output-file
              Specifies the output file to use. By default stdout is used.

       -v, --version
              Prints version information and exits.

       -r, --raw element
              Outputs the element as it is stored in the drg file, where
              element must be 1 for header, 2 for title, 3 for image, 4 for
              description or 5 for sbagen data.
