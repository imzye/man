NAME
       WMBlueClock - a dockapp which displays time and date

SYNOPSIS
       wmblueclock [OPTIONS]

DESCRIPTION
       WMBlueClock is a clock dockapp for window managers which support
       dockapps. It displays current time and date.

OPTIONS
       The following options can be passed to WMBlueCPU on the command line:

       -h, --help
              Display basic usage info and exit

       -v, --version
              Display version number and exit

       -display <DISPLAY>
              The X11 display to connect to.

       -m <MILISECS>
              The number of miliseconds between screen updates

       -w, --window
              Run in own window (also good for other windowmanagers)

       -nw, --no-window
              Don't run in own window

       -s, --shape
              Use the XShape extension

       -ns, --no-shape
              Don't use the XShape extension. The window will be placed on
              it's own panel.

       -a, --ampm
              Use 12 hour mode.

       -na, --no-ampm
              Don't use 12 hour mode, use 24 hour mode.
