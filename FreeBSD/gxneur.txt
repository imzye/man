SYNOPSIS
       gxneur

DESCRIPTION
       This manual page documents briefly gxneur command.

       gXNeur is a GTK frontend for xneur keyboard layout switcher. gXNeur
       runs in system tray and shows XNeur's state. It also allows to
       configure XNeur via GUI dialog.

OPTIONS
       This program accepts no options.
