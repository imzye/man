NAME
       WMBlueCPU - a dockapp which displays cpu usage

SYNOPSIS
       wmbluecpu [OPTIONS]

DESCRIPTION
       WMBlueCPU is a cpu monitoring dockapp for window managers which support
       dockapps. It shows the current cpu usage with numbers and a graphical
       history list.

OPTIONS
       The following options can be passed to WMBlueCPU on the command line:

       -h, --help
              Display basic usage info and exit

       -v, --version
              Display version number and exit

       -display <DISPLAY>
              The X11 display to connect to.

       -m <MILISECS>
              The number of miliseconds between screen updates

       -w, --window
              Run in own window (also good for other windowmanagers)

       -nw, --no-window
              Don't run in own window

       -s, --shape
              Use the XShape extension

       -ns, --no-shape
              Don't use the XShape extension. The window will be placed on
              it's own panel.
