NAME
       Text::Emoticon - Factory class for Yahoo! and MSN emoticons

SYNOPSIS
         use Text::Emoticon;

         my $emoticon = Text::Emoticon->new('MSN', { strict => 1, xhtml => 0 });
         print $emoticon->filter('Hello ;)');

DESCRIPTION
       Text::Emoticon is a factory class to dispatch MSN/YIM emoticon set.
       It's made to become handy to be used in other applications like
       Kwiki/MT plugins.
