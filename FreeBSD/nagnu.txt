SYNOPSIS
     nagnu [-c file] [-e file] [-s seconds]

DESCRIPTION
     Nagnu is a dashboard for nagios 3.x or 4.x showing a listing of all host
     and service problems.

OPTIONS
     -c      Nagnu will look for the configuration file in ./, /etc and
             /usr/local/etc. If it is located elsewhere please specify with
             this flag.

     -e      Excludes file. Newline separated list of service_descriptions or
             hostnames that should be excluded from the output.

     -s      Update interval in seconds. Default is 10.
