NAME
       mfsnetdump - dump network traffic as mfs packets

SYNOPSIS
       mfsnetdump [-s] [-i net device] [-f pcap_filter] [-c packet count] [-e
       commands] [-o commands]

DESCRIPTION
       mfsnetdump is the supervisor program of Moose File System. It is needed
       to start a completely new system or a system after a big crash. It can
       be also used to force select a new leader master.

OPTIONS
       -s     set standard PCAP filter (dump only mfs ports -
              default:9419-9422)

       -i     dump packets using given network interface

       -f     define pcap filter to filter packets

       -c     dump only 'count' packets and exit

       -e     do not display given commands

       -o     show only given commands
