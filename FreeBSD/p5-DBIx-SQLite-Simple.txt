NAME
       DBIx::SQLite::Simple - easy access to SQLite databases using objects

ATTRIBUTES
       db  Used to store the filename containing the SQLite database.

METHODS
       new(db => 'filename.db')
           Object creator. Takes one argument, and sets the global variable
           $Dbo to the newly created database handler.

       commit
           Changes made on created database are not automatically commited.
           You must call this method if you want to commit pending changes.

       close
           When you're done using the database, you can disconnect from it.
           This method will not commit changes, so do it before closing.
