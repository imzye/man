CMAP_CONTEXT_GET(3)                Corosync Cluster Engine Programmer's Manual



NAME
       cmap_context_get - Gets the context variable for a CMAP instance


SYNOPSIS
       #include <corosync/cmap.h>


       cs_error_t cmap_context_get (cmap_handle_t handle, const void
       **context);


DESCRIPTION
       The cmap_context_get function is used to retrieve the context variable
       previously stored using cmap_context_set(3).


RETURN VALUE
       This call returns the CS_OK value if successful, otherwise an error is
       returned.


SEE ALSO
       cmap_context_set(3), cmap_overview(3)
