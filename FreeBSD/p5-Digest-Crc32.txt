NAME
       Digest::CRC32 - Cyclic Redundency Check digests implementation

VERSION
       0.01

SYNOPSIS
               use Digest::CRC32;

               my $crc = new Digest::CRC32();

               # Digest for a string
               printf $crc->strcrc32("Hello world");

               #Digest for a file
               print $crc->filecrc32($myfile);

DESCRIPTION
       This module provides a perl implementation to generate 32 bits CRC
       digests for buffers and files.
