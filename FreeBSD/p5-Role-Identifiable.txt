Role::Identifiable::HasIdent(3)            User Contributed Perl Documentation


NAME
       Role::Identifiable::HasIdent - a thing with an ident attribute

VERSION
       version 0.007

DESCRIPTION
       This is an incredibly simple role.  It adds a required "ident"
       attribute that stores a simple string, meant to identify exceptions.

       The string has to contain at least one character, and it can't start or
       end with whitespace.
