BSDE_RULE_TO_STRING(3)                        FreeBSD Library Functions Manual

NAME
     bsde_rule_to_string – convert a ugidfw rule into its text representation

LIBRARY
     File System Firewall Interface Library (libugidfw, -lugidfw)

SYNOPSIS
     #include <ugidfw.h>

     int
     bsde_rule_to_string(struct mac_bsdextended_rule *rule, char *buf,
         size_t buflen);

DESCRIPTION
     The bsde_rule_to_string() function converts a rule in its internal
     representation (struct mac_bsdextended_rule) into its text
     representation, and writes up to buflen bytes of it to buf (including the
     terminating NUL).

RETURN VALUES
     The bsde_rule_to_string() function returns -1 if the conversion was
     truncated; otherwise the value 0 is returned.
