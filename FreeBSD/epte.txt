NAME
       epte - Electronic Periodic Table of the Elements


SYNOPSIS
       epte -[xyz]


DESCRIPTION
       epte  gives a periodic table of the elements, as is well known in
       chemistry, and provides subscreens which show common chemical  data of
       various sorts.

COMMAND-LINE OPTIONS
       The command-line options for epte have not changed from previous
       versions.

       The command-line arguments should be preceeded by a `-' character.


       v    Do not display the orbital filling for each element.


       x    Do not display the periodic table with atomic weights.


       y    Do not display the periodic table with element focus.


       z    Do not display the periodic table with group labels.


FILES
       /usr/local/bin/epte Executable image.

       /usr/local/man/man1/epte.1 Manual page.
