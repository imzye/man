NAME
       Tempate::Plugin::encoding - Template plugin to specify encoding

SYNOPSIS
         [% USE encoding 'euc-jp' -%]
         <?xml version="1.0" encoding="[% encoding %]"?>

DESCRIPTION
       Template::Plugin::encoding is a Template plugin to declare the encoding
       of template files. This plugin doesn't actually do anything but
       Template::Provider::Encoding scans the usage of this module to find the
       encoding of templates. As a bonus, you can use "encoding" variable in
       the template to specify file encoding, which might be useful for XML or
       HTML meta tag.
