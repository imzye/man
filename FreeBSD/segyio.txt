NAME
       segyio-catb - Concatenate SEG-Y binary header and print on seismic unix
       format

SYNPOSIS
       segyio-catb [OPTION]...  [FILE]...

DESCRIPTION
       segyio-catb Concatenate the SEG-Y binary header in FILEs.


OPTIONS
       -n, --nonzero
              only print fields with non-zero values


       -d, --description
              print with byte offset and field description


       --version
              output version information and exit


       --help display this help and exit
