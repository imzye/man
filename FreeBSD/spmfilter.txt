NAME
       spmfilter - spmfilter control program

SYNOPSIS
       spmfilter [-d] [-f config_file]


DESCRIPTION
       The spmfilter(1) command controls the operation of the spmfilter
       framework.


OPTIONS
       The following command-line options are implemented:

       -f config_file (or --file)
              Read alternate configuration file instead of the default
              (/etc/spmfilter.conf).  Use this option to distinguish between
              multiple spmfilter instances on the same host.


       -d (or --debug)
              Enable verbose logging for debugging purposes.

       -? (or --help)
              Print spmfilter usage.
