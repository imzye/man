NAME
     libphk – Poul-Henning Kamps utility functions

LIBRARY
     library “libphk”

DESCRIPTION
     libphk contains a diverse range of utility functions written by Poul-
     Henning Kamp which may or may not be useful to other than himself.

     phk_argv(3) contains functions for parsing a string into "char **argv"
     style arguments.

SEE ALSO
     phk_argv(3)

HISTORY
     Written by Poul-Henning Kamp.
