NAME
       NLL - Simple language for programming learning

SYNOPSIS
       nll [<options>] [<filename>|-|{--|stdin}] [<arguments>...]

DESCRIPTION
       Simple script language for beginners for programming learning.

       Complicated sentence structures are abolished and It is designed to
       understand if you can go on reading it by a line unit.

       Integer arithmetic, character string processing, a floating decimal
       operation are possible.

       This has text drawing, graphic and sound function and can make simple
       games.

       NLL is one implementation in NLUX of programming language HOBBY.

OPTIONS
       -h|-help
               Output help message.

       -v|-version
               Output version.

       -q|-quit
               Auto quit after execution of the input file.

       -n|-number
               Auto numbering mode of the input file.

       -nr|-no-run
               Disable auto execution of the input file.

       -na|-no-argv
               Not give ARGV[] for arguments.

       -ns|-no-system
               Disable system functions.

       -nc|-no-syscall
               Disable SYSCALL() function.

       -nn|-no-network
               Disable network functions.

       -nf|-no-fixed
               Disable one time only calculation of the fixed value.

       -e <command>
               Execute the command.

       -       The file name not to read.

       --|stdin
               Input the commands from standard input.

EXAMPLES
       $ nll   Start NLL and output the prompt 'nll>'.

       $ nll sample.nll
               Start NLL, load and run the program.

       $ nll -q -e 'PRINT 1+2'
               Execute the program in one linear mode.

       $ echo "abc" | nll -e 'S=FGETS(0);S+="def";FPUTS(0,S)'
               Use NLL as the filter program.

PROGRAMMING
       Input the following lines in turn from the top, and see the saved
       program by LIST command on the way.

       nll> A=1; B=2; PRINT A + B
               Run the program in direct mode.

       nll> 1 A=1
               Save the program in program mode.

       nll> 2 B=2
               Insert the next line into the saved program.

       nll> 10 PRINT A + B
               Insert the next line into the saved program.  The line number
               is renumbered in automatically.

       nll> 1 1 A=2
               Modify the saved line. (Two times of line number)

       nll> 2 1
               Move the saved line.

       nll> 1  Delete the saved line.

COMMANDS
       nll> RUN
               Run the saved program.

       nll> LIST
               See the saved program.

       nll> NEW
               Clear the saved program.

       nll> LOAD "sample.nll"
               Load the program.

       nll> QUIT
               Exit from NLL.
