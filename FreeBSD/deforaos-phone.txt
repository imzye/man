NAME
       gprs - Desktop telephony application

SYNOPSIS
       gprs

DESCRIPTION
       gprs is a telephony application for desktop environments and embedded
       devices. It allows connecting to Internet with the pppd(8) daemon.

OPTIONS
       gprs does not accept any options from the command line.
