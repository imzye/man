NAME
       Net::Snort::Parser::File - a module for handling the parsing of snort
       files

SYNOPSIS
       use Net::Snort::Parser::File; my (@lines) = parse_file('snort.conf');

DESCRIPTION
       Net::Snort::Parser::File is a module that provides a simple mechanism
       for handling the include statements that snort supports.

NOTES
       This module does not handle multiline statements.  It should.
