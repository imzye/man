NAME
       bin2rec - converts biosignal data produced with rec2bin, into a gdf
       file. rec2bin and bin2rec are complementary functions, which can be
       useful to dump data, edit date and revert it back into the original
       format.


SYNOPSIS
       REC2BIN  [OPTIONS]  SOURCE  [DEST]



DESCRIPTION
        SOURCE is the source file
        DEST is the destination file
        Supported OPTIONS are:
        -v, --version
            prints version information
        -h, --help
            prints this information
        -VERBOSE=#, verbosity level #
            0=silent, 9=debugging
