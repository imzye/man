NAME
       glib - The Graphics LIBrarian.

SYNOPSIS
       glib [ -d ] [ -s ] [ -t ] [ -u ] [ -e ] [ -a ] libname [ files...  ]

DESCRIPTION
       glib manages .gl ("gee"-"el") files which are GRASP animations which
       can be played using xgrasp(1).

OPTIONS
       -d   delete a file from library

       -s   extended file list

       -t   quick file list

       -u   update (add) file to library

       -e   extract file from library

       -a   extract all files from library

SEE ALSO
          X(1), xgrasp(1).
