NAME
       rgbPaint - A fork of mtPaint

SYNOPSIS
       mtpaint [option] [imagefile ... ]

DESCRIPTION
       rgbPaint is a very basic GTK+2 based painting program created after
       forking version 3.09 of mtPaint.

OPTIONS
       rgbPaint can accept the following options:

         --help          Output usage information
         --version       Output version information
         -s              Grab a screenshot
         -u <limit>      Set the undo limit in megabytes 1-500, default=32
         -stamps <...>   All remaining files passed are loaded as stamps
         -thumb <size>   Set the maximum width/height of the stamp thumbnails 32-256, default=40

HOMEPAGE
       http://mtpaint.sourceforge.net/
