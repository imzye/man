NAME
       sparkline - manual page for sparkline 1.4

DESCRIPTION
       usage: sparkline [-h] [--version] [--min MIN] [--max MAX] [--rows ROWS]

              [data ...]

       Reads from command line args or stdin and prints a sparkline from the
       data.  Requires at least 2 data points as input.

   positional arguments:
       data   Floating point data, any delimiter.

   optional arguments:
       -h, --help
              show this help message and exit

       --version, -v
              Display the version number and exit.

       --min MIN
              Set smaller values to MIN.

       --max MAX
              Set larger values to MAX.

       --rows ROWS, -r ROWS
              Number of rows high the graph will be.
