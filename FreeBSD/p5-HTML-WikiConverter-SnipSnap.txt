NAME
       HTML::WikiConverter::SnipSnap - Convert HTML to SnipSnap markup

SYNOPSIS
         use HTML::WikiConverter;
         my $wc = new HTML::WikiConverter( dialect => 'SnipSnap' );
         print $wc->html2wiki( $html );

DESCRIPTION
       This module contains rules for converting HTML into SnipSnap markup.
       See HTML::WikiConverter for additional usage details.
