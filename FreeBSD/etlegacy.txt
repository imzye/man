NAME
       etl - ET: Legacy client and listen server


SYNOPSIS
       etl [options]


DESCRIPTION
       etl (ET: Legacy) is an open source project based on the code of
       Wolfenstein: Enemy Territory, which was released in 2010 under the
       terms of GPLv3.

OPTIONS
       -v,--version
              Prints version info


       +      Passes commands into the game. Syntax is +<command> <param>
              <value>. F.e. '+set g_warmup 30' or '+exec etl_server.cfg'. Use
              space between more than one command.
