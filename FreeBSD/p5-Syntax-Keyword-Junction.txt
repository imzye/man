NAME
       Syntax::Feature::Junction - Provide keywords for any, all, none, or one

SYNOPSIS
         use syntax 'junction';

         if (any(@grant) eq 'su') {
           ...
         }

         if (all($foo, $bar) >= 10) {
           ...
         }

         if (qr/^\d+$/ == all(@answers)) {
           ...
         }

         if (all(@input) <= @limits) {
           ...
         }

         if (none(@pass) eq 'password') {
           ...
         }

         if (one(@answer) == 42) {
           ...
         }

       or if you want to rename an export, use Sub::Exporter options:

         use syntax 'junction' => {
           any => { -as => 'robot_any' }
         };

         if (robot_any(@grant) eq 'su') {
           ...
         }

       The full documentation for this module is in Syntax::Keyword::Junction.
       This is just a way to use the sugar that syntax gives us.
