NAME
       PPIx::EditorTools - Utility methods and base class for manipulating
       Perl via PPI

VERSION
       version 0.21

SYNOPSIS
           See PPIx::EditorTools::*

DESCRIPTION
       Base class and utility methods for manipulating Perl via PPI. Pulled
       out from the "Padre::Task::PPI" code.

METHODS
       new()
           Constructor. Generally shouldn't be called with any arguments.

SEE ALSO
       "PPIx::EditorTools::*", Padre, App::EditorTools, Padre, and PPI.
