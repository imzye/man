NAME
       fdnscname - resolve the canonical name of a host

SYNOPSIS
       fdnscname <hostname>

DESCRIPTION
       fdnsname will attempt to look up the canonical name of hostname.  If
       one is found, it is printed to standard output and the program exits
       with return code 0.  If not, nothing is printed and the program exits
       with return code 1.
