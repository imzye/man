NAME
       bingo.pl

SYNOPSIS
       % bingo.pl

DESCRIPTION
       This is a console bingo game application. With bingo_print.pl from
       Games::Bingo::Print you can generate cards and then you are actually
       ready to play.

SEE ALSO
       Games::Bingo
       Games::Bingo::Print
