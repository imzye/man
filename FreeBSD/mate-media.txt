MATE-VOLUME-CONTROL-APPLET(1)                  FreeBSD General Commands Manual

NAME
       mate-volume-control-applet - The MATE Volume Control Applet

SYNOPSIS
       mate-volume-control [OPTIONS]

DESCRIPTION
       The MATE Volume Control Applet is used for adjusting audio levels from
       the notification area.

OPTIONS
       -v, --version
              Output version information and exit.

       -d, --debug
              Enable debugging code.

       --display=DISPLAY
              X display to use.

       -?, -h, --help
              Print standard command line options.

       --help-all
              Print all command line options.

       This program also accepts the standard GTK+ options.

       --help-gtk
              Print GTK+ options.
