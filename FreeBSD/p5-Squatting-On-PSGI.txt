NAME
       Squatting::On::PSGI - Run Squatting app on PSGI

SYNOPSIS
         # app.psgi
         use App 'On::PSGI';
         App->init;

         my $handler = sub {
             my $env = shift;
             App->psgi($env);
         };

DESCRIPTION
       Squatting::On::PSGI is an adapter to run Squatting apps on PSGI
       implementations.
