NAME
     jail-primer – Third Generation Jail System Solution.

DESCRIPTION
     A simplified prospective on jail configuration and usage. Complete easy
     to understand detailed documentation on creating a Third Generation Jail
     System Solutiom which is based on a single filesystem that contains all
     of the required operating system executable libraries which is shared
     with each of the individual jails.

     The legacy rc.conf method, Modern rc.conf method, and the jail(8)
     jail.conf methods are documented. Script are included that perform the
     tasks explained in the documentation

JAIL-PRIMER
     The jail-primer is comprised of 15 scripts and one html file. The html
     file contains all the documentation. You may point your browser at
     http://jail-primer.sourceforge.net/ which is the recommended method or
     /usr/local/share/doc/jail-primer/jail-primer.html
