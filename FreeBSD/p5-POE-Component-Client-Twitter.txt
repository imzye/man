NAME
       POE::Component::Client::Twitter - POE chat component for twitter.com

SYNOPSIS
         use POE::Component::Client::Twitter;

DESCRIPTION
       POE::Component::Client::Twitter is a POE component for Twitter API. See
       <http://groups.google.com/group/twitter-development-talk/web/api-documentation>
       for more details about Twitter API.

       This module is in its beta quality and the API and implementation will
       be likely changed along with the further development.
