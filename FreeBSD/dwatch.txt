NAME
       dwatch


SYNOPSIS
       dwatch [-dv] [-f conf]


EXAMPLE
       dwatch -f /etc/dwatch.conf


DESCRIPTION
       Dwatch is a program that watches over other programs and performs
       actions based on conditions specified in a configuration file.

       It is meant to be run from cron at regular intervals.


OPTIONS
       -d     Debugging (repeat -d for more).

       -v     Verbose, i.e. print actions on stdout. Useful when running from
              cron.

       -f conf
              Read configuration from the specified file.
