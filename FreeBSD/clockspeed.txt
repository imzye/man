NAME
       clockadd - adjust the local UNIX clock, once

SYNOPSIS
       clockadd

DESCRIPTION
       clockadd reads a TAI64NA time adjustment.  It then adds that adjustment
       to the local UNIX clock.

       The adjustment must be a single 16-byte packet.

       clockadd must be run by root.
