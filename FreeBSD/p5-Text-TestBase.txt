NAME
       Data::Section::TestBase - Parse Test::Base format from DATA section

SYNOPSIS
           use Data::Section::TestBase;

           my @blocks = blocks;

DESCRIPTION
       This module parse a DATA section as Test::Base format by
       Text::TestBase.

FUNCTIONS
       my @blocks = blocks([section_name]);
           Get a list of blocks from the DATA section. The elements of @list
           are instances of Text::TestBase::Block.  If "section_name" is
           provided, only return blocks that have a section with that name.
