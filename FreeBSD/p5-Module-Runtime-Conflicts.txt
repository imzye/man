NAME
       Module::Runtime::Conflicts - Provide information on conflicts for
       Module::Runtime

VERSION
       version 0.003

SYNOPSIS
           `moose-outdated`

       or

           use Module::Runtime::Conflicts;
           Module::Runtime::Conflicts->check_conflicts;

DESCRIPTION
       This module provides conflicts checking for Module::Runtime, which had
       a recent release that broke some versions of Moose. It is called from
       Moose::Conflicts and "moose-outdated".
