     etherswitch â€“ Ethernet switch framework

SYNOPSIS
     To compile the framework into the kernel, place the following lines in
     the kernel configuration file:

           device etherswitch
           device miiproxy
           device iicbus

DESCRIPTION
     The etherswitch driver provides a framework for Ethernet switch devices.

FILES
     /dev/etherswitch?      etherswitch device nodes

SEE ALSO
     arswitch(4), iicbus(4), ip17x(4), rtl8366rb(4), e6060sw(4), adm6996fc(4),
     ksz8995ma(4), ukswitch(4), etherswitchcfg(8)

HISTORY
     The etherswitch framework first appeared in FreeBSD 10.0.
