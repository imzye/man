Pthread::GetThreadId(3)                    User Contributed Perl Documentation


SYNOPSIS use Pthread::GetThreadId;
         my $systhread_id = getthreadid_np();

DESCRIPTION
       getthreadid_np
           Returns kernel level POSIX thread id as integer value.
