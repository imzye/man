NAME
       pftpd - Anonymous FTP daemon.

SYNOPSIS
       pftpd [options]


DESCRIPTION
       This manual page documents briefly the pftpd command.

       pftpd is a daemon that implements an anonymous-only FTP server.

SEE ALSO
       adcd (1), ftpd (1) and ftp (1).
