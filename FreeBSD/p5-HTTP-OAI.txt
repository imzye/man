NAME
       oai_browser - Command line OAI repository browser

DESCRIPTION
       The oai_browser utility provides a command-line tool to browse an OAI-
       compliant repository.

SYNOPSIS
       oai_browser.pl [options] arguments

ARGUMENTS
       baseURL
           Specify baseURL to connect to.

OPTIONS
       --help  Show this page.

       --silent
               Don't display data harvested from the repository - only shows a
               record count.

       --trace Turn on trace debugging.

       --tracesax
               Turn on trace debugging of SAX calls.

       --skip-identify
               Don't perform an initial Identify to check the repository's
               baseURL.
