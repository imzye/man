NAME
       Mouse::Meta::Attribute::Custom::Trait::Array - Shortcut for ArrayRef
       trait

DESCRIPTION
       This module is an alias to MouseX::NativeTraits::ArrayRef, which allows
       you to refer the trait as "Array".
