NAME
       MooseX::MethodAttributes - Code attribute introspection

VERSION
       version 0.32

SYNOPSIS
           package MyClass;

           use Moose;
           use MooseX::MethodAttributes;

           sub foo : Bar Baz('corge') { ... }

           my $attrs = MyClass->meta->get_method('foo')->attributes; # ["Bar", "Baz('corge')"]

DESCRIPTION
       This module allows code attributes of methods to be introspected using
       Moose meta method objects.
