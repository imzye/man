NAME
       Net::YASA - Interface to YASA (Yet Another Suffix Array)

VERSION
       Version 0.03

SYNOPSIS
       This module will submit content to the YASA WebService to return a list
       of terms and corresponding frequencies.

           use Net::YASA;

           my $foo = Net::YASA->new();
           my $termset = $foo->extract(<some_of_utf8_words>);
           print 'TermSet 1: ', $$termset[0], "\n";
           print 'TermSet 2: ', $$termset[1], "\n";
           ...
