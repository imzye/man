NAME
       Data::Perl::Bool::MooseLike - data::Perl::Bool subclass that simulates
       Moose's native traits.

VERSION
       version 0.001009

SYNOPSIS
           # you should not be consuming this class directly.

DESCRIPTION
       This class provides a wrapper and methods for interacting with a
       boolean. All methods are written to emulate/match existing behavior
       that exists with Moose's native traits.
