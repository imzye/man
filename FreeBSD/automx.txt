NAME
       automx-test - automx command line client

SYNOPSIS
       automx-test localpart@domainpart

DESCRIPTION
       The automx-test command line client requests autoconfiguration data in
       Mozilla and Microsoft schema requests and outputs the server response
       to STDOUT.
