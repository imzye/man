NAME
       epdfview - view PDF documents

SYNOPSIS
       epdfview [ OPTION... ] [ FILE ]

DESCRIPTION
       This manual page documents briefly the epdfview command.

       The aim of epdfview is to make a simple PDF document viewer, in the
       lines of evince(1) but without using the GNOME libraries.

OPTIONS
   Help Options
       -h, --help
              Show help options

       --help-all
              Show all help options

       --help-gtk
              Show GTK+ Options

   GTK+ Options
       --class=CLASS
              Program class as used by the window manager

       --name=NAME
              Program name as used by the window manager

       --screen=SCREEN
              X screen to use

       --sync Make X calls synchronous

       --gtk-module=MODULES
              Load additional GTK+ modules

       --g-fatal-warnings
              Make all warnings fatal

   Application Options
       --display=DISPLAY
              X display to use
