NAME
       ftpasswd - Perl script for managing AuthUserFiles and AuthGroupFiles

SYNOPSIS
       ftpasswd [ options ]

DESCRIPTION
       ftpasswd is a Perl script designed to create and manage AuthUserFiles
       and AuthGroupFiles of the correct format for proftpd.

       Full documentation for ftpasswd can be found at:
       http://www.proftpd.org/docs/utils/ftpasswd.html
