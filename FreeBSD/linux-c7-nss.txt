NAME
       cert8.db - Legacy NSS certificate database

DESCRIPTION
       cert8.db is an NSS certificate database.

       This certificate database is in the legacy database format. Consider
       migrating to cert9.db and key4.db which are the new sqlite-based shared
       database format with support for concurrent access.

FILES
       /etc/pki/nssdb/cert8.db

       cert9.db(5), key4.db(5), pkcs11.txt(5),
