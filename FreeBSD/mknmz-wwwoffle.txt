NAME
       mknmz-wwwoffle - wwwoffle cache indexer for namazu2

SYNOPSIS
       mknmz-wwwoffle [options] targets...

DESCRIPTION
       mknmz-wwwoffle is wwwoffle cache indexer for Namazu Version 2.

OPTIONS
       This programs follow the usual GNU command line syntax, with long
       options starting with two dashes (`-').  A summary of options is
       included below.

       And mknmz options are available.

       For a complete description, see the README.ja files.

       -h, --help
              Show summary of options.

       -v, --version
              Show version of program.

SEE ALSO
       mknmz(1), namazu(1).
