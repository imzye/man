NAME
       Font::TTF - Perl module for TrueType Font hacking

DESCRIPTION
       This module allows you to do almost anything to a TrueType/OpenType
       Font including modify and inspect nearly all tables.

AUTHOR
       Martin Hosken <http://scripts.sil.org/FontUtils>.  (see CONTRIBUTORS
       for other authors).

       Repository available at <https://github.com/silnrsi/font-ttf.git>
