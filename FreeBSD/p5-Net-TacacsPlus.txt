NAME
       Net::TacacsPlus - Tacacs+ library

SYNOPSYS
               use Net::TacacsPlus qw{ tacacs_client };

               my $client = tacacs_client(
                       'host' => 'tacacs.server',
                       'key'  => 'secret',
               );

DESCRIPTION
       Tacacs+ client implemented by Net::TacacsPlus::Client.

FUNCTIONS
       tacacs_client(@arg)
           Returns Net::TacacsPlus::Client object created with @arg.
