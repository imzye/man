NAME
       waypaper - GUI wallpaper manager for Wayland and X11

SYNOPSIS
       waypaper [ OPTION ]

DESCRIPTION
       Waypaper is a TUI wallpaper setter for Wayland and Xorg window
       managers, with support for gif and video wallpapers. It works as a
       frontend for popular wallpaper backends like swaybg, swww, hyprpaper,
       wallutils, mpvpaper, and feh.
       https://anufrievroman.gitbook.com/waypaper

OPTIONS
       -v     Print version.


       -h     Print help.


       --restore
              Restore the last chosen wallpaper. Useful at launch of the
              window manager.


       --random
              Set a random wallpaper.


       --wallpaper XXX
              Set the wallpaper from the provided path and save it in the
              config.


       --folder
              Set the folder of wallpaper images.


       --monitor
              Set the desired monitor. Should be used together with either
              --random or --wallpaper.


       --backend XXX
              Specify which backend to use, which can be either swaybg, swww,
              feh, or wallutils. Useful if you use waypaper on both Wayland
              and Xorg on the same machine. By default, last used backend is
              used.


       --state-file XXX
              Set the path to state file.


       --fill XXX
              Specifies filling type, which can be eiher fill, stretch, fit,
              center, or tile.


       --list
              List wallpapers, monitors, and backend in json format.


       --no-post-command
              Prevent running the post command that was set in the config.


COMMANDS DURING USE
       Press '?' during use to get a list of keybindings.
