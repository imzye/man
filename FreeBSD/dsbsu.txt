SYNOPSIS
     dsbsu [-m message] [-u user] command
     dsbsu -a [-c command]
     dsbsudo [sudo options] command [args ...]

DESCRIPTION
     DSBSu is a Qt front end which allows executing commands as another user
     using su(1).  dsbsudo is a wrapper script for sudo(8) which uses dsbsu to
     ask for a password if necessary.

OPTIONS
     -a      Writes the entered password to stdout, and exits.

     -c      Shows the given command string in the password window.

     -m      Shows the given message in the password window.

     -u      Runs the command as the given user.  If -u is not defined, root
             is assumed.
