NAME
       Test::Declare - declarative testing

SYNOPSIS
           use strict;
           use warnings;
           use Test::Declare;
           plan tests => blocks;

           describe 'foo bar test' => run {
               init {
                   # init..
               };
               test 'foo is bar?' => run {
                   is foo, bar;
               };
               cleanup {
                   # cleanup..
               };
           };

DESCRIPTION
       Test::More, Test::Exception, Test::Deep, Test::Warn and Test::Output
       wrapper module.

METHOD
   describe outline setting.
   blocks get test block count.
   init definition of init block.
   test definition of test block.
   run run test code.
   cleanup definition of cleanup block.
