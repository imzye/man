NAME
       Template::Plugin::Filter::Minify::JavaScript - JavaScript::Minifier
       filter for Template Toolkit

VERSION
       version 0.94

SYNOPSIS
         [% USE Filter.Minify.JavaScript %]

         [% FILTER minify_js %]
           $(document).ready(
               function() {
                   $('body').append('<div>Hello World!</div>');
               }
           );
         [% END %]

DESCRIPTION
       This module is a Template Toolkit filter, which uses
       JavaScript::Minifier to compress javascript code from filtered content
       during template processing.

SEE ALSO
       JavaScript::Minifier, Template::Plugin::Filter, Template
