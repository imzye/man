NAME
     hutrans – hutrans command manual page

SYNOPSIS
     hutrans [-i input] [-o output]

DESCRIPTION
     The hutrans command converts 7-bit ASCII HTML representation of non-ASCII
     characters, such as &#1234; (decimal) or &#xabcd; (hexadecimal) into
     UTF-8 encoded text.

     The options are as follows:

     -i input    Specify input file (default: stdin).

     -o output   Specify output file (default: stdout).

ENVIRONMENT
     The hutrans command ignores environment variables.

EXAMPLES
     The following is an example of a typical usage of the hutrans command:

           % hutrans -i input -o output

SEE ALSO
     utrans(1), uhtrans(1), tuc(1), libutf-8(3)

     G. Adam Stanislav, Whiz Kid Technomagic i18n Tools,

STANDARDS
     ANSI X3.159-1989 (“ANSI C89”), UTF-8, RFC2279

DIAGNOSTICS
     Exit status is 0 on success, 1 on invalid usage, and 2 if a file cannot
     be opened.
