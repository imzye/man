DESCRIPTION
     The bmd-plugin-avahi publishes VM's vnc port using `avahi-publish`
     command. Clients can lookup the vnc port through the mDNS protocol. If
     you use MacOS, open Finder and Network placement folder of it, then you
     can find the VM name. And click it to connect to the remove vnc display.
     The gtk client `bvnc` is also available in 'avahi-gtk3' package.

     This plugin doesn't have any configurations. Just install and restart the
     bmd, this plugin starts to work.

REQUIREMENTS
     This plugin requires 'avahi-app' package.
