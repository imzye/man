NAME
       Log::Log4perl::EasyCatch - Easy Logging with Log4perl, catching errors
       and warnings, using configfile

VERSION
       Version 0.1.x

SYNOPSIS
         use Log::Log4perl::EasyCatch;

         INFO "Startup!";

         ERROR "There is an error: $error" if $error;

         DEBUG "Internal state: $status";

         ...

DESCRIPTION
       Everything from Log::Log4perl in easy mode, plus: Logging of warnings
       and Exceptions; default config file.
