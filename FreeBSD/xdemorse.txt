NAME
       xdemorse - a GTK+ Morse Code Decoder

SYNOPSIS
              xdemorse [-hv]

DESCRIPTION
              [-h: print this usage information and exit]
              [-v: print xdemorse version number and exit]
SEE ALSO
       The full documentation for xdemorse is maintained as an html document.
       If the xdemorse program is properly installed at your site, the command

              firefox /usr/share/doc/xdemorse/xdemorse.html

       should give you access to the complete manual.
