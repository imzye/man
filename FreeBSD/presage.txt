NAME
       presage_dbus_python_demo - presage dbus demo program

SYNOPSIS
       presage_dbus_python_demo [OPTION]...

DESCRIPTION
       At the prompt, type in some text. Hit enter to generate a prediction.
       Any text input is valid, including no text, a single character, or a
       long string.

       -c, --config CONFIG
              use config file CONFIG

       -h, --help
              display this help and exit

       -v, --version
              output version information and exit
