NAME
       HTML::WikiConverter::Socialtext - Convert HTML to Socialtext markup

SYNOPSIS
         use HTML::WikiConverter;
         my $wc = new HTML::WikiConverter( dialect => 'Socialtext');
         print $wc->html2wiki( $html );

DESCRIPTION
       This module contains rules for converting HTML into Socialtext markup.
       See HTML::WikiConverter for additional usage details.
