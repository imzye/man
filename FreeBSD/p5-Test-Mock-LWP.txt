NAME
       Test::Mock::HTTP::Request - Mocks HTTP::Request

SYNOPSIS
       Make HTTP::Request to make testing easier.

       See Test::Mock::LWP manpage for more details.

       This class uses Test::MockObject, so refer to it's documentation as
       well.
