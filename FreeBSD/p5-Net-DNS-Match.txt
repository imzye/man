NAME
       Net::DNS::Match - Perl extension for testing domains against another
       list of domains (similar to Net::Patricia but for FQDNs)

SYNOPSIS
         use Net::DNS::Match;
         use Data::Dumper;
         my $addr = 'img.yahoo.com';

         my $obj = Net::DNS::Match->new();
         $obj->add([
             'yahoo.com',
             'google.com',
             'www.facebook.com',
          ]);

        die Dumper($obj->match($addr));

DESCRIPTION
       This module was initially created to test a list of domains against a
       whitelist (eg: the Alexa top 1000 list).

   EXPORT
       None by default.
