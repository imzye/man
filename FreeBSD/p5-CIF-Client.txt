NAME
       CIF::Client - Perl extension that extends REST::Client for use with the
       CI-Framework REST interface

SYNOPSIS
         use CIF::Client;
         my $client = CIF::Client->new({
           host        => $url,
           timeout     => 60,
           apikey      => $apikey,
         });

         $client->search($query);
         die('request failed with code: '.$client->responseCode()) unless($client->responseCode == 200);

         my $text = $client->responseContent();

         print $client->table($text) || die('no records')

COMMAND-LINE
         $> cif -h
         $> cif -q example.com
         $> cif -q domain -p bindzone
         $> cif -q 192.168.1.0/24
         $> cif -q infrastructure/network -p snort
         $> cif -q url -s low | grep -v private

CONFIG FILE
       Your config should be stored in ~/.cif (default)

         [client]
         host = https://example.com:443/api
         apikey = xx-xx-xx-xx-xx
         timeout = 60
         #severity = medium

DESCRIPTION
       Simple extension of REST::Client for use with the CI-Framework REST
       based interface. Implements apikeys support and sample table output.
