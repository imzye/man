NAME
       ip2location - query IP address information from local IP2Location BIN
       data file


SYNOPSIS
       ip2location --data-file [IP2LOCATION BIN DATA PATH] --ip [IP ADDRESS]
       [OPTIONS]


DESCRIPTION
       ip2location read the IP address, IPv4 or IPv6 address format, and get
       the information from IP2Location BIN data file for display.

       User may provide the IP address as standard input or file containing
       one IP address per line. Several output formats supported such as CSV,
       TAB and XML.

       You can download the IP2Location BIN data file from
       http://www.ip2location.com or http://lite.ip2location.com

EXAMPLES
       ip2location --data-file [IP2LOCATION BIN DATA PATH] --ip [IP ADDRESS]
              Query an IP address and display the results on screen

       ip2location --data-file [IP2LOCATION BIN DATA PATH] --input-file [INPUT
       FILE PATH]
              Query all IP addresses from an input file and display the
              results on screen

       ip2location --data-file [IP2LOCATION BIN DATA PATH] --input-file [INPUT
       FILE PATH] --format XML
              Query all IP addresses from an input file and display the
              results in XML format on screen

       ip2location --data-file [IP2LOCATION BIN DATA PATH] --input-file [INPUT
       FILE PATH] --format TAB --output-file [OUTPUT FILE PATH]
              Query all IP addresses from an input file and output the result
              to a file in TAB format

       ip2location --data-file [IP2LOCATION BIN DATA PATH] --ip [IP ADDRESS]
       --field country_code -0-heading
              Query an IP address and display the country_short result

       ip2location --data-file [IP2LOCATION BIN DATA PATH] --ip [IP ADDRESS]
       --field country_code,city_name -0-heading --format TAB
              Query an IP address and display the country_short and city
              result


OPTIONS
       -b, --bin-version
           Print the IP2Location BIN database version.

       -d, --data-file
           Specify the path of IP2Location .BIN data file.

       -i, --input-file
           Specify an input file with IP address list.

       -p, --ip
           Specify an IP address query (Supported IPv4 and IPv6 address).

       -o, --output-file
           Specify output file for query results.

       -f, --format
           Specify output format. Supported formats are:
               - CSV (default)
               - TAB
               - XML

       -h, -?, --help
           Display this help file

       -v, --version
           Display the version number

       -e, --field
           Specify the field to be displayed. Supported values are:
               - ip
               - country_code
               - country_name
               - region_name
               - city_name
               - isp
               - latitude
               - longitude
               - domain
               - zip_code
               - time_zone
               - net_speed
               - idd_code
               - area_code
               - weather_station_code
               - weather_station_name
               - mcc
               - mnc
               - mobile_brand
               - elevation
               - usage_type
               - address_type
               - category
               - district
               - asn
               - as

       -n, -0-heading
           Suppress the heading display.
