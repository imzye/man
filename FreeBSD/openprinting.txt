NAME
            psm-ipp - Internet Print Protocol Print Service Module

SYNOPSIS
       /usr/local/libexec/psm-ipp.so


DESCRIPTION
       The Internet Print Protocol(IPP) Print Service Module(psm) provides
       applications using the Free Standards Group(FSG) OpenPrinting API(PAPI)
       a means of interacting with IPP based print services.




SEE ALSO
       papi(3)
