NAME
       dir-dump - the File::Dir::Dumper command line application.

SYNOPSIS
       quadp [-o output-file.txt] /path/to/directory/to/dump

       (quadp --man displays the man page)

DESCRIPTION
       Run this program with the path to the directory to dump to produce a
       dump on the standard output.

       Run it with -o output-file.txt to write to output-file.txt.

SEE ALSO
       <http://search.cpan.org/dist/File-Dir-Dumper/> - File-Dir-Dumper on the
       CPAN.
