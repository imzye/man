NAME
     kdata_array_alloc – allocate array data for plotting

LIBRARY
     library “libkplot”

SYNOPSIS
     #include <cairo.h>
     #include <kplot.h>

     struct kdata *
     kdata_array_alloc(struct kpair *pairs, size_t sz);

DESCRIPTION
     The kdata_array_alloc function creates a reference-counted data sources
     for kplot(3).  It is set to an array of data points pairs unless passed a
     NULL pointer, in which case x-values are set to the pair index from zero
     and y-values to zero.  These values can be later reset with
     kdata_array_fill(3), or directly modified with kdata_array_add(3) or
     kdata_array_set(3).

RETURN VALUES
     kdata_array_alloc returns NULL if memory allocation failed.

SEE ALSO
     kdata_array_fill(3), kplot(3)
