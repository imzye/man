NAME
       Scalar::Util::Numeric::PP - Pure-perl drop-in replacement/approximation
       of Scalar::Util::Numeric

VERSION
       This document describes version 0.04 of Scalar::Util::Numeric::PP (from
       Perl distribution Scalar-Util-Numeric-PP), released on 2016-01-22.

SYNOPSIS
DESCRIPTION
       This module is written mainly for the convenience of Data::Sah, as a
       drop-in pure-perl replacement for the XS module Scalar::Util::Numeric,
       in the case when Data::Sah needs to generate code that uses PP modules
       instead of XS ones.

       Not all functions from Scalar::Util::Numeric have been provided.

FUNCTIONS
   isint
   isfloat
   isnum
   isneg
   isinf
   isnan
