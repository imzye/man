NAME:
       Apache::TS - a collection of Perl interfaces to manage Apache Traffic
       Server installations.

SYNOPSIS
         #!/usr/bin/perl
         use Apache::TS::AdminClient;

         my $cli = Apache::TS::AdminClient->new(%input);
         my $string = $cli->get_stat("proxy.config.product_company");
         print "$string\n";

DESCRIPTION
       This is the main module for Apache::TS, in includes the following sub-
       modules
         Apache::TS::AdminClient
         Apache::TS::Config
         Apache::TS::Config::Records

       This module, Apache:TS, only defines a set of constants, which are used
       by both the sub-modules as well as scripts.
