NAME
     hpdoc – Documentation for HP 21xx/1000 and HP3000 computers

DESCRIPTION
     hp2100 and hp3000 are simulators for a number of HP computers.  See the
     appropriate manual pages for more details.

     Much of the documentation is simulator specific, but there is also some
     generic documentation.

     It can be found at /usr/local/share/doc/hpdoc,
     and also at http://simh.trailing-edge.com/hp/#Documentation.

SEE ALSO
     hp2100(1), hp3000(1).
