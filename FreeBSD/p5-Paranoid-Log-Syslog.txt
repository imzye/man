NAME
       Paranoid::Log::Syslog - Log Facility Syslog

VERSION
       $Id: lib/Paranoid/Log/Syslog.pm, 2.00 2016/05/13 19:51:02 acorliss Exp
       $

SYNOPSIS
         use Paranoid::Log;

         startLogger( 'syslog_local1', 'Syslog', PL_WARN, PL_EQ,
           { facility => 'local1', ident => 'myproc' });

DESCRIPTION
       This module implements UNIX syslog support for logging purposes.  The
       options hash is entirely optional.  The facility defaults to user if
       omitted, and ident defaults to the process name.

       This module does support the use of multiple syslog loggers, with each
       being able to set their own facility and ident.

SUBROUTINES/METHODS
       NOTE:  Given that this module is not intended to be used directly
       nothing is exported.

   init
   logMsg
   addLogger
   delLogger

DEPENDENCIES
       o   Paranoid

       o   Paranoid::Debug

       o   Unix::Syslog
