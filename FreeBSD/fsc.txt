DESCRIPTION
     fscd.conf is the configuration file for the FreeBSD service control
     daemon fscd(8).  Its format is one service per line without trailing nor
     foregoing spaces or tabs.

     Lines starting with a hash (#) will be ignored.

FILES
     /usr/local/etc/fscd.conf  standard configuration file for fscd(8).

SEE ALSO
     fscadm(8), fscd(8)
