NAME
       NIFTICLIB - Neuroimaging Informatics Technology Initiative data format


DESCRIPTION
       Nifticlib is a set of tools and libraries for maintaining a common data
       format among neuroimaging applications.  For documentation, please
       visit http://nifti.nimh.nih.gov/


SEE ALSO
       afni(1), fsl(1), caret(1)
