NOM
       ccutter - éditeur de musique SID

SYNOPSIS
       ccutter [OPTION]... [FICHIER]

DESCRIPTION
       CheeseCutter (C) 2009-17 Abaddon Publié sous GNU GPL.

OPTIONS
       -b [valeur]
              Définir la taille du tampon de lecture (déf=2048)

       -f     Démarrer en mode plein écran

       -nofp  Ne pas utiliser l'émulation resid-fp

       -fpr [x]
              Spécifie le préréglage de filtre. x = 0..16 pour 6581 et 0..1
              pour 8580

       -i     Désactive l'interpolation resid (utilise le mode rapide à la
              place)

       -m [0|1]
              Spécifie le modèle SID pour reSID (6581/8580) (déf=0)

       -n     Active le mode NTSC

       -r [value]
              Paramètre la fréquence de lecture (déf=48000)

       -y     Utiliser la superposition vidéo YUV
