NAME
       Amon2::Plugin::Web::MobileAgent - HTTP::MobileAgent plugin for Amon2

SYNOPSIS
           package MyApp::Web;
           use parent qw/MyApp Amon2::Web/;
           __PACKAGE__->load_plugins('Web::MobileAgent');
           1;

           # in your controller
           $c->mobile_agent();

DESCRIPTION
       This plugin integrates HTTP::MobileAgent and Amon2.

       This module adds "$c->mobile_agent()" method to the context object.
       The agent class is generated by "$c->req".
