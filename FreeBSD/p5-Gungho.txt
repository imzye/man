NAME
       gungho - An Extensible, High-Performance Web Crawler Framework

SYNOPSIS
          gungho -c config.yml
          gungho -v
          gungho -h

DESCRIPTION
       gungho is the command line tool to run the Gungho web crawler
       framework.

OPTIONS
   --config | -c
       Specify the config file to read from. By default, gungho attempts to
       read a config file named config.yml in the current directory

   --version | -v
       Print out the version and exit

   --help | -h
       Print out this help message and exit
