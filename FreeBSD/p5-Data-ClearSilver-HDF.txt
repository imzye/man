NAME
       Data::ClearSilver::HDF - Convert from Perl Data Structure to
       ClearSilver HDF

VERSION
       version 0.04

SYNOPSIS
         use Data::ClearSilver::HDF;

         my $data = {
           foo => {
             bar => 1,
             baz => [0 .. 5]
           },
           obj => bless { foo => "xxx", bar => "yyy" }
         };

         my $hdf = Data::ClearSilver::HDF->hdf($data);

         print $hdf->getValue("obj.foo", undef); # xxx

PROPERTIES
   $USE_SORT
       Sorting each keys hieralcally. default false;

METHODS
   hdf($data)
       The argument $data must be reference.  In the data, all of value
       excluded ARRAY, HASH, blessed reference will be ignored.

       Blessed reference will be unblessed by Data::Structure::Util's unbless
       functon.

   hdf_dump($hdf)
       Dump as string from ClearSilver::HDF object.  This method will create
       temporary file.

   hdf_scalar($hdf, $keys, $data)
       Translate scalar data to hdf.  Please don't call directory.

   hdf_array($hdf, $keys, $data)
       Translate array reference data to hdf.  Please don't call directory.

   hdf_hash($hdf, $keys, $data)
       Translate hash reference data to hdf.  Please don't call directory.
