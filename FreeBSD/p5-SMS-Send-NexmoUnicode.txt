NAME
       SMS::Send::NexmoUnicode - SMS::Send driver for www.nexmo.com

SYNOPSIS
         use SMS::Send;

         my $sender = SMS::Send->new('NexmoUnicode',
                         _username   => 'UserName',
                         _password   => 'Password',
                         _type       => 'unicode',
                         _from       => 'author',
                       );

         my $sent = $sender->send_sms(
                         text => 'My very urgent message',
                         to   => '0912345678',
                    );

DESCRIPTION
       SMS::Send::NexmoUnicode is a SMS::Send driver which allows you to send
       unicode messages through <http://www.nexmo.com/>.

METHODS
   new
       The "new" method takes a few parameters. "_username" , "_password",
       "_type", and "_from" are mandatory.

   send_sms
       Takes "to" as recipient phonenumber, and "text" as the text that's
       supposed to be delivered.
